# External Tools for Progress in Human Geography

A PiHG review is a **critical synthesis of others' scholarship**, so the tooling is bibliographic and
organizational, not estimation software. Confirm any deposit/format requirement on the SAGE PiHG
author pages (检索于 2026-06；以官网为准).

## Literature discovery & coverage

| Tool / source | Use |
|---------------|-----|
| **Web of Science** / **Scopus** | forward-citation snowballing (who cites the seeds) and cross-discipline reach |
| **GeoBase / GEOBASE** (Elsevier) | the discipline's core human-and-physical-geography indexing database; keyword + subject sweep |
| **Google Scholar** | broad forward/backward snowballing, working papers, grey literature |
| **JSTOR** | back-catalogue access to canonical human geography |
| **SocArXiv / OSF Preprints / SSRN** | preprints and forthcoming work (the frontier) |
| **PiHG archive on SAGE Journals** | locate the standing **progress reports** on the subfield (so you build on, not duplicate, the series) |
| **Sibling-journal archives** (Annals of the AAG, Transactions of the IBG, Antipode, Dialogues in Human Geography) | check the venue before citing any candidate exemplar |

## Synthesis & reference management

| Tool / source | Use |
|---------------|-----|
| **Zotero** / **EndNote** | build the citation corpus; export to the evidence matrix |
| Spreadsheet matrix | the evidence matrix (claim/concept / tradition / move / appraisal / tension) |
| **PRISMA / PRISMA-ScR templates** | only if the piece is a genuine systematic review of a tractable evidence base — *not* the default for a critical PiHG review |
| **bibliometrix (R)** / **VOSviewer** | bibliometric/citation-map exhibits where the review computes one (rare for PiHG) |

## Deposit (only if the review runs its own bibliometric analysis)

| Repository | Use |
|------------|-----|
| **OSF** / **Zenodo** | deposit any bibliometric/citation data + code; cite the DOI |

## Method notes

- PiHG reviews report **no primary estimates and no detailed cases** — keep no replication package of your own data; the analogous artefacts are the **scholarly apparatus, positionality, and coverage account**.
- Document the search like a record: databases, dates, terms *and* theoretical-tradition names, selection logic, saturation.
- Sweep deliberately across **traditions** — political-economic, feminist, poststructural, postcolonial, more-than-human, quantitative/GIScience — siloing by tradition is a recurring PiHG referee complaint.
- A formal PRISMA tally is usually the wrong form for a critical/theoretical review; prefer a transparent narrative account (see `proghg-transparency-and-reproducibility`).
- Verify the venue of every candidate exemplar before citing it as a PiHG article (sibling-journal guard: Annals of the AAG / Transactions of the IBG / Antipode / Dialogues in Human Geography are not PiHG).
