## Official journal / society pages

- **https://economicdynamics.org/info-for-authors/** — SED "Information for Authors" page. Verifies:
  RED publishes original articles applying dynamic economic theory to economics problems, with related
  measurement and empirical papers welcome; submission routes through the journal's ScienceDirect page;
  the review process starts only after the submission fee is received; a reduced **USD 100** rate applies
  only when every coauthor is a full-time student at submission; first invited resubmissions pay a new
  fee; second-and-later resubmissions have no submission fee; authors must keep the editor informed
  about related submissions or publication plans; comments on published RED articles require first
  sending the comment to the original author.
- **https://www.sciencedirect.com/journal/review-of-economic-dynamics/publish/guide-for-authors** —
  Elsevier Guide for Authors. Verifies: the active submission link points to
  **www.editorialmanager.com**; RED supports open access; the Guide lists the active RED submission fee
  as **USD 195** (with the same **USD 100** all-student reduction, first invited resubmission fee, and
  second-and-later resubmission exemption); manuscripts are single-anonymized with an editor suitability
  screen and at least two reviewers when sent out; the abstract must not exceed **250 words** and should
  stand alone; keywords are **1 to 6**; LaTeX submissions are accepted; RED uses Elsevier's author-year
  reference pattern and data/software reference guidance; generative-AI use must be disclosed, with
  stricter limits on AI-generated or AI-altered figures; Option C research-data instructions require
  repository deposit and article citation/linking where possible, or a data-availability statement
  explaining why sharing is not possible.
- **https://economicdynamics.org/our-editors/** — SED editors page. Verifies: **Loukas Karabarbounis**
  is RED's **Coordinating Editor**; the page lists Editors, Editorial Advisory Board members, and
  Associate Editors, including **Christian Zimmermann**.
- **https://www.sciencedirect.com/journal/review-of-economic-dynamics/about/editorial-board** —
  ScienceDirect editorial board page. Cross-checks the current coordinating editor, editorial-board
  structure, and the ScienceDirect submit/guide entry points.
- **https://economicdynamics.org/info-for-reviewers/** — SED reviewer page. Verifies: reviewers assess
  appropriateness for publication, review is confidential and anonymous from the reviewer's side, reviewers
  should notify the editor if they cannot act within six weeks, reports normally include a report for
  authors and a separate letter to the editor, and decisions include reject, accept conditional on minor
  revisions, and revise-and-resubmit.
- **https://economicdynamics.org/availability-of-data-and-computer-code-for-published-papers/** —
  SED "Availability of Data and Computer Code for Published Papers" policy. Verifies: before final
  acceptance, authors provide the data and computer code used to generate the paper's results, sufficient
  for replication; empirical papers provide final datasets and data-manipulation code plus construction
  details; numerical papers provide final programs and a readme; readme.txt must describe software/OS,
  special computing setup, execution order, expected computation time, random seeds/random-number
  generator where applicable, and file roles; archive packages should use open standards such as
  **.zip**, **.gz**, or **.gzip**; file names should avoid spaces; files are posted on the RED website,
  indexed on RePEc, and backed up; exemptions require Coordinating Editor approval at submission.
- **https://ideas.repec.org/s/red/ccodes.html** — IDEAS/RePEc "Review of Economic Dynamics - Computer
  Codes" series. Verifies: RED maintains a dedicated RePEc-indexed computer-codes series with recent
  "Code and data files for ..." entries and the series handle **RePEc:red:ccodes**.

## Fees, APCs, and fee-source precedence

- **Submission fee used by this pack: USD 195.** The Elsevier Guide for Authors is the payment-process
  source and lists **USD 195**. The SED information page still lists an older **175 US dollars** amount. For live submission
  preflights, use the Elsevier Guide/payment workflow amount and record the SED-page discrepancy as a
  source-maintenance note rather than charging authors the older amount.
- **All-student reduced submission fee: USD 100.** Both the SED page and Elsevier Guide agree that every
  coauthor must be a full-time student at submission.
- **Resubmission fee rule.** First invited resubmissions pay a new fee; second-and-later resubmissions
  are exempt. Elsevier also states that transferred manuscripts through its article-transfer service do
  not require another submission fee.
- **https://assets.ctfassets.net/o78em1y1w4i4/5ZH2ckERsr69JYpnMue94T/c40ff39a9d21ba89acacdb5c48685370/article-publishing-charge.pdf** —
  Elsevier APC price list, prices excluding taxes as of **2025-05-09**. Lists RED (ISSN 1094-2025) as
  **Hybrid** with optional open-access APC **USD 3,320 / EUR 3,110 / GBP 2,660 / JPY 411,780**. This is
  an acceptance-stage open-access charge, not a submission fee.

## Scope, style, and submission guardrails encoded in skills

- RED is method/lens-defined: dynamic economic theory applications, quantitative dynamic models, related
  measurement work, and empirical papers are in scope when they discipline a dynamic-economics object.
- The pack encodes **single-anonymized** submission: author identity remains visible to reviewers, so do
  not scrub the manuscript as for double-blind review.
- The abstract limit is **250 words**; the pack does not encode a RED-specific manuscript word/page cap.
  Use the current Guide for Authors and Editorial Manager prompts for live length or file constraints.
- References and front matter follow the current Elsevier Guide: author-year reference pattern, 1 to 6
  keywords, editable source files, LaTeX accepted, data/software references when relevant, and AI-use
  declaration when applicable.
- RED has two overlapping reproducibility layers: the SED RED archive/data-code policy before final
  acceptance, and Elsevier Option C research-data instructions in the Guide. A robust submission prepares
  both the RED code package and repository/data-statement metadata early.
- Historic SED newsletters report fast first-decision performance in earlier years, but this pack avoids
  treating an old newsletter statistic as a current guarantee. Current skills describe RED as
  fast-oriented and advise planning around the six-week reviewer-availability signal and live journal
  guidance rather than promising a fixed decision date.
