# FOCS Resources

Working materials behind the FOCS skill pack. The `skills/` directory carries
the advice; this directory carries what the advice stands on — the verified
source trail, award-anchored exemplars, a rebuilt opening, and the small
amount of tooling a proofs-first venue justifies.

## Contents

| Resource | Use it to... |
| --- | --- |
| [`official-source-map.md`](official-source-map.md) | Trace every FOCS fact in the pack to a dated official source; see the access-method note and the explicit 待核实 list before repeating any date or rule. |
| [`exemplars/library.md`](exemplars/library.md) | Position your draft against Test of Time and Machtey Award papers whose FOCS attribution is venue-verified, and dodge the FOCS/STOC misattribution traps. |

| [`code/README.md`](code/README.md) | Apply the few shared-kit tools that survive at a theorems venue, plus FOCS-native PDF, statement-freeze, and certificate checks. |

## Scope note

This is a theoretical-computer-science conference pack. There is no
experiment matrix to fill, no checklist to attach, and no artifact badge to
chase — importing those rituals from ML-conference packs is the venue
confusion this pack exists to prevent. The evidence objects that matter here
are proofs, certificates, and the public full version.

## Suggested route

1. Decide the venue honestly with
   [`../skills/focs-topic-selection`](../skills/focs-topic-selection/SKILL.md),
   using [`exemplars/library.md`](exemplars/library.md) as the durability
   benchmark.
2. Build checkability in from the start:
   [`../skills/focs-reproducibility`](../skills/focs-reproducibility/SKILL.md)
   plus the statement-freeze tooling in [`code/README.md`](code/README.md).
3. Write for the window:
   [`../skills/focs-writing-style`](../skills/focs-writing-style/SKILL.md) and
   [`../skills/focs-supplementary`](../skills/focs-supplementary/SKILL.md),
   with [`worked-examples/01-introduction.md`](worked-examples/01-introduction.md)
   as the before/after model.
4. Before upload, run
   [`../skills/focs-submission`](../skills/focs-submission/SKILL.md) against
   the current CFP re-opened via
   [`official-source-map.md`](official-source-map.md) — never against this
   pack's snapshot alone.
