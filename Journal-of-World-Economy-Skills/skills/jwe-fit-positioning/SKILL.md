# 对口判断与定位（jwe-fit-positioning）

> 《世界经济》由中国世界经济学会与中国社会科学院世界经济与政治研究所（IWEP）共同主办、中国社会科学院主管（1978 年创刊，月刊；ISSN 1002-9621，CN 11-1138/F），定位即"世界经济"——它要的是**开放经济中的因果**，不是优秀的国内实证。这条定位线是本刊与《经济研究》/《金融研究》/《中国工业经济》最硬的区别，也是本 skill 要先把关的。基础事实见 `resources/journal-profile.md`。

## 触发时机

- 拿不准稿子的"国际 / 开放"维度够不够、对不对口本刊
- 题目里有"全球 / 国际 / 开放"字样，但主体看着像国内实证
- 在《世界经济》《金融研究》《中国工业经济》之间犹豫

## 一句话标准

**去掉"开放 / 外部"维度，文章还成立吗？**
- 成立 → 这是国内稿，本刊不对口，改投。
- 不成立（核心问题、识别、机制都依赖国际 / 开放） → 对口。

## 对照表：对口 vs 不对口

| 维度 | 对口《世界经济》 | 实为国内稿（改投） |
|------|------------------|--------------------|
| 处理变量 | 关税 / 贸易协定 / 汇改 / 外资准入 / 全球冲击 | 国内补贴 / 户籍 / 地方政策 |
| 因变量 | 出口边际 / 进口品价格 / FDI / GVC 位置 / 跨境资本 | 国内产出 / 就业 / 信贷 |
| 机制 | 贸易 / 投资 / 金融 / 预期渠道（跨境） | 纯国内传导 |
| 数据 | 海关微观 / 跨国面板 / 投入产出 GVC | 一般省级 / 上市公司面板 |
| 文献 | 国际贸易 / 国际金融经典与前沿 | 仅国内实证 |

## 改投决策（与跨刊配合）

- 落点在**货币 / 银行 / 资本市场 / 公司金融** → 《金融研究》（journal-of-financial-research）
- 落点在**国内产业 / 企业行为 / 产业政策** → 《中国工业经济》（china-industrial-economics）
- **纯国内干净因果 / 一般均衡理论** → 《经济研究》/《经济学季刊》
- 仅"出海 / 跨国经营 / 对外投资 + 跨境机制" → 留在《世界经济》

## 如果偏弱：如何把开放维度真正立住（而非贴标签）

- 把外生变动锚在**国际 / 制度性冲击**上（不是国内政策的副产品）
- 因变量或机制中至少一环是**跨境**的（贸易额、进口品、汇率敞口、GVC 联系）
- 引言前段就交代"为什么这是开放经济问题"，不是结尾补一句

## 自检清单

- [ ] 去掉开放维度后文章不成立
- [ ] 处理 / 机制 / 数据 至少两项带国际属性
- [ ] 引言**前两段**已立住开放维度，非硬贴
- [ ] 已确认不更适合《金融研究》/《中国工业经济》

## 反模式

- 国内问题 + 结尾一句"对全球治理有启示"
- 用了 GVC / 全球价值链词汇但全文无跨境数据或分解
- "受国际环境影响"作为唯一国际性，处理变量仍是国内政策

## 《世界经济》对口判定决策表

本刊（中国社会科学院 IWEP 与中国世界经济学会主办的 CSSCI 权威顶级期刊）最硬的把关线是"开放经济中的因果"。下表把"判不对口→改投去向"显性化，便于一次定位。

| 文章特征 | 判定 | 建议去向 |
|----------|------|----------|
| 处理=国际/制度性冲击，机制跨境 | 对口本刊 | 进入 `jwe-topic-international` |
| 货币/银行/资本市场/公司金融为核心 | 不对口 | 《金融研究》 journal-of-financial-research |
| 国内产业/企业行为/产业政策为核心 | 不对口 | 《中国工业经济》 china-industrial-economics |
| 纯国内干净因果或一般均衡理论 | 不对口 | 《经济研究》/《经济学季刊》 |
| 仅"出海/跨国经营+跨境机制" | 对口本刊 | 留本刊，强化跨境机制 |

## 微型走查：一篇"全球化与就业"稿件的对口判定（示意）

设稿件用省级面板做"全球化对中国制造业就业的影响"，国际性靠进口额作控制变量。按本 skill 走查：

```
一句话标准 → 去掉"开放/外部"维度文章还成立吗？
  处理变量 → 实为省级产业政策，进口额仅作控制 → 去掉国际项主结论不变 → 成立
判定 → 这是国内稿，本刊不对口
救稿路径（若坚持本刊）：
  上移处理 → 改用关税暴露度（进口渗透×初期份额）作准外生冲击
  暴露度构造（示意）→ 城市层进口渗透暴露，分布右偏，高暴露城市集中沿海
  机制跨境化 → 就业经"进口竞争→本地劳动力市场"传导（ADH 谱系）
  去开放维度复检 → 此时去掉关税冲击主结论不成立 → 转为对口
不救则改投 → 若落点仍是国内产业政策 → china-industrial-economics
```

> 走查中的暴露度分布为示意描述，仅演示对口判定与救稿决策，非真实数据。

## 审稿人追问模式 + 本刊语境修法

- 追问"这题的国际性是不是只在引言一句话" → 用一句话标准自检：去掉开放维度若仍成立，则上移处理变量或改投。
- 追问"为什么不投《金融研究》/《中国工业经济》" → 明确核心机制是否跨境；纯国内金融/产业落点应跨刊改投。
- 追问"开放维度立在哪" → 要求处理/机制/数据至少两项带国际属性，并在引言前两段交代。

## 校准锚点（本刊已刊论文形态）

本刊论文的开放维度通常在引言前段即立住，处理变量多为国际/制度性冲击，机制与数据至少一环跨境；纯国内题即便冠以"全球化"也较难通过对口关。具体栏目边界与对口口味以编辑部最新稿约为准。

## 输出格式

```
【匹配度】高 / 中 / 低
【国际维度】贸易 / 投资 / 金融 / 汇率 / GVC | 偏弱（需上移）
【去开放维度后】成立（改投）/ 不成立（对口）
【建议去向】世界经济 / 金融研究 / 中国工业经济 / 经济研究
【下一步】jwe-topic-international 或 跨刊改投
```
