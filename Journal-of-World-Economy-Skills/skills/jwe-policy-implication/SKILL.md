# 开放政策含义（jwe-policy-implication）

## 触发时机

- 文末政策含义写成国内"加强 / 完善 / 推进"操作清单
- 政策落点与文章的国际 / 开放维度脱节
- 不确定本刊政策含义该写到什么层级

## 核心：落在对外开放政策层

《世界经济》（中国世界经济学会 + IWEP 主办，中国社会科学院主管）的政策含义应对应**开放政策工具**（贸易/汇率/资本账户/产业开放/GVC），且必须**由本文识别出的机制 / 边界推出**，不是泛泛号召。落到国内"加强完善推进"的操作清单，即偏离本刊定位——这是本刊与国内导向期刊在政策落点上的硬区别。

| 政策域 | 工具 | 写法（由证据推出） |
|--------|------|--------------------|
| **贸易政策** | 关税、贸易协定、出口退税、贸易便利化 | "高暴露行业承受更大转嫁 → 谈判中应优先保护 X 类中间品关税" |
| **汇率政策** | 汇率弹性、汇率传递、外汇管理 | "高敞口企业响应更强 → 增强弹性时应配套敞口对冲工具" |
| **资本账户** | 跨境资本流动管理、宏观审慎 | "对全球金融周期敏感度高 → 资本流入期宜用逆周期 MPM" |
| **产业开放** | 外资准入、负面清单、FDI 政策 | "准入放开提升下游加成率 → 负面清单应优先减 X 环节" |
| **GVC / 供应链** | 国内增加值、供应链安全 | "DVAR 提升集中在 X 环节 → 补链应聚焦该上游" |

## 写作要点

- 每条政策含义对应**一个本文发现**，可回指到某张表 / 某个机制
- 区分**结论层**（我们发现什么）与**政策层**（因此政策应如何），不混写
- 用"在 X 条件下宜 Y"的有边界表述，而非无条件号召
- 兼顾对内（国内调整）与对外（谈判 / 规则参与）两个面向时，分别说清

## 自检清单

- [ ] 政策落点是开放政策工具（贸易 / 汇率 / 资本账户 / 产业开放）
- [ ] 每条含义可回指本文具体证据 / 机制
- [ ] 有边界条件，不是无条件"应大力推进"
- [ ] 没有滑回纯国内操作清单
- [ ] "加强 / 完善 / 推进 / 深化"四件套命中 = 0

## 反模式

- "建议政府高度重视对外开放" 式空话
- 政策建议与文章的贸易 / 金融机制无对应
- 写成地方政府操作手册
- 把局部均衡结果直接当福利结论推全局政策

## 《世界经济》政策含义审稿期待表

本刊（IWEP + 中国世界经济学会主办的 CSSCI 权威顶级期刊）的政策段，审的是"开放政策含义是否由本文识别出的因果与边界推出"。下表把退稿/退修常见信号显性化。

| 审稿期待 | 达标写法 | 退修/退稿信号 |
|----------|----------|----------------|
| 落在对外开放政策工具 | 关税谈判、汇率弹性配套、跨境资本逆周期、负面清单环节 | 国内"加强完善推进深化"操作清单 |
| 每条含义回指本文证据 | "高敞口组响应更强（表 X）→ …" | 政策条与贸易/金融机制无对应 |
| 区分结论层与政策层 | 先述发现，再给"在 X 条件下宜 Y" | 把相关当因果直接开政策处方 |
| 福利口径自洽 | 局部均衡结果不外推全局福利，需 GE 时点明 | 用局部估计推全国关税福利结论 |

## 微型走查：FDI 负面清单放开的政策含义（示意）

设稿件发现"服务业外资准入放开后，下游制造企业加成率上升，集中在高进口中间品依赖组"。按本 skill 推政策含义：

```
本文发现（示意）：负面清单删减 → 下游加成率 +0.03（高依赖组），低依赖组不显著
推政策含义：
  政策域 → 产业开放（外资准入/负面清单）
  含义-证据对应 → "放开效应集中在高中间品依赖环节（表 4）→ 后续负面清单应优先减该上游服务环节"
  边界条件 → "在中间品可贸易、下游竞争充分时成立"，非无条件全面放开
  对内/对外 → 对内：配套竞争政策；对外：服务贸易谈判中以此为筹码
四件套自检 → 全文"加强/完善/推进/深化"命中 0 处
```

> 走查中的加成率增幅为示意占位，仅演示"由发现推开放政策含义"的链条，非真实估计。

## 审稿人追问模式 + 本刊语境修法

- 追问"政策建议是不是放之四海皆准的空话" → 给"在 X 条件下宜 Y"的有边界表述，回指具体异质性组。
- 追问"局部均衡结果凭什么推全国福利" → 明确区分集约边际证据与一般均衡福利，需要时补 GE/福利口径或限定结论范围。
- 追问"这条政策和文章的开放机制有关系吗" → 删除与本文贸易/汇率/资本/GVC 机制无关的泛化建议。

## 校准锚点（本刊已刊论文形态）

本刊政策含义段通常紧扣本文识别出的开放经济机制，给出贸易谈判、汇率与资本账户管理、产业开放节奏等层面的有边界建议，并避免滑入地方政府操作手册式表述；政策段篇幅与表述规范以编辑部最新稿约为准。

## 输出格式

```
【政策域】贸易 / 汇率 / 资本账户 / 产业开放 / GVC
【含义-证据对应】[政策条 → 表/机制]
【边界条件】有 / 无
【四件套命中】X 处
【下一步】jwe-submission
```
