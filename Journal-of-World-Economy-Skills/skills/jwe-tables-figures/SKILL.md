# 表格与图（jwe-tables-figures）

## 触发时机

- 主表列数过多、信息过载，审稿人抓不到主结果
- 表注不规范（标准误层次、显著性、样本量缺失）
- 有事件研究 / 引力 / GVC 分解但展示方式不专业

## 主表原则

- **一张主表讲一个故事**：基准 + 关键稳健，控制 vs 不控制对比，4–6 列为宜，别堆到 10+ 列
- 每列报告：系数、标准误（注明聚类层次）、固定效应行、观测数、R²/拟合
- 显著性符号统一（*/**/***）并在表注定义；系数保留合理小数位
- 完整控制变量、次要稳健性放附录或备索

## 开放经济专属图表

| 对象 | 规范要点 |
|------|----------|
| **事件研究图** | 横轴相对处理期，含置信区间，处理前系数应近零（平行趋势） |
| **引力回归表** | 注明 PPML、固定效应结构（出口国×年、进口国×年、国家对）、零贸易处理 |
| **GVC 分解** | 注明分解法（KWW）、用哪张 MRIO 与年份、单位口径 |
| **shift-share** | 报告 exposure-robust 标准误，必要时附 Rotemberg 权重表 |
| **暴露度地图 / 分布** | 标注暴露度定义与数据来源 |

## 图的规范

- 《世界经济》为中文月刊（IWEP + 中国世界经济学会主办）：图表标题 / 注 用中文；变量定义清晰；可黑白打印仍可读（投稿前以官网最新版式要求为准）
- 不用花哨配色掩盖信息；坐标轴、单位、图例齐全
- 数据来源与样本期在图注标明

## 执行桥（StatsPAI / Stata MCP）

表格图形**从拟合结果生成**，不要手抄数字。完整映射见
[`execution-with-mcp`](../../../shared-resources/empirical-methods/execution-with-mcp.md)。《世界经济》是国际经济实证刊，跨国/企业面板；强调识别与聚类。

- **表：**`etable`（多列）或 `did_summary_to_latex` 直接从 `result_id` 生成。
- **图：**`plot_from_result` / `enhanced_event_study_plot` / `event_study_table`，坐标单位与
  标准误/聚类注记自带。
- **每个表注**写明估计量与聚类层次，并以可解释单位报告经济量级。

完整“拟合结果 → 图表”链见 [JF 执行 walkthrough](../../../Journal-of-Finance-Skills/resources/worked-examples/02-execution-walkthrough.md)。
## 自检清单

- [ ] 主表 ≤ 6 列，主结果一眼可见
- [ ] 每表注明标准误聚类层次与显著性定义
- [ ] 固定效应、观测数、拟合优度齐全
- [ ] 事件研究图含 CI 且处理前近零
- [ ] 引力 / GVC / shift-share 表注明方法与口径
- [ ] 所有图表有数据来源与样本期

## 反模式

- 主表 10+ 列堆所有变量
- 表注不写聚类层次
- 事件研究图无置信区间
- 引力表不写是否 PPML

## 《世界经济》制表审稿期待表

下表把"本刊审稿人盯图表时最容易扣分的点"显性化。《世界经济》（IWEP + 中国世界经济学会主办的 CSSCI 权威顶级期刊）的图表评审，本质是检验"开放经济口径"是否在版面上落到实处：海关微观、引力 PPML、KWW 增加值分解、shift-share 暴露度这些对象一旦展示不规范，开放属性就在最显眼处露怯。

| 审稿期待 | 达标 | 常见退稿/退修信号 |
|----------|------|--------------------|
| 主表聚焦开放经济主结果 | 基准+关键稳健 4–6 列，暴露度系数一眼可见 | 10+ 列堆国别×行业×边际全交互，主结论被淹没 |
| 标准误层次匹配暴露结构 | 注明按行业/地区/暴露单元聚类，shift-share 用 exposure-robust | 海关微观默认企业聚类，不提暴露相关性 |
| 引力表自证 PPML 口径 | 表注写 PPML、多边阻力 FE、零贸易保留 | 只给 OLS-log，零贸易消失却不说明 |
| GVC 分解可追溯 | 注明 KWW、MRIO（WIOD/ADB/OECD ICIO）与年份 | 只标"增加值贸易"，分解法与表来源缺失 |
| 事件研究图可证平行趋势 | 横轴相对处理期、处理前系数近零含 CI | 仅给点估计无置信带，或省略处理前期 |

## 微型走查：关税冲击对出口企业加成率的主表（示意数字）

设一篇投本刊的稿件，处理变量为 2018 年美对华加征关税的企业进口中间品暴露度，因变量为加成率（De Loecker-Warzynski 法估），海关库×工业企业库匹配。按本 skill 走一遍：

```
原稿主表：12 列（全样本/分行业/分边际/分所有制各 3 列），无 FE 行，标准误未注层次
走查诊断：
  列数 12 > 6 → 砍至基准 + 高/低暴露分组 + 安慰剂 = 5 列，其余移附录
  暴露度系数（示意）：高暴露组 -0.043（se 0.012，按 HS2×省聚类）；低暴露组 -0.008（se 0.011）
  组间差异检验（示意）：交互项 p≈0.02，写入表注
  FE 行补全：企业□ 年份□ 行业×年□
  事件研究图：处理前 t=-3..-1 系数（示意）均落在 [-0.01,0.01] 含零，处理后逐期走低
修订主表骨架：列1 基准 / 列2 高暴露 / 列3 低暴露 / 列4 加并购控制 / 列5 安慰剂年份
表注模板：标准误按 HS2×省聚类于括号内；*/**/*** 为 10/5/1%；加成率以 DLW 法估算
```

> 走查内所有系数、标准误、p 值均为示意占位，仅演示决策规则，非真实估计结果。

## 审稿人追问模式 + 本刊语境修法

- 追问"加成率为何用 DLW 而非会计利润率" → 表注与正文交代生产函数估计法及产出弹性来源，附会计利润率作稳健性列。
- 追问"暴露度系数的标准误是否吸收了 shift-share 结构" → 若暴露度为份额×冲击构造，改报 exposure-robust SE，并在表注点明。
- 追问"事件研究图处理前不平的那一期怎么解释" → 补预期效应讨论或重设基准期，勿删点掩盖。
- 追问"引力表多边阻力固定效应是否齐" → 列出口国×年、进口国×年、国家对三类 FE 行，缺则补。

## 校准锚点（本刊已刊论文形态）

本刊贸易实证类论文的主表通常控制在一页内、列数克制，正文主表配 1–2 张事件研究/暴露度分布图，引力与 GVC 分解多以独立表呈现并在表注交代方法口径；具体版式（字号、三线表线型、图注语言）以编辑部最新稿约与录用后排版要求为准。

## 输出格式

```
【主表列数】X（建议 ≤6）
【表注规范】聚类□ 显著性□ FE□ N□
【专属图表】事件研究□ 引力□ GVC□ shift-share□
【数据来源标注】齐 / 缺
【下一步】jwe-policy-implication
```
