# 国际视角选题（jwe-topic-international）

## 触发时机

- 有数据 / 现象，但讲不出"开放经济中的因果问句"
- 选题像国内问题，国际性只是背景
- 想从"描述全球趋势"升级为"可识别的因果 / 一般均衡命题"

## 好选题的三种范式

1. **外部冲击 → 国内结果**：关税战 / 全球需求冲击 / 美联储加息 → 企业出口、就业、价格、投资
2. **开放政策 → 资源再配置**：入世 / 自贸区 / 外资负面清单 / RCEP → 企业进入退出、GVC 嵌入、加成率
3. **中国 ↔ 世界双向互动**：中国冲击对外溢出（China shock 的反向）、对外直接投资的母国效应、人民币国际化

## 把问句立准（开放经济版）

- 谁是处理组 / 暴露组？暴露度如何由**国际**结构（进口渗透、出口依赖、GVC 上下游）度量？
- 反事实是什么？没有这次开放 / 冲击会怎样？
- 因变量落在哪个**贸易 / 投资 / 金融**边际上（扩展 vs 集约边际、价格 vs 数量、母国 vs 东道国）？
- 是局部均衡问题还是需要**一般均衡 / 福利**口径（关税的福利、GVC 重构）？

## 选题升级对照

| 偏弱（描述/国内腔） | 升级（开放经济因果） |
|----------------------|----------------------|
| "全球化对中国经济的影响" | "关税暴露度更高的城市，制造业就业如何变化（local labor market）" |
| "一带一路促进了贸易" | "OFDI 进入沿线国后，母国企业出口的产品结构如何调整" |
| "汇率影响出口" | "汇改作为外生冲击，高汇率敞口企业的定价与加成如何响应" |
| "GVC 很重要" | "用 KWW 分解的 GVC 位置上移，是否提高企业国内增加值率" |

## 自检清单

- [ ] 问句含明确的**处理 / 暴露 + 反事实**
- [ ] 暴露度由国际结构（进口渗透 / 出口依赖 / GVC）度量
- [ ] 因变量落在具体贸易 / 投资 / 金融边际
- [ ] 已区分局部均衡与一般均衡 / 福利口径
- [ ] 贡献能写成一句"修正 / 扩展了国际经济学的某个判断"

## 反模式

- "研究 XX 的影响"式无方向、无反事实的题目
- 把中国当封闭系统，外部只作控制变量
- 用宏大词（全球治理、百年变局）替代可识别命题

## 《世界经济》选题对口期待表

《世界经济》（中国社会科学院世界经济与政治研究所 IWEP 与中国世界经济学会主办的 CSSCI 权威顶级期刊）收的是"开放经济视角下的可识别命题"，而非把全球趋势当背景板的国内题。下表把本刊对选题的隐性期待显性化。

| 选题维度 | 本刊期待 | 高风险（易判不对口） |
|----------|----------|----------------------|
| 处理来源 | 国际/制度性冲击（关税、汇改、负面清单、RCEP） | 纯国内政策，国际仅作时代背景 |
| 暴露度构造 | 由国际结构度量（进口渗透、出口依赖、GVC 上下游、汇率敞口） | 用省份虚拟变量或规模分组冒充暴露 |
| 因变量边际 | 落在贸易/投资/金融的具体边际（加成率、DVAR、外汇敞口） | 笼统"经济发展""高质量发展" |
| 均衡口径 | 明确局部均衡 vs 一般均衡/福利（关税福利、GVC 重构） | 把局部结果直接当全局福利推断 |

## 微型走查：汇率传递选题打磨（示意）

设作者手握"汇率影响出口"的模糊想法。按本 skill 把它立成本刊级问句：

```
原始想法：人民币汇率波动对出口的影响
走查升级：
  范式判定 → 外部冲击→国内（汇改作准外生冲击）
  处理/暴露 → 2015 年"8·11"汇改；暴露=企业外币计价出口占比（示意分布：均值 0.34）
  反事实 → 无此次中间价机制改革，高敞口企业定价路径如何
  因变量边际 → 出口产品的目的国货币定价加成（price-to-market），而非笼统出口额
  均衡口径 → 局部均衡（企业定价响应），暂不外推至整体经常账户福利
立准问句：汇改提高汇率弹性后，外币敞口更高的出口企业，其加成率与目的国定价如何调整
```

> 走查中的暴露占比等数字为示意，仅演示从模糊想法到可识别问句的决策链，非实证结果。

## 审稿人追问模式 + 本刊语境修法

- 追问"这题去掉'国际'还成立吗" → 回到 `jwe-fit-positioning` 自检；若成立则上移处理变量到国际冲击，否则改投。
- 追问"暴露度凭什么外生" → 在选题阶段就把暴露锚在初期份额×外部共同冲击上，为后续 shift-share 推断留接口。
- 追问"为什么是这个因变量边际" → 用异质性企业（Melitz）或定价（Gopinath 主导货币）理论指认应观察的边际，而非可得即用。

## 校准锚点（本刊已刊论文形态）

本刊近年贸易/开放宏观实证题，多以一个清晰的国际冲击或开放政策为处理、以国际结构度量暴露、落到企业或产品层的具体边际，并在引言前段即交代开放经济属性；选题的"国际比较/对中国对外开放议题的实证"双重底色较突出。具体选题口味与栏目偏好以编辑部最新稿约为准。

```
【范式】外部冲击→国内 / 开放政策→再配置 / 中国↔世界

【暴露度度量】<进口渗透/出口依赖/GVC/汇率敞口>
【均衡口径】局部 / 一般均衡-福利
```
