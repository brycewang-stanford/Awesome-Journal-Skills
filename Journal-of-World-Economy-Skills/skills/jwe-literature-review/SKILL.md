# 国际文献对话（jwe-literature-review）

## 触发时机

- 综述只堆近年国内回归，缺国际经典与前沿
- 说不清自己站在国际贸易 / 国际金融哪条脉络上
- 审稿人很可能问"和 XX 的国际文献有何对话"

## 核心原则

《世界经济》看重**与国际经济学脉络的对话**。作为中国世界经济学会与 IWEP 主办的国际经济学旗舰刊（1978 年创刊），它默认作者站在国际贸易/国际金融的世界文献里说话——这正是它与纯国内导向期刊的分野：只引国内实证 ≈ 没进门。要按"源流 → 进展 → 本文位置"组织，而非罗列。

## 必备脉络锚点（按子领域，举例非穷尽）

| 子领域 | 经典锚点 | 前沿方向 |
|--------|----------|----------|
| 贸易理论 | 比较优势、新贸易理论（Krugman）、异质性企业（Melitz） | 企业-产品-目的国微观、加成率与质量 |
| 贸易实证 | 引力模型（Anderson-van Wincoop）、China shock（Autor-Dorn-Hanson） | 本地劳动力市场效应、贸易战实证（关税转嫁 Amiti 等） |
| 全球价值链 | 垂直专业化（Hummels）、增加值贸易（KWW: Koopman-Wang-Wei） | GVC 上游度、国内增加值率、MRIO 分解 |
| 国际金融 | Mundell-Fleming、不可能三角、汇率传递 | 全球金融周期（Rey）、资本流动顺周期、外汇敞口 |
| 开放宏观 | 实际汇率、经常账户失衡 | 主导货币定价（Gopinath）、人民币国际化 |
| FDI | OLI 范式、水平 vs 垂直 FDI | OFDI 母国效应、负面清单准入 |

> 锚点是脉络坐标，不是凑引用。务必落到"本文相对该脉络补了什么"。

## 中外并重，但国际为主轴

- 国际经典 / 前沿是**主轴**，必须对话
- 中国相关文献用来定位"中国情境的独特性 / 贡献"
- 避免只引中文、或只引五年内 working paper

## 自检清单

- [ ] 至少进入一条清晰的国际理论脉络（贸易 / 金融 / GVC / 开放宏观）
- [ ] 引了该脉络的**经典**与**近 3–5 年前沿**各若干
- [ ] 综述按源流 / 进展 / 本文位置组织，非编号罗列
- [ ] 明确写出"本文相对国际文献的边际贡献"
- [ ] 涉及 GVC / 关税 / 汇率传递等，引到对应方法论原文（KWW、引力等）

## 反模式

- 全是中文实证，无一篇国际前沿
- 引了 Melitz / 引力但全文方法与之无关（贴金）
- "国内尚无研究"作为唯一贡献论证

## 《世界经济》文献综述审稿期待表

本刊（中国社会科学院 IWEP 与中国世界经济学会主办的 CSSCI 权威顶级期刊）默认作者站在国际贸易/国际金融的世界文献里说话。下表把综述环节的退稿信号显性化。

| 审稿期待 | 达标 | 退稿/退修信号 |
|----------|------|----------------|
| 进入一条清晰国际脉络 | 明确站在引力/异质性企业/GVC/开放宏观某脉络 | 只堆近年国内回归，未进国际门 |
| 经典+前沿双锚 | 既引脉络奠基作，也引近 3–5 年前沿 | 只引五年内 working paper 或只引经典 |
| 源流式组织 | 按"源流→进展→本文位置"叙述 | 编号罗列、文献堆砌无对话 |
| 边际贡献相对国际文献 | 写清"相对该脉络补了什么" | "国内尚无研究"作为唯一贡献论证 |

## 微型走查：China shock 反向溢出选题的综述定位（示意）

设稿件研究"中国 OFDI 进入东道国后对母国企业出口结构的影响"。按本 skill 给它定脉络坐标：

```
走查定位：
  主脉络 → 贸易实证（China shock 谱系）+ FDI（OFDI 母国效应）
  经典锚点 → 异质性企业 Melitz；引力 Anderson-van Wincoop；China shock Autor-Dorn-Hanson；OLI 范式
  前沿对话 → ADH 后续本地劳动力市场文献的"反向溢出"延伸；OFDI 母国生产率/出口升级近作
  中国情境定位 → 以中国为母国的 OFDI 微观数据少见，可补"对外投资—母国出口结构"证据
  边际贡献一句话 → 把 China shock 文献从"中国冲击世界"扩到"中国对外投资反作用于母国出口边际"
反模式自检 → 是否只为贴金引 Melitz 却全文方法无关？否，异质性企业含义直接用于分组
```

> 走查中的脉络与锚点为演示性举例，非穷尽文献清单，具体引用以作者核实原文为准。

## 审稿人追问模式 + 本刊语境修法

- 追问"你和 XX 的国际文献到底有何对话" → 把综述从"罗列"改为"源流→进展→缺口→本文"，在缺口处嵌入本文。
- 追问"引了 Melitz/引力却没用上" → 删贴金引用，或让该理论真正进入设定与异质性预期。
- 追问"中国文献和国际文献谁是主轴" → 国际经典/前沿为主轴对话，中文文献用于定位中国情境的独特贡献。

## 校准锚点（本刊已刊论文形态）

本刊论文综述通常以国际贸易/国际金融脉络为主轴展开对话，中文文献用于锚定中国对外开放议题的情境贡献，并在综述末尾给出相对国际前沿的边际贡献；引用密度与中外文献配比以编辑部最新稿约及栏目惯例为准。
