# 异质性分析（jwe-heterogeneity）

> 《世界经济》（IWEP + 中国世界经济学会主办，月刊）偏好**国际经济学意义上的切分**：暴露度（进口渗透/出口依赖/GVC 联系/汇率敞口）、贸易类型（一般 vs 加工贸易）、贸易边际（扩展 vs 集约）、目的国/伙伴。东中西、国企民企这类通用维度在本刊价值有限——它们换到任何国内刊都成立，立不住本刊的开放属性。基础事实见 `resources/journal-profile.md`。

## 触发时机

- 异质性只切了东中西 / 国企民企，缺国际经济学意义的维度
- 主效应 + 渠道已立，需要刻画"对谁更强"
- 审稿人会问"效应在不同暴露 / 贸易类型间是否一致"

## 开放经济专属的切分维度（优先这些）

| 维度 | 切分方式 | 为什么有意义 |
|------|----------|--------------|
| **暴露度** | 高 vs 低进口渗透 / 出口依赖 / GVC 联系 / 汇率敞口 | 直接检验冲击经暴露通道传导 |
| **贸易类型** | 一般贸易 vs 加工贸易；中间品 vs 最终品出口商 | 加工贸易对关税 / 汇率响应不同 |
| **企业属性** | 内资 vs 外资；高 vs 低生产率（Melitz 含义） | 异质性企业理论可检验含义 |
| **目的国 / 伙伴** | 发达 vs 发展中；签 vs 未签协定国；远 vs 近 | 引力 / 协定效应的异质性 |
| **贸易边际** | 扩展边际（进入退出）vs 集约边际（量价） | 区分谁在调整 |
| **GVC 位置** | 上游 vs 下游；高 vs 低国内增加值率 | 全球生产网络中的位置差异 |

## 做实异质性（而非堆回归）

- 每个切分都要有**理论预期**：为什么这一组应更强 / 更弱
- 用交互项 + 系数差异检验，别只贴两张分样本表
- 异质性方向应与主渠道一致（高暴露组效应更大 → 支持该渠道）
- 谨防"切到显著为止"的数据探查

## 自检清单

- [ ] 至少一个维度是国际经济学专属（暴露 / 贸易类型 / GVC / 目的国）
- [ ] 每个切分有事前理论预期
- [ ] 报告系数差异的统计检验（交互项 / 组间检验）
- [ ] 异质性方向与主渠道自洽
- [ ] 没有把异质性当"凑显著"的捞针

## 反模式

- 只切东中西 / 大小企业等通用维度
- 切了一堆维度但无理论解释
- 分样本显著差异不做组间检验
- 异质性方向与机制矛盾却不解释

## 《世界经济》异质性审稿期待表

本刊（IWEP + 中国世界经济学会主办的 CSSCI 权威顶级期刊）要的异质性，是"国际经济学意义上的切分"——换到任何国内刊都成立的东中西、国企民企维度，在本刊立不住开放属性。下表把退修信号显性化。

| 审稿期待 | 达标 | 退修/退稿信号 |
|----------|------|----------------|
| 至少一维属开放经济专属 | 暴露度/贸易类型/GVC 位置/目的国 | 仅东中西、规模、所有制等通用维度 |
| 每维有事前理论预期 | 引 Melitz/加工贸易含义说明方向 | 切一堆维度无理论解释 |
| 报告系数差异检验 | 交互项或组间检验给 p 值 | 只贴两张分样本表说"更大" |
| 方向与主渠道自洽 | 高暴露组更强 → 支持该渠道 | 异质性方向与机制矛盾不解释 |

## 微型走查：贸易战转嫁的异质性切分（示意）

设主效应为"进口中间品关税上升压低出口企业加成率"。按本 skill 把异质性做实：

```
原稿异质性：东部 vs 中西部、大企业 vs 小企业（两张分样本表，无组间检验）
走查改造：
  换专属维度 → 加工贸易 vs 一般贸易；高 vs 低进口中间品依赖；高 vs 低生产率
  事前预期 → 加工贸易"两头在外"，关税转嫁敏感度更高（理论方向：更负）
  交互检验（示意）：关税×加工贸易交互项系数 -0.021（p≈0.03）
  组间对照（示意）：加工组 -0.05 vs 一般组 -0.02，差异检验 p≈0.04
  自洽核验 → 高依赖、加工组效应更大 → 与"贸易渠道经中间品传导"主机制一致
保留维度：加工/一般、进口依赖高/低；剔除纯东中西切分
```

> 走查中的交互系数与分组估计为示意占位，仅演示"通用维度→开放经济维度"的改造逻辑，非真实结果。

## 审稿人追问模式 + 本刊语境修法

- 追问"东中西差异说明了什么开放经济含义" → 替换为暴露度/贸易类型/GVC 位置等可检验渠道含义的维度。
- 追问"两组系数差异显著吗" → 补交互项或 Chow/组间检验，给出 p 值，勿凭点估计大小下结论。
- 追问"异质性方向和你的机制对得上吗" → 让高暴露/高敞口组效应更强，方向与主渠道一致；矛盾处需解释或重审机制。

## 校准锚点（本刊已刊论文形态）

本刊实证论文的异质性多沿贸易类型、暴露度、企业生产率（Melitz 含义）、目的国/伙伴等国际经济学维度展开，并配交互项或组间检验佐证；每一维通常对应一条事前理论预期。具体维度选择与呈现规范以编辑部最新稿约为准。

## 输出格式

```
【维度】暴露度 / 贸易类型 / 企业 / 目的国 / 贸易边际 / GVC位置
【理论预期】<每维度一句>
【差异检验】交互项□ 组间检验□
【与渠道一致性】是 / 否 <说明>
【下一步】jwe-tables-figures
```
