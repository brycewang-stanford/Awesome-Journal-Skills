## 用法

- **学叙事与定位，不学具体数字。** 看这些论文如何把"中国政策准实验"写成现象先行的故事，
  如何把识别策略与机制嵌进引言（对照 `worked-examples/01-introduction.md` 与
  `../../skills/cie-fit-positioning/SKILL.md`）。
- **机制以江艇(2022)为准绳。** 本刊对三步中介持保留态度，机制首选分渠道回归与调节效应；起
  机制部分前先读 `../../skills/cie-mechanism/SKILL.md`。
- **识别按本刊红线做满。** 交错 DID 必上事件研究图 + 安慰剂 + 异质性稳健估计（Bacon 分解 /
  CS / SA / dCDH），见 `../../skills/cie-did-identification/SKILL.md`；方法规范底线另见
  [`../../../shared-resources/empirical-methods/reporting-standards.md`](../../../shared-resources/empirical-methods/reporting-standards.md)。
- **引用前核对刊名。** 凡记不清是 CIE 还是《经济研究》/《管理世界》/《数量经济技术经济研究》
  的篇目，先回 CNKI（GGYY）或官网（`ciejournal.ajcass.com`）核对，再写进参考文献。

## 排除项（核实为其他venue或暂未核实，勿误记为 CIE）

- 多数**纯计量方法名篇**与宏观 / 一般均衡题材常发在**《经济研究》**——引用前务必核对刊名，
  不要因为"方法很顶"就默认是《中国工业经济》。
- 偏**管理理论 / 案例 / 问卷量表**的论文发在**《管理世界》**（本包 `cie-fit-positioning` 的
  改投对照表亦如此分流）。
- 偏**计量技术 / 技术经济**方法的论文常发在**《数量经济技术经济研究》**。
- 题名相近但本轮未能逐项核实卷期的"智能制造 / 宽带中国 / 全要素生产率"类论文——一律按
  `待核实 / TO VERIFY` 处理，不入正表。
