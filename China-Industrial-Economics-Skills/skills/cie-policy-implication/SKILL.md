# 政策含义（cie-policy-implication）

## 触发时机

- 文末政策建议是"加强、完善、推进、深化"四件套
- 建议谁都能写，与本文实证发现无关
- 建议停在"意义层"，没说具体怎么做

## 本刊与《经济研究》的关键差异

投稿指南："'启示/政策建议'**来源于结论又高于结论**…体现理论对于实践的指导意义，求新、求深、求实。"本刊办刊理念"**实践立地**"——政策建议必须**可操作**：

> **谁（主体）、在哪个环节（节点）、做什么（动作）、依据本文哪条发现 / 哪个异质性。**

这与《经济研究》"政策含义偏意义层"形成鲜明对比。

## 可操作建议四要素

| 要素 | 反例（不可操作） | 正例（可操作） |
|------|------------------|----------------|
| 主体 | "政府应…" | "试点城市发改部门 / 行业主管部门" |
| 节点 | "加强支持" | "在试点验收的资金拨付环节" |
| 动作 | "完善机制" | "将拨付与企业研发投入增量挂钩" |
| 依据 | （无） | "因本文发现效应集中于融资约束高的企业" |

## 从发现到建议（映射）

- **主效应** → 该政策值得推广 / 调整的总体判断
- **机制** → 建议应作用在机制环节（如经由融资约束 → 配套融资支持）
- **异质性** → **差异化施策**（对哪类主体加力、对哪类退出），这是本刊最看重的政策落点
- 注意边界：本文未检验的不要外推承诺

## 自检清单

- [ ] 每条建议有**主体 + 节点 + 动作**
- [ ] 每条建议**对应本文一条发现 / 一个异质性维度**
- [ ] 有**差异化政策**（来自异质性），不是一刀切
- [ ] 无"加强/完善/推进/深化"空泛四件套
- [ ] 不超出实证证据外推
- [ ] 与摘要、结论的核心观点一致，不重复堆砌

## 反模式

- "建议政府高度重视 X"——谁都能写
- 政策建议与实证发现脱节（"两张皮"）
- 一刀切建议，浪费了异质性结果
- 把"启示"写成对结论的简单复述

## 本刊政策建议审稿期待与退稿模式

| 审稿期待（"实践立地"口径） | 达标证据 | 退稿/退修模式 |
|----------------------------|----------|----------------|
| 可操作 | 每条含主体+节点+动作 | "加强/完善/推进/深化"四件套 |
| 有依据 | 每条挂本文一条发现/异质性 | 建议谁都能写、与发现无关 |
| 差异化 | 由异质性推出"对谁加力/退出" | 一刀切，浪费异质性结果 |
| 不外推 | 只建议证据覆盖到的范围 | 把未检验结论包装成承诺 |

> "两张皮"（模型/实证/政策各说各话）是本刊明确反对、且高频退修的问题；具体审稿尺度以编辑部最新意见为准。

## 微型走查：智能制造试点 × TFP 的政策建议

承接主效应 +0.043（约样本均值 6.5%）、机制为"数字化改造主渠道"、异质性为"融资约束高的民企效应更大"。把发现逐条翻译成可操作建议：

1. **总体（来自主效应）**：试点显著提升 TFP，建议**工信主管部门**在**下一批试点遴选环节**扩大覆盖面，而非笼统"推进智能制造"。
2. **机制配套（来自数字化渠道）**：建议**地方发改/工信部门**在**试点验收的资金拨付环节**，将补贴与企业**数字化改造投入增量**挂钩，强化主渠道。
3. **差异化（来自异质性）**：因效应集中于融资约束高的民企，建议**地方金融监管与政策性银行**对**入选民企的技改环节**定向配套低息技改贷款；对预算软约束的国企则侧重考核约束而非补贴加码。
4. **边界**：本文未检验对就业、能耗的影响，故不在建议中承诺这些维度。

四条建议条条含主体+节点+动作+依据，接 `cie-submission`。

## 审稿人追问 × 本刊语境修法

- 追问"政策建议太空，谁都能写。" → 修法：每条补齐"主体—节点—动作"，删掉"加强支持""高度重视"。
- 追问"建议和你的发现什么关系？" → 修法：在每条建议后用括号标注"依据本文表 X / 第 Y 维异质性"，消除两张皮。
- 追问"异质性那么丰富，建议却一刀切。" → 修法：把差异化施策写实——对哪类主体加力、对哪类退出或改用考核。
- 追问"建议外推到本文没验证的领域。" → 修法：加边界声明，只对证据覆盖范围提建议。

## 校准锚点

- "启示来源于结论又高于结论""实践立地"等为本刊投稿指南口径；具体措辞与篇幅以官网最新《投稿（修改）指南》为准。
- 上述四条建议中的部门名称、挂钩机制为示意性表达，真实建议须据本文样本与发现具体化。
- 与《经济研究》"政策含义偏意义层"的对比是经验性概括，非绝对规则。

## 输出格式

```
【可操作性】高 / 中 / 低
【建议条目】每条：主体 + 节点 + 动作 + 依据发现
【差异化】来自异质性 √ / 一刀切（需改）
【四件套命中】X 处（应为 0）
【外推风险】无 / 有 <点>
【下一步】cie-submission
```
