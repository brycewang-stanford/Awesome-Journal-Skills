# 《计算机学报》录用后定稿与校样

稿件在《计算机学报》(Chinese Journal of Computers, CJC) 通过主编终审录用后，进入**定稿→排版→
校样(清样)→出版**阶段。这一阶段的目标不是继续改进研究，而是把一篇已被接受的长文**零差错地送进
正式出版**。CJC 是 CCF A 类综合性中文月刊，正式出版会被 EI、Scopus、CSCD 等收录，因此定稿与
校样的规范程度直接影响见刊质量与后续检索。事实核验日期 2026-07-09，缴费与排期等以编辑部通知为准，
未确认项标 **待核实**。

## 一、录用后的即时事项

- **著作权/版权协议**：按编辑部要求签署著作权转让或授权协议，全体作者签字齐全。
- **确认作者信息终稿**：署名顺序、通信作者、单位署名、基金项目及编号一经定稿不宜再动，务必此刻核准。
- **审理费/版面费**：按通知缴纳相关费用（审理费口径为 150 元，属报告值；版面费金额 **待核实**，
  以编辑部正式通知为准），保留缴费凭证。

## 二、定稿排版核对清单

用官方模板做最终排版，逐项核对：

| 部件 | 定稿检查点 |
|---|---|
| 中文题名 / 英文题名 | 两者含义一致、无笔误、可独立检索 |
| 中文摘要 / English Abstract | 报道性四要素齐全，中英文内容对应、时态与术语统一 |
| 关键词 / Keywords | 中英文一一对应，数量与顺序一致 |
| 中图分类号 | TP 类号准确，与主题匹配 |
| 作者简介 / 基金项目 | 研究方向、资助编号无误 |
| 图表 | 图题表题中英文规范，量与单位用法定计量单位，分辨率达印刷要求 |
| 参考文献 | GB/T 7714 著录完整，中文文献中英文对照，序号与正文引用一致 |

## 三、校样(清样)阶段纪律

编辑部返回校样后，作者的职责是**校对而非改写**：
- **只做编辑性勘误**：改正排版错误、错别字、公式符号错排、图表串位、参考文献著录疏漏。
- **不做实质性改动**：录用后不得新增结论、替换实验、改动核心数据；确有必要须经编辑部同意。
- **公式与符号重点校**：中文排版易在下标、上标、正斜体、矢量加粗、希腊字母处出错，逐一核对。
- **交叉引用核对**：图 X、表 Y、第 Z 节、文献[n] 的编号在最终排版后仍然一一对应。
- **按时返回**：校样通常有紧的返回时限，逾期会影响见刊排期。

## 四、勘误标注方式

- 在校样 PDF 上用统一批注列出勘误：位置(页/栏/行) + 原文 + 改为。
- 汇总成一份**勘误清单**随校样返回，便于排版人员精确定位，避免"改一处漏一处"。
- 若发现录用稿本身的可修正笔误（非实质），一并列出并说明。

## 五、常见见刊差错对照

| 差错 | 后果 | 防范 |
|---|---|---|
| 英文摘要与中文不一致 | 检索与国际可见性受损 | 定稿时中英文逐句比对 |
| 基金编号/作者单位错排 | 影响署名权益与资助核销 | 录用即锁定并双人核对 |
| 参考文献序号错位 | 引证失真 | 排版后重跑正文—文献对应 |
| 校样阶段擅自改数据 | 违反出版规范，延期或撤稿风险 | 只做编辑性勘误 |
| 逾期未返校样 | 顺延到后续期次 | 收到即排期，预留缓冲 |

## 六、出版排期与见刊后

- 与编辑部确认**拟刊期次/卷期页码**与见刊时间（月刊按月排期）。
- 见刊后核对正式版 DOI、卷期页码与作者信息；如发现印刷错误，按编辑部渠道申请正式勘误(erratum)。
- 保存正式版 PDF、DOI 与引用信息，供后续 GB/T 7714 规范引用与成果登记。

## 七、需向编辑部确认的易变项

- 版面费与审理费的**具体金额与缴纳方式**（**待核实**）。
- 是否要求提供开放的代码/数据链接或补充材料的最终托管地址（见 `cjc-reproducibility`、
  `cjc-supplementary`）。
- 校样返回的**具体时限**与是否提供二校机会。

## 八、署名与版本管理

- **署名一经定稿不宜再改**：录用后增删作者或调整顺序需全体作者书面同意并经编辑部批准，属敏感事项，
  务必在录用前解决好署名争议。
- **版本单一化**：以编辑部返回的录用稿为唯一基线，作者本地不要再分叉多个版本；所有勘误集中在校样
  一次性提交，避免"两份稿子对不上"。
- **保留可追溯记录**：把录用稿、每次校样、勘误清单、缴费凭证、著作权协议归档留存。

## 九、校样勘误样例

```text
[勘误清单]
P511 左栏 第8行: "卷积和网络" → "卷积神经网络"
P512 式(7): 下标 i 应为 j，与式(6)一致
P514 表3 第2列表头: 单位缺失，应补 "(ms)"
P518 文献[15]: 缺页码，应补 "623-637"；中文文献缺英文对照，已补
```

逐条给"位置→原文→改为"，排版人员可精确定位，避免改一处漏一处。

## 十、见刊后的存档与推广

- 见刊后核对正式版 **DOI、卷期、页码**与作者信息，保存正式 PDF。
- 按 GB/T 7714 生成本文的标准引用条目，供个人主页、成果登记与他人引用。
- 在遵守本刊版权协议的前提下，可在个人主页或机构库放置合规版本；如需自存档，先确认协议允许的版本
  与延迟期(**待核实** 本刊自存档政策)。
- 将录用后固定版本的代码/数据链接更新为**永久链接**（见 `cjc-artifact-evaluation`）。
