# English Social-Science Breadth Resources

This layer supports routing across the English economics, finance, management,
accounting, marketing, operations, and information-systems fit cards.

## What to Load

- `official-source-map.md` - official-source anchors for live author-guideline
  checks.
- `source-basis.md` - source discipline and coverage basis for this breadth
  bundle.
- `worked-examples/venue-routing.md` - diagnostic examples for common
  venue-selection decisions.
- `exemplars/selection-patterns.md` - field-specific routing archetypes and
  sibling-journal traps.

## Usage Rule

Start with `en-journal-workflow` when the venue is uncertain. Once the target is
known, use the per-journal breadth skill for quick fit and then, when available,
route into the corresponding first-party depth pack for lifecycle work.
