# NAACL Resources

Working materials that complement the twelve skills in
[`../skills/`](../skills/). The skills advise; these files let an agent
verify facts, benchmark against real NAACL papers, study the target prose
register, and assemble a reproducibility package.

## Scope note

This pack covers the *ACL-family conference pipeline as it applies to NAACL:
ARR cycles, commitment, Findings, and Anthology publication. It is not a
journal pack (no editor correspondence, no APC logic) and not a generic ML
pack — for experiment scaffolding it defers to the shared ML-conference kit
via `code/README.md` rather than duplicating it.

## Suggested route through the pack

1. **Decide the venue and the timing** with
   [`../skills/naacl-topic-selection/SKILL.md`](../skills/naacl-topic-selection/SKILL.md)
   and [`../skills/naacl-workflow/SKILL.md`](../skills/naacl-workflow/SKILL.md),
   verifying the calendar via [`official-source-map.md`](official-source-map.md).
2. **Build the evidence** against
   [`../skills/naacl-experiments/SKILL.md`](../skills/naacl-experiments/SKILL.md)
   and [`../skills/naacl-reproducibility/SKILL.md`](../skills/naacl-reproducibility/SKILL.md),
   packaging with [`code/README.md`](code/README.md).
3. **Write and file** using
   [`../skills/naacl-writing-style/SKILL.md`](../skills/naacl-writing-style/SKILL.md),
   the [`worked-examples/01-introduction.md`](worked-examples/01-introduction.md)

4. **Survive review and publish** with the response, review-process,
   camera-ready, and supplementary skills, benchmarking prose against
   [`exemplars/library.md`](exemplars/library.md).
