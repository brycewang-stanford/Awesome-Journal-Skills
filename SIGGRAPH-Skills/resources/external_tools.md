# External Tools - ACM SIGGRAPH Technical Papers

Access date: 2026-07-09

Use these surfaces after reopening the current SIGGRAPH cycle pages. SIGGRAPH mechanics — the two
annual cycles, the page budget, the rebuttal length, the dual-track boundary, the acmart revision —
are decided per edition, and the `s2026.siggraph.org` / `asia.siggraph.org` / `dl.acm.org` gateway
may block direct fetches (read via search renderings and cross-check ACM DL and dblp). Do not
confuse the ACM SIGGRAPH conference with unrelated uses of the word "graph."

## Official workflow

- SIGGRAPH 2026 Technical Papers: https://s2026.siggraph.org/program/technical-papers/
- SIGGRAPH 2026 Technical Papers submissions FAQ: https://s2026.siggraph.org/technical-papers-submissions-faq/
- SIGGRAPH 2026 reviewer instructions & ethics (review model, rebuttal, decisions): https://s2026.siggraph.org/technical-papers-reviewer-instructions-ethics/
- SIGGRAPH Asia 2026 Technical Papers (next-cycle target): https://asia.siggraph.org/2026/submissions/technical-papers/
- ACM SIGGRAPH author instructions (acmart, representative image, supplemental): https://www.siggraph.org/preparing-your-content/author-instructions/
- ACM Transactions on Graphics journal home: https://dl.acm.org/journal/tog
- ACM TOG author guidelines (rolling direct submission via ScholarOne): https://dl.acm.org/journal/tog/author-guidelines
- Submission system: Linklings (linked from the current cycle's submission page)
- Graphics Replicability Stamp Initiative (GRSI): https://www.replicabilitystamp.org/
- Code Replicability in Computer Graphics (CRCG): https://replicability.graphics/

## Cross-check surfaces (when the gateway blocks the primary page)

- ACM DL TOG tables of contents — SIGGRAPH 2026 = **Vol. 45, Issue 4**; SIGGRAPH Asia appears as a
  separate issue in the same volume year.
- dblp ACM Transactions on Graphics stream: https://dblp.org/db/journals/tog/index.html
- dblp SIGGRAPH proceedings series (pre-TOG editions and exemplar verification).
- Ke-Sen Huang's SIGGRAPH/TOG paper listings (community index of accepted papers per year).
- ACM SIGGRAPH blog and press releases for submission/acceptance counts and award announcements.

## Author-side checks

- Linklings profile, conflicts, co-author list, topic areas, and the **submission-form** step that
  locks authors before the paper-upload deadline.
- ACM `acmart` (>= 2.16) compliance, the current Conference-track page budget (<= 7 pages excluding
  references and up to two figures-only pages), embedded fonts, and the paper ID on every page.
- The supplemental package: representative image, results **video** (rights cleared on every
  frame), comparison gallery, code/data, appendix.
- Rebuttal readiness: the <= 1000-word, **text-only** author response (no images/video/URLs).
- Reproducibility/replicability intent: a runnable code release, deterministic result regeneration,
  and Software Heritage / Zenodo archiving for a Graphics Replicability Stamp.

## Do not infer

- Do not infer the deadline: SIGGRAPH (North America, ~January) and SIGGRAPH Asia (~May) are
  separate cycles. Confirm which one and its exact **UTC** cutoff on the live submissions FAQ.
- Do not assume the page budget, acmart revision, rebuttal length, dual-track availability, or
  blinding policy carry over between editions — SIGGRAPH review has historically been single-blind,
  but confirm each cycle (**待核实** for the exact 2026 wording).
- If pages disagree, treat the newest technical-papers call, the submissions FAQ, or a direct
  Technical Papers Chair announcement as controlling, and record the conflict.
