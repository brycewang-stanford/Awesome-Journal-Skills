# Nature Geoscience — Pre-Submission Checklist

> Verify all volatile specifics (word/display-item/reference limits, content types, Reporting
> Summary, repository lists, portal) on the official Nature Geoscience author pages before
> submitting (Checked: 2026-07-16).

## 1. Scope and venue

- [ ] The result is a broad-interest Earth-system advance (see `ngeo-scope-fit`)
- [ ] You have honestly considered a community journal / *Communications Earth & Environment*
- [ ] The single headline advance is stated in one sentence
- [ ] Every headline claim is grounded in data or a data-validated model, with quantified uncertainty

## 2. Length and structure

- [ ] Main text within the ~3,000-word Article limit (excludes abstract/Methods/refs/legends)
- [ ] Abstract ≤ ~200 words, unreferenced, uses "Here we show"
- [ ] Title ≤ ~90 characters
- [ ] Display items within the 4–6 ceiling
- [ ] References near/under ~50
- [ ] Results/Discussion divided by 3–4 topical subheadings (Article)
- [ ] Online Methods present and divided by topical subheadings

## 3. Format

- [ ] Correct content type selected (Article / Brief Communication)
- [ ] SI units; consistent significant figures; uncertainty on every quantity
- [ ] Every acronym/symbol defined on first use; notation consistent across text / Methods / figures / SI
- [ ] Nature reference style applied
- [ ] Accessible to non-specialists (cross-check `ngeo-writing-style`)

## 4. Figures and display items

- [ ] 4–6 display items only; each essential
- [ ] Fig. 1 conveys the Earth-system advance at a glance
- [ ] Uncertainty / significance shown (error bars, envelopes, stippling)
- [ ] Maps state projection, scale, units, data source
- [ ] Perceptually-uniform, color-blind-safe colormaps; no rainbow/jet; not color-only
- [ ] Captions define all symbols/units and are tight

## 5. Supplementary Information and Methods

- [ ] The main text stands alone without the SI
- [ ] No headline-advance evidence lives only in the SI
- [ ] Reproducible protocol + full uncertainty budget are in online Methods
- [ ] Every "see Supplementary" / "see Methods" pointer resolves
- [ ] SI items labeled Fig. Sn / Table Sn and each cited from the text or Methods

## 6. Compliance (gates acceptance)

- [ ] Data availability statement: repository + accession/DOI (community repo where available, e.g. PANGAEA)
- [ ] Code availability statement for custom analysis code
- [ ] Nature Portfolio Reporting Summary completed and consistent with Methods
- [ ] Ethics / sample-provenance / permit statements (fieldwork, cores, meteorites) if applicable
- [ ] Competing-interests statement
- [ ] Author-contributions statement

## 7. Files to upload

- [ ] Main manuscript (main text + online Methods)
- [ ] All figures at production resolution
- [ ] Supplementary Information file(s)
- [ ] Reporting Summary
- [ ] Cover letter (justifies broad interest; not seen by reviewers)
- [ ] Source data for main figures if required

## 8. Editorial Manager metadata and integrity

- [ ] Corresponding author, ORCID, and all affiliations complete
- [ ] Title / abstract / author list consistent with the manuscript
- [ ] Originality / not-under-consideration-elsewhere confirmed
- [ ] Prior preprint (ESSOAr / EarthArXiv) disclosed
- [ ] Related manuscripts under consideration disclosed
- [ ] Double-blind opt-in selected and manuscript anonymised, if chosen

## 9. Referees

- [ ] Suggested referees can judge correctness, evidence type, AND breadth
- [ ] At least one suggested referee from the claimed neighboring field
- [ ] No conflicts of interest among suggestions
- [ ] Excluded referees (short list, within any cap) with brief, factual rationale

## 10. Self-consistency

- [ ] Figure / table / reference numbering continuous and matching in-text callouts
- [ ] Cover-letter referee section matches the Editorial Manager form
- [ ] Cross-references intact after any last-minute trimming
