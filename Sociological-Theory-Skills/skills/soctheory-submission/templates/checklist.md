# Sociological Theory Pre-Submission Checklist

ST is a **theory-focused** journal: no hypothesis test, no estimation, no results section, no
data-availability statement. Any data is illustrative only. This checklist reflects that. Verify
all volatile specifics (length cap, abstract limit, processing fee, ASA Style edition, editorial
team, portal URL) on the official ST / SAGE author page before submitting (检索于 2026-06；以官网为准).

## 1. Scope & fit

- [ ] The manuscript makes a conceptual contribution (new theory / re-read tradition / metatheory / formal construction / synthesis)
- [ ] It intervenes in one specific theoretical conversation with an explicit move (extend / challenge / reconstruct / synthesize)
- [ ] There is no hypothesis test, no estimation, no results section; any data is illustrative
- [ ] The "before → after" theoretical contribution can be stated in one sentence
- [ ] Confirmed against the current ST aims & scope; not a better fit for ASR / AJS / Sociological Methodology / ARS / Theory and Society

## 2. Format

- [ ] Standard 12-pt font, double-spaced, generous margins (ST describes ~1.5-inch margins, ~40 manuscript pages — verify)
- [ ] Within the length cap: max ~14,500 words inclusive of footnotes, references, tables, figures, and appendices (verify)
- [ ] Separate title page (kept out of the anonymized body)
- [ ] Abstract within the current word limit (verify)
- [ ] Page/line numbering as required by the portal
- [ ] Figures meet SAGE resolution and format requirements

## 3. Masking (double-anonymous review)

- [ ] No author names, affiliations, or identifying phrases in the body
- [ ] Self-citations neutral ("a prior study (Author 2019)"), never "our earlier study"
- [ ] Acknowledgments and funding identifiers removed (added after acceptance)
- [ ] Document file properties scrubbed of author metadata
- [ ] File names free of author/institution names

## 4. Abstract & front matter

- [ ] Abstract states the problem, the theoretical move, and the contribution
- [ ] No data/results/hypothesis-test language in the abstract (there is none)
- [ ] Keywords selected per the journal's conventions

## 5. Theory quality

- [ ] Every concept is defined, bounded by its extension, and distinct from neighbors
- [ ] Assumptions stated explicitly
- [ ] Each proposition is preceded by reasoning and a named social mechanism
- [ ] The propositions are internally consistent
- [ ] Boundary conditions stated as part of the theory (scope / level / context)
- [ ] The strongest rival theory is engaged on theoretical grounds; a counter-case is handled
- [ ] Level of analysis is consistent or its transitions are theorized

## 6. Conceptual exhibits (no data plots)

- [ ] Each figure is a mechanism diagram / typology / process model / concept map — never a data plot or results table
- [ ] Every box/cell/node is a defined concept; every arrow names its mechanism
- [ ] The argument is recoverable from figure + caption
- [ ] Figure labels and numbering match the text exactly; exhibits respect the length cap

## 7. References

- [ ] ASA Style Guide author–date in-text citations; full reference list (7th edition, June 2022 — verify)
- [ ] Every in-text citation is in the list and vice versa
- [ ] The recent, live conversation is cited (not only the classics)
- [ ] Every attributed paper's venue is verified (no ST/ASR/AJS misattribution)
- [ ] All references are real and accurately represented

## 8. Ethics & portal (Manuscript Central / SAGE Track)

- [ ] Original, unpublished, and not under review elsewhere (ASA: simultaneous submission is unethical)
- [ ] Authorship and order agreed by all authors; each made a genuine contribution
- [ ] Prior conference/working-paper version disclosed if required
- [ ] Similarity check run; self-overlap disclosed
- [ ] ORCID iD provided for the corresponding author if required
- [ ] Manuscript processing fee paid (reported as $25, waived for ASA student members — verify) via SAGE Track
- [ ] Registered at mc.manuscriptcentral.com/soct; correct article type selected
- [ ] Files staged: anonymized manuscript + separate title page + any required forms
- [ ] Within portal file-size and file-type limits
