# Language (LSA) Manuscript Skeleton

A section-by-section skeleton for a *Language* **General Research Article**, anonymized for
double-anonymous review and conformant to the Language Style Sheet, Leipzig glossing, and IPA
conventions. Replace bracketed guidance; delete the notes before submission. An Overleaf/CUP template
is the authoritative source for exact formatting — verify against it.

## Manuscript file (anonymized)

### Title
[Descriptive and general enough that a non-specialist sees the topic. No author identifiers.]

### Abstract
[~150 words (verify current limit). State the question/puzzle, the data/language(s), the analysis, and
the finding. No citations if avoidable; readable on its own.]

### 1. Introduction
[Front-load the contribution: (a) the empirical generalization stated theory-neutrally; (b) the analysis
and its key prediction; (c) why a linguist outside the subfield should care; (d) the rival account it
adjudicates. Introduce a lead numbered example early.]

Example format:

```
(1)  a.  object-language-form           ← italics; object line
         morph-morph-morph              ← Leipzig gloss line, aligned; small-caps labels (PST, 3SG, ERG)
         'free translation'             ← single quotes
     b.  ...                            (source: Author fieldnotes / Corpus 2019)
```

### 2. Background / the generalization
[State the empirical pattern fully, with numbered, glossed examples. Give the theory-neutral statement
before any formalism.]

### 3. Analysis
[The account: assumptions explicit, non-vacuous. Show it derives the attested cases and blocks the
unattested ones. State observable predictions.]

### 4. Adjudication / competing analyses
[Engage the strongest rival framework(s) fairly. Give the prediction on which they diverge and show
which the data support.]

### 5. Data & methods
[For quantitative/experimental/corpus work: participants/consultants, stimuli/sample, task, coding,
measurement (e.g., forced alignment, Praat settings). For fieldwork: consultants, elicitation, archive.]

### 6. Results
[Report the model (mixed-effects where clustered), effect sizes, and intervals — not stars alone. Lead
with the exhibit carrying the main generalization. Numbered examples and figures with alt text.]

### 7. Discussion
[What the analysis contributes to the general question; scope conditions; what remains open.]

### 8. Conclusion
[Restate the contribution and its cross-subfield significance in a few lines.]

### Glossing abbreviations
[Alphabetical list of every non-standard grammatical label used, per Leipzig conventions.]

### Appendices
[Supplementary tables/derivations, labeled A1, A2, ... Count toward the inclusive word limit.]

### References
[Language / Unified Style Sheet author-date form. Excluded from the word count. Not APA/Chicago.]
