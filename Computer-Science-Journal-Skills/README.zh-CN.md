# 《计算机科学》(Computer Science, JSJKX) 投稿技能包 · 中文详版

本页是 [`README.md`](README.md) 的中文详版，展开介绍《计算机科学》(Computer Science, 简称 JSJKX) 的定位、
投稿要点与 12 个技能如何配合使用。**《计算机科学》是一本期刊(journal)，不是会议**，请勿混淆。

## 一、期刊定位（面向计算机全学科的中文综合月刊）

《计算机科学》(Computer Science) 创刊于 1974 年 1 月，由重庆西南信息有限公司主办，为月刊；国内统一刊号
CN 50-1075/TP、国际刊号 ISSN 1002-137X。它是 **中国计算机学会(CCF) 会刊**、**CCF 推荐 B 类中文科技期刊**、
**CCF 计算领域高质量科技期刊分级目录 T2 级**，同时被 **CSCD**（核心库）、**北大中文核心** 与 **中国科技核心**
收录。相较于 CCF-A、以高门槛原创长文为主的《计算机学报》(Chinese Journal of Computers, CJC)，《计算机科学》
的鲜明特点是**覆盖面广、发文量较大、专题(专栏)与综述活跃**，明确鼓励综述与长文投稿，报道国内外计算机科学
与技术涉及面广的方法论与技术、以及能起承先启后作用的研究成果。

主办沿革在不同来源表述不一（重庆西南信息有限公司／原科技部西南信息中心等），此项及"主编/荣誉主编 陈国良
院士"的准确头衔，以官网《期刊介绍》最新表述为准，本包标 **待核实**（见 `resources/official-source-map.md`）。

## 二、栏目与专题

官网《期刊介绍》所列栏目覆盖：计算机科学理论、计算机体系结构、计算机软件、数据库/大数据/数据科学、
计算机图形学/多媒体、人工智能、人机交互/普适计算、计算机网络、信息安全、交叉与前沿等。除常规栏目外，
本刊常以**特约、组稿**形式推出**专题(专栏)**，围绕某一前沿方向集中刊发一组文章，并单独发布**征稿通知**。
作者可据此走两条路：把稿件投向匹配的常规栏目，或对接正在征稿的专题专栏（含综述）。详见
[`csj-topic-selection`](skills/csj-topic-selection/SKILL.md)。

## 三、写作与格式（中文科技规范）

- **中英文对照**：需报道性中文摘要与英文摘要(Abstract)、中英文关键词(Keywords)、中英文题名与作者单位。
- **中图分类号**：计算机学科通常为 TP 类，需据文给出。
- **参考文献**：按 **GB/T 7714—2015 顺序编码制**（正文上标数字、文后按引用先后编号），中文文献建议中英文
  对照。量与单位使用法定计量单位。
- **篇幅**：投稿一般建议不少于 8000 字，鼓励长文与综述；具体上限以官方模板与《投稿须知》为准。
详见 [`csj-writing-style`](skills/csj-writing-style/SKILL.md)。

## 四、投稿与审稿

- **在线投稿**：仅通过官网 `www.jsjkx.com` 的"作者投稿"系统提交，本刊只接受该系统投稿。
- **单盲审稿制**：作者不知审稿人姓名，审稿人知作者姓名。因此署名/单位在外审阶段可见，需重视利益冲突声明
  与学术形象。
- **流程**：编辑初审→专家外审→复审退修→主编终审→修回定稿；具体轮次与周期以编辑部通知为准（**待核实**）。
- **版面费**：第一作者为 CCF 会员的录用稿件版面费享 8.5 折；具体金额 **待核实**。
- **见刊周期**：约 13 个月（**待核实**，随版面情况调整）。
详见 [`csj-submission`](skills/csj-submission/SKILL.md) 与 [`csj-review-process`](skills/csj-review-process/SKILL.md)。

## 五、12 个技能的编排

建议按投稿弧线调用：`csj-topic-selection` → `csj-writing-style` → `csj-related-work` → `csj-experiments`
→ `csj-reproducibility` / `csj-supplementary` / `csj-artifact-evaluation` → `csj-submission`
→ `csj-review-process` → `csj-author-response` → `csj-camera-ready`，全程由 `csj-workflow` 统筹时间线。
