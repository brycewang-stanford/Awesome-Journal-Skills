# 《计算机科学》技能包 · Computer Science (Journal, JSJKX) Skills

> English readers: this is a Chinese-language pack for the Chinese computer-science
> **journal**《计算机科学》(Computer Science, JSJKX) — a monthly, not a conference.
> A fuller Chinese guide is in [`README.zh-CN.md`](README.zh-CN.md).

面向 **《计算机科学》(Computer Science, 简称 JSJKX)** 投稿的 12 个中文 agent 技能。
《计算机科学》是由重庆西南信息有限公司主办、创刊于 **1974 年 1 月** 的**计算机全学科中文综合月刊**，
国内统一刊号 CN 50-1075/TP、国际刊号 ISSN 1002-137X，是 **CCF 会刊**、**CCF 推荐 B 类中文科技期刊**、
**CCF 计算领域高质量科技期刊分级目录 T2 级**，同时是 **CSCD 来源期刊**、**北大中文核心** 与 **中国科技核心**。
本刊覆盖面广、发文量较大，**专题(专栏)与综述活跃**，鼓励综述和长文投稿——这是它区别于门槛更高、以 CCF-A
长文为主的《计算机学报》(Chinese Journal of Computers) 的核心定位。本包覆盖一次投稿的完整弧线：判断选题
是否契合本刊综合定位与栏目、把中文稿件写到符合体例、搭建经得起单盲外审的实证证据、按官方口径完成投稿、
应对多轮修回，直到录用定稿与见刊。

官方依据核验日期：**2026-07-09**。已通过官网 `www.jsjkx.com`、CNKI 期刊主页与 CCF 推荐/分级目录**双源
交叉核实**关键事实；核验环境对官网直接抓取返回 403，故通过搜索引擎渲染阅读官网《投稿须知》《期刊介绍》
并与 CNKI/CCF 目录交叉核对，完整记录（含全部 **待核实** 项）见
[`resources/official-source-map.md`](resources/official-source-map.md)。凡未双源确认的影响因子、版面费金额、
审稿周期、主办沿革与主编等一律标 **待核实**，本包不杜撰。

> 名称与类型辨析：本刊《计算机科学》英文作 **Computer Science**，是一本**期刊(journal)**，缩写常用 JSJKX
> (取自官网域名)，**不是学术会议**，请勿与同名的各类 "Computer Science" 会议或英文刊混淆；也不同于
> CCF-A 的《计算机学报》(Chinese Journal of Computers) 与《软件学报》。

## 本刊相对兄弟中文刊的特色

- **计算机全学科 + 覆盖面广、发文量大**：栏目涵盖计算机科学理论、体系结构、软件、数据库/大数据/数据科学、
  图形学/多媒体、人工智能、人机交互/普适计算、计算机网络、信息安全、交叉前沿等，面向全学科而非单一方向。
- **专题(专栏)与综述活跃**：常以特约/组稿方式推出专题专栏，明确鼓励综述与长文；这给了作者"综述立题"与
  "搭专题快车"两条区别于兄弟刊的路径。
- **单盲审稿制**：作者不知审稿人姓名、审稿人知作者姓名（与《计算机学报》双盲外审不同），署名与单位在外审
  阶段可见，需注意学术形象与利益冲突声明。
- **中文科技规范**：报道性中英文摘要、中英文关键词、中图分类号(TP)、参考文献 **GB/T 7714—2015 顺序编码制**、
  量与单位用法定计量单位。
- **面向应用与广度的定位**：相对《计算机学报》更高门槛的原创长文，本刊更强调覆盖面、时效与栏目组织，
  见刊周期约 13 个月（**待核实**，以官网刊期为准）。

## 12 个技能

| 技能 | 作用 |
|---|---|
| [`csj-topic-selection`](skills/csj-topic-selection/SKILL.md) | 判断选题是否契合本刊综合定位、栏目/专题匹配、与兄弟刊分工 |
| [`csj-workflow`](skills/csj-workflow/SKILL.md) | 从选题到见刊的全流程时间线与技能编排 |
| [`csj-writing-style`](skills/csj-writing-style/SKILL.md) | 中文体例：中英文摘要、关键词、中图分类号、GB/T 7714 |
| [`csj-related-work`](skills/csj-related-work/SKILL.md) | delta 优先的相关工作与文献综述，兼顾综述型稿件 |
| [`csj-experiments`](skills/csj-experiments/SKILL.md) | 实验设计与呈现：RQ、公平基线、统计严谨、消融 |
| [`csj-reproducibility`](skills/csj-reproducibility/SKILL.md) | 可复现性：环境/种子/数据固定、污染防范、复现包 |
| [`csj-supplementary`](skills/csj-supplementary/SKILL.md) | 按"决定性"划分正文与附录/补充材料/多媒体附件 |
| [`csj-submission`](skills/csj-submission/SKILL.md) | 投稿就绪审计：模板、六件套、查重、栏目归属、退稿风险分诊 |
| [`csj-review-process`](skills/csj-review-process/SKILL.md) | 单盲审稿流程建模（初审→外审→复审→终审→修回）与作者杠杆 |
| [`csj-author-response`](skills/csj-author-response/SKILL.md) | 审稿意见逐条答复与修回说明 |
| [`csj-artifact-evaluation`](skills/csj-artifact-evaluation/SKILL.md) | 代码与数据可用性（讲清本刊现状与做法） |
| [`csj-camera-ready`](skills/csj-camera-ready/SKILL.md) | 录用后定稿、校样、缴费与出版排期 |
