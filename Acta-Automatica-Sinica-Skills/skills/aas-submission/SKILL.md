# 《自动化学报》投稿就绪审计

在把稿件上传到《自动化学报》(Acta Automatica Sinica, AAS) 在线投审稿系统之前，先跑完这份审计。
AAS 是由中国自动化学会与中国科学院自动化研究所共同主办、创刊于 1963 年的控制与自动化学科中文
旗舰月刊，聚焦控制理论、模式识别与智能系统、机器人与人工智能在自动化中的应用。它按期刊稿件走
**双盲三审制**，因此投稿被当作一篇要经受编辑初审、双盲外审、主编终审的完整学术论文来评判。以下
事实于 2026-07-09 通过官网、CNKI 期刊主页与中国自动化学会/CCF 目录交叉核验（见
`resources/official-source-map.md`），凡本刊未公开确认的数字均标注 **待核实**，请以投审稿系统内
最新《投稿须知》为准。

## 一、投审稿系统与账号

- 本刊**只接受在线投稿**：中文稿投审稿系统入口 `http://mc03.manuscriptcentral.com/aas-cn`
  (ScholarOne Manuscripts)，或经官网 `http://www.aas.net.cn` 右侧"投审稿系统"图标进入。
- 注意勿投错系统：英文刊《IEEE/CAA Journal of Automatica Sinica》系统为 `.../ieee-jas`，与本刊
  中文《自动化学报》(Acta Automatica Sinica) 的 `aas-cn` 不同。
- 注册填写的题名、作者、单位、通信作者与研究方向决定编辑分派外审专家的匹配度，务必用**真实终稿
  信息**；系统内可随时查询稿件评审状态。

## 二、双盲匿名化（投稿前必做）

本刊实行**双盲制审稿**，投稿前须彻底删除稿件中的**所有作者信息**：

| 位置 | 需清理的内容 |
|---|---|
| 页眉/页脚 | 姓名、单位、期刊模板残留的身份字段 |
| 标题下方 | 作者署名与单位行 |
| 尾注/脚注 | 通信作者、基金致谢中的可识别信息 |
| 论文最后 | 作者简介、联系方式 |
| 文件名 | 姓名、单位、基金号 |
| 文档属性 | PDF/Word 元数据中的作者、公司字段 |
| 图表 | 图源路径、水印、账户名 |

自引处改为第三人称中性表述（"文献[x]提出"而非"我们此前提出"），避免暴露身份。

## 三、作者承诺（前置，收到才初审）

- 官网"下载中心"下载《自动化学报》**作者承诺**；**全部作者亲笔签名**，并**加盖第一作者单位公章**。
- 承诺件须**与论文同时提交**；**编辑部只有收到作者承诺后才对稿件进行初审**——漏交等于卡在初审前。
- 承诺内容通常包括：原创、未一稿多投、署名与贡献真实、无学术不端等。

## 四、原创性与署名红线

- 稿件须为**未在国内外公开出版物发表**的原始稿件、**非一稿多投**；本刊**不接受任何语种的翻译稿件**。
- **不接收将 AI 工具作为署名作者**的稿件；若使用生成式 AI 辅助，按官网当期政策据实披露（细则
  **待核实**）。
- 若为作者已发表会议论文的期刊扩展版，须显著声明扩展关系、指明新增贡献（新增理论/实验/系统），
  并规范引用会议版本。

## 五、格式与栏目（初投从简）

- **初次投稿无需使用本刊模板**：可按任意模板排版，只要结构完整、方便编委会与外审专家评阅即可；
  录用后再按本刊体例定稿排版。
- 稿件须**结构完整、论点明确、论据可靠、论证合理、层次分明、可读性强**。
- 选定栏目/文章类型：**综述 / 论文 / 短文**，按对应体例撰写（详见 `aas-writing-style`）。
- 中文科技论文标准部件仍需齐备（下表）。

| 组件 | 要求 |
|---|---|
| 中文题名 + 英文题名 | 英文题名可独立检索、与中文一致 |
| 中文摘要 + English Abstract | 报道性四要素：目的、方法、结果、结论 |
| 中文关键词 + Keywords | 中英文一一对应 |
| 中图分类号 | 自动化/计算机类多以 **TP** 打头，据主题选取 |
| 参考文献 | **GB/T 7714** 顺序编码制，著录完整 |

## 六、学术规范与查重

- 投稿前用知网/维普/万方查重自查，把文字复制比压到安全线以下；重点排查**自引**与**前序会议版本**
  的重叠。
- 具体复制比红线以本刊/出版方当期口径为准（**待核实**具体阈值）；无论如何应远低于常见 30% 警戒线。
- 署名、贡献、基金、数据来源须真实，不得伪造、篡改、买卖数据或代写。

## 七、退稿风险分诊表

| 审计发现 | 风险等级 | 处置 |
|---|---|---|
| 未删作者信息 / 违反双盲 | 初审退回 | 逐位置清理身份信息与元数据后再投 |
| 缺《作者承诺》或未盖公章 | 卡在初审前 | 补齐全部签名与第一作者单位公章 |
| 为翻译稿 / 一稿多投 | 直接不受理/严重违规 | 撤回，改投或重写为原创中文稿 |
| 将 AI 列为署名作者 | 不受理 | 调整署名，据实披露 AI 辅助 |
| 无新意、重复前人、观点模糊 | 初审退稿理由 | 回到 `aas-topic-selection` 重估贡献 |
| 参考文献不符 GB/T 7714 | 需补正 | 逐条重排著录格式 |
| 主题超出控制/自动化范围 | 不符宗旨 | 改投更契合的专业刊 |

## 八、图表、公式与源文件

- **图表**：矢量图优先，位图保证印刷分辨率；图题在图下、表题在表上，中英文规范；黑白打印下用线型
  或纹理区分而非仅靠颜色；控制类图（Bode 图、相图、响应曲线）坐标与单位标注清晰。
- **公式**：用公式编辑器或 LaTeX 排版而非截图；对齐、编号连续；正文引用公式用编号；控制律、
  Lyapunov 函数等关键式独立编号。
- **源文件**：录用后若用 LaTeX 定稿，本地完整编译一次，确认无缺字体、无未定义引用、参考文献正常
  生成。

## 九、投稿信与系统字段

- 系统通常要求填写题名、摘要、关键词、领域分类、作者与通信作者、基金、推荐/回避审稿人等，逐项据实
  填写；领域分类要与稿件主题精确匹配，直接影响外审专家分派。
- 回避审稿人仅出于真实利益冲突并简述理由，不得用来排除严格评审者。
- 投稿信简明说明：本文主要贡献、为何适合《自动化学报》(Acta Automatica Sinica, AAS)、是否为会议
  扩展版及其增量、无一稿多投与利益冲突声明。

## 十、投稿前最后一轮操作顺序

1. 冻结正文论证主线与核心贡献陈述；参考文献可微调。
2. 逐位置做双盲匿名化，清理文件名与文档元数据。
3. 备好《作者承诺》：全部签名 + 第一作者单位公章，扫描件随稿提交。
4. 跑查重，压到安全线以下，逐条核对自引与会议版本重叠。
5. 校对中英文摘要、关键词、中图分类号、参考文献与图表公式。
6. 在系统内据实填字段，上传后重新下载 PDF 冷读一遍，确认是最终版、无身份泄露、无乱码。

## 输出格式

```text
[AAS 投稿状态] 就绪 / 阻塞 / 待完善
[系统] 投 aas-cn(中文刊)？非 ieee-jas？账号与领域分类是否匹配主题？
[双盲] 页眉页脚/标题下/尾注/文末/文件名/元数据/图表 身份是否全部清理？
[承诺] 全部签名 + 第一作者单位公章？随稿同时提交？
[原创] 非翻译稿/非一稿多投？AI 未列为署名作者？会议扩展版已声明增量？
[格式] 中英文摘要/关键词/中图分类号/GB/T 7714 齐全？栏目：综述/论文/短文
[查重] 文字复制比 ___%（自查安全线以下）
[待办队列] <按优先级排序，标注负责人与时限>
```
