# 《自动化学报》审稿流程建模

本技能把《自动化学报》(Acta Automatica Sinica, AAS) 的审稿流程拆成可预期的环节，并指出每个环节
作者能施加的杠杆。AAS 实行**双盲三审制**。以下流程于 2026-07-09 核验（见
`resources/official-source-map.md`），精确周期以官网《稿件处理流程》为准（**待核实**）。

## 一、流程全链路

```text
在线投稿(aas-cn) + 作者承诺
   → 编辑初审（是否送外审）
   → 双盲外审（专家评审，删除作者信息）
   → 退修/复审（多轮；大修常重新送审）
   → 主编终审（录用与否）
   → 录用 → 定稿校样 → 见刊 + 稿酬
```

## 二、各环节说明与作者杠杆

| 环节 | 谁在做 | 作者杠杆 |
|---|---|---|
| 作者承诺 | 编辑部核收 | 投稿即备好全部签名+第一作者单位公章，否则不初审 |
| 编辑初审 | 责任编辑 | 稿件对口、匿名合规、格式齐全，降低初审退回率 |
| 双盲外审 | 外审专家 | 稿件自身质量；投稿时精确的领域分类帮助匹配专家 |
| 退修 | 作者 | 逐条答复 + 修改对照（见 `aas-author-response`），按期提交 |
| 复审 | 外审/编辑 | 修改聚焦审稿人核心关切，减少反复 |
| 主编终审 | 主编/编委 | 前序环节质量的累积结果 |

## 三、双盲对作者的约束

- 外审阶段作者与审稿人互不知身份，故投稿前须彻底匿名化（见 `aas-submission`）。
- 修回时也要保持匿名口径：答复信中不自我暴露身份，自引仍用中性第三人称。
- 违反双盲（正文/元数据/附件泄露身份）可能被退回补正，浪费一轮时间。

## 四、时间进度（经验值）

| 环节 | 经验时长 | 备注 |
|---|---|---|
| 初审 | 数日~数周 | 编辑判断是否送外审 |
| 外审 | 约 1~3 个月/轮 | **待核实**精确周期 |
| 退修 | 编辑部给定期限 | 大修需时更长 |
| 复审 | 大修常重新送审 | 视修改幅度 |
| 终审 | 复审通过后 | 主编定裁 |

个案差异大；可在 aas-cn 系统内随时查询稿件状态。

## 五、可能的决议类型

| 决议 | 含义 | 应对 |
|---|---|---|
| 退稿(初审/外审) | 不符宗旨/质量不足 | 重估选题或改投（见 `aas-topic-selection`） |
| 大修 | 有价值但需实质修改 | 逐条整改，常重新送审 |
| 小修 | 接近录用，细节问题 | 快速精确修订 |
| 录用 | 通过终审 | 进入定稿（见 `aas-camera-ready`） |

## 六、作者在各环节的检查表

```text
[投稿] 稿件对口本刊？匿名合规？作者承诺(签名+公章)随稿？
[初审后] 送外审/退回？若退回补正，逐项修复再投
[外审后] 大修/小修/退稿？阅读全部意见，分类整理
[退修] 在期限内提交修改稿 + 逐条答复；重大改动说明重新送审预期
[复审] 修改是否回应核心关切？是否引入新问题？
[终审] 保持沟通，等待主编定裁
[状态] 定期在 aas-cn 系统查询
```

## 七、常见流程失误

- 漏交/未盖章作者承诺，稿件卡在初审前——投稿即备齐。
- 匿名不彻底被退回补正——投稿前逐位置清理。
- 退修拖到超期，可能退稿或按新稿处理——收到意见即排期。
- 大修改动过大、反复多轮——聚焦审稿人核心关切。
- 投到英文 ieee-jas 系统而非中文 aas-cn——确认目标是《自动化学报》(Acta Automatica Sinica)。

## 八、编辑初审关注什么

编辑初审是第一道闸门，通过率取决于几个可控项：

| 初审关注 | 作者可控 |
|---|---|
| 是否对口本刊宗旨 | 选题落在控制/自动化/智能系统（见 `aas-topic-selection`） |
| 匿名是否合规 | 彻底删除作者信息与元数据 |
| 作者承诺是否齐备 | 全部签名 + 第一作者单位公章，随稿提交 |
| 格式是否基本齐全 | 中英文摘要、关键词、中图分类号、参考文献 |
| 是否明显学术不端 | 投稿前自查查重 |
| 是否一稿多投/翻译稿 | 确保原创、唯一在审 |

初审退回多为形式性问题，逐项修复即可再投；初审退稿多为宗旨/新意问题，需回选题重估。

## 九、外审意见的常见维度

外审专家通常从以下维度评审，作者可据此预判并提前补强：

- **创新性**：相对已有工作的实质增量（见 `aas-related-work`）。
- **正确性**：理论证明是否严谨、假设是否合理。
- **实验充分性**：基线是否公平、统计是否严谨、消融是否到位（见 `aas-experiments`）。
- **可复现性**：结果能否复现（见 `aas-reproducibility`）。
- **写作与规范**：结构、中英文摘要、图表、GB/T 7714 著录。

## 十、多轮退修的心态与节奏

- 双盲三审制下**多轮退修是常态**，大修后重新送审并不意味着否定，而是走向录用的路径。
- 每轮把审稿人核心关切一次性回应到位，减少往返轮次。
- 记录每轮意见与修改，形成可追溯的答复档案（见 `aas-author-response`）。

## 十一、与编辑部的沟通礼仪

- 稿件状态可在 aas-cn 系统查询；非必要不频繁催稿，给外审合理时间。
- 若远超经验周期仍无进展，可礼貌邮件问询稿件状态，附稿号与题名。
- 收到编辑部通知（补正、退修、录用）及时响应，按给定期限行动。
- 一切沟通保持专业；退修阶段的实质讨论走答复信（见 `aas-author-response`）。

## 十二、预期管理

《自动化学报》(Acta Automatica Sinica, AAS) 的双盲三审制意味着录用是多环节累积的结果，作者能做的
是在每个环节把可控项做到位：对口选题、彻底匿名、齐备承诺、按期高质量退修。把 `aas-review-process`
与 `aas-author-response`、`aas-workflow` 联动，管理好时间与预期，并始终确认稿件在中文
Acta Automatica Sinica 的 aas-cn 系统中流转，而非英文 JAS 系统。
