# 《自动化学报》退修说明与答复信

本技能帮你为《自动化学报》(Acta Automatica Sinica, AAS) 撰写退修说明与逐条答复。AAS 双盲三审制下
大修常重新送审，一封清晰、逐条、可核对的答复信能显著提高录用概率。以下方法于 2026-07-09 核验框架
（见 `resources/official-source-map.md`）；退修期限以编辑部通知为准（**待核实**）。

## 一、答复信总体结构

```text
致编辑与各位审稿专家：
感谢对本文的评审。我们已按意见逐条修改，主要改动概述如下：（3~5 句总述）
下文对每位审稿人的每条意见逐一回应，[原文位置]标注对应修改处。

—— 审稿人 1 ——
意见1.1：<原文引用>
回应：<接受/说明> + <具体改动> + <改后位置>
...
```

## 二、逐条答复的三段式

每条意见都用固定三段，方便审稿人核对：

| 段 | 内容 |
|---|---|
| 复述意见 | 原样引用审稿人意见，不曲解 |
| 回应 | 接受并改 / 部分接受 / 据理说明分歧 |
| 落点 | 指明修改稿中的具体位置（章节/图表/公式号） |

## 三、修改对照（改前→改后）

对关键修改给出"改前→改后"引文，让审稿人一眼看到变化：

```text
[改前] 原句/原公式
[改后] 修改后句/公式
[位置] 第X节 / 式(y) / 图z
```

## 四、大修 vs 小修策略

| 类型 | 策略 |
|---|---|
| 大修 | 系统性回应，可能补实验/补证明；预期重新送审，改动要经得起复核 |
| 小修 | 精确快速修订，逐条确认，避免引入新问题 |

大修时在答复信开头用一段总述"本轮主要新增了什么"（新实验、新证明、新对比），帮助外审快速把握。

## 五、对分歧意见的处理

- 若不同意某条意见，**据理礼貌说明**：先感谢，再用证据/文献/逻辑解释，必要时做折中（如补充实验
  回应关切）。
- 避免情绪化或简单拒绝；也避免为迎合而做损害论文正确性的改动。
- 若两位审稿人意见冲突，说明本文的权衡取舍并请编辑裁量。

## 六、保持双盲的匿名口径

- 答复信在外审阶段同样匿名：不透露作者身份、单位；自引仍用中性第三人称。
- 不在答复中写"我们实验室/我们学校"等可识别信息。

## 七、修改稿组织

- 提供**修改稿**（clean 版）与必要时的**改动标记版**（便于核对）。
- 修改与答复信一一对应，位置标注准确。
- 在编辑部给定**期限内**提交；如需延期，提前与编辑沟通。

## 八、答复信自检清单

```text
[逐条] 每条意见都回应？无遗漏？
[三段式] 复述→回应→落点 齐全？
[对照] 关键改动给出改前→改后与位置？
[大修总述] 新增实验/证明/对比是否概述？
[分歧] 不同意处据理礼貌说明，未简单拒绝？
[匿名] 答复信无身份泄露、自引中性？
[期限] 在编辑部期限内提交？clean+标记版齐备？
```

## 九、常见答复失误

- 笼统说"已修改"却不指位置，审稿人无法核对——补章节/图表号。
- 遗漏某条意见——外审会追问，务必逐条。
- 对分歧简单拒绝或情绪化——改为据理说明 + 折中。
- 大修改动大却无总述，外审难把握——加改动概述。
- 答复中暴露作者身份，违反双盲——保持匿名口径。

## 十、意见分类与优先级

收到意见后先分类，再决定回应顺序：

| 类别 | 处理优先级 | 做法 |
|---|---|---|
| 影响录用的核心质疑（正确性/创新性） | 最高 | 补证明/补实验，正面回应 |
| 实验充分性（基线/消融/统计） | 高 | 补对照或统计，用数据说话 |
| 写作与规范（结构/著录/图表） | 中 | 逐条修订，快速可清 |
| 表述/笔误 | 低 | 一并订正 |

先把核心质疑回应到位，这是重新送审能否通过的关键。

## 十一、答复信语气与格式细节

- 开头感谢、结尾致意，全程专业克制，即使面对严厉意见也不辩驳性对抗。
- 用可视区分标注：引用意见用一种格式，回应用另一种，改动位置加粗或标号。
- 一条意见对应一段回应，避免把多条揉在一起导致遗漏。
- 篇幅上宁可具体详尽，不要笼统含糊；具体到"第X节式(y)"最有说服力。

## 十二、多轮退修的档案管理

```text
[版本记录] R1 意见→R1 答复→R1 修改稿；R2 意见→...
[未决项] 上一轮未完全解决、本轮继续跟进的意见
[新增证据] 每轮新补的实验/证明清单
[一致性] 修改是否同步到复现包与补充材料（见 aas-reproducibility）
```

## 十三、与本刊定位

《自动化学报》(Acta Automatica Sinica, AAS) 的多轮退修是提升论文质量的过程。把审稿意见当作改进
机会，逐条、可核对、据理地回应，配合 `aas-review-process` 的预期管理与 `aas-experiments` 的补充
证据，能在重新送审中稳步走向录用。全程保持双盲匿名口径，确认稿件在中文 Acta Automatica Sinica
的流程中流转。
