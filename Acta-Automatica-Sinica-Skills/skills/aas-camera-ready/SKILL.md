# 《自动化学报》录用后定稿与校样

稿件被《自动化学报》(Acta Automatica Sinica, AAS) 录用后，进入定稿、校样与出版环节。本技能给出
从匿名稿到见刊的收尾清单。AAS 见刊后赠样刊 2 本、3 个月内一次性支付稿酬。以下口径于 2026-07-09
核验（见 `resources/official-source-map.md`），模板版本与费用细节以编辑部通知为准（**待核实**）。

## 一、从匿名稿恢复完整信息

投稿版是匿名的，录用后须补齐：

| 项 | 内容 |
|---|---|
| 作者署名 | 姓名、排序、通信作者标注 |
| 单位 | 各作者单位、地址、邮编 |
| 基金项目 | 资助项目名称与编号 |
| 作者简介 | 姓名、出生年、职称/学位、研究方向 |
| 致谢 | 恢复被匿名删除的致谢 |

## 二、套用本刊定稿模板

- 初投可用任意模板，**录用后按本刊体例定稿排版**（模板见官网，版本 **待核实**）。
- 按模板校对：中英文题名、报道性中英文摘要、中英文关键词、中图分类号、正文结构、公式编号、图表
  位置、参考文献 GB/T 7714。
- 若用 LaTeX，本地完整编译一次，确认无缺字体、无未定义引用、参考文献正常生成。

## 三、清样(校样)逐处核对

| 核对项 | 要点 |
|---|---|
| 文字 | 无错别字、无乱码、术语统一 |
| 公式 | 编号连续、符号正确、排版无错位 |
| 图表 | 图题表题、编号、清晰度、法定单位 |
| 参考文献 | GB/T 7714 著录完整、编号与正文对应 |
| 中英文摘要 | 对应一致、可独立检索 |
| 作者/基金 | 署名、单位、基金号准确无误 |

校样阶段一般只做**必要更正**，不做实质性内容改动；重大改动需与编辑部沟通。

## 四、稿酬与样刊

- 见刊后赠**当期样刊 2 本**。
- 刊发后**3 个月内一次性支付稿酬**（含印刷版、光盘版、网络版稿酬）。
- 版面费是否收取及金额 **待核实**（本刊向作者支付稿酬，与很多收版面费的刊不同）。

## 五、出版排期

- 编辑部综合投稿时间与研究方向安排发表期次；录用到见刊通常需数月。
- 关注 aas-cn 系统与编辑部邮件通知，按时返回校样与确认信息。

## 六、定稿自检清单

```text
[署名] 作者/排序/通信作者/单位/基金/简介 补齐无误？
[模板] 按本刊体例定稿排版？LaTeX 干净编译？
[摘要] 中英文对应、可独立检索？
[公式] 编号连续、符号正确？
[图表] 图下表上、清晰、法定单位？
[参考文献] GB/T 7714 著录完整、编号对应？
[清样] 逐处核对无错别字/乱码？只做必要更正？
[流程] 校样按期返回？稿酬信息确认？
[待核实] 模板版本 / 版面费 / 排期：待核实
```

## 七、常见收尾失误

- 忘记恢复被匿名删除的作者/基金/致谢信息——按清单逐项补齐。
- 校样阶段做大改动引发返工——只做必要更正，重大改动先沟通。
- 参考文献编号与正文错位——终校逐条核对。
- LaTeX 在编辑部环境编译失败——提交前干净编译。
- 混淆中文《自动化学报》(Acta Automatica Sinica) 与英文 JAS 的模板——用本刊中文定稿模板。

## 八、校样阶段的沟通纪律

- 校样(清样)一般只允许**必要更正**（错字、公式排版、图表清晰度、著录错误），不做实质内容改动。
- 若确需实质更正（如发现数据笔误），**先与编辑部书面沟通**，说明原因与影响，经同意再改。
- 按编辑部给定的**校样返回期限**返回；逾期可能影响排期。
- 通信作者是校样阶段的对接人，负责汇总各作者的校对意见后统一返回，避免多头反馈。

## 九、录用后到见刊的关键交接物

| 交接物 | 说明 |
|---|---|
| 定稿源文件 | Word 或 LaTeX 全套（.tex/.bib/图源/宏），按本刊模板 |
| 完整署名信息 | 作者、单位、通信作者、基金、作者简介 |
| 版权/出版协议 | 按编辑部要求签署（细则 **待核实**） |
| 校样确认 | 逐处核对后签署确认 |
| 联系方式 | 通信作者稿酬与样刊寄送信息 |

## 十、见刊后事项

- 收到**样刊 2 本**；核对见刊版与定稿是否一致，如有印刷错误及时反馈编辑部。
- 见刊后**3 个月内**收到一次性稿酬（印刷版、光盘版、网络版）；留意稿酬发放所需的信息填报。
- 见刊后可在 CNKI/万方等平台确认收录；引用本文时用正式的卷期页信息。
- 若此前有匿名复现仓库，见刊后可去匿名化并在正式渠道公开，提升可复现性与影响力。

## 十一、录用后收尾总检查

```text
[信息] 署名/单位/基金/简介/致谢 全部恢复且准确？
[模板] 本刊定稿模板套用完成、干净编译？
[清样] 逐处核对、只做必要更正、按期返回？
[协议] 版权/出版协议签署？
[稿酬] 通信作者信息填报、样刊寄送地址确认？
[见刊] 核对见刊版、必要时反馈印刷错误？
[待核实] 模板版本 / 出版协议细则 / 版面费：待核实
```

## 十二、定稿阶段的参考文献终校

参考文献是清样最易出错处，定稿单独过一遍：

- 逐条核对 GB/T 7714 著录要素齐全（作者、题名、来源、年、卷期页/DOI、类型标识）。
- 正文上标编号与文末列表**一一对应**、无断号、无多余。
- 中文文献著录完整；英文文献大小写、缩写规范一致。
- 退修中新增/删除的引用是否已同步更新编号。

## 十三、与本刊定位

《自动化学报》(Acta Automatica Sinica, AAS) 的定稿环节强调体例规范与清样零差错。把
`aas-camera-ready` 与 `aas-writing-style` 联动做终校，确保见刊版在中英文摘要、公式、图表与
GB/T 7714 参考文献上完全合规，顺利完成从录用到见刊与稿酬发放的最后一段。始终用本刊中文
Acta Automatica Sinica 的定稿模板，勿误用英文 JAS 模板。
