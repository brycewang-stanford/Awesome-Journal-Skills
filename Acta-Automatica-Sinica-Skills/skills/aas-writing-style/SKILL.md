# 《自动化学报》中文科技写作规范

本技能给出投向《自动化学报》(Acta Automatica Sinica, AAS) 的中文稿体例要求。AAS 有中英文双语与
综述长文传统，报道性摘要与规范著录是外审的第一印象。以下要求于 2026-07-09 核验（见
`resources/official-source-map.md`），具体模板样式以官网当期为准（**待核实**）。

## 一、标题与署名（注意双盲）

- 中文题名简明、可检索；配**英文题名**，与中文一致、可独立检索。
- 投稿版**删除全部署名与单位信息**（双盲，见 `aas-submission`）；录用定稿再补齐作者、单位、基金、
  作者简介与通信作者。

## 二、报道性摘要（中英文并重）

摘要采用**报道性**写法，四要素齐全，中英文一一对应：

| 要素 | 写什么 |
|---|---|
| 目的/问题 | 研究的控制/自动化问题与动机 |
| 方法 | 提出的模型/控制律/算法与分析工具（如 Lyapunov） |
| 结果 | 关键定量结论（稳定性/性能指标改进，给数字） |
| 结论 | 意义与适用范围 |

- 中文摘要一般 200~300 字左右（**待核实**具体字数），不引文献、不用未定义缩略语。
- English Abstract 与中文对应、可独立检索，术语用本领域通行英文（consensus、robust control、
  pattern recognition 等）。

## 三、关键词与中图分类号

- **中英文关键词**一般 3~8 个，中英文一一对应，选可检索的领域术语。
- **中图分类号**：自动化/计算机类多以 **TP** 打头（如 TP13 自动控制理论、TP18 人工智能、
  TP24 机器人、TP391 信息处理/模式识别），据主题精确选取；可含 1~2 个。

## 四、正文结构（论文栏目）

推荐"问题形式化—理论分析—实验验证"三段闭环：

1. **引言**：漏斗式，从背景到 gap 再到可数贡献（见 `resources/worked-examples/01-introduction.md`）。
2. **问题描述**：系统模型、假设、控制目标形式化。
3. **方法与分析**：控制律/算法设计，稳定性/收敛性/最优性证明。
4. **仿真/实物实验**：对比基线、指标、鲁棒性与消融。
5. **结论**：贡献回收、局限与展望。

综述栏目改为：背景 → 分类框架 → 各类方法对比与批判 → 未解问题与趋势；短文压缩为单点创新的紧凑
结构。

## 五、公式规范

- 用公式编辑器或 LaTeX 排版，**不用截图**；变量斜体、矩阵/向量按惯例、常量与函数名正体。
- 公式**连续编号**，正文用编号引用；控制律、Lyapunov 函数、稳定性条件等关键式独立编号。
- 首次出现的符号给出定义；复杂推导可移入附录（见 `aas-supplementary`）。

## 六、图表规范

| 项 | 要求 |
|---|---|
| 图题/表题 | 图题在图下、表题在表上，中英文规范 |
| 矢量优先 | 曲线图/框图用矢量，位图保证印刷分辨率 |
| 黑白可辨 | 用线型/纹理/标记区分而非仅靠颜色 |
| 控制类图 | Bode 图、相轨迹、阶跃响应坐标与单位标注清晰 |
| 量与单位 | 用**法定计量单位**，量符号斜体、单位正体 |

## 七、参考文献 GB/T 7714

- 采用 **GB/T 7714 顺序编码制**：正文按引用顺序上标编号，文末按序著录。
- 各类型文献著录要素齐全（作者、题名、刊名/出版者、年、卷期页/DOI）；文献类型标识规范
  （[J]/[C]/[M]/[D] 等）。
- 中文文献著录完整；是否要求中英文对照以官方模板为准（**待核实**）。用 Zotero/EndNote 的 GB/T 7714
  样式统一生成后逐条校对。

## 八、语言与用词

- 术语前后统一，缩略语首次出现给全称；中文表达准确、避免口语化与"很好/很多"式空话。
- 时态与人称：中文科技写作多用客观陈述；自引改中性第三人称以保双盲。
- 数字、公式、单位遵循国标；避免中英文标点混用。

## 九、体例自检清单

```text
[题名] 中/英文题名齐全、可检索？（投稿版已匿名）
[摘要] 报道性四要素？中英文对应、可独立检索？
[关键词] 中英文 3~8 个一一对应？
[中图分类号] TP 打头、匹配主题？
```

## 十、英文摘要单独打磨

英文摘要是本刊双语传统的重要部分，也是国际检索的入口：

- 与中文摘要**信息对等**，但按英文科技写作习惯行文，不逐字硬译。
- 术语用领域通行英文（consensus control、adaptive control、pattern recognition、robustness）。
- 时态：方法与结果多用一般现在时/一般过去时，保持一致；避免中式英语与语法错误。
- 可请英语好的同行或母语者润色，语言问题会影响外审与国际读者印象。

## 十一、量、单位与数字规范

| 项 | 规范 |
|---|---|
| 量符号 | 斜体（如 t、x、K） |
| 单位 | 正体，用法定计量单位（s、m、rad/s） |
| 数字 | 遵循国标，三位分节视规范；量值与单位间留空 |
| 百分数 | 明确基数，避免歧义 |
| 公差/误差 | 给出置信区间或方差口径 |

## 十二、常见体例问题

- 摘要写成"背景介绍"而非报道性结论——补齐方法与量化结果。
- 中图分类号乱填或缺失——据主题选 TP 子类。
- 参考文献格式混乱、缺类型标识——用 GB/T 7714 样式统一重排。
- 图表仅靠颜色区分、黑白打印失效——加线型/标记。
- 忘记这是《自动化学报》(Acta Automatica Sinica) 的双语要求，只写中文摘要——补 English Abstract。
- 英文摘要中式英语、术语不统一——请母语者润色、术语对齐。
