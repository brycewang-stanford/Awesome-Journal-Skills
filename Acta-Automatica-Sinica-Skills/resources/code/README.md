# 复现包与自查脚本 — 《自动化学报》(Acta Automatica Sinica, AAS)

本目录说明如何为投向《自动化学报》(Acta Automatica Sinica, AAS) 的控制/自动化/模式识别类稿件
组织**复现包**，以及投稿前的**查重与匿名化自查**思路。本包不放置真实脚本，仅给出可照做的结构与
清单；具体政策以官网当期《投稿须知》为准（本刊现状：**待核实**是否强制提交代码/数据）。

## 复现包目录建议

```text
repro/
  README.md            # 一键复现说明：环境、数据、运行、预期结果
  env/                 # environment.yml / requirements.txt / Dockerfile
  data/                # 数据或获取脚本（大数据给下载脚本+校验和）
  src/                 # 方法与基线实现
  configs/             # 每个实验/图表对应的配置与随机种子
  scripts/             # run_all.sh：从原始数据到论文图表
  results/             # 关键中间结果与最终指标（可复现校验）
```

## 控制/自动化类的复现要点

- **系统与仿真环境**：写清被控对象模型、仿真器（如 MATLAB/Simulink、Python 控制库）版本、步长、
  求解器；实物实验注明硬件平台、采样周期、传感器与标定。
- **随机性固定**：强化学习/模式识别实验固定随机种子，报告多次运行的均值与方差，而非单次最好值。
- **超参数与调参协议**：给出搜索空间与选择准则，避免在测试集上调参造成的乐观偏差。
- **指标一致**：控制类的稳态误差、超调、收敛时间、鲁棒性；识别类的准确率/召回/F1 等，口径统一。

## 投稿前自查（对应双盲三审制）

| 自查项 | 做法 |
|---|---|
| 文字复制比 | 用知网/维普/万方查重自查，压到安全线以下，重点查自引与前序会议版本 |
| 匿名化 | 删除正文、页眉页脚、尾注、致谢、文件名中的作者/单位/基金号；清理 PDF 元数据 |
| 图表匿名 | 图源路径、水印、账户名不得泄露身份 |
| 代码/数据链接 | 若附链接，用匿名仓库（如匿名化 Git 快照），勿暴露作者主页 |
| 一键复现 | 在干净环境重跑 run_all.sh，确认无缺依赖、结果可复现 |
