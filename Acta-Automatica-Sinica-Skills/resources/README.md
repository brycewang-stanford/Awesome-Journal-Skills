## 使用纪律

- 本包为控制与自动化学科**中文期刊**技能包，正文以中文写作，面向《自动化学报》(Acta Automatica
  Sinica) 的双盲三审制与综述长文传统。
- 引用任何影响因子、版面费、审稿周期、录用率、主编前，先回到官网当期《投稿须知》与 CNKI/学会目录
  双源核对；无法确认即标 **待核实**。
- 注意把中文《自动化学报》(AAS) 与英文《IEEE/CAA Journal of Automatica Sinica》(JAS) 分清，二者
  投审稿系统与语种均不同。
