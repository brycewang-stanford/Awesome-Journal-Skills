# 外部工具与官方入口 — 《自动化学报》(Acta Automatica Sinica, AAS)

本表汇集《自动化学报》(Acta Automatica Sinica, AAS) 作者侧常用的官方入口与自查工具。所有网址与
口径以官网当期公告为准；带 **待核实** 者表示未双源确认，勿写入面向作者的确定结论。

## 官方入口

| 用途 | 入口 | 备注 |
|---|---|---|
| 期刊官网 | https://www.aas.net.cn/ | 投稿须知、稿件处理流程、下载中心、当期目录 |
| 投稿须知 | https://www.aas.net.cn/news/tgxz.htm | 投审稿系统、双盲、作者承诺、栏目、稿酬口径 |
| 论文模板 | https://www.aas.net.cn/news/lunwenmoban.htm | 录用后定稿排版参考；初投无需套用 |
| 中文稿投审稿系统 | http://mc03.manuscriptcentral.com/aas-cn | ScholarOne Manuscripts；在线投稿与状态查询 |
| 主办学会 | https://www.caa.org.cn/ | 中国自动化学会；期刊分级与学术活动 |
| 主办研究所 | http://www.ia.cas.cn/ | 中国科学院自动化研究所 |

> 名称辨析：英文刊《IEEE/CAA Journal of Automatica Sinica (JAS)》投审稿系统为
> `mc03.manuscriptcentral.com/ieee-jas`，与本刊中文《自动化学报》的 `aas-cn` 不同，请勿投错系统。

## 作者侧自查工具（非官方）

| 用途 | 工具类别 | 说明 |
|---|---|---|
| 学术不端检测 | 知网 AMLC/PMLC、维普、万方查重 | 投稿前自查文字复制比，压到安全线以下 |
| 参考文献著录 | Zotero/EndNote + GB/T 7714 样式 | 顺序编码制；核对中文文献著录完整 |
| 公式排版 | LaTeX / MathType | 公式用编辑器排版而非截图，编号连续 |
| 矢量制图 | Origin / Matplotlib / TikZ | 图内文字随缩放清晰，黑白可辨 |
| 匿名化自查 | PDF 属性清理、文件名检查 | 双盲：删除作者/单位/基金号与文档元数据 |
