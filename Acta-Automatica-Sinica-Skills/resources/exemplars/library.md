# 范文库使用方法 — 《自动化学报》(Acta Automatica Sinica, AAS)

本文件说明如何**自行挑选与核验**《自动化学报》(Acta Automatica Sinica, AAS) 的范文，用作写作与
选题参照。本包**不杜撰**任何具体篇目、作者、期号或获奖属性；请按下述方法从官方渠道取样并自行核验。

## 为什么要读范文

《自动化学报》有**中英文双语与综述长文传统**，栏目分为综述、论文、短文。读三五篇近两年、与自己
主题相近的**已见刊范文**，能快速校准：本刊的问题设定粒度、控制/模式识别方向的论证深度、实验与
稳定性/收敛性分析的呈现习惯、中英文摘要写法、参考文献 GB/T 7714 著录样式，以及综述稿与研究长文
的体例差异。

## 如何取样（官方优先）

1. 打开官网 https://www.aas.net.cn/ 的**当期目录**与**过刊浏览**，按栏目（综述/论文/短文）筛选。
2. 在 CNKI/万方/维普检索本刊近 1~2 年、与自己主题（如自适应控制、多智能体、强化学习控制、模式
   识别、机器人）相近的高被引文。
3. 每个栏目至少取 2 篇：一篇方法型研究长文、一篇综述，观察二者结构差异。

## 核验清单（每篇范文都要过一遍）

| 维度 | 要看什么 |
|---|---|
| 栏目归属 | 属综述/论文/短文哪类？体例与篇幅如何随之变化 |
| 问题设定 | 控制/自动化问题如何形式化（系统模型、假设、目标） |
| 贡献陈述 | 摘要与引言最后如何用 1~3 条明确贡献收束 |
| 理论深度 | 稳定性/收敛性/最优性是否给出证明或严格分析 |
| 实验呈现 | 仿真/实物实验、对比基线、指标、统计与鲁棒性 |
| 中英文摘要 | 报道性四要素是否齐全，英文摘要是否可独立检索 |
| 参考文献 | GB/T 7714 顺序编码制著录是否完整、中英文处理 |

## 使用纪律

- 只把从官方渠道**亲自核验**过的篇目当范文；不要引用来路不明的"获奖名单"或二手榜单。
- 范文用于**校准写法与深度**，不是照抄结构；本刊查重红线严格，切勿套用他人段落。
- 若无法确认某篇的栏目或属性，标 **待核实**，不要臆断。
- 注意区分中文《自动化学报》(AAS) 与英文《IEEE/CAA Journal of Automatica Sinica》的范文，二者
  语种与体例不同，勿混作同一刊参照。
