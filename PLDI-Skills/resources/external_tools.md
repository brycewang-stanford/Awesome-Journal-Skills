Access date: 2026-07-08

Open these systems only after rereading the current cycle's track pages. PLDI resets
its deadlines, HotCRP instance, and artifact rules every edition, and the PLDI 2027
site was still a placeholder at the access date.

## Author-side checks before touching HotCRP

- Correct HotCRP instance for the current cycle, with profile, conflicts, and topics set.
- Unmodified `acmsmall` single-column format; page count within the current cap
  measured on text excluding bibliography.
- Double-blind sweep across PDF metadata, acknowledgements, grant numbers, tool
  names, benchmark-suite hosting, and repository URLs.
- Anonymous supplementary material and any anonymized code links.
- Artifact plan drafted before acceptance, since evaluation starts right after
  notification and badge criteria reward early packaging discipline.

## Do not infer

- Do not project PLDI 2026 dates onto PLDI 2027; the 2027 CFP was unpublished at
  the access date.
- Do not assume page caps, response-window mechanics, badge criteria, or the
  HotCRP URL carry over between editions.
- Do not treat PACMPL's journal formatting as license to write journal-length
  prose: the submission cap is enforced with summary rejection.
- If two official pages disagree, prefer the newer CFP or a direct statement from
  the current Program Chair, and record the conflict in your notes.
