# AOS Pre-Submission Checklist

> Verify every volatile item (portal behavior, article types, artwork specs, OA pricing, editors) on the official AOS Guide for Authors at sciencedirect.com/journal/accounting-organizations-and-society/publish/guide-for-authors and in Editorial Manager (editorialmanager.com/AOS) before submitting. Facts below reflect a check on 2026-07-16; see `../../../resources/official-source-map.md`.

## 1. Scope sanity (the AOS-specific gate)

- [ ] The paper studies accounting's social, organizational, behavioral or institutional operation — not capital-markets pricing dressed in accounting data
- [ ] A named theoretical tradition does real work in the findings (institutional theory / governmentality / Bourdieu / ANT / practice theory / psychological process)
- [ ] The contribution is conceptual and would survive the "no security prices" test
- [ ] The paper engages AOS's own canon (see `../../../resources/exemplars/library.md`), not only a sibling journal's

## 2. Anonymization (double-anonymized review)

- [ ] Manuscript file contains no author names, affiliations, acknowledgments, or grant numbers
- [ ] Self-citations in the third person ("prior work (Author, 20xx)"), never "our earlier study"
- [ ] No field-access narratives that identify the authors or the organization
- [ ] Site pseudonyms consistent; no logos, letterheads, or identifiable document scans in exhibits
- [ ] File names and document metadata/properties scrubbed
- [ ] Tracked changes and comments removed from all files

## 3. Files for Editorial Manager

- [ ] Anonymized manuscript (source file; EM builds the review PDF)
- [ ] Separate title page: title, authors, affiliations, corresponding author, acknowledgments, funding
- [ ] Declaration of competing interests (completed even if "none")
- [ ] Tables in editable format; figures at required resolution per current artwork instructions
- [ ] Supplementary files (instrument, online appendix, extra exhibits) uploaded separately
- [ ] Cover letter: two or three sentences on fit with AOS's interdisciplinary mission; overlap with any working/conference versions disclosed
- [ ] Correct article type selected (regular vs special issue / conference track)

## 4. Ethics & consent

- [ ] Human-participant work (interviews, experiments, surveys) reports institutional approval and informed consent in the manuscript
- [ ] Approval documentation retained and producible on editor request
- [ ] Field-site anonymity commitments honored — nothing quoted beyond what consent covers
- [ ] For historical work: archive permissions and citation conventions respected

## 5. Evidence & analysis (tradition-specific core)

**Qualitative field / historical**
- [ ] Data-inventory table (interviews: n, roles, durations, period; observations; documents)
- [ ] Claims traceable to material; evidence/coding exhibits present
- [ ] Counter-evidence acknowledged and handled
- [ ] Method section describes the analytic process concretely (coding, iteration, theory–data movement)

**Experimental / survey**
- [ ] Design, pool, randomization, manipulation and attention checks reported
- [ ] Cell means, SDs, per-cell n, effect sizes; process tests (mediation/moderation) with bootstrap CIs
- [ ] Exclusion rules pre-specified and reported; pre-registration linked if used
- [ ] (Survey) reliabilities, validity evidence, common-method-bias handling

**Archival-with-theory**
- [ ] Sample screens, variable construction, clustering documented; interpretation stays institutional/organizational

## 6. Writing & format

- [ ] Abstract self-contained: question, design, conceptual answer, consequence
- [ ] Keywords serve both accounting and social-science indexing
- [ ] Terminology identical across abstract, text, exhibits, appendices
- [ ] References complete, author–date style per the current guide
- [ ] Length disciplined — no hard cap published, so every section must earn its pages (re-check the current guide)
- [ ] Generative-AI use declared per Elsevier policy (if any)

## 7. Final pass

- [ ] Abstract, introduction, and discussion state the same claim in the same words
- [ ] Every exhibit passes the one-sentence test and is self-contained
- [ ] Data statement drafted: what is shareable, what consent/anonymity restricts, stated plainly
- [ ] OA decision made deliberately; APC confirmed in the submission system if electing gold OA
- [ ] All authors approved the final files and the submission
