# Journal of Economic Theory Skills 中文说明

这是面向 **Journal of Economic Theory (JET)** 的 12 个投稿技能包。JET 是 Elsevier
旗下经济理论期刊，适合严谨、原创的理论贡献：机制设计、信息经济学、决策理论、博弈论、
匹配、市场设计、政治经济学，以及金融/宏观/货币理论等方向。

本包是 theory-first 工作流，不是经验经济学模板。核心判断标准是新模型、新定理、新刻画或
新的证明/一般性；经验、实验、计算或校准工作只有在牢固服务于理论贡献时才适合。

## 技能列表

| Skill | 用途 |
| --- | --- |
| `jet-workflow` | 判断下一步该调用哪个 JET 技能。 |
| `jet-topic-selection` | 判断是否是 JET 范围内的理论贡献。 |
| `jet-literature-positioning` | 将结果与最近的理论文献和定理比较。 |
| `jet-identification-strategy` | 检查假设、结果、证明架构和一般性。 |
| `jet-contribution-framing` | 用定理和经济含义表达贡献。 |
| `jet-data-analysis` | 处理从属于理论的数值例子或计算。 |
| `jet-tables-figures` | 准备理论图示、机制图、表格和符号展示。 |
| `jet-writing-style` | 打磨 Elsevier/LaTeX 的 theorem-proof 写作。 |
| `jet-replication-and-data-policy` | 准备 Option C 数据声明、可复现计算和 AI 使用声明。 |
| `jet-review-process` | 理解 single-anonymized 审稿和理论 referee 期待。 |
| `jet-submission` | 进行 Editorial Manager 投稿前检查。 |
| `jet-rebuttal` | 回复关于正确性、一般性、贡献和表达的审稿意见。 |
