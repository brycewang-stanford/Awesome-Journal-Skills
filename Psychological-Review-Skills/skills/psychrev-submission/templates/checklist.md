# Psychological Review Pre-Submission Checklist

Psychological Review is a **theory journal**: the contribution is a new theory, formal/
computational model, or major synthesis — never an empirical report. Data appear only to
**motivate or constrain** the theory. There is no experimental dataset or empirical
replication package to deposit (for computational models, you share your **own model code**).
Verify all volatile specifics (editor, exact length/abstract limits, portal URL, APA edition,
fees/open-access, AI policy) on the official APA Psychological Review author page before
submitting. 检索于 2026-06；以官网为准.

## 1. Scope & fit

- [ ] The manuscript makes a THEORETICAL contribution (new theory / formal model / synthesis)
- [ ] Data appear only to motivate or constrain the theory; no new experiment is the contribution
- [ ] It systematically evaluates the standing alternative theories
- [ ] Not a literature survey, not a methods/design paper, not an empirical report
- [ ] Confirmed against the current Psychological Review aims & scope

## 2. Format (APA)

- [ ] APA 7th-edition formatting: double-spaced, 12-pt, 1-inch margins (verify current edition)
- [ ] Separate title page (authors, affiliations, ORCID iDs, contact) kept out of the masked body
- [ ] Abstract ≤ 250 words on a separate page (verify current limit)
- [ ] No fixed upper length; texts > 15,000 words include a justification (verify)
- [ ] Equations numbered; notation defined at first use
- [ ] Manuscript in Word / OpenOffice if the portal requires it (verify)

## 3. Masking (anonymous review)

- [ ] No author names, affiliations, or identifying phrases in the body
- [ ] Self-citations neutral ("prior research (Author, 2019)"), never "our earlier study"
- [ ] Acknowledgments and funding identifiers removed (added after acceptance)
- [ ] Document file properties scrubbed of author metadata
- [ ] File names free of author/institution names

## 4. Abstract & front matter

- [ ] Abstract states the problem, the theoretical move, and the contribution
- [ ] No data/results language in the abstract (there is no experiment)
- [ ] If a replication is included, it is mentioned in the abstract (per journal policy)
- [ ] Keywords selected per APA conventions

## 5. Theory quality

- [ ] Core commitments separated from auxiliary/implementational assumptions
- [ ] The mechanism (the why) is explicit before formalization
- [ ] Every parameter/construct has a psychological interpretation and stated range/domain
- [ ] Predictions are *derived* (not asserted) and at least one is risky and novel
- [ ] Head-to-head comparison with rival models on diagnostic phenomena
- [ ] Scope, identifiability (recovery/mimicry), and a "does NOT explain" list are stated

## 6. Figures & model code (conceptual only)

- [ ] Each figure is a model/architecture diagram or a simulation-of-model-behavior plot — never an experiment
- [ ] Every diagram box is a defined construct/module; every arrow a specified process
- [ ] Simulation captions disclose parameters and whether they were fit or set a priori
- [ ] (Computational models) author's own model code prepared, reproduces figures/fits, with a README
- [ ] Figure numbering and labels match the text exactly

## 7. References

- [ ] APA 7th author–date in-text citations; full reference list (verify edition; see source map)
- [ ] Every in-text citation is in the list and vice versa
- [ ] The current, strongest form of each rival model is cited (not only its original version)
- [ ] All references are real and accurately represented

## 7b. Contribution bar

- [ ] Contribution stated as before → after against a *named* prior model
- [ ] Advance is a mechanism / unification / adjudication / predictive formalization — not a relabel or refit
- [ ] Tied to diagnostic phenomena and a novel falsifiable prediction
- [ ] Generality claimed only to the extent the derivations and scope support

## 8. Ethics & portal (Editorial Manager)

- [ ] Original, unpublished, not under review elsewhere (APA Ethics Code)
- [ ] Authorship and order agreed by all authors; each made a genuine contribution
- [ ] Prior conference/preprint version disclosed if required; AI-assistance disclosed per current policy
- [ ] Similarity check run; self-overlap disclosed
- [ ] Registered in Editorial Manager (editorialmanager.com/rev — verify); correct article type (Theoretical article vs. Theoretical Note)
- [ ] Fees / open-access (APA Open) policy checked on the author page (do not assume)
- [ ] Files staged: masked manuscript + separate title page + model code/forms if applicable
- [ ] Within portal file-size and file-type limits
