# External Tools - IJCAI

Access date: 2026-06-01

Use these tools after reopening the current IJCAI cycle pages. IJCAI rules can change by
year and by track.

## Author-side checks

- Chairing Tool account, author role, title, abstract, keywords, author information, ORCID,
  coauthor acknowledgement, and conflicts.
- Current author kit, PDF style, page count, reference pages, and optional ethics statement.
- Anonymous paper, no acknowledgements or contribution statement, no identifying PDF
  metadata, and anonymized supplement/code/data.
- Technical Appendix, Resubmission File, and any code/data ZIP within current size/type
  limits.
- Recent-rejection disclosure package for the previous 12 months if applicable.
- Author response using the current official template and current response deadline.
- Camera-ready package, copyright/metadata, registration, and in-person presentation plan.

## Do not infer

- Do not infer future-cycle dates from IJCAI-ECAI 2026.
- Do not assume the same page limits, response length, PPI fee rules, submission system,
  review phases, author limits, LLM policy, supplementary-material rules, or presentation
  obligations without reopening current official pages.
- If official pages disagree, use the newest author kit, CFP update, or direct chair
  communication as the controlling source and record the conflict.
