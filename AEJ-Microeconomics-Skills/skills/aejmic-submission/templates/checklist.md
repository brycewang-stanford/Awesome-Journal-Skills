# AEJ: Micro Pre-Submission Self-Check (8 sections)

A last-pass checklist for an *American Economic Journal: Microeconomics* (AEJ: Micro)
manuscript before submitting via the AEA submission system. AEJ: Micro is the AEA's
theory-first microeconomics journal (founded 2009 — 待核实, quarterly, **single-blind**).
Verify volatile specifics (submission fee, abstract limit, current Data Editor, co-editor
list) on the official AEA author pages — 检索于 2026-06；以官网为准.

## 1. Venue & scope fit
- [ ] The central contribution is a microeconomic-**theory** result (or theory-grounded experiment), not an empirical paper
- [ ] The result is of **broad** micro interest (not specialist-only → JET/GEB; not abstract/methodological → TE/Econometrica; not empirical applied → AEJ: Applied)
- [ ] The one-sentence claim is sharp

## 2. Authorship & review type (single-blind — no anonymization)
- [ ] Review is **single-blind**: the referee sees the author; no anonymization is required
- [ ] Author names, affiliations, and contact info included as the system requests
- [ ] Self-citations phrased professionally, not promotionally
- [ ] Acknowledgments and funding stated normally

## 3. Front matter
- [ ] **Abstract ≤100 words** (hard limit)
- [ ] **JEL codes** present (per AEA practice; also routes to the right co-editor)
- [ ] Keywords present

## 4. Results & proofs
- [ ] Main results stated as **numbered** propositions/theorems, self-contained
- [ ] Key proofs in the paper (main text or appendix); none exiled to supplementary files
- [ ] Each proof matches exactly what its proposition claims; no "it can be shown"

## 5. House statistical presentation (any empirical/experimental content)
- [ ] **No significance asterisks**; standard errors / confidence sets reported
- [ ] Numerical examples state which proposition they illustrate

## 6. Data & code (AEA Data and Code Availability Policy)
- [ ] If the paper has data, code, experiments, or numerical results: deposit prepared for the AEA Data and Code Repository (openICPSR)
- [ ] One master script regenerates all tables, figures, and numerical examples; versions pinned; seeds set/reported
- [ ] **Pure-theory papers:** code behind numerical examples/figures still prepared
- [ ] Experiment materials (instructions, code, pre-registration, data) included

## 7. Fee & files
- [ ] AEA-member submission fee ready (confirm current amount — volatile)
- [ ] Anonymized manuscript PDF + separate title page as the system requires
- [ ] AEA/LaTeX formatting per current author guidelines

## 8. Declarations & final
- [ ] Disclosure statement per AEA policy (financial interests, funding)
- [ ] Confirmed not under review elsewhere; AI not listed as an author
- [ ] Co-author sign-off; corresponding author designated
