# AEJ: Microeconomics — Official Source Map

Primary and authoritative sources behind every AEJ: Micro-specific fact injected into
this skill pack. **Access date for all entries: 2026-06.** AEJ: Micro is owned and
operated by the **American Economic Association (AEA)**; replication is administered by
the **Office of the AEA Data Editor** and deposited in the **AEA Data and Code Repository
on openICPSR**. Re-verify volatile items (fees, abstract limit, editors, policy effective
dates) before relying on them — AEA pages change. Items not confirmed from a primary fetch
are marked **需复核 (to be verified)**.

## Official AEA / AEJ: Micro pages

- **https://www.aeaweb.org/journals/mic** — AEJ: Micro journal home. Verifies: published
  by the **American Economic Association**; **four issues per year** (quarterly); **Print
  ISSN 1945-7669 / Online ISSN 1945-7685**; scope — the journal "publishes theoretical
  work as well as both empirical and experimental work with a theoretical framework,"
  concentrating on microeconomic theory, industrial organization, and the microeconomic
  dimensions of international trade, political economy, and finance. Accessed 2026-06.
- **https://www.aeaweb.org/journals/mic/editors** — AEJ: Micro editorial leadership.
  Verifies: **Editor: Navin Kartik (Yale)**; **Coeditors: S. Nageeb Ali (Penn State),
  Alessandro Bonatti (MIT), Kareen Rozen (Brown), Daniel Yi Xu (Duke)**; **Data Editor:
  Lars Vilhuber (Cornell)**; Managing Editor: Michelle DeBlasi (AEA); plus a Board of
  Editors. Accessed 2026-06. (Roster is volatile — re-confirm.)
- **https://www.aeaweb.org/journals/mic/submissions** — AEJ: Micro Submission Guidelines.
  Verifies: **abstract of 100 or fewer words required**; **submission fee** — AEA members
  **$200** (high-income) / **$100** (middle-income) / **$0** (low-income), nonmembers
  **$300 / $200 / $0** (fee scaled by membership and country income classification);
  **papers rejected without review refunded 50%**; **publication fee $15 per typeset page**
  (page proofs, for papers submitted after Feb 1, 2024); "Prior to acceptance, authors of
  papers that contain **empirical work, simulations, or experimental work** must provide
  the data, code, and other details sufficient to permit replication. All submissions must
  comply with the **AEA Data and Code Availability Policy**." Accessed 2026-06.
- **https://www.aeaweb.org/journals/mic/editorial-policy** — AEJ: Micro Editorial Policy.
  Verifies: **single-blind** peer review — "the name of the author is revealed to the
  referee(s), while the referee(s) remain anonymous to authors." Accessed 2026-06.
- **https://www.aeaweb.org/journals/mic/reviewers** — AEJ: Micro Reviewer Guidelines.
  Corroborates the single-blind review process. Accessed 2026-06.

## AEA data / code regime (shared across AEA journals)

- **https://www.aeaweb.org/journals/data/faq** and the **Office of the AEA Data Editor**
  (**https://aeadataeditor.github.io/**) — AEA Data and Code Availability Policy. Verifies:
  the AEA adopted a revised Data and Code Availability Policy (2019) with **pre-publication
  reproducibility verification** by the Data Editor team; **Lars Vilhuber** has served as
  AEA Data Editor since 2018; materials are deposited in the **AEA Data and Code Repository
  on openICPSR** (each receives a DOI and metadata including JEL codes). The policy applies
  to papers with data, code, experiments, or numerical results; pure-theory papers still
  deposit any numerical/simulation code. Accessed 2026-06.
