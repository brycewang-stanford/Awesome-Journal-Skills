# 范式校准文献清单 — 《财贸经济》

> **用途：校准本刊实证范式，不是"本刊已刊目录"。** 下表汇集财政、税收、货币金融、贸易等
> 领域的**经典 / 代表性文献**，多数发表于国际顶刊或国内综合经济学刊，用来示范《财贸经济》
> 偏好的"真实财经问题 + 政策 / 制度准实验识别 + 机制 + 政策含义"范式。**引用前务必回到原文
> 核对卷期页码**；下表年份可靠，个别卷期 / 页码为便于定位而给粗略值，标 `以原文为准`。
>
> **相邻刊辨析（勿张冠李戴）。**《财贸经济》**不等于**：
> - 《金融研究》——中国金融学会主办的金融专门刊，纯资产定价 / 公司金融更契合它。
> - 《世界经济》——中国世界经济学会主办，国际贸易 / 开放宏观更契合它。
> - 《财政研究》《税务研究》——更专门的财税政策刊。
> - 《经济研究》——综合性经济学顶刊，理论贡献更一般时更契合它。

## 用法

- **财经政策评估是核心。** 本刊偏好"真实财经问题 + 企业 / 城市 / 银行层面准实验识别 + 机制 +
  异质性"的实证范式：先讲现象与问题、交代中国制度背景，再讲识别，机制与异质性互证，最后落到
  可操作的财经政策含义（`../../skills/cte-identification`、`../../skills/cte-mechanism`、
  `../../skills/cte-heterogeneity`、`../../skills/cte-policy-implication`）。
- **正视政策内生性。** 财税 / 金融 / 贸易政策往往非随机投放，上表范式多以政策分批推行（DID）、
  资格线断点（RDD）、份额移动 / 历史地理工具（IV）或双重机器学习（DML）回应内生性，本包
  `../code/`（多时点 DID、事件研究、IV、DML、聚类与安慰剂）正为此而备。
- **写作落到本刊房风格。** 摘要五句法、机制落到可识别渠道、避免空泛"意义"句式，见
  `../../skills/cte-abstract`、`../../skills/cte-style`。
- **核对出处与相邻刊。** 引用前回到原文复核卷期页码，避免与《金融研究》《世界经济》
  《财政研究》《经济研究》的同题材文章混淆。

> **核验状态（2026-07-16）。** 上表为国际 / 国内经典范式文献，年份可靠、用于范式校准；确切卷期
