# 财贸经济选题（cte-topic-selection）

## 触发时机

- 用户说"我想冲《财贸经济》，但不确定选题够不够'财经'"
- 边际贡献写不出来 / 写了一段但审稿人 buy 不了
- 不确定该投《财贸经济》、《金融研究》、《世界经济》还是综合性经济学期刊

## 三个硬约束

《财贸经济》对选题的硬约束：

1. **财经议题**：选题必须真正落在财政、税收、货币金融、贸易、流通或消费上，第一段就要说明"这是一个什么财经问题、为什么现在做"
2. **理论贡献 + 政策现实并重**：仅"现象 + 描述统计 + 政策口号"不够，要能挂上一支理论（财政分权、最优税收、货币传导、金融摩擦、贸易与产业组织、消费与福利等），同时紧扣一项中国重大财经政策现实
3. **边际贡献可枚举**：能拆成 3–5 条独立、互不重叠的贡献

## 选题范围自检（落到财经）

本刊关注的主流议题（用来判断"是否对口"）：

- 财政与政府间关系（财政分权、转移支付、地方债、财政可持续、支出结构）
- 税收（税制改革、减税降费、增值税 / 所得税、税收征管、税负与企业行为）
- 货币与金融（货币政策传导、利率市场化、金融监管、系统性风险、影子银行、数字金融）
- 贸易与开放（关税、贸易政策、出口企业、全球价值链、外资与开放红利）
- 流通与市场（商贸流通、供应链、市场一体化、平台经济、要素市场）
- 消费与居民福利（消费升级、收入分配、社会保障、住房、养老、共同富裕）

如果选题在上述清单里找不到落点，**多半不对口本刊**。

## 理论贡献自检

写完一句话摘要后，逐条自检：

- [ ] 本文挂在哪一支理论上？（财政分权 / 最优税收 / 货币传导 / 金融摩擦 / 贸易组织 / 消费福利）
- [ ] 该理论分支的代表性文献是哪 2–3 篇（含财经领域中文权威与国际理论）？
- [ ] 本文相对于该理论的边际差异是什么？（修正 / 扩展 / 引入新机制 / 提供首个中国制度证据）
- [ ] 中国财经制度是否提供了海外文献没有的识别机会（如某项财税 / 金融改革的准实验冲击）？

少于 3 个 yes → 选题深度不够，回到财经理论文献再读两轮。

## 边际贡献四句式

```
本文的边际贡献主要体现在以下四个方面：
第一，[理论维度]——本文将……纳入[财政分权 / 货币传导 / 金融摩擦等理论框架]，区别于[已有研究]……。
第二，[识别策略维度]——本文利用[财税 / 金融 / 贸易政策冲击]构造……，缓解了[政策内生 / 自选择 / 反向因果]带来的内生性问题。
第三，[机制维度]——本文揭示了[企业 / 银行 / 居民 / 地方政府层面]的……机制，回答了已有文献"知其然不知其所以然"的问题。
第四，[政策意义维度]——本文为理解[财税体制 / 金融改革 / 扩内需 / 共同富裕等议题]提供了……证据。
```

四条都齐才算合格。少于四条 → 选题深度不够。

## 期刊匹配判断

|  | 财贸经济 | 金融研究 | 世界经济 | 综合经济学期刊 |
|---|---|---|---|---|
| 学科 | 财政 / 金融 / 贸易 / 流通 / 消费 | 金融专门 | 国际经济 / 开放宏观 | 一般经济学 |
| 中国财经制度场景 | **必备** | 偏市场机制 | 偏国际维度 | 不要求 |
| 因果识别 | 高门槛 | 高门槛 | 高门槛 | 高门槛 |
| 政策现实相关性 | **强** | 中 | 中 | 中 |

若选题偏纯资产定价 / 公司金融 → 《金融研究》；偏国际贸易 / 开放宏观 → 《世界经济》；理论贡献更一般 → 《经济研究》。

## 反模式

- 选题脱离中国财经制度背景，讲一个可移植到任何国家的通用故事
- "现象 + 描述统计 + 政策建议"三段式，无理论、无识别
- 边际贡献写成一段散文，拆不出 3–5 条

```
【期刊匹配建议】财贸经济 / 金融研究 / 世界经济 / 其他

```
