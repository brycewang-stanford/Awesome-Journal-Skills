# 财贸经济文献综述（cte-literature-review）

## 触发时机

- 综述缺财经领域中文权威文献，或全部是英文文献
- 综述缺**理论文献**（财政分权、货币传导、金融摩擦、贸易组织等理论根基）
- 综述写成了"罗列式"而非"对话式"

## 结构规范

《财贸经济》偏好**三层结构**的综述：

1. **第一层：理论文献**（中外并重）——建立问题的理论坐标（如财政联邦主义、最优税收、货币政策传导、金融加速器、贸易与异质性企业）
2. **第二层：财经领域实证文献**（中文为主、英文为辅）——梳理已有的中国财税 / 金融 / 贸易微观与宏观实证发现
3. **第三层：本文位置**——明确指出"本文与现有研究的差异"，尤其相对于本刊及《金融研究》《世界经济》《财政研究》近年同主题文章的差异

### 与综合经济学期刊综述的差异

《财贸经济》尤其看重**财经领域文献是否到位、是否扣住中国制度**：既要有国际理论与方法文献，也要充分对话国内财经学界的权威研究与政策文献。一篇只引国际顶刊、不引国内财经经典与制度背景的稿件，会被审稿人判为"未扎根本土文献 / 与中国制度脱节"。

## 中外文献配比

经验法则（导向是"中外并重 + 理论扎实 + 扣住中国制度"）：

- 总引文 40–80 条（投稿前到官网"投稿须知"核对当年参考文献条数上限）
- 中文文献占比 **40%–60%**（财经领域本土实证与制度文献不能缺位）
- **理论文献占比 ≥ 20%**（理论推导 / 综述，不是实证应用）
- 五年内文献占比 **≥ 40%**
- 既覆盖国际顶刊（AER / QJE / JPE / JF / RFS / JPubE / JIE 等），也覆盖国内权威（《经济研究》《管理世界》《金融研究》《世界经济》《财政研究》《数量经济技术经济研究》等）

## 必引避坑清单

- [ ] 是否引用了所属理论分支的**经典理论文献**（财政分权 / 最优税收 / 货币传导 / 金融摩擦 / 贸易组织的原始 paper）？
- [ ] 是否引用了**财经领域的中文权威研究**（不是只引国际文献）？
- [ ] 是否交代了相关**中国财经制度 / 政策背景**（改革时点、政策文件）？
- [ ] 是否引用了近 3 年本刊或相邻财经刊的同主题论文？
- [ ] 引用是否覆盖三类：理论、方法、本土财经实证？

## 综述句式模板

```
[作者 (年份)] 利用[企业 / 城市 / 银行数据 + 方法]研究了[财经问题]，发现……。
然而，该研究[局限]，未考虑[本文要补的空白，如政策内生 / 制度异质 / 传导机制]。
与之相比，本文……。
```

每段综述至少出现一次"与之相比 / 区别于 / 本文延伸了 / 本文补充了"。

## 对话式 vs 罗列式

《财贸经济》偏好**对话式**综述（研究之间相互回应），而非**罗列式**（按作者顺序点名）。

### 错误（罗列式）

> 张三（2020）研究了减税，李四（2021）研究了企业投资，王五（2022）研究了融资约束。

### 正确（对话式 — 按主题分组、表明分歧、给出本文位置）

> 减税能否刺激企业投资，已有研究存在分歧（张三，2020；李四，2021）：一类研究认为减税通过放松现金流约束提高投资，另一类则强调预期不稳定削弱了减税的激励效果。两类判断的差异部分源于对政策内生性与企业预期的不同处理。本文沿这一思路，使用[数据]并构造[识别策略]，进一步考察了……。

判断标准：把作者名都隐去后，综述还能读出"研究 A 怎么说、研究 B 怎么反驳、本文怎么定位"，就是对话式。

## 与本文贡献阐述衔接

综述末尾必须导出本文的差异化定位：

> 本文与现有文献的重要区别体现在以下几个方面：
> 第一，[识别策略上的差异——如何处理政策内生 / 反向因果]；
> 第二，[数据或样本上的差异——更新 / 更细的企业 / 城市 / 银行数据]；
> 第三，[机制或异质性上的差异，落到中国财经制度]。

**与本刊及相邻财经刊已发表文章对比**是审稿人的关注点——综述若不提近 3 年同主题文章，会被视为"未充分对话"。

## 反模式

- 把综述写成"财税 / 金融政策背景介绍"而非学术对话
- 综述完全围绕国际文献，无国内财经权威研究与制度背景
- 中文文献只引一两条凑数
- 综述里出现一长串并列引用 —— 拆开按贡献分述
- 自引扎堆

## 输出格式

```
【结构】三层 / 两层 / 一层
【中外配比】中文 X% / 英文 Y%
【理论文献占比】X%
【财经本土权威文献】到位 / 缺位
【制度背景交代】到位 / 缺位
【五年内文献比例】X%
【缺漏清单】[...]
【下一步】cte-identification
```
