# ESR Pre-Submission Self-Check (7 sections)

A last-pass checklist for a *European Sociological Review* (ESR) manuscript before submitting via
**ScholarOne Manuscripts** (`mc.manuscriptcentral.com/esr`). Verify volatile specifics (exact word/
abstract caps, policy effective dates, editors, any OA charges) on the official OUP/ESR pages — items
not directly confirmed are marked 待核实.

## 1. Type & scope
- [ ] Decided: research **Article** / **Comment-Reply** / **Data Brief**
- [ ] Quantitative contribution with a portable mechanism and comparative or longitudinal leverage
- [ ] Not a single-country description or a pure methods/theory demonstration

## 2. Length & front matter
- [ ] Article around **8,000 words including endnotes and references** (tables/figures excluded)
- [ ] If over ~8,000, the contribution-justifies-length case is explicit (待核实 on any stricter cap)
- [ ] Abstract **≤200 words**, non-identifying; keywords included
- [ ] Written in acceptable English (author's responsibility)

## 3. Anonymity (double-blind)
- [ ] No author names, affiliations, or identifying acknowledgments anywhere in the manuscript
- [ ] Self-citations neutralized ("in a prior study," not "in our prior study")
- [ ] Identifying funding/grant numbers moved to the system fields, not the body
- [ ] Document properties / file metadata stripped of author identity

## 4. Style & exhibits
- [ ] Author-date citations, parenthetical (e.g., "(Gans, 1962)"), one consistent style
- [ ] Tables and figures self-contained, anonymous, carry magnitude + uncertainty
- [ ] Estimates translated substantively (predicted probabilities / marginal effects), not stars-only

## 5. Modeling readiness (what reviewers check)
- [ ] Standard errors at the correct level; few-cluster (~20-30 countries) inference handled (wild bootstrap / Bayesian / df correction)
- [ ] Measurement equivalence reported for cross-national latent comparisons (invariance level + what it licenses)
- [ ] Robustness incl. leave-one-country-out / alternative harmonization where relevant

## 6. Transparency (required)
- [ ] **Data Availability Statement** drafted — names archive/provider, data version/edition, access condition
- [ ] **Replication package** assembled for statistical/computational work (master script + data or access path + harmonization code + seeds + README), or qualitative **exemption** noted
- [ ] Package output re-run end-to-end and matches every reported table/figure (待核实 on current effective dates)

## 7. Ethics & posting
- [ ] Not under review at any other journal; prior appearances disclosed
- [ ] No OA charge surprises if open access is selected (待核实 on current APC)
- [ ] Corresponding-author contact and ORCID entered in the ScholarOne system fields (待核实 on ORCID requirement)
