# Governance — Pre-Submission Checklist

Run this before uploading to the **Wiley submission system** ("Wiley Authors" / Research Exchange).
*Governance: An International Journal of Policy, Administration, and Institutions* is **double-blind**;
verify every volatile item (caps, portal, citation style, ORCID, fees) on the live author-guidelines
page before relying on it (检索于 2026-06；以官网为准).

## 1. Scope fit (desk-rejection guard)
- [ ] Subject is executive politics / public policy / public administration / organization of the state
- [ ] **NOT corporate governance** (firm boards, shareholders, corporate compliance) — hard out of scope
- [ ] **NOT a literature review, state-of-the-field essay, or pure bibliometric study** — out of scope
- [ ] A clear, original **comparative/institutional intervention** is stated by the end of the introduction
- [ ] The claim **travels** across countries, levels of government, or policy domains
- [ ] Right journal vs. siblings: not PAR (practitioner), JPART (PA theory), JPAM (policy analysis),
      Regulation & Governance (regulatory studies)

## 2. Length & abstract
- [ ] Manuscript ≤ **9,000 words excluding citations/bibliography** (working cap; 待核实 against the
      official page — a third-party listing cites 3,000–5,000)
- [ ] Word count recorded (on the title page)
- [ ] Abstract **~150 words**, stating question + comparative approach + finding
- [ ] Robustness grids, full specifications, and balance tables moved to **supplementary material**

## 3. Anonymization (double-blind)
- [ ] No author names, affiliations, or acknowledgments anywhere in the manuscript
- [ ] No obvious self-references ("as we showed in…"); self-citations neutralized
- [ ] Funding/grant acknowledgments removed from the anonymized manuscript
- [ ] Identifying **file metadata stripped** (document properties, author tags, tracked changes, comments)
- [ ] Separate, **non-anonymous title page** prepared as a distinct file (authors, affiliations, ORCID,
      corresponding-author contact, word count, acknowledgments)

## 4. Data Availability Statement (required at submission)
- [ ] **Data Availability Statement included** in the manuscript
- [ ] Statement matches the correct case: openly available / restricted-with-access / not applicable
- [ ] If shared: a **DOI-citable repository** named (Harvard Dataverse / OSF / Zenodo), with the DOI/URL
- [ ] If restricted: reason given + exact access route (application, provider contact, data-use agreement)
- [ ] Replication master script reproduces every table/figure; README + seeds + pinned versions included
- [ ] Synthetic data provided where the real data cannot be shared (where feasible)
- [ ] (Note: *Governance* requires the **statement** and encourages sharing, but does not run a verified
      pre-publication reproduction check like APSR's Dataverse)

## 5. References & format
- [ ] Citations in a single consistent **author-date** style per the live Wiley/journal guidelines (待核实)
- [ ] Reference list complete; DOIs included where available; references excluded from the word count
- [ ] Headings, tables, and figures formatted to the journal/Wiley guidelines
- [ ] Figures/tables **self-contained, colorblind-safe, grayscale-legible**; cross-country comparison
      visible in the layout

## 6. ORCID & author info
- [ ] Corresponding-author **ORCID iD** ready (per Wiley norms; 待核实)
- [ ] Author order and contributions agreed
- [ ] Conflict-of-interest / funding statements prepared for the title page (not the anonymized file)

## 7. Supplementary & special files
- [ ] Anonymized **pre-analysis plan** supplied as supplementary material if a preregistration is referenced
- [ ] Supplementary material (extended robustness, appendices, qualitative source tables) staged
- [ ] Ethics / IRB / human-subjects compliance addressed where applicable

## 8. Files for the Wiley portal
- [ ] Anonymized main manuscript (no identifying metadata)
- [ ] Separate non-anonymous title page
- [ ] Figures/tables per portal requirements (format and resolution)
- [ ] Supplementary material file(s)
- [ ] Data Availability Statement present in the manuscript (and DOI links live)
- [ ] Preprint status checked against Wiley's preprint policy

## 9. Final gate
- [ ] Re-read the introduction: is the comparative/institutional contribution unmissable?
- [ ] Live portal confirmed (Wiley submission system, not legacy ScholarOne)
- [ ] All 待核实 / volatile items verified against the official author-guidelines page today
