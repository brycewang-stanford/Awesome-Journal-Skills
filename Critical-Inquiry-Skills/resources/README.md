# Critical Inquiry Resources

Action-oriented resources for the *Critical Inquiry* (CI) skill pack. The `skills/` give advice; this
directory lets an agent **model** a CI-style essay and **benchmark** against verified exemplars. Pair
these with the relevant `skills/ci-*/SKILL.md`.

*Critical Inquiry* is an **interdisciplinary humanities / critical-theory** journal: its evidence is
**objects read closely through theory** — texts, images, artworks, films, cases — **never data,
statistics, or code**. Accordingly this pack ships **no code kit** (there is nothing to compute), in
deliberate contrast to the empirical-economics packs.

## No code kit (humanities venue)

Unlike the empirical-journal packs, there is **no `code/` directory and no vendored pipeline**. CI
publishes humanistic argument, not statistical results; the analytic work is close reading, not
estimation. If a task asks for a "data pipeline" or "regression" for a CI submission, that is a
category error — redirect to [`exemplars/library.md`](exemplars/library.md) and
[`../skills/ci-evidence-and-objects/SKILL.md`](../skills/ci-evidence-and-objects/SKILL.md), which
explain that CI's evidence is objects, not datasets.

## Suggested workflow

1. Forge the claim with
   [`../skills/ci-argument-and-intervention`](../skills/ci-argument-and-intervention/SKILL.md) and
   situate it with
   [`../skills/ci-scholarly-positioning`](../skills/ci-scholarly-positioning/SKILL.md).
2. Choose and read objects with
   [`../skills/ci-evidence-and-objects`](../skills/ci-evidence-and-objects/SKILL.md); architect the
   essay with [`../skills/ci-structure-and-exposition`](../skills/ci-structure-and-exposition/SKILL.md)
   and model the opening on
   [`worked-examples/01-introduction.md`](worked-examples/01-introduction.md).
3. Polish prose to the word cap with
   [`../skills/ci-writing-style`](../skills/ci-writing-style/SKILL.md) and
   [`../skills/ci-citation-and-style`](../skills/ci-citation-and-style/SKILL.md).
4. Benchmark against verified CI essays in [`exemplars/library.md`](exemplars/library.md); confirm
   submission, word-cap, and image/permission policy in
   [`official-source-map.md`](official-source-map.md).
