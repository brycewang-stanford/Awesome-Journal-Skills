# 原创理论贡献（ssc-theory-contribution）

## 触发时机

- 有发现但贡献只写成"本文发现 X 影响 Y"
- 想冲本刊却拿不出"原创命题"
- 评审/自评："贡献只是验证了已有理论"

## 贡献的三种合格形态

1. **新概念**：命名一个此前无名的机制/现象，并给出界定与边界
2. **新命题**：提出可被讨论/检验的一般性判断（非就事论事）
3. **新框架**：给出组织若干现象的分析框架，重排已有认识

> 验证已有结论 ≠ 理论贡献。必须能说清**"本文改变了我们对什么的理解"**。

## 自主知识体系定位

- 从**中国经验**提炼一般性理论，而非把中国当国外理论的"又一个样本"
- 说清：中国情境如何**改写理论前提/适用边界**，或贡献了何种新机制
- 与既有（含西方主流）理论**平等对话**：继承什么、修正什么、超越什么
- 警惕两个极端：①全盘套用国外框架；②只讲特殊性、拒绝一般化（不可对话）

## 贡献陈述写法

```
既有研究就 <议题> 形成了 <判断A/分歧>。本文提出 <概念/命题/框架>，
论证 <核心机理>；区别于 <既有判断>，本文表明 <新理解>。
这一命题的适用边界是 <条件>，对 <理论T> 构成 <修正/扩展/整合>。
```

## 自检清单

- [ ] 贡献是概念/命题/框架之一，不是"发现 X→Y"
- [ ] 能说清"改变了对什么的理解"
- [ ] 与既有理论实质对话（继承/修正/超越各有所指）
- [ ] 中国经验进入了理论（改写前提或贡献机制），不是采样地
- [ ] 给出了命题的适用边界，避免无限外推

## 命题压力测试

把核心命题写成三句话：

1. 既有理论怎样解释 X。
2. 中国经验或本文材料显示这一解释在哪个条件下不充分。
3. 本文提出的概念、命题或框架如何重排解释关系。

然后检查两个边界：这个命题是否能与外部理论对话，是否又不把中国经验简化成"例证"。
若只能写成"本文发现 A 影响 B"，说明还停留在经验结论，需要回到概念命名或机制抽象。

## 本刊贡献分量的校准

观察《中国社会科学》刊文的贡献形态：常见的是给机制命名的概念（往往进入后续文献成为通用词）、重排既有认识的分析框架，以及对经典判断的条件化修正。三个校准问题：

- 你的概念五年后会出现在别人的标题里吗？
- 你的框架能组织你自己没研究过的现象吗？
- 你的修正改变的是结论、前提，还是适用边界？——能改写前提者分量最重

## 微型提炼演练

经验发现："宅基地制度改革试点地区农户财产性收入显著上升。"逐层提炼：

```text
第一层（经验）：改革 → 收入上升
第二层（机制）：产权确权激活了沉睡资产的抵押与流转功能
第三层（命题）："产权激活"命题——农村要素市场化的关键不是私有化
  程度，而是产权束中"可交易束"的制度化拆分
第四层（对话）：修正了将产权改革等同于私有化进程的既有判断，为
  渐进式改革理论补充"拆分—激活"机制
```

到第三、四层才构成本刊意义上的贡献；停在前两层的稿件，更适合对口实证刊。

## 评审拷问三连及预演

1. "这个概念与××的区别是什么？"——预演：写出与两个最近邻概念的界分表
2. "该命题在什么条件下不成立？"——预演：主动写出两个失效情境并纳入正文边界讨论
3. "除了中国，该命题还能解释什么？"——预演：给出一个可比情境的延伸讨论，哪怕只有一段；这正是"自主知识体系"与"可对话性"并立的要求

## 反模式

- "丰富了 XX 研究 / 提供了新视角"——空贡献
- 只罗列发现，把理论提炼留给读者
- 把"特殊国情"当贡献，却拒绝任何一般化

## 输出格式

```
【贡献形态】新概念 / 新命题 / 新框架
【一句话命题】<…>
【对话关系】继承<…> 修正<…> 超越<…>
【自主知识体系】中国经验如何进入理论：<…>
【适用边界】<条件>
【下一步】ssc-literature-dialogue
```
