# 理论脉络对话（ssc-literature-dialogue）

## 触发时机

- 文献只引近五年实证，没有经典/流派
- 综述写成"张三发现…李四发现…"的流水账
- 评审："缺乏与基础理论的对话"

## 综合刊的文献观

《中国社会科学》看的是**学术脉络感**：你站在哪个理论传统里、回应谁、推进什么。跨学科稿尤其要让相关学科的同行看出"你读过这一脉的根"。

## 三段式综述（替代流水账）

1. **理论源流段**：勾勒该议题的经典传统与主要流派（点到关键作者/著作）
2. **当代进展段**：近年研究在哪几条线上推进，分歧何在
3. **本文位置段**：本文承接哪条线、补哪个缺口、与哪个判断争论

## 经典与本土并重

- 触及**理论源流**（学科奠基作者与代表作），不能只有近期论文
- 跨学科稿：引入相关学科的经典对话对象，避免"自说自话"
- 重视中文学术传统中的本土理论资源，纳入对话而非装饰

## 对话矩阵

写综述前先填三列：

| 理论传统 | 关键判断 | 本文如何回应 |
|----------|----------|--------------|
| 经典源流 | 它如何定义问题 | 继承/修正/反驳 |
| 当代争论 | 分歧在哪里 | 补机制/补证据/换框架 |
| 本土资源 | 中国经验提供什么概念增量 | 提炼而非装饰 |

综述正文只保留能进入第三列的文献。

## 分学科的"源流"指什么

本刊实行跨学科评审，"理论脉络"在不同学科有不同所指，综述要让该学科同行认出门牌：

| 学科 | 源流的形态 | 流水账的形态 |
|------|------------|--------------|
| 哲学 | 经典文本、诠释传统与问题史 | 只引近年期刊论文 |
| 经济学 | 理论流派及其分析传统 | 只引近五年顶刊实证 |
| 社会学 | 奠基理论家与中层理论谱系 | 变量文献清单 |
| 法学 | 法教义学传统与法理论争 | 政策文件加注释 |
| 史学 | 史料传统与史学史上的论争 | 平铺研究综述 |

## 微型改写示例

流水账原文：

```text
关于社会资本的研究，张三（2020）发现其促进集体行动；李四（2021）利用
面板数据证实了这一结论；王五（2022）则在西部样本中得到类似发现。
```

进入脉络后：

```text
社会资本研究存在两个传统：一支沿帕特南一脉将其视为公民社会的存量，
一支沿布迪厄一脉强调其再生产不平等。近年中文实证多在前一传统内累积
（张三，2020；李四，2021），却未回应后一传统的批评——社会资本由谁
占有、向谁封闭。本文从中国村庄经验切入这一分歧，论证……
```

差别：文献从"并列的发现"变成"传统之间的争论"，本文位置随之显形。

## 本土理论资源的入场方式

- 概念入场：以中文学术传统中的经典提炼（如"差序格局"一类概念）作为分析起点，并交代其谱系与后续争论
- 争论入场：把中文学界内部的实质分歧写成对话的一极，而非附录式的"国内研究综述"
- 材料入场：以中国经验材料对国际理论判断构成压力测试，结论回馈到一般理论层面

三种方式共同要求"入场后承担论证功能"；只在综述里礼节性引用本土文献，本刊评审一眼即知是装饰。

## "对话不足"评语的修复路径

1. 先判断缺的是哪一层：缺经典源流、缺对立流派，还是缺本土理论资源——三者修法不同
2. 补对话对象必须落到正文论证处，而非只扩综述节；本刊评审会核对综述承诺与正文兑现是否一致
3. 跨学科稿至少让两个学科的同行能在综述里找到自己脉络的入口，否则外审抽到外学科专家时极易失分

## 自检清单

- [ ] 综述进入了理论脉络（有经典/流派，不只近年实证）
- [ ] 按"源流—进展—本文位置"组织，而非按作者罗列
- [ ] 指明了本文介入的具体分歧/缺口
- [ ] 跨学科对话对象到位，相关学科同行能认领
- [ ] 引用为论证服务，无堆砌式并列引注

## 反模式

- "（张三，2020；李四，2021；王五，2022）"式并列，不分述
- 只综述方法/数据演进，回避理论分歧
- 把综述写成"无人反对本文"的铺垫

## 输出格式

```
【理论源流】<传统/流派/关键作者>
【当代分歧】<两三条线及其对立>
【本文位置】承接<…> 补<…> 争论<…>
【脉络感】足 / 不足（缺哪个传统）
【下一步】ssc-argumentation
```
