# 学理文风（ssc-style）

## 触发时机

- 通读后觉得"像政策文件 / 工作汇报 / 教材综述"
- 一段里出现 ≥ 2 个"具有重要意义 / 提供新视角"
- 文风要么口号化，要么纯技术腔，缺学理质感

## 目标语域：宏观视野 + 学理克制

既不流于口号与宏大抒情，也不沦为纯技术报告。每个判断都有学理依据，每个大词都能兑现。

## 对照表（黑名单 → 白名单）

| 问题 | 黑名单 | 白名单 |
|------|--------|--------|
| 空洞价值 | "具有重大的理论价值" | "为理解 X 提供了 Y 框架，修正了 Z 判断" |
| 口号化 | "必须高度重视 / 大力推进" | "这一机制表明，制度设计应在 X 处优先于 Y" |
| 套话贡献 | "丰富了相关研究" | "提出 <概念>，重排了对 X 的既有认识" |
| 炫技 | "本文运用复杂的 XX 方法" | "借助 XX 方法，本文得以分离出 Y 机制" |
| 文献堆砌 | "（A,2020；B,2021；C,2022）" | 按源流/进展/位置分述 |
| 大词空转 | "推动文明进步" | 落到具体机制与边界，少用不可兑现的大词 |

## 行文规范

- **一段一概括**：每段首句是概括句，审稿人读首句即可复原全文逻辑
- 概念全文一致，慎用未界定的大词
- 谦逊与边界：经验研究说"提供了证据支持"，不说"证明了"
- 跨学科稿照顾他学科读者：少用本学科黑话或首次出现即解释

## 自检清单

- [ ] "具有重要意义"类空话命中 = 0
- [ ] 无政策汇报腔（"加强/完善/推进/深化"四件套）
- [ ] 每段首句为概括句
- [ ] 大词都能兑现为具体机制/边界
- [ ] 方法描述不长于思想阐释

## 学理改写顺序

不要先做句子润色，先做段落功能检查：

1. 标出每段首句，判断它是在提出判断、推进论证，还是只在转述材料。
2. 把"重要意义"类句子改成"本文改变了对 X 的哪一层理解"。
3. 把政策建议句改成机制句：先说为什么，再说在何种条件下可能怎么办。
4. 把方法句压到服务论证的位置：方法只说明如何看见机制，不替代理论发现。
5. 通读标题、小标题、摘要、结论，确保核心概念命名一致。

这一顺序能避免把一篇政策报告只改成更漂亮的政策报告。目标是让文字承载理论命题。

## 学科语域的校准

本刊各学科文章语域不同，但共享"学理克制"的底色。改稿前先认领学科语域，再做句子手术：

- 哲学/思想史稿：允许长句与概念密度，但每个术语须有出处或界定，忌散文化抒情
- 经济学/定量稿：忌翻译腔与符号堆砌，结果句要还原为关于社会过程的中文判断
- 法学稿：规范分析语言务求精确，忌把论证写成普法宣传
- 史学稿：叙事可以从容，但每段叙事都须服务论题，忌为细节而细节
- 跨学科稿：以最克制的学科为基准定调，避免段落间语域跳变

## 标题与小标题的学理化

本刊文章标题常见"概念/命题主标题——限定副标题"的结构，主标题点出理论对象而非政策对象：

| 政策腔标题 | 学理化标题 |
|------------|------------|
| 关于加强基层治理能力建设的研究 | 嵌入性自治：基层秩序生成的一种机制 |
| ××政策实施效果评估 | 制度弹性与政策执行的再生产逻辑——以××为例 |

小标题同理：用判断句或概念短语，连读应能复原论证链。

## 微型改写演练

原段（政策汇报腔）：

```text
当前，必须高度重视数字治理工作，大力推进平台监管体系建设，不断完善
相关法律法规，这对推动国家治理现代化具有重大意义。
```

学理化改写：

```text
平台监管的难点不在规则缺位，而在监管权的配置：当算法决策内化于平台，
传统科层监管的"事后查处"逻辑即告失灵。这意味着治理现代化的关键变量，
是监管权在国家与平台之间的再分配方式。
```

改写动作：删除"高度重视/大力推进"四件套；把"重大意义"换成具体机制判断；让段落承载一个可争论的命题，而非一组号召。改完自测：这段话能被反驳吗？不能被反驳的段落多半还是口号。

## 反模式

- 通篇"重要意义 / 高度重视"
- 用排比大词替代论证
- 方法段长于结论段

## 输出格式

```
【空话命中】X 处：[...]
【政策腔】有 / 无
【段首概括率】X / 总段数
【大词兑现】到位 / 空转 [...]
【下一步】ssc-abstract-keywords
```
