# 选题与问题意识（ssc-topic-problematic）

## 触发时机

- 选题有方向但说不清"为什么重要、与谁争论"
- 引言写成"研究背景介绍"而非"问题的提出"
- 被评"问题太小 / 像政策评估"

## 重大问题的三个维度

1. **理论争论**：领域内存在对立判断或未解之问，本文介入其中
2. **现实分量**：连着国家发展、制度变迁、文明/治理这一层级的真问题
3. **时代之问**：能回应"中国之问、世界之问、人民之问、时代之问"中的某一面

> 黄金句式：本文不只问"X 是否影响 Y"，而是问"X→Y 揭示了什么一般机理 / 修正了哪个既有理论判断 / 中国经验对该判断意味着什么"。

## 把小问题"上移"的手法

| 小问题（高风险） | 上移后（够格） |
|------------------|----------------|
| 某政策对某结果的评估 | 该政策背后的制度逻辑及其对 X 理论的修正 |
| 某地区某现象的描述 | 该现象折射的结构性矛盾与一般机制 |
| 验证国外理论在中国成立 | 中国情境如何改写该理论的适用边界/前提 |

## 问题提出段的结构（引言前 1/3）

1. 摆出现象/张力 → 2. 指明既有理论的解释缺口或分歧 → 3. 提出本文的核心问题 → 4. 预告本文命题（不展开论证）

## 选题层级判定表

| 层级 | 例 | 本刊接受度 |
|------|----|------------|
| 行业/技术问题 | 某行业数字化转型的绩效表现 | 低——改投专业期刊 |
| 政策问题 | 某政策是否达到预期目标 | 低——除非上移到制度逻辑 |
| 制度问题 | 该类政策为何在此制度结构中反复出现 | 够格的起点 |
| 文明/理论问题 | 中国式现代化对既有现代化理论的改写 | 本刊核心区 |

判定时自下而上问一句：这个问题若消失，哪一层的讨论会受损？只有某行业受损的选题，不必投本刊。

## 微型上移演练

原问题："短视频平台使用是否降低青少年学业成绩？"逐级上移：

```text
政策层：平台监管新规能否改善青少年发展结果？
制度层：注意力商业化与教育制度的目标冲突，如何在家庭中被调解？
理论层：当算法成为与家庭、学校并列的社会化机构，经典社会化理论
  是否需要重写？
```

上移到理论层后，原实证设计降格为证据之一，引言改写为与社会化理论的对话。但要量力：若上移后与作者的材料和学养脱节，宁可停在制度层立一个扎实的中层问题，也不要悬空喊理论——空喊比问题小更致命。

## "时代之问"的具体落点

把口号落为可论证的问题张力，避免贴标签：

| 之问 | 贴标签写法（无效） | 落点写法（有效） |
|------|--------------------|------------------|
| 中国之问 | "本文回应中国之问" | 中国经验在该议题上与既有理论预测的具体偏离 |
| 世界之问 | "具有世界意义" | 该机制在何种可比情境下同样成立或失效 |
| 时代之问 | "立足新时代" | 何种结构性变化使旧判断失效、新问题由此产生 |

检验法：删掉所有"之问"字样后，问题是否仍然成立？若不成立，说明问题是靠标签撑着的。

## 跨学科接口自检

本刊读者横跨哲学、经济学、法学、史学等多个学科，问题的提出应让外学科读者在前两页内明白"这与我何干"：

- 经济学选题给出制度与治理意涵，而不止效率结论
- 法学/政治学选题点明经验后果，而不止规范推演
- 史学选题指出对当下理论争论的回响，而不止史实补正

## 自检清单

- [ ] 问题连着制度/文明/治理层级，不是细分行业
- [ ] 点明了领域内的**理论分歧或缺口**（与谁争论）
- [ ] 有"中国问题"的实质内核，不是贴标签
- [ ] 引言是"问题的提出"，不是"研究背景综述"

## 反模式

- 用"具有重要意义"替代真实的问题张力
- 把"数据可得 / 政策新出台"当作选题理由
- 问题与结论之间没有理论命题，只有经验关系

## 输出格式

```
【核心问题】<一句话>
【理论分歧】与 <谁/哪个判断> 对话
【现实分量】制度/文明/治理层级：是 / 否（需上移）
【中国内核】实质 / 仅标签
【下一步】ssc-theory-contribution
```
