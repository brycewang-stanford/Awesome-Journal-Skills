# 外审回复（ssc-rebuttal）

## 触发时机

- 收到《中国社会科学》外审/编辑部修改意见
- 需要逐条回复并对应修改正文

## 顺序：先改正文，后写回复

回复信不能跑在正文修订前面。**先动正文，再写"已如何修改 + 在何处"**，否则空头承诺会被识破。

## 本刊审稿的关注点（据其定位）

- 原创命题是否成立、是否被过度外推
- 理论对话是否到位（经典/流派、跨学科）
- 论证链是否严谨、概念是否清晰
- 思想分量与现实/理论意义是否匹配
- 体例（页下注、引文准确性）

## 逐条回复模板

```
【意见N】<审稿人原话要点>
【回应】<接受/部分接受/审慎商榷——给出学理理由>
【修改】正文第X节第Y段：<具体改动>
【证据/引文】<新增论证/文献/材料>
```

## 三类意见的处理

| 意见类型 | 策略 |
|----------|------|
| 命题/理论质疑 | 优先正面回应：补论证、明确边界，必要时收窄命题（不硬扛） |
| 文献/对话不足 | 补理论脉络与跨学科对话，落到正文，不只在回复信里说 |
| 体例/表述 | 照单全收并修订，回复从简 |

## 本刊外审意见的跨学科特征

《中国社会科学》的稿件常由不同学科背景的专家评审，意见形态与单学科期刊不同：

| 意见形态 | 背后逻辑 | 处理 |
|----------|----------|------|
| 他学科专家质疑概念用法 | 该概念在其学科另有定谱 | 不视为外行意见；增设界定句，说明本文用法与该学科传统的关系 |
| 两位审稿人意见相互冲突 | 学科评价标准不同 | 分别作答，并向编辑部说明取舍的学理依据 |
| 要求"提升理论高度" | 综合刊门槛所在，并非客套 | 回 `ssc-theory-contribution` 实质重写，不能只在回复信抬高调门 |
| 引文与注释逐条核查 | 本刊对页下注与引文准确性要求严格 | 全部修订并逐条注明修改位置 |

## 微型回复示例（命题质疑）

```
【意见2】"嵌入性自治"命题与既有"行政吸纳社会"判断区别不明，疑为旧说新名。
【回应】部分接受。二者均承认行政力量在场，但"行政吸纳"预设自治被消解，
本文命题强调自治能力在制度接口处被激活，方向相反。已增写界分段落。
【修改】第二节第3段新增与该判断的三点区分；结论第2段补充适用边界。
【证据/引文】新增对该判断原始文献的页下注引证，及两个方向相反的个案。
```

## 回复信的开篇结构

逐条回复之前，先用半页给出修改总览，帮编辑部与复审专家快速建立全局：

```text
一、命题层面：将原命题收窄为 <…>，新增适用边界讨论（第五节）。
二、对话层面：补充与 <学科/流派> 的对话（第一、二节）。
三、论证层面：删除原第三节的材料铺陈，新增反例处理一节。
四、体例层面：页下注格式、引文页码、英文摘要已全面修订。
```

总览只列实质修改，不写客套；每条与后文的逐条回复一一对应。重投稿件如同重新论证：编辑部要看到的是"问题被解决的结构"，而不是"作者很努力的痕迹"。

## 商榷的分寸感

- 商榷只用于学理分歧，且必须给出文献或论证依据；体例与表述类意见一律照改
- 不要"口头接受、实质未改"——复审常由原审稿人承担，逐条核对是常态
- 修改说明按正文章节组织而非按意见散列，便于编辑部与复审专家定位
- 若复审周期较长，耐心等候并按编辑部流程询问，时限规则以编辑部现行说明为准

## 礼仪与分寸

- 对编辑/审稿人保持尊重，分歧处"审慎商榷"并给理由，不情绪化
- 真接受就真改，别口头接受、正文不动
- 重大修改在回复信开头做"修改总览"，便于复审定位

## 自检清单

- [ ] 每条意见都有回应 + 对应正文改动位置
- [ ] 命题质疑被正面处理（补证/收窄边界）
- [ ] 理论/跨学科对话的补充落到了正文
- [ ] 体例问题全部修订
- [ ] 语气专业、商榷有据

## 反模式

- 回复信承诺修改，正文未动
- 对命题质疑硬扛或回避，不调整边界
- 把审稿人意见情绪化对待

## 输出格式

```
【意见总数】N　【已逐条回应】N
【修改总览】<3–5 条主要修改>
【命题处理】正面回应 / 收窄边界 / 商榷
【未采纳项】<条目 + 学理理由>
【复审就绪】是 / 否
```
