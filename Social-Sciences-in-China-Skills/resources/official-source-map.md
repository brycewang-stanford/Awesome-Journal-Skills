## 二、已据公开信息确认的特征（高置信度，仍建议官方复核）

1. **主办与出版**：《中国社会科学》由中国社会科学院主办，中国社会科学杂志社编辑出版。
2. **创刊年**：1980 年。
3. **刊期**：月刊，每月 25 日出版。
4. **期刊性质**：中国哲学社会科学综合性顶级学术期刊，覆盖文学、历史学、哲学、经济学、法学、政治学、社会学等多学科——非单一经济学 / 实证类期刊。
5. **摘要与关键词**：摘要约 200–300 字，关键词 3–5 个，配中英文标题 / 摘要 / 关键词（具体字数以最新《投稿须知》为准）。
6. **注释体例**：采用页下注（脚注），遵该刊引文注释规范，不使用「著者-出版年」夹注制——这是它区别于经济学、心理学、管理学等专业实证期刊（后者多用夹注 + 文末参考文献 / APA）的硬性差异。
7. **审稿与遴选导向**：实行专家评审；选题强调重大理论与现实问题、学术原创性与思想性，鼓励构建中国自主知识体系、提炼中国学派与标识性概念。
8. **拒稿雷区**：就事论事、缺乏理论提升的政策评估 / 纯实证 / 纯技术性写作不受欢迎；「思想分量 > 方法复杂度」是其隐性门槛。

## 三、投稿前 live-check 清单

- [ ] 通过 `sscp.cssn.cn/tsgpt/` 或杂志社官网“我要投稿”进入当前投稿系统，确认账号注册、作者承诺书和上传步骤。
- [ ] 打开最新《投稿须知》，确认脚注格式、是否另列参考文献、文章/专题/笔谈/讨论与评议等栏目篇幅。
- [ ] 复核双向匿名要求：正文不得出现作者个人信息，行文也应避免透露作者身份。
- [ ] 复核费用规则：2025-10-31 投稿须知明确本社刊物不向作者、发文单位收取任何形式费用，也未委托商业机构征稿组稿。
- [ ] 复核审阅周期：2025-10-31 投稿须知给出“一般为 3 个月”的审阅周期表述。
- [ ] 核对中文版与英文版 ISSN / CN 刊号、期刊栏目和最新目录。
- [ ] 如需列举代表性论文，先用官网目录、CNKI 或国家哲学社会科学文献中心核对题名、期次和页码。

## 四、核实方法建议

1. 在可访问中国境内网站的网络环境下打开中国社会科学院 / 中国社会科学杂志社官网与中国社会科学网。
2. 定位《中国社会科学》期刊页面，下载 / 查阅最新《投稿须知》与引文注释规范。
3. 通过投稿系统 sscp.portal.founderss.cn（或官网入口）确认注册与提交流程，并截图留存版本日期。
4. 以中国知网（CNKI）等数据库核对该刊近年目录与代表性论文的脚注格式实例。
5. 核对无误后，据最新规定同步更新 `journal-profile.md` 与各 SKILL.md 中的体例描述。
