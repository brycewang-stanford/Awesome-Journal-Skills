# PMLA Resources

Discipline-appropriate resources for the *PMLA* (Publications of the Modern Language Association) skill
pack. The `skills/pmla-*/SKILL.md` give advice; this directory lets an agent **model** a PMLA-style
section and **benchmark** against verified exemplars. Pair each resource with the relevant
`skills/pmla-*/SKILL.md`.

**No code kit (humanities venue).** PMLA is the generalist flagship of **literary and language
studies**: an essay is **argument, close reading, theoretical framing, and engagement with the critical
conversation**, in **MLA style**. There is **no dataset, no statistical analysis, and no replication
package** — so, unlike the empirical-science packs, this directory deliberately ships **no `code/`
directory**. The evidence is the text. See [`external_tools.md`](external_tools.md) §7 ("This is not a
data pack").

## What's here

| Resource | Use it to... |
| --- | --- |
| [`worked-examples/01-introduction.md`](worked-examples/01-introduction.md) | See a before→after rewrite of a literary-studies essay **introduction** in PMLA house style (significant problem → thesis → conversation → stakes, intervention early; close reading in the foreground — no data, no hypotheses). Clearly-marked **fictional** essay on the public-domain *Frankenstein*; no real quotations. |
| [`exemplars/library.md`](exemplars/library.md) | Benchmark against **real, web-verified PMLA essays** organized by field × method. Positioning and exemplar value only — no reproduced or invented readings. Includes a **sibling-venue** do-not-misattribute list. |
| [`official-source-map.md`](official-source-map.md) | **PMLA-specific policy & facts:** word range (6,000–9,000; notes counted, Works Cited/translations excluded), ScholarOne submission, MLA membership, blind review, AI-disclosure, special features, scope/values — with `待核实` flags on volatile items. The authoritative pack source. |
| [`external_tools.md`](external_tools.md) | Texts, editions, archives, the MLA International Bibliography, MLA-style citation tools, and prose aids — matched to materials and method, not to estimation or code. |

## Suggested workflow

1. **Scope and frame** with [`../skills/pmla-topic-selection`](../skills/pmla-topic-selection/SKILL.md)
   and [`../skills/pmla-scholarly-positioning`](../skills/pmla-scholarly-positioning/SKILL.md); model the
   introduction on [`worked-examples/01-introduction.md`](worked-examples/01-introduction.md).
2. **Build the argument and the reading** with
   [`../skills/pmla-argument-development`](../skills/pmla-argument-development/SKILL.md),
   [`../skills/pmla-textual-evidence-and-close-reading`](../skills/pmla-textual-evidence-and-close-reading/SKILL.md),
   and [`../skills/pmla-theory-and-method`](../skills/pmla-theory-and-method/SKILL.md); source texts and
   bibliography via [`external_tools.md`](external_tools.md).
3. **Shape and polish** with
   [`../skills/pmla-structure-and-exposition`](../skills/pmla-structure-and-exposition/SKILL.md) and
   [`../skills/pmla-writing-style`](../skills/pmla-writing-style/SKILL.md) (mind the word range — notes
   count).
4. **Benchmark** against verified essays in [`exemplars/library.md`](exemplars/library.md); confirm
   scope, venues, submission, and review policy in
   [`official-source-map.md`](official-source-map.md).
