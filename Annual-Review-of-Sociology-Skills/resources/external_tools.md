# External Tools for Annual Review of Sociology

An ARSoc review is a **systematic synthesis of others' work**, so the tooling is bibliographic and
organizational, not estimation software. Confirm any deposit/format requirement on the Annual
Reviews author pages (检索于 2026-06；以官网为准).

## Literature discovery & coverage

| Tool / source | Use |
|---------------|-----|
| **Sociological Abstracts** (ProQuest) | the discipline's core indexing database; primary keyword + subject sweep |
| **Web of Science** / **Scopus** | forward-citation snowballing (who cites the seeds) and cross-discipline reach |
| **JSTOR** | back-catalogue access to canonical sociology (ASR, AJS, *Social Forces*, etc.) |
| **Google Scholar** | broad forward/backward snowballing, working papers, grey literature |
| **SocArXiv** / **SSRN** | preprints and forthcoming work (the frontier) |
| **ASA journal archives** (ASR, *Contexts*, *Socius*, *Sociological Theory*) | locate sibling-venue work and check venue before citing |

## Synthesis & reference management

| Tool / source | Use |
|---------------|-----|
| **Zotero** / **EndNote** | build the citation corpus; export to the evidence matrix |
| **PRISMA / PRISMA-ScR templates** | document the coverage account (flow diagram, search log) where a near-systematic review is warranted |
| **Rayyan** / spreadsheet matrix | screening and the evidence matrix (claim / method-mode / finding / tradition / credibility / tension) |
| **metafor / robumeta (R)** | only if the review contributes its own meta-analysis of commensurable estimates |
| **bibliometrix (R)** / **VOSviewer** | bibliometric/citation-map exhibits where the review computes one |

## Deposit (only if the review runs its own analysis)

| Repository | Use |
|------------|-----|
| **OSF** / **Dataverse** / **Zenodo** | deposit meta-analysis or bibliometric data + code; cite the DOI |

## Method notes

- ARSoc reviews report **no primary estimates** — keep no replication package of your own data; the analogous artefacts are the **coverage account** and any **meta-analysis** you compute.
- Document the search like a dataset: databases, dates, terms *and* theoretical-tradition names, inclusion/exclusion, saturation, counts.
- Sweep deliberately across **modes** — quantitative, qualitative/ethnographic, computational, demographic, theoretical — siloing by method is a recurring ARSoc referee complaint.
- Verify the venue of every candidate exemplar before citing it as an ARSoc article (sibling-journal guard: ASR / AJS / *Sociological Theory* / the sister Annual Reviews are not ARSoc).
