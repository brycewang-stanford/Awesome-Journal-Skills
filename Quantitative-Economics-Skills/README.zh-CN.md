# Quantitative Economics Skills

面向 **Quantitative Economics (QE)** 的 12 个 agent skills。QE 是 Econometric Society 的
quantitative economics 期刊，关注理论、实证、实验与计算材料的清晰量化贡献。

官方依据核验日期：2026-06-20。来源见 `resources/official-source-map.md`，包括 QE submission
instructions、editorial procedures、publication instructions、ES data/code policy 与 Wiley journal metadata。

## Skills

- `qe-workflow`：路由 QE 全流程。
- `qe-topic-selection`：判断适合 QE、Econometrica、ET 或 field journal。
- `qe-literature-positioning`：定位 economics / econometrics literature。
- `qe-identification-strategy`：审查理论、假设、识别与实证策略。
- `qe-data-analysis`：处理 quantitative evidence、simulation、experiment 与 empirical work。
- `qe-tables-figures`：按 QE 格式处理 exhibits。
- `qe-contribution-framing`：明确 quantitative-economics contribution。
- `qe-writing-style`：处理 abstract、spacing、appendix 与 significance-star norms。
- `qe-replication-and-data-policy`：准备 ES data/code materials 与 README。
- `qe-review-process`：理解 editorial review、desk rejection 与 data-editor checks。
- `qe-submission`：核查 membership、fee、PDF、format 与 disclosures。
- `qe-rebuttal`：撰写回复与修改计划。
