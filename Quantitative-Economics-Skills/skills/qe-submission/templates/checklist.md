# QE Pre-Submission Self-Check (8 sections)

A last-pass checklist for a *Quantitative Economics* (QE) manuscript before
submitting via the Econometric Society online portal. Verify volatile specifics
(fee, page charge, editors, policy effective dates) on the official ES pages.

## 1. Eligibility & membership
- [ ] At least one author is a current Econometric Society member
- [ ] Decided: regular fee (US$100) / student fee (US$50) / low-income exemption / Econometrica transfer (fee waived)
- [ ] If transferring from Econometrica: cover letter explains differences and how referee comments were addressed

## 2. Format (hard rules)
- [ ] Single PDF; font ≥12pt; 1.5 or double spacing throughout (incl. references and appendices)
- [ ] Margins ≥1.25in on all sides; ≤32 lines per page; pages numbered
- [ ] Figures and tables in-text on the relevant page; references at the very end

## 3. Front matter
- [ ] Abstract ≤150 words
- [ ] Title page: keywords + complete author affiliations (JEL codes optional)

## 4. House statistical presentation
- [ ] No asterisks or boldface for statistical significance anywhere
- [ ] Standard errors and confidence/coverage sets reported instead
- [ ] No certified random author ordering (circled-r); contributions in a footnote if noted

## 5. Appendices
- [ ] Supplemental Appendix ≤25 pages, for additional results/robustness only
- [ ] Proofs of key results are in the main paper, which is self-contained

## 6. Experimental / own-data (effective Jan 1, 2026)
- [ ] Detailed experimental instructions / survey transcripts included at submission
- [ ] Own-data studies (RCT / lab / lab-in-the-field / online) pre-registered (AEA RCT Registry / AsPredicted / OSF), or prior editor exemption obtained

## 7. Reproducibility readiness (ES Data Editor checks before acceptance)
- [ ] Raw data, code, and documentation assembled; one master script regenerates all results
- [ ] Seeds set/reported; any partial-check scope noted for the README
- [ ] Exemption/limited-access request (if any) stated now, at initial submission

## 8. Declarations & posting
- [ ] Disclosure statement (US$10,000 interested-party threshold; funding sources); conflict-of-interest checked
- [ ] Multi-author responsibility statement prepared (effective Jan 2026)
- [ ] Manuscript posted to a public website; link ready; AI not listed as an author
