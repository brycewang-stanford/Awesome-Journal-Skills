# RSS Resources

Working materials for the RSS (Robotics: Science and Systems) skill pack. The `skills/`
directory tells an agent what to do; this directory supplies the reference layer — a
verified source map, award-anchored exemplars, a demo-to-science rewrite, and a
reproducibility adapter for the computational side of a robotics submission.

## Scope note

This pack serves a single-track robotics conference whose bar is a scientific claim
backed by embodied evidence. It is not an economics journal pack (no Stata/DiD/IV kit)
and not a mega-conference pack — routing between RSS and its higher-volume siblings is
handled in [`../skills/rss-topic-selection`](../skills/rss-topic-selection/SKILL.md)
and [`../skills/rss-workflow`](../skills/rss-workflow/SKILL.md).

## Suggested route

1. **Frame** — `rss-topic-selection` for the fit verdict, then the worked example to
   see what "science-first" means on the page.
2. **Build evidence** — `rss-experiments` and `rss-reproducibility` while the robot is
   still available; regret about missing trials is unfixable in January.
3. **Benchmark** — compare your first page against the exemplars library; every PDF is
   free at roboticsproceedings.org.
4. **Verify** — reopen [`official-source-map.md`](official-source-map.md) and
   [`external_tools.md`](external_tools.md) before quoting any deadline or format rule.
