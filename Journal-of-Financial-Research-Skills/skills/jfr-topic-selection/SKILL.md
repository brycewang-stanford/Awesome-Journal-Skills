# 选题与贡献（jfr-topic-selection）

## 触发时机

- 有金融数据/发现但讲不出"为什么是金融问题、贡献在哪"
- 选题像"X 与 Y 的关系"，缺机理与制度抓手
- 想判断该题是否值得冲《金融研究》而非一般金融刊

## 《金融研究》偏好的四类选题

1. **货币政策与金融稳定**：政策工具与传导、宏观审慎、系统性风险、流动性
2. **银行与信贷**：信贷配置、银行风险承担、资本/拨备监管、利率市场化、存款竞争
3. **公司金融与治理**：融资约束、杠杆与投资、并购、机构投资者、信息环境、ESG 融资后果
4. **金融市场与创新**：资产定价异象、市场微观结构、金融科技、绿色/普惠金融、债市/汇市

## 把"发现"升级为"金融问题"

三步走：

1. **锚定金融机理**：这是哪条渠道的故事？（信贷/风险承担/资产负债表/信息）
2. **锚定中国制度抓手**：哪条监管规则/市场结构使该机理在中国可识别、有现实意义？
3. **锚定贡献增量**：相对既有文献，你**新增了哪条机制、哪个边界、哪个被忽略的主体**？

只有第 1 步 → 像综述；只有第 3 步 → 像炫数据；三步齐 → 像《金融研究》。

## 贡献话术（避免空泛）

- 弱："本文研究了 X 对 Y 的影响。" → 强："本文识别出 X 通过<风险承担渠道>影响 Y，并刻画了其在<利率市场化>后的边界。"
- 弱："丰富了相关研究。" → 强："修正了'X 一律缓解融资约束'的判断，揭示在<国有/民营>分层下方向相反。"

## 自检清单

- [ ] 一句话能说清"哪条金融渠道 + 哪个中国制度抓手"
- [ ] 贡献是新机制/新边界/新主体，而非新样本
- [ ] 落点在金融体系/政策意义，不止于公司绩效
- [ ] 题目可被可信识别策略支撑（见 jfr-identification）
- [ ] 不与该刊近年高被引文重复（差异说得出）

## 微型走查：把"我有一份绿色信贷数据"打磨成选题

示意稿，演示三步升级：

- **起点（炫数据）**："我们有银行绿色信贷余额数据，想看它和企业表现的关系。" —— 只有数据，没有金融问题。
- **第 1 步·锚定机理**：绿色信贷指引收紧高污染企业信贷 → **信贷渠道**（信贷可得性再配置）。
- **第 2 步·锚定制度抓手**：2012 年绿色信贷指引按"两高一剩"名录划线，制造外生信贷收紧 → 可识别、有现实意义。
- **第 3 步·锚定增量**：既有研究多看企业总体融资，**忽略绿色信贷在不同所有制企业间的再配置效应**；本文补这条主体边界——民营高污染企业受冲击更大约 1.5 倍。
- **成形选题**："绿色信贷指引通过信贷渠道在所有制间再配置信贷，民营高污染企业融资约束被放大，揭示环境监管的金融分配后果。"

## 选题强度自检矩阵

| 三步是否齐 | 表现 | 处置 |
|------------|------|------|
| 只有第 1 步 | 有机理无识别、像综述 | 补中国制度抓手 |
| 只有第 3 步 | 炫数据无机理 | 补金融渠道命名 |
| 缺第 2 步 | 机理有但中国不可识别 | 找监管阈值/政策冲击 |
| 三步齐 | 渠道 × 制度 × 增量 | 推进到识别 |

## 审稿人追问模式与本刊语境下的修法

| 审稿人追问 | 背后担心 | 本刊语境修法 |
|------------|----------|--------------|
| "这是金融问题还是产业问题？" | 落点漂移 | 把金融渠道立为主线（回 jfr-fit-positioning） |
| "新数据/新指标就是新贡献吗？" | 以数据冒充问题 | 增量必须是新机制/新边界/新主体 |
| "选题太大，边界在哪？" | 宏大无边界 | 收缩到具体渠道 × 具体制度抓手 |
| "和近年高被引文有何不同？" | 重复风险 | 明确差异化的机制或主体维度 |

## 校准锚点

本刊偏好"金融渠道 + 中国制度抓手 + 明确增量"的中观选题，而非"金融发展促进 X"式宏大命题，且选题须被可信识别策略支撑。栏目侧重与选题口味以编辑部最新稿约与近期目录为准。

## 反模式

- "金融发展促进经济增长"式宏大无边界选题
- 用新数据/新指标冒充新问题
- 选题落点其实在产业或会计（应回 jfr-fit-positioning）
- 只有相关性故事，识别策略无从谈起
- 三步只占其一就匆忙立题

## 输出格式

```
【一句话选题】<渠道> × <中国制度抓手> → <金融结果>
【线别】宏观金融 / 微观公司金融
【贡献类型】新机制 / 新边界 / 新主体
【现实意义对象】货币当局 / 监管 / 金融机构 / 市场
【可识别性】强 / 中 / 弱
【下一步】jfr-literature-review
```
