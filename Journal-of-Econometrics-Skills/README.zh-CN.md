# Journal of Econometrics Skills

面向 **Journal of Econometrics (JoE)** 的 12 个 agent skills。JoE 是 Elsevier 的计量经济学方法旗舰，
关注 identification、estimation、testing、decision、prediction 与经济研究中的方法应用。

官方依据最新核验日期：2026-06-20。来源见 `resources/official-source-map.md`，包括
ScienceDirect Guide for Authors、JoE 自有指南、Editorial Express resubmission 页面，以及当前
“新投稿入口 vs. Editorial Express 指引”并存的操作性冲突。

## Skills

- `joe-workflow`：路由 JoE 全流程。
- `joe-topic-selection`：判断是否是真正的 methodological contribution。
- `joe-literature-positioning`：定位最近 estimator/test/theory 文献。
- `joe-identification-strategy`：审查 assumptions、regularity conditions、asymptotics 与 proof plan。
- `joe-data-analysis`：设计 Monte Carlo 与 empirical illustration。
- `joe-contribution-framing`：说明为什么结果对 econometrics 有贡献。
- `joe-tables-figures`：处理 size/power tables 与 method figures。
- `joe-writing-style`：压缩 abstract、intro 与 proofs。
- `joe-replication-and-data-policy`：处理 Elsevier data citation 与可复现材料。
- `joe-review-process`：理解 single-anonymized review、tracks 与 referee expectations。
- `joe-submission`：核查 live portal、fee、files/PDF 与 track。
- `joe-rebuttal`：撰写修改回复。
