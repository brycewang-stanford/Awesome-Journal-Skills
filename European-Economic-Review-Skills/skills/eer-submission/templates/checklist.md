# EER Pre-Submission Self-Check (8 sections)

A last-pass checklist for a *European Economic Review* (EER) manuscript before
submitting via Editorial Manager (`editorialmanager.com/eerev`). Verify volatile
specifics (fee, abstract limit, reference style, OA APC, replication-policy
details) on the official Elsevier / ScienceDirect guide for authors —
**检索于 2026-06；以官网为准**.

## 1. Scope & fit
- [ ] The question is general-interest across economics (survives the editor desk-screen)
- [ ] Breadth + novelty are visible in the abstract and first paragraph
- [ ] Sibling decision made (EER vs JEEA / The Economic Journal / Elsevier field journal)

## 2. Manuscript format (Elsevier economics)
- [ ] Editable source (LaTeX or Word) — required to typeset the final article
- [ ] Title page: title, authors, affiliations, corresponding author, declaration of interest, funding
- [ ] Re-confirm any length/word guidance on the guide for authors (no single fixed cap verified — 待核实)

## 3. Front matter
- [ ] Abstract present (re-confirm any word limit — 检索于 2026-06)
- [ ] JEL codes + keywords accurate (cross-field where the result travels)
- [ ] Research highlights included (a few concise bullet findings — Elsevier requirement)

## 4. Statistical presentation
- [ ] Standard errors (and/or CIs) reported under every estimate
- [ ] Significance stars do not replace SEs or magnitudes anywhere
- [ ] One headline table carries the main result; key figure has uncertainty bands

## 5. Identification & robustness
- [ ] Design-appropriate estimator (heterogeneity-robust DiD / weak-IV-robust IV / bias-corrected RDD)
- [ ] Key identifying assumption stated and the main threat pre-empted
- [ ] Coefficient-stability / sensitivity battery included; fragilities stated honestly

## 6. Replication readiness (mandatory policy)
- [ ] `run_all` master script regenerates every reported table and figure from raw data
- [ ] Environment pinned (package versions) and seeds set/reported
- [ ] Data-availability statement matches the deposit; restricted-data path documented
- [ ] Deposit location confirmed against the current EER/Elsevier policy (检索于 2026-06)

## 7. Fees & files (Editorial Manager)
- [ ] Submission fee ready (EUR 125 / EUR 100 PhD), or Research4Life voucher obtained **before** submitting
- [ ] Cover letter (scope/contribution; suggested/excluded referees if invited)
- [ ] Supplementary appendix and editable/vector figure source uploaded

## 8. Declarations & integrity
- [ ] Declaration of interest + funding statement included
- [ ] Authorship / CRediT roles set; AI not listed as an author
- [ ] Confirmed not under review elsewhere; permissions for any third-party material cleared
