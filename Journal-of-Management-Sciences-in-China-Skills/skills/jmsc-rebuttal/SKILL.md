# 外审回复 / R&R（jmsc-rebuttal）

## 触发时机

- 收到《管理科学学报》外审意见，需要写回复信
- 修改稿已就绪，要做逐条回应
- 审稿质疑集中在假设、证明、算法或数值（本刊典型关切）

## 核心：先改正文，再写回复

**正文未修订前不要生成回复**。回复信的每一条都要指向修改稿的**具体命题/公式/附录/页码**，让审稿人能核验。回应数理稿的关键是：把"我们认为合理"换成"我们已证明/已补充分析"。

## 本刊高频审稿意见与应对

| 审稿意见 | 应对要点 |
|----------|----------|
| 假设 A_i 不合理/太强 | 辩护其类型与必要性；或放松后重证，报告影响 |
| 命题证明有跳步/不完整 | 补全证明（含边界/唯一性），指明附录位置 |
| 缺存在性或唯一性 | 分别补证；若不唯一，刻画解集/给精化 |
| 算法无复杂度/收敛 | 补 O(·) / 收敛性证明（到何点、何条件、何速率） |
| 启发式无质量保证 | 补近似比或与下界/最优的 gap |
| 数值实验单薄 | 扩参数范围 + 加基准 + 报重复统计量 |
| 管理启示空泛 | 改写为参数化决策规律（jmsc-managerial-insights） |
| 与已有模型区别不清 | 明确新在何处（新结构/新性质/新算法） |

## 回复信结构

1. **致谢 + 总述**：概括主要修改（新增命题、补全证明、扩充实验）。
2. **逐条回应**：原意见 → 回应 → 修改位置（命题 X / 式 (Y) / 附录 Z / 第 N 页）。
3. **修改稿标注**：用颜色/修订标记，便于核对。

## 回应语气与策略

- 承认有理的意见并落实修改，不狡辩。
- 不同意时给**数理理由**（反例、定理、文献），而非"我们坚持认为"。
- 每条都让审稿人能定位到修改处，不空口承诺。

## 自检清单

- [ ] 正文已实际修订，回复信才动笔
- [ ] 每条意见都有回应，无遗漏
- [ ] 每条回应指向具体命题/公式/附录/页码
- [ ] 涉及证明/算法的意见，用补证/补分析回应（非口头保证）
- [ ] 不同意处给数理依据，语气专业克制
- [ ] 修改稿有标注，便于核验

## 反模式

- 正文没改就先写"已修改"
- "我们认为合理"对抗"假设不合理"，不给证据
- 回应不指位置，审稿人无从核对
- 把数值实验当证明来回应"证明不完整"
- 选择性回应，回避最硬的那条

## 本刊回复信审稿期待与扣分模式

《管理科学学报》的 R&R 多围绕假设、证明、算法、数值四类硬关切。下表对齐本刊常见扣分点与修法：

| 扣分信号 | 根因 | 本刊期望的修法 |
|----------|------|----------------|
| "回应空洞、未落实修改" | 口头承诺无正文改动 | 先改正文，回复指向命题/式/附录/页码 |
| "用仿真回应证明缺口" | 把验证当证明 | 补解析证明，仿真仅作佐证 |
| "分歧处只说坚持" | 不给数理依据 | 给反例/定理/文献支撑分歧立场 |
| "回避最硬意见" | 选择性回应 | 逐条回应，最硬一条放显著位置正面处理 |
| "假设质疑未化解" | 仅重申合理 | 辩护类型与必要性，或放松重证报告影响 |

> 锚点：本刊回复信常以"修改总述 + 逐条回应（原意见→回应→修改位置）+ 修订标注稿"三段式组织，分歧处用定理或反例说话。具体格式以编辑部最新要求为准。

## 微型走查：一封回复信的逐条回应

虚构稿件收到两位审稿人意见。按"先改正文、再逐条指位"走一遍（示意仅作演示）：

- **修订状态**：正文已修订（新增命题 4、补全命题 2 唯一性证明、数值新增敏感性一节），方动笔回复。
- **R1-1（假设 A3 太强：信息完全不现实）**：回应——感谢指出；我们在第 5 节新增不完全信息扩展，证明命题 2 的阈值结构在信号噪声 σ_ε≤0.3 时仍成立（新增命题 4，附录 B.3）；并讨论 A3 作为基准的简化作用。指向命题 4 / 附录 B.3 / 第 14 页。
- **R1-2（命题 2 只证存在未证唯一）**：回应——已补唯一性证明，利用目标严格凹（Π''<0，式 (12)），见附录 A.2 第 3 段；正文命题 2 表述相应改为"存在且唯一"。
- **R2-1（算法无复杂度）**：回应——已补复杂度分析，列生成定价子问题为 O(T·|R|)，整体轮数随规模的经验曲线见图 6；见第 4.3 节。
- **分歧处（R2-2 建议改用元启发式）**：回应——我们保留精确列生成，因其在 n≤30 时 6 分钟内达 3% gap（表 4），元启发式无解质量保证；以数值与界论证，而非"我们坚持"。

审稿人若复议"不完全信息扩展是否改变核心结论"，回应应明确：阈值方向不变、仅阈值水平随 σ_ε 平移，给出命题 4 的比较静态作为依据，避免含糊。

## 输出格式

```
【修订状态】正文已改？是/否（否→先改正文）
【意见清单】R1-1 … R1-2 … R2-1 …
【逐条】意见→回应→位置（命题/式/附录/页）
【补证/补分析】<列出新增的证明/复杂度/实验>
【分歧处】<给出的数理依据>
【标注】修改稿已标记？是/否
【结论】可回 / 待补 <清单>
```
