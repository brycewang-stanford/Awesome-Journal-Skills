# Journal of Human Resources Skills

Twelve Codex/Claude skills for manuscripts targeted at the *Journal of Human
Resources* (JHR), a leading empirical-microeconomics journal published by
University of Wisconsin Press and housed at the UW-Madison Institute for Research
on Poverty.

Despite the name, JHR is not a human-resource-management journal. It publishes
policy-relevant empirical microeconomics in labor, education, health,
development, discrimination, retirement, and related fields. This pack is tuned
for scope discipline, credible empirical design, the page-based format rule,
per-author disclosure statements, and the JHR data/replication archive policy.

## Skills

| Skill | Use it for |
| --- | --- |
| `jhr-workflow` | Route the manuscript to the next JHR skill. |
| `jhr-topic-selection` | Confirm empirical-micro scope and avoid HR-management mismatch. |
| `jhr-literature-positioning` | Position the contribution against policy-relevant applied micro work. |
| `jhr-identification-strategy` | Stress-test causal design and reconciliation with prior estimates. |
| `jhr-data-analysis` | Build samples, estimates, robustness, and comparative analyses. |
| `jhr-contribution-framing` | Make the policy-relevant JHR contribution explicit. |
| `jhr-tables-figures` | Prepare applied-micro tables, figures, and appendix exhibits. |
| `jhr-writing-style` | Polish concise JHR prose within the page limit. |
| `jhr-replication-and-data-policy` | Prepare archive plan, CC0 deposit, DAS, code, and waiver requests. |
| `jhr-review-process` | Understand desk screening, single-anonymous review, and review recycling. |
| `jhr-submission` | Run msubmit.net preflight. |
| `jhr-rebuttal` | Organize R&R response and reconciliation work. |
