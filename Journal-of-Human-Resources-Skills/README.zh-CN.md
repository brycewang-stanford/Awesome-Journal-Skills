# Journal of Human Resources Skills 中文说明

这是面向 *Journal of Human Resources* (JHR) 的 12 个投稿技能包。JHR 是 University
of Wisconsin Press 出版、UW-Madison Institute for Research on Poverty 相关的实证微观经济学期刊。

注意：JHR 不是人力资源管理期刊。它关注具有公共政策意义的实证微观研究，包括劳动、教育、
健康、发展、歧视、退休等领域。本包覆盖选题适配、识别策略、与既有估计的 reconciliation、
页面限制、作者披露、数据归档、投稿和 R&R 回复。

## 技能列表

| Skill | 用途 |
| --- | --- |
| `jhr-workflow` | 判断下一步该调用哪个 JHR 技能。 |
| `jhr-topic-selection` | 确认实证微观范围，避免 HR 管理误投。 |
| `jhr-literature-positioning` | 在政策相关 applied micro 文献中定位贡献。 |
| `jhr-identification-strategy` | 检查因果识别和与既有估计的 reconciliation。 |
| `jhr-data-analysis` | 设计样本、估计、稳健性和比较分析。 |
| `jhr-contribution-framing` | 明确 JHR 关心的政策贡献。 |
| `jhr-tables-figures` | 准备 applied micro 表格、图和附录展示。 |
| `jhr-writing-style` | 在页数限制内打磨精炼写作。 |
| `jhr-replication-and-data-policy` | 准备归档计划、CC0 数据、DAS、代码和豁免申请。 |
| `jhr-review-process` | 理解 desk screen、单匿名审稿和 review recycling。 |
| `jhr-submission` | 进行 msubmit.net 投稿前检查。 |
| `jhr-rebuttal` | 组织 R&R 回复和 reconciliation 工作。 |
