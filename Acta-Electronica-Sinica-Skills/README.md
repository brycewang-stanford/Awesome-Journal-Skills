# 《电子学报》技能包 · Acta Electronica Sinica (AES) Skills

> English note: this is a Chinese-language depth pack for **《电子学报》(Acta Electronica Sinica, AES)**.
> 完整中文说明见 [README.zh-CN.md](./README.zh-CN.md)。

面向 **《电子学报》(Acta Electronica Sinica, AES)** 的 12 技能中文深度包，帮助作者把一份电子/信息/
通信/信号处理方向的中文原创论文，从选题、写作、投稿、双盲外审、修回一路打磨到录用出版。

## 关于本刊

《电子学报》(Acta Electronica Sinica, AES) 是由中国科学技术协会主管、**中国电子学会
(Chinese Institute of Electronics, CIE)** 主办、创刊于 **1962 年** 的电子与信息科学**综合性**中文
旗舰学术**月刊**（ISSN 0372-2112，CN 11-2087/TN，主编 郝跃）。本刊被 EI、CSCD 核心、北大核心、
CSTPCD 等收录，学科覆盖电子科学与技术、信息与通信工程、信号与信息处理、电路与系统、电磁场与
微波、集成电路等方向，实行双盲评审与多级审理。中国电子学会同时主办英文姊妹刊
**Chinese Journal of Electronics (CJE)**（创刊 1992 年、被 SCI 收录），二者定位不同，同一稿件不可
同时投递。以上事实于 2026-07-09 通过官网 (ejournal.org.cn)、CNKI/万方期刊主页与 CCF 推荐中文
科技期刊目录交叉核验；凡未公开确认者在各技能中标注 **待核实**（见
[resources/official-source-map.md](./resources/official-source-map.md)）。

## 与兄弟刊的区别

《电子学报》聚焦**电子信息学科**（区别于《计算机学报》《软件学报》的计算机类、《自动化学报》的
控制类、《通信学报》的通信类、《中国科学:信息科学》的综合类），有中英文摘要、原创论文与综述并重、
严格双盲多级评审，并与英文版 Chinese Journal of Electronics 形成中英文分流。投稿前请先用
`aes-topic-selection` 判断选题是否对口本刊学科。

## 12 个技能

| 技能 | 用途 |
|---|---|
| `aes-submission` | 投稿就绪审计（系统/模板/中英文部件/双盲/评审费） |
| `aes-author-response` | 审稿意见答复与修回说明 |
| `aes-camera-ready` | 录用后定稿、揭盲回填与校样 |
| `aes-artifact-evaluation` | 代码/数据/硬件制品的可用性披露 |
| `aes-reproducibility` | 可复现性与实验可重复 |
| `aes-supplementary` | 附录与补充材料组织 |
| `aes-review-process` | 多级审理流程与稿件状态研判 |
| `aes-writing-style` | 中文科技写作规范（摘要/中图分类号/GB/T 7714） |
| `aes-related-work` | 相关工作与文献综述 |
| `aes-experiments` | 实验/仿真设计与呈现 |
| `aes-workflow` | 投稿全流程编排 |
| `aes-topic-selection` | 选题与栏目学科定位 |

## 快速开始

1. 动笔前：`aes-topic-selection` 判断对口度 → `aes-related-work` 定位贡献。
2. 研究写作期：`aes-experiments`、`aes-reproducibility`、`aes-writing-style`、`aes-supplementary`。
3. 投稿前：`aes-artifact-evaluation`（如适用）→ `aes-submission` 跑就绪审计。
4. 投稿后：`aes-review-process` 跟踪 → `aes-author-response` 修回 → `aes-camera-ready` 定稿。
5. 全程用 `aes-workflow` 做阶段调度。
