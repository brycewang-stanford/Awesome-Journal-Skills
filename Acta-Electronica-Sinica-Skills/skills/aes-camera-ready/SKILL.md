# 《电子学报》录用后定稿与校样

当《电子学报》(Acta Electronica Sinica, AES) 编辑部发出**录用通知**，工作重心从"说服评审"转向
"零差错出版"。本刊由中国电子学会 (Chinese Institute of Electronics) 主办、为月刊，录用后会进入
编辑加工、排版、校样 (清样) 与付印流程。此阶段任何错误都会印在正式出版物上难以挽回，需以校对员
的严谨对待。以下要点于 2026-07-09 经官网 (ejournal.org.cn) 与期刊主页核验；未公开确认者标 **待核实**，
以录用信与编辑部清样说明为准。

> 心态切换：这不再是学术争辩，而是工程质检。Acta / Electronica / Sinica 的每一处字符、公式、
> 参考文献都要与你的最终意图逐字符一致。

## 一、录用后第一批事务

- [ ] 回签**版权/著作权转让协议**（本刊要求，具体形式以编辑部为准，**待核实**）。
- [ ] 缴纳**版面费/发表费**（金额与时点以录用信公示为准，**待核实**）。
- [ ] 确认所有作者署名顺序、单位、通信作者与联系方式为最终版。
- [ ] 补全双盲阶段隐去的**基金项目名称与编号**、致谢、作者简介。

## 二、揭盲信息回填清单

双盲评审期间被隐去的内容此时必须**完整、准确**回填：

| 项目 | 回填要求 |
|---|---|
| 作者姓名/单位 | 与投稿系统登记一致，外文单位名用规范译名 |
| 基金项目 | 名称 + 编号完整，多项基金逐一列出 |
| 通信作者 | 标注符号 + 邮箱 |
| 作者简介 | 姓名、出生年、性别、职称、研究方向 |
| 致谢 | 对个人/机构的感谢，不与基金重复 |

## 三、正文终校核对表

- [ ] 中文题名 / 英文题名一致、无错别字，英文实词大小写规范。
- [ ] 中文摘要 / 英文 Abstract 与正文结论一致，数字口径一致。
- [ ] 中文关键词 / 英文 Keywords 一一对应。
- [ ] 中图分类号（电子信息类多为 TN 大类）与主题相符。
- [ ] 所有公式连续编号、符号定义齐全，正文引用式号正确。
- [ ] 所有图表编号连续、图注表注完整，正文均有引用。
- [ ] 量与单位使用国家法定计量单位，斜体正体规范。

## 四、参考文献终校（GB/T 7714）

- 逐条核对著录项：作者、题名、刊名/出版者、年、卷(期)、页码、文献类型标识码。
- 中英文文献著录格式统一，英文作者姓前名后、名缩写规范。
- 核对正文引用序号与文末列表一一对应，无断号、无多余条目。
- 确认 DOI（若著录）准确无误。

## 五、图表与公式的印刷级质量

- 图件分辨率满足印刷要求（线条图矢量优先，位图足够 dpi）；黑白印刷下可辨。
- 图中文字字号、坐标标注、图例清晰；避免过小字号在缩排后不可读。
- 公式使用公式编辑器排版而非截图，转换排版后须复核符号未走样。
- 表格跨栏/跨页时表头重复，三线表规范。

## 六、校样（清样）流程

- 编辑部发来**校样 PDF**后，通常给定较短的返校期限（**待核实**具体天数）。
- 校样阶段一般只允许**订正错误**，不允许大幅改写或新增内容；如确需改动实质内容须与编辑沟通。
- 采用"清样—作者校—编辑复核"的一校二校模式；每次校对用清晰标注列出勘误位置与正确内容。
- 逐字校对：题名、作者、摘要、公式、数据、参考文献是高错区，务必二人以上交叉校对。

## 七、出版时序与 DOI

- 确认稿件排入的**卷(期)与出版时间**；月刊按期出版，具体见编辑部安排。
- 确认 DOI 与在线优先出版（若有）状态。
- 出版后核对官网、CNKI/万方数据库中的电子版无误，发现错误及时联系编辑部勘误。

## 八、定稿总检清单

- [ ] 版权协议已签、版面费已缴。
- [ ] 揭盲信息（作者/基金/致谢/简介）完整回填。
- [ ] 中英文题名、摘要、关键词、中图分类号全部核对。
- [ ] 公式、图表、参考文献逐项终校。
- [ ] 校样勘误已提交并经编辑确认。
- [ ] 卷期、出版时间、DOI 确认。

## 九、易错高发区专项复核

录用后的差错多集中在少数几处，值得二人交叉专项复核：

| 高发区 | 典型错误 |
|---|---|
| 作者与单位 | 姓名拼写、单位译名、通信作者标注遗漏 |
| 基金编号 | 揭盲回填时编号错位、多项基金漏列 |
| 公式符号 | 排版转换后上下标走样、正斜体混乱 |
| 数据数字 | 摘要与正文、图与表之间数值口径不一致 |
| 参考文献 | 断号、卷期页码错、中英格式不统一 |
| 图注表注 | 编号与正文引用不对应、量纲缺失 |

对以上每一区，建议用"作者自校 + 合作者复校"的双人机制，逐字比对最终意图与清样。

## 十、出版后事务

- 出版后核对官网、CNKI、万方等数据库的电子版是否与清样一致；发现错误立即联系编辑部勘误。
- 保存正式 PDF 与 DOI，用于后续引用、成果登记与基金结题。
- 如需在个人主页或预印本平台传播，遵守本刊版权协议约定的自存档政策（**待核实**具体条款）。

## 十一、定稿阶段沟通要点

- 与编辑部沟通一律经官方渠道（投稿系统/编辑部邮箱），附稿号，一次说清一件事。
- 校样返回务必在期限内；如需延时，提前说明并给出可返回时间。
- 对编辑加工中改动了原意的地方，礼貌指出并给出正确表述，不要默认接受。
- 所有作者对最终版署名、单位、致谢知情并同意，避免出版后署名争议。

## 输出格式（定稿交付确认）

```
【《电子学报》(AES) 定稿/校样确认】
稿号：<XXXX>  拟刊卷期：<20XX 年第 X 期>
版权协议：<已签>  版面费：<已缴 / 待缴，金额待核实>
揭盲回填：<作者/基金/致谢/简介 已补全>
中英文部件：<题名/摘要/关键词/中图分类号 已校>
参考文献：<GB/T 7714 已终校，N 条无断号>
校样：<一校已返 / 二校待返>
待核实：<返校期限 / 版面费金额 / 出版时序>
```

按此清单逐项签字确认，可确保稿件在《电子学报》(Acta Electronica Sinica, AES) 正式出版时
零排版差错、零信息缺失。
