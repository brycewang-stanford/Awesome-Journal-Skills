# 《电子学报》审稿流程全解

理解《电子学报》(Acta Electronica Sinica, AES) 的审理链路，能帮你在漫长的评审等待中做出正确判断。
本刊由中国电子学会 (Chinese Institute of Electronics) 主办、为月刊，采用中文学报典型的**多级审理制**
并实行**双盲评审**。下面把从投稿到录用的每一环、每个决策点讲清楚。以下描述综合官网 (ejournal.org.cn)
渲染与中文学报通行流程；本刊未逐条公开的细节标 **待核实**，以投稿系统内稿件状态与编辑部说明为准
（核验日 2026-07-09）。

> 心态：学报审稿比会议慢、比会议深。Acta / Electronica / Sinica 的每一轮都在做实质学术判断，
> 耐心、按节奏配合，比频繁催稿更有效。

## 一、审理链路总览

```
投稿上传 → 编辑部受理/形式审查(初审)
        → 双盲外审(2 位及以上同行专家)
        → 复审(编辑/栏目/副主编汇总意见)
        → 退修 ↔ 作者修回(可能多轮)
        → 主编/编委会终审
        → 录用 → 编辑加工/校样 → 出版
```

## 二、各阶段详解

| 阶段 | 谁在做 | 可能结果 |
|---|---|---|
| 初审（形式/学术门槛） | 编辑部/栏目编辑 | 送外审 / 直接退稿(不对口、形式不合、不端) |
| 外审（双盲同行评议） | ≥2 位领域专家 | 录用 / 小修 / 大修(退修) / 退稿 |
| 复审 | 编辑/副主编汇总 | 汇总专家意见形成处理建议 |
| 修回与再审 | 作者↔专家 | 一轮或多轮退修再审 |
| 终审 | 主编/编委会 | 最终录用 / 退稿 |
| 编辑加工 | 编辑部 | 校样、定稿、排期出版 |

## 三、关键时限与红线

- **6 个月红线**：一般情况下，若投稿 6 个月后仍未收到稿件刊用与否的答复，作者有权自行处理稿件。
  这是本刊公开的重要时限承诺，也提醒作者**勿一稿多投**。
- 外审与修回的具体周期以稿件状态为准（**待核实**具体天数）；月刊排期意味着录用到见刊还有出版周期。
- 修回截止时间在退修信中给定，逾期可能按退稿或重新投稿处理，务必按时或提前申请延期。

## 四、如何解读稿件状态

- "编辑部收稿/受理"：进入初审队列。
- "外审中/审理中"：已送同行专家，耐心等待，此阶段频繁催稿收效甚微。
- "退修/修改后再审"：实质进展，按 `aes-author-response` 认真修回。
- "复审/主编审"：接近终审，通常无需作者动作。
- "录用/待发表"：进入 `aes-camera-ready` 流程。
- "退稿"：阅读意见，判断是改投兄弟刊还是大改后另投。

## 五、礼貌催稿的时机与方式

- 在**明显超出常规周期**（结合 6 个月红线）后再礼貌询问，避免刚送审就催。
- 通过投稿系统站内信或编辑部邮箱，一次询问一件事，语气客气、附稿号。
- 不要向审稿人施压（双盲下你也不知道是谁）；一切沟通经编辑部。

## 六、与英文姊妹刊 CJE 的分流

- 《电子学报》(中文) 与英文姊妹刊 **Chinese Journal of Electronics (CJE)** 同由中国电子学会主办；
  CJE 为英文期刊、被 SCI 收录，创刊于 1992 年。
- 若目标是国际英文读者与 SCI 检索，可考虑改投或另投 CJE；若面向中文学术共同体、追求中文旗舰
  学报影响力，则投《电子学报》。二者定位不同，勿同一稿件同时投递（属一稿多投）。

## 七、决策点应对策略

| 收到 | 应对 |
|---|---|
| 直接退稿(不对口) | 依意见改投更对口的兄弟刊或 CJE |
| 大修/退修 | 逐条认真修回，补实验、补文献 |
| 小修 | 快速精修，勿引入新问题 |
| 矛盾意见 | 礼貌说明取舍，请编辑裁量 |
| 长期无进展 | 结合 6 个月红线礼貌询问 |

## 八、流程认知自检

- [ ] 我知道稿件当前处于哪一审级。
- [ ] 我理解每个决策点的可能结果。
- [ ] 我掌握 6 个月红线与修回截止时间。
- [ ] 我不会一稿多投（含 CJE）。
- [ ] 我按状态而非焦虑决定是否催稿。

## 九、外审意见的类型与含义

读懂外审给出的整体判定，有助于预判走向并准备回应：

| 判定 | 含义 | 走向 |
|---|---|---|
| 录用/小修 | 认可创新与质量，仅细节 | 很可能录用 |
| 大修/退修 | 有价值但需补强 | 认真修回可争取录用 |
| 退稿(可改投) | 本刊不合适或问题较大 | 改投或大改另投 |
| 退稿(根本缺陷) | 创新性/正确性硬伤 | 需重新审视选题与方法 |

不同专家判定不一致时，编辑的复审与主编终审会做综合权衡，作者应针对分歧点重点回应。

## 十、审理期的作者纪律

- **一稿一投**：投稿期间不得将同一稿件投向其他期刊（含英文姊妹刊 CJE），否则违反学术规范。
- **保持畅通**：确保通信作者邮箱与系统站内信畅通，避免错过退修通知与截止时间。
- **耐心配合**：学报审理以质量为先，节奏偏慢属正常；把精力放在提升稿件而非频繁催问。

## 十一、审理流程认知小结

- 学报走多级审理：初审把形式与门槛，外审做同行评议，复审汇总，终审定夺。
- 6 个月是本刊公开的答复红线，也是作者一稿一投的时间承诺。
- 状态"外审中"时耐心等待最有效；"退修"是实质进展，全力修回。
- 中文《电子学报》与英文姊妹刊 CJE 定位不同，按读者与检索需求择一投递。
