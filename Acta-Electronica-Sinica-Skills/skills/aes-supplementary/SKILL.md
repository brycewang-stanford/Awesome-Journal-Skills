# 《电子学报》附录与补充材料

《电子学报》(Acta Electronica Sinica, AES) 作为中国电子学会 (Chinese Institute of Electronics) 主办的
月刊学报，正文版面有限而学术完整性要求高，二者的张力靠**附录/补充材料**来化解：正文承载主线与
核心证据，附录承载完整推导、扩展实验与可复现细节。本技能帮你判断"什么进正文、什么下沉"，并规范
补充材料的组织与提交。以下要点对齐中文科技期刊通行做法；本刊对在线补充材料的具体接收形式标 **待核实**，
以投稿系统与编辑部说明为准（核验日 2026-07-09，官网 ejournal.org.cn）。

> 判断原则：正文回答"为什么可信、贡献是什么"；附录回答"细节如何、如何重复"。凡是读者理解主线
> 不需要、但复现或深究时需要的内容，都属于 Acta / Electronica / Sinica 稿件的补充材料范畴。

## 一、正文 vs 附录的分工

| 内容 | 归属 |
|---|---|
| 核心方法、主结果图表、关键对比 | 正文 |
| 完整定理证明 / 冗长推导 | 附录 |
| 全部超参数表、额外数据集结果 | 附录/补充材料 |
| 扩展消融、参数敏感性完整曲线 | 附录/补充材料 |
| 信号波形、音视频、动态演示 | 多媒体补充附件 |
| 大规模原始数据表 | 在线补充/外链 |

## 二、补充材料的组织结构

```
附录 A  完整推导与证明
   A.1 定理 1 的证明
   A.2 式 (12) 的详细推导
附录 B  实验补充
   B.1 完整超参数设置表
   B.2 附加数据集上的结果
   B.3 参数敏感性完整曲线
附录 C  数据与制品说明
   C.1 数据采集参数与预处理细节
   C.2 代码/数据托管位置（录用后署名）
```

- 附录编号与正文引用一一对应；正文在需要处写"详见附录 A.1 / 补充材料 B.2"。
- 补充材料不是正文的垃圾桶：仍需有小标题、图表编号与说明，能被独立读懂。

## 三、公式推导与证明的呈现

- 正文给出结论式与关键中间步骤，完整逐步推导放附录。
- 证明写清前提、引理依赖、每步依据；符号与正文严格一致。
- 长证明分小节、给编号，方便审稿人定位核对。

## 四、扩展实验与数据表

- 把因篇幅无法进正文、但支撑结论稳健性的实验放附录：更多数据集、更多基线、完整敏感性分析。
- 大数据表用附录表格或在线补充；超长表可给外链 + 校验和。
- 附录实验须与正文口径一致（数据、指标、参数），避免自相矛盾。

## 五、多媒体与信号类附件

- 音视频、信号波形动画、交互演示等以本刊接受的**多媒体附件**形式提交（形式与大小限制 **待核实**）。
- 每个多媒体附件配一段文字说明：内容、时长/规模、对应正文哪一处结论。
- 确保附件可脱离正文被理解，命名规范、格式通用（如 mp4/wav/csv）。

## 六、双盲阶段的匿名化

- 补充材料同样需**匿名**：附录致谢、代码链接、数据出处不得泄露作者/单位。
- 多媒体附件的元数据、文件属性、水印中清除身份信息。
- 录用后再在附录 C 补上署名的正式仓库与数据地址（见 `aes-camera-ready`）。

## 七、与正文的一致性核对

- [ ] 每处"详见附录/补充材料"都指向真实存在、编号正确的位置。
- [ ] 附录图表编号在全篇内唯一、连续，不与正文冲突。
- [ ] 附录数据与正文报告的数值口径一致。
- [ ] 补充材料可独立读懂，非碎片堆砌。

## 八、补充材料自检清单

- [ ] 正文/附录分工合理，主线在正文清晰可读。
- [ ] 完整证明、超参数表、扩展实验已下沉附录。
- [ ] 多媒体附件格式通用、配文字说明、对应正文结论。
- [ ] 双盲期匿名化完成，元数据无泄露。
- [ ] 所有交叉引用与编号核对无误。
- [ ] 大数据/大附件用外链+校验和，符合本刊接收形式（待核实）。

## 九、下沉决策的实操判据

面对"这段该进正文还是附录"的纠结，可用三个判据快速决定：

| 判据 | 进正文 | 下沉附录 |
|---|---|---|
| 是否影响理解主线 | 是 | 否 |
| 是否核心证据 | 是 | 否（属佐证/完整性） |
| 篇幅是否挤占主线 | 精炼保留 | 冗长则下沉 |

一条经验：正文让审稿人"信服贡献"，附录让审稿人"核查细节"。若删掉某段后主线仍完整、结论仍成立，
它多半属于补充材料。

## 十、补充材料的可引用性

- 附录同样是正式出版内容的一部分，需保证可被独立引用与检索（编号、标题规范）。
- 在线补充材料若单独存放，确认其在出版后有稳定访问入口（**待核实**本刊具体承载方式）。
- 多媒体与大数据外链要长期有效，配校验和，避免"链接已失效"损害论文可信度与复现价值。

## 输出格式（补充材料清单）

```
【《电子学报》(AES) 附录/补充材料清单】
正文保留：<核心方法+主结果，主线清晰>
附录 A：<证明/推导 条目>
附录 B：<扩展实验/超参数表>
附录 C：<数据与制品说明>
多媒体附件：<列表 + 对应正文位置>
双盲匿名：<通过 / 存在泄露点>
交叉引用核对：<全部指向正确>
待核实：<本刊多媒体附件形式与大小限制>
```

用附录与补充材料把《电子学报》(Acta Electronica Sinica, AES) 正文的篇幅压力与学术完整性同时
照顾好，是让审稿人既读得顺、又查得实的关键安排。
