# 《电子学报》投稿就绪审计

在把稿件上传到《电子学报》(Acta Electronica Sinica, AES) 在线投稿系统之前，请先完整跑完这份
审计。《电子学报》是由中国科学技术协会主管、**中国电子学会 (Chinese Institute of Electronics, CIE)**
主办、创刊于 1962 年的电子与信息科学**综合性**中文旗舰月刊 (ISSN 0372-2112，CN 11-2087/TN)。
它以高水平原创学术论文为主，按学报稿件走多级审理，因此投稿被当作一篇要经受编辑初审、专家外审、
复审与主编终审的完整论文来评判。以下事实于 2026-07-09 通过官网 (ejournal.org.cn)、CNKI/万方
期刊主页与 CCF 推荐中文科技期刊目录交叉核验（见 `resources/official-source-map.md`）；凡本刊未
公开确认的数字均标注 **待核实**，请以投稿系统内最新《投稿须知》为准。

> 一句话定位：这是 electronics 与 information science 学科的中文旗舰，Acta / Electronica / Sinica
> 三个英文词对应的正是"电子·学报"，投稿前请对号入座到本刊学科侧重，而非泛泛的计算机或控制类刊。

## 一、投稿系统与账号

- 本刊**只接受网上投稿**，投稿入口为官网 `https://www.ejournal.org.cn/`；请勿邮寄纸稿或以邮件正文投稿。
- 首次投稿需在系统内**注册作者账号**，如实填写姓名、单位、通信作者邮箱与研究方向；这些信息决定
  编辑分派外审专家的匹配度，务必用**真实终稿信息**，不要用占位标题或临时邮箱。
- 注册与稿件上传建议安排在同一工作时段完成，避免账号信息与稿件信息脱节。
- 稿件上传后一般会在数日内收到编辑部对稿件是否符合基本受理条件的初步反馈。

## 二、稿件类型与栏目归属

《电子学报》接收的稿件通常包括：

| 稿件类型 | 说明 |
|---|---|
| 学术论文 | 以电子与信息科学为主体、有创新性的原创研究长文，为本刊主体 |
| 技术报告 | 有科学依据和可靠数据的工程/系统性成果 |
| 阶段性成果报告 | 前沿方向的初步但扎实的结果 |
| 综述/展望评论 | 属前沿学科、对学科发展有指导意义的评论性文稿（多为约稿或高资历作者） |

学科侧重覆盖：电子科学与技术、信息与通信工程、信号与信息处理、电路与系统、电磁场与微波、
集成电路、物理电子学等。投稿前请判断稿件是否落在这些方向内——若属纯软件工程或纯自动控制，
应改投《软件学报》《自动化学报》等更对口的兄弟刊。

## 三、中文科技论文的硬性组件

《电子学报》稿件必须齐备以下中文科技期刊标准部件（详见 `aes-writing-style`）：

| 组件 | 要求 |
|---|---|
| 中文题名 + 英文题名 | 英文题名与中文含义一致、可独立检索，实词首字母规范 |
| 中文摘要 + 英文摘要 (Abstract) | 结构完整、可独立成篇；英文摘要信息量应与中文对应 |
| 中文关键词 + 英文 Keywords | 一般 3–8 个，取自规范术语 |
| 中图分类号 | 电子信息类多为 TN 大类，据主题细分 |
| 文献标识码 / DOI | 由编辑部编排 |
| 基金项目 | 注明项目名称与编号（双盲阶段见下条处理） |
| 作者简介 | 姓名、出生年、职称、研究方向、通信作者邮箱 |
| 参考文献 | 按 **GB/T 7714** 顺序编码制著录，中英文著录规范一致 |

## 四、双盲评审与匿名化

本刊《投稿须知》渲染显示采用**双盲评审**：投稿上传的稿件正文及附件中**请勿出现作者姓名、单位、
基金编号等可识别信息**（双盲的具体执行以系统最新说明为准，标注 **待核实**）。匿名化清单：

- [ ] 正文、页眉页脚、致谢中删除作者与单位名称。
- [ ] 基金项目在盲审版中以"某基金资助"占位，录用后再补全编号。
- [ ] 自引改为第三人称（"文献[x]提出"而非"我们此前提出"）。
- [ ] 文件属性 (PDF/Word Metadata) 清除作者名。
- [ ] 图表、致谢、代码仓库链接中不泄露单位或个人身份。

## 五、格式与篇幅

- 使用本刊**官方模板**（通常提供 Word 与 LaTeX 两种）。中文正文按 A4 幅面排版，提交 PDF，
  必要时附源文件与图件源文件。
- 《电子学报》为一级学报，篇幅以把方法、推导、系统设计与充分实验完整讲透为准；具体页数/版面
  上限以模板与编辑部要求为准（**待核实**具体数值）。
- 图表须清晰、可黑白打印可辨；公式用公式编辑器排版并连续编号；量和单位遵循国家法定计量单位。

## 六、费用与时限

- 投稿时通常需缴纳**稿件评审费**：渲染显示中文稿约 **199 元/篇**、英文稿约 **260 元/篇**
  （**待核实**，以投稿系统内公示为准）；版面费/发表费标准另行公示（**待核实**）。
- 时限红线：一般情况下若 **6 个月**后仍未获稿件刊用与否的答复，作者有权自行处理稿件。请据此
  规划一稿一投，勿一稿多投。

## 七、伦理与学术规范

- 稿件不得涉及国家机密；涉及国家或单位机密内容须先经**脱敏/脱密**处理并附单位保密审查证明。
- 作者须保证拥有合法著作权、无一稿多投、无抄袭与不当署名；本刊使用学术不端检测系统。
- 数据须真实可靠，图像不得不当拼接或篡改；如涉及人类/动物实验或敏感数据须说明合规性。

## 八、提交前退稿风险分诊（逐项自检）

- [ ] 主题落在《电子学报》电子/信息学科范围内，而非计算机或控制类刊更对口的选题。
- [ ] 创新点在摘要首段一句话说清，且区别于英文姊妹刊 Chinese Journal of Electronics (CJE) 的定位。
- [ ] 中英文摘要、关键词、中图分类号、GB/T 7714 参考文献齐备且格式正确。
- [ ] 双盲匿名化清单全部勾选。
- [ ] 使用官方模板，图表公式规范，计量单位合规。
- [ ] 涉密审查、著作权与不端检测无问题。
- [ ] 评审费准备就绪，通信作者邮箱畅通。

## 九、常见形式退稿原因（编辑初审阶段）

- 未使用官方模板，或中英文摘要、关键词、中图分类号等标准部件缺失。
- 双盲版泄露作者/单位/基金编号，触发匿名化不合规退回。
- 参考文献著录混乱，中英文文献格式不统一，未按 GB/T 7714 顺序编码制。
- 主题与本刊学科不符（更适合计算机类或控制类兄弟刊）。
- 一稿多投或与作者已发表工作重复度过高，被不端检测拦截。
- 图表分辨率低、公式为截图、计量单位不规范。
- 未缴纳稿件评审费导致稿件未进入送审队列。

把以上七条逐一排除后，稿件基本能顺利越过《电子学报》的初审形式门槛，进入外审环节。

跑完本清单再上传，可显著降低《电子学报》(Acta Electronica Sinica, AES) 编辑初审阶段的形式退稿概率。
