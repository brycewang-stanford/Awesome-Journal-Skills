# 《电子学报》审稿意见答复与修回

收到《电子学报》(Acta Electronica Sinica, AES) 编辑部的**退修/修改后再审**通知，意味着稿件已越过
初审、进入实质性学术评价，这是好信号。本刊由中国电子学会 (Chinese Institute of Electronics) 主办、
走多级审理，退修通知通常附有**若干位外审专家**的分条意见和编辑的综合处理意见。你的任务是提交
两份东西：**修改后的稿件**与一份**逐条修改说明 (Response Letter)**。以下要点于 2026-07-09 经官网
(ejournal.org.cn) 与期刊主页交叉核验；凡未公开确认者标 **待核实**，以退修信与投稿系统为准。

> 核心原则：审稿人不是敌人。把每条意见当作提高论文质量、也提高录用概率的机会；用数据与文献回应，
> 而非用情绪或辩解回应。这是 Acta / Electronica / Sinica 学报评审文化里最看重的作者素养。

## 一、开工前的准备

- 通读退修信，区分三类意见：**必须修改**（否则退稿）、**建议修改**（可解释可采纳）、**可讨论**
  （有分歧、需要礼貌据理说明）。
- 建立一张主表：把每位专家的每条意见拆成独立条目并编号（如 R1-1、R1-2、R2-1）。
- 记录本刊给的**修回截止时间**与允许的**再审轮次**（**待核实**，通常以退修信为准）；如需延期，
  提前邮件向编辑部申请，不要逾期沉默。

## 二、修改说明的标准结构

```
尊敬的编辑与各位审稿专家：
  感谢各位对本文（稿号：XXXX）的宝贵意见。我们已逐条修改，
  以下"[意见]/[回应]/[修改位置]"三段式说明每处改动。正文中修改
  部分以蓝色/修订模式标出。

【审稿专家一】
[意见 R1-1] 原文……（原样引用专家意见）
[回应] 感谢指出。我们……（说明如何修改、依据是什么）
[修改位置] 见修改稿第 X 页第 Y 段 / 图 Z / 式 (n)。

[意见 R1-2] ……
...
【审稿专家二】
...
```

每一条都必须有回应，**不允许跳过**任何一条，即使是"表述建议"这类小意见。

## 三、逐条回应的写法

| 意见类型 | 推荐回应策略 |
|---|---|
| 指出方法/推导错误 | 承认并改正，给出更正后的公式或证明，说明影响范围 |
| 要求补充对比实验 | 尽量补做；列出新增表格/图，报告新结果 |
| 质疑创新性不足 | 用一段"贡献重述"澄清与已有工作（含 CJE 相关文献）的区别 |
| 要求补充文献 | 补引并在相关工作中讨论其与本文关系，而非机械堆砌引用 |
| 表述/语言问题 | 直接采纳并致谢，列出修改后的句子 |
| 相互矛盾的意见 | 礼貌指出分歧，说明本文选择哪条及理由，请编辑裁量 |
| 不便采纳的意见 | 提供数据/文献/篇幅依据，礼貌保留，绝不无视 |

## 四、正文改痕规范

- 提交两版：**改痕版**（修订模式或蓝色标注所有改动）+ **干净版**。
- 大段重写时，在修改说明中摘录关键新增段落，方便专家快速定位。
- 新增图表须编号连续，不要打乱原有编号造成对照困难。
- 若删减内容，说明删减理由（如冗余、超篇幅），避免专家误以为回避问题。

## 五、语气与措辞红线

- 全程使用"感谢""我们已……""如专家所建议"等**合作性措辞**。
- 禁止出现"审稿人误解了""这个问题不重要""众所周知"等对抗或轻慢表述。
- 即便坚持己见，也要先肯定意见价值，再用证据说明，最后把裁量权交回编辑。
- 中英文摘要、关键词若因修改而变动，同步更新，保持中英一致。

## 六、补做实验的取舍

- 优先补做**能直接回应质疑**的实验；无法在修回期内完成的大实验，说明现状并给出可行的替代证据
  （如小规模验证、理论分析或引用同类结果）。
- 新增实验须与原实验在数据集、指标、参数上口径一致，避免引入新的可比性问题（详见 `aes-experiments`）。
- 把新增结果清楚标注为"应审稿意见新增"，便于专家识别工作量。

## 七、修回自检清单

- [ ] 每条意见都有独立编号与三段式回应，无遗漏。
- [ ] 正文改痕版与干净版都已准备，改动可一一对应。
- [ ] 补做实验/文献已落到正文并在说明中指明位置。
- [ ] 措辞全程合作、专业、无对抗性表达。
- [ ] 中英文摘要、关键词、图表编号随修改同步更新。
- [ ] 双盲若仍在进行，回应信中不泄露作者身份。
- [ ] 在截止时间前提交，必要时已申请延期。

## 八、多轮再审的节奏把握

《电子学报》的退修常不止一轮。把握多轮节奏能避免"越修越乱"：

| 轮次 | 特征 | 策略 |
|---|---|---|
| 第一轮大修 | 意见多、涉及方法与实验 | 系统性重构，补关键实验，全面回应 |
| 第二轮 | 聚焦第一轮未彻底解决处 | 精准回应残留质疑，勿引入新改动 |
| 第三轮及以后 | 多为细节/表述 | 快速精修，稳定全文，避免反复 |

每轮都保留上一轮的修改说明，形成可追溯的答复档案；新一轮回应时可引用"如上一轮所述并进一步…"，
让编辑与专家看到工作的连续性与诚意。

## 九、被退稿后的理性选择

- 若终审退稿，先冷静阅读全部意见，区分"根本性缺陷"与"可补救不足"。
- 根本性问题（选题不对口、创新性不足）宜改投更合适的兄弟刊或英文姊妹刊 CJE，而非原样再投。
- 可补救的不足（实验不足、表述不清）可大改后另投层次匹配的期刊，把审稿意见当作免费的高质量评审。

## 输出格式（修回交付物清单）

```
【《电子学报》(AES) 修回交付物】
稿号：<XXXX>  轮次：<第 N 轮外审后修回>
修改说明：<逐条三段式，覆盖 R1/R2/... 全部意见>
改痕稿：<已备>  干净稿：<已备>
新增实验/文献：<列表>
矛盾意见处理：<如何裁量并请编辑定夺>
未采纳意见：<条目 + 保留理由>
时限：<截止 YYYY-MM-DD，已/未申请延期>
待核实：<允许再审轮次上限 / 延期政策>
```

一份条理清晰、态度诚恳、以证据说话的修改说明，是稿件在《电子学报》(Acta Electronica Sinica, AES)
从"退修"走向"录用"的关键一步，也是外审专家判断作者专业度的重要依据。
