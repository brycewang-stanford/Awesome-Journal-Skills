# 《电子学报》选题与栏目定位

选题决定成败的一半。一篇优秀但**不对口**的稿件，会在《电子学报》(Acta Electronica Sinica, AES)
初审阶段就被建议改投。本刊由中国电子学会 (Chinese Institute of Electronics) 主办，是电子与信息
科学的**综合性**中文旗舰月刊，学科边界清晰。本技能帮你在动笔前把选题放进本刊的正确坐标系。
以下学科范围综合官网 (ejournal.org.cn) 与期刊简介渲染；本刊具体栏目设置标 **待核实**，以最新
刊物目次为准（核验日 2026-07-09）。

> 一句话：先问"这属于电子信息学科吗、够不够本刊层次、和 CJE 怎么分"，再动笔。
> 这是把稿件对进 Acta / Electronica / Sinica 坐标的第一步。

## 一、本刊学科侧重

《电子学报》主要覆盖（与纯计算机、纯自动控制类刊相区别）：

| 学科方向 | 典型主题 |
|---|---|
| 电子科学与技术 | 物理电子、微电子、光电子器件 |
| 信息与通信工程 | 通信理论、编码、网络、无线传输 |
| 信号与信息处理 | 语音/图像/雷达信号处理、检测与估计 |
| 电路与系统 | 模拟/数字电路、系统设计与优化 |
| 电磁场与微波 | 天线、微波器件、电磁兼容 |
| 集成电路 | 芯片设计、EDA、SoC、FPGA/ASIC |

若稿件核心是软件工程、数据库、纯算法理论或自动控制系统，通常更适合《软件学报》《计算机学报》
《自动化学报》《通信学报》等兄弟刊。

## 二、创新性与层次自评

- 本刊为一级学报，选题需有**明确创新点**（新方法/新理论/新器件/新系统），而非增量复现或工程堆叠。
- 自问三问：解决了什么真问题？相比最强已有方法强在哪、强多少？能否公平且可复现地证明？
- 若创新点只是"换个数据集重跑"或"小幅调参"，选题层次可能不足以支撑本刊。

## 三、对口度决策树

```
主题是否属电子/信息/通信/信号/电路/电磁/集成电路？
  ├─ 否 → 改投对口兄弟刊（计算机/控制/软件类）
  └─ 是 → 创新点是否清晰且达一级学报层次？
           ├─ 否 → 继续打磨或投层次匹配的刊
           └─ 是 → 目标读者与检索需求？
                    ├─ 中文学术共同体/中文旗舰影响力 → 《电子学报》
                    └─ 国际英文读者/SCI 检索 → Chinese Journal of Electronics (CJE)
```

## 四、《电子学报》与英文姊妹刊 CJE 的选择

- 二者同由中国电子学会主办；**《电子学报》为中文月刊**，面向中文学术共同体、被 EI 等收录；
  **CJE 为英文期刊、被 SCI 收录**，面向国际读者。
- 面向中文读者、追求中文旗舰学报影响力与中文表达，选《电子学报》。
- 面向国际同行、需要 SCI 检索，选 CJE。同一稿件**不可**同时投两刊（属一稿多投）。

## 五、原创论文 vs 综述

| 类型 | 定位 |
|---|---|
| 学术论文（原创） | 本刊主体，需有创新方法/结果与充分实验 |
| 技术报告 | 有可靠数据的工程/系统性成果 |
| 综述/展望 | 属前沿方向、对学科有指导意义，需高视野与系统梳理，多为约稿或资深作者 |

新人作者一般从原创论文切入；综述需要对领域有全局把握，贸然投稿易因深度不足被退。

## 六、选题打磨动作

- 用 `aes-related-work` 的检索方法确认选题在本刊/CJE 及国际文献中的位置与空白。
- 用一句话写清创新点，能否通过"同行电梯测试"（30 秒讲清做了什么、比谁好）。
- 对照本刊近年目次，判断选题是否契合当前关注的方向与栏目。

## 七、选题自检清单

- [ ] 主题落在电子/信息/通信/信号/电路/电磁/集成电路学科内。
- [ ] 创新点清晰、达一级学报层次，非增量复现。
- [ ] 已判断投《电子学报》还是英文姊妹刊 CJE，且不一稿多投。
- [ ] 稿件类型（原创/技术报告/综述）定位准确。
- [ ] 选题在本刊/国际文献中的空白已确认。
- [ ] 一句话创新点能通过电梯测试。

## 八、选题风险类型与对策

选题阶段的风险可归为几类，早识别早止损：

| 风险类型 | 表现 | 对策 |
|---|---|---|
| 不对口 | 主题偏计算机/控制学科 | 改投兄弟刊，勿硬投本刊 |
| 层次不足 | 仅增量复现/小幅调参 | 深化创新或投层次匹配刊 |
| 撞题 | 与近期已发表工作高度重合 | 找差异化角度或提高增量 |
| 过窄 | 问题太小、普适性弱 | 扩展到更一般设定或多场景 |
| 过大 | 一篇讲不透多个问题 | 聚焦一个核心贡献 |

## 九、与本刊近期方向对齐

- 浏览《电子学报》近一两年目次，感知本刊当前活跃的方向与栏目偏好（**待核实**具体栏目名）。
- 关注中国电子学会及本刊发布的重点征稿/专题（如有），契合专题的对口稿件更易获得关注。
- 判断选题是"本刊读者关心的问题"还是"只有我关心的问题"——前者才有传播与引用价值。

选题对了，后面每一步都事半功倍；选题错了，写得再好也是投错门。

## 十、选题定位小结

- 三问定位：属电子信息学科吗？创新达一级学报层次吗？投中文本刊还是英文 CJE？
- 主题偏计算机/控制学科时果断改投兄弟刊，别硬投《电子学报》。
- 一句话讲清创新点并通过"电梯测试"，是选题成熟的信号。
- 对照本刊近期目次与专题，判断选题是否是本刊读者真正关心的问题。
