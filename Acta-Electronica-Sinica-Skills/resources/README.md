# 资源目录 · 《电子学报》(Acta Electronica Sinica, AES) 技能包

本目录收录支撑 12 个 `aes-*` 技能的参考资料。所有事实以
[official-source-map.md](./official-source-map.md) 为准，标注 **待核实** 者请以本刊官网
(ejournal.org.cn) 最新《投稿须知》为最终依据。

## 文件清单

| 文件 | 用途 |
|---|---|
| [official-source-map.md](./official-source-map.md) | 关键事实的来源、核实方法与待核实清单 |
| [exemplars/library.md](./exemplars/library.md) | 范例库：题名/摘要/参考文献/修改说明示意 |
| [external_tools.md](./external_tools.md) | 写作、检索、投稿、可复现辅助工具 |
| [worked-examples/01-introduction.md](./worked-examples/01-introduction.md) | 引言写作分步示例 |
| [code/README.md](./code/README.md) | 研究制品打包骨架说明 |

## 使用建议

1. 先读 `official-source-map.md`，了解哪些是已核实事实、哪些标 **待核实**。
2. 写作时对照 `exemplars/library.md` 与 `worked-examples/` 的示意。
3. 准备代码/数据时参考 `code/README.md` 的制品骨架。
4. 需要工具时查 `external_tools.md`。

本资源为独立第三方辅助材料，非《电子学报》(Acta Electronica Sinica, AES) 官方出品。
