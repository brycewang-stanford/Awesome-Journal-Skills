# 外部工具清单 · 《电子学报》(Acta Electronica Sinica, AES) 写作与投稿辅助

以下工具可辅助面向《电子学报》(Acta Electronica Sinica, AES) 的选题、写作、检索、投稿与可复现工作。
均为第三方工具，非本刊指定；使用时以本刊官网 (ejournal.org.cn) 最新《投稿须知》与官方模板为准。

## 一、文献检索

| 工具 | 用途 |
|---|---|
| CNKI（知网） | 中文文献主库，检索本刊及兄弟刊论文 |
| 万方 / 维普 | 中文文献补充检索 |
| IEEE Xplore | 英文电子/信息类文献与英文姊妹刊 CJE |
| dblp | 计算机/信息领域文献题录 |
| Google Scholar | 引用追踪与跨库补漏 |

配合 `aes-related-work` 的关键词矩阵法系统检索，降低漏引风险。

## 二、写作与排版

| 工具 | 用途 |
|---|---|
| 本刊官方 Word/LaTeX 模板 | 唯一正式排版依据（从投稿系统获取） |
| LaTeX + BibTeX/BibLaTeX | 公式与参考文献排版，配 GB/T 7714 样式 |
| 公式编辑器（LaTeX/MathType） | 公式排版，避免截图 |
| 三线表工具 | 规范表格 |

参考文献务必按 **GB/T 7714 顺序编码制**，中英文著录统一（见 `aes-writing-style`）。

## 三、可复现与制品

| 工具 | 用途 |
|---|---|
| Git + 匿名托管 | 双盲期匿名开放代码 |
| Docker / conda | 环境固化，锁定依赖版本 |
| 校验和工具(md5sum/sha256sum) | 大数据外链完整性校验 |
| 随机种子管理 | 固定并记录种子，多次运行 |

## 四、学术规范

| 工具 | 用途 |
|---|---|
| 学术不端检测系统 | 投稿前自查重复率 |
| ORCID | 作者唯一标识 |
| 单位保密审查 | 涉密内容脱敏/脱密证明 |

## 五、投稿与流程管理

- 本刊在线投稿系统（ejournal.org.cn）：注册、上传、缴评审费、查状态。
- 投稿档案目录（本地）：稿件各版本、修改说明、审稿意见、往来邮件、缴费凭证（见 `aes-workflow`）。

所有工具仅为辅助，最终格式与流程以《电子学报》(Acta Electronica Sinica, AES) 官方规定为准。
