# 《计算机辅助设计与图形学学报》相关工作与文献综述

本技能帮助为《计算机辅助设计与图形学学报》(Journal of Computer-Aided Design & Computer
Graphics, JCAD&CG) 撰写**delta 优先**的相关工作。本刊是 CCF A 类图形学/CAD 方向中文月刊，
外审专家多为图形学/几何/可视化领域同行，最看重你**与最接近工作的差异(delta)**是否清楚、
是否漏引关键工作(含本刊近年相关论文)。核验见 `resources/official-source-map.md`（2026-07-09）。

## 一、相关工作的目标：论证创新性，而非罗列

相关工作不是文献清单，而是为本文贡献**清场**：读者读完应确信"这个图形/几何问题此前没被
这样解决"。每一簇相关工作都要回答：它们做了什么、局限是什么、本文如何超越。

## 二、delta 优先的写法

1. 先定位**最接近的 2~3 个工作**（同问题/同表示/同评估），逐一说清其做法与局限。
2. 明确本文的 delta：是**新方法/新表示/新约束/新评估协议/新应用**中的哪一类？
3. 把 delta 与本文贡献分条对应，避免"我们也做了 X"式模糊表述。

> 反例："已有很多网格简化方法[1-20]，本文提出一种新方法。" 正例："QEM 类方法[3]以平均
> 误差最优为目标，对薄壳自穿透无约束；特征感知方法[7]开销高且未显式处理穿透——本文以
> 特征-穿透联合约束填补该空缺。"

## 三、按图形学贡献类型分类梳理

| 贡献类型 | 相关工作应覆盖的簇 |
|---|---|
| 几何处理/建模 | 网格/点云表示、重建、简化、参数化的代表方法与其精度/鲁棒性局限 |
| 渲染/图像合成 | 传统渲染、神经渲染、真实感/时序稳定性方法及其开销 |
| CAD/CAE/CAM | 参数化建模、约束求解、仿真、拓扑优化、一体化流程 |
| 可视化/可视分析 | 任务分类、视觉编码、交互与可扩展性工作 |
| VR/AR/MR/交互 | 沉浸感、延迟、舒适度、交互范式相关研究 |
| 图像视频处理 | 处理/增强/生成方法及数据集与评测 |

按"方法簇"而非"逐篇"组织，每簇末尾点出共同局限，引出本文。

## 四、文献选择与引用规范

- 优先引**本领域高质量文献**与**本刊(JCAD&CG)近年相关工作**，体现对期刊社群的了解；漏引
  本刊明显相关工作易引起外审反感。
- 兼顾中英文文献；中文文献按 GB/T 7714 给**中英文对照**。
- 引用要准确：不歪曲他人方法、不把弱基线当作全部现状。
- 会议(CAD/CG、ChinaGraph、SIGGRAPH、CVPR 等)与期刊文献均可引，标注文献类型([C]/[J])。

## 五、与实验/写作技能的衔接

- 相关工作中点名的"最强基线"必须在 `jcadcg-experiments` 中真的比较，口径一致。
- delta 陈述与摘要/引言中的贡献分条(`jcadcg-writing-style`)一一对应。
- 若综述型稿件，需升级为系统分类框架 + 批判性比较 + 趋势展望（见 `jcadcg-topic-selection`
  对综述的要求）。

## 六、自查清单

- [ ] 找到并逐一区分最接近的 2~3 个工作，delta 明确。
- [ ] 按方法簇组织，每簇点出共同局限。
- [ ] 覆盖本文涉及的所有图形/几何子方向，无明显漏引。
- [ ] 引了本刊近年相关工作。
- [ ] 中文文献 GB/T 7714 中英文对照；文献类型标识齐全。
- [ ] 相关工作点名的基线在实验中真的比较、口径一致。
- [ ] 不歪曲、不树稻草人、不把预印本当已证实结论。

## 七、常见问题对照

| 问题 | 后果 | 修正 |
|---|---|---|
| 文献罗列无 delta | 创新性存疑 | 改 delta 优先、每簇点局限 |
| 漏引本刊/领域关键工作 | 外审反感 | 补检 CNKI/本刊近年目录 |
| 基线选弱、回避强工作 | 被质疑不公平 | 纳入最强可得基线 |
| 中文文献缺英文对照 | 不符 GB/T 7714 | 逐条补齐 |
| 把相关工作写成背景介绍 | 未清场 | 聚焦"他人局限→本文超越" |

## 八、检索与查全的方法

图形学/CAD 领域文献分散在期刊与顶会，漏引强相关工作是外审失分点。建议多源检索并交叉：

- **中文源**：CNKI/万方检索本刊(JCAD&CG)、《计算机学报》、《软件学报》、《中国图象图形
  学报》近年相关论文。
- **英文源**：图形学/视觉顶会与期刊(SIGGRAPH/TOG、Eurographics、CVPR/ICCV/ECCV、
  IEEE TVCG、CAD/CAGD 等)。
- **顺链检索**：从最接近工作的引文与被引前后追溯，补齐方法簇。
- **本刊自查**：确认引用了本刊近年同方向工作，体现对期刊社群的了解。

## 九、相关工作的篇幅与位置

- 本刊论文相关工作可独立成节，也可在引言中扼要交代、方法节再展开对比——依贡献密度定。
- 无论何种组织，**最接近工作的 delta 必须在读者遇到方法前讲清**，避免读完方法才知道创新点。
- 综述稿的"相关工作"升级为**系统分类框架**：给出维度、分类树、代表方法归位与批判性比较
  (见 `jcadcg-topic-selection` 对综述的要求)。

## 十、引用诚信与常见雷区

| 雷区 | 说明 | 规避 |
|---|---|---|
| 树稻草人 | 只比弱基线、回避强工作 | 纳入最强可得工作并公平陈述 |
| 曲解他人方法 | 简化或误述以凸显本文 | 准确复述，必要时引原文表述 |
| 预印本当定论 | 把未评审结果当已证实 | 标注来源性质，谨慎引用 |
| 过度自引 | 堆砌自己论文充数 | 自引须真相关，避免刷引用 |
| 漏引本刊/领域基石 | 显得不了解领域 | 多源检索 + 本刊自查 |

## 十一、与后续技能的衔接

- 相关工作确立的 delta 与基线清单 → 直接进入 `jcadcg-experiments` 逐项比较、口径一致。
- delta 陈述回填摘要/引言贡献分条 → `jcadcg-writing-style`。
- 若发现选题与最接近工作差异不足，回到 `jcadcg-topic-selection` 重估创新性与落点。

## 十二、输出格式

```text
[JCAD&CG 相关工作] 就绪 / 需加强
[最接近工作] W1:__(局限__)；W2:__(局限__)；W3:__(局限__)
[本文 delta] 属于：新方法/新表示/新约束/新评估/新应用（___）
[方法簇覆盖] 几何/渲染/CAD/可视化/VR/图像视频（是否有漏引：___）
[本刊工作] 是否引本刊近年相关论文？（是/否）
[GB/T 7714] 中文文献中英文对照、文献类型标识齐全？（是/否）
[待办] <逐条列出需补的簇/文献/delta>
```
