# 《计算机辅助设计与图形学学报》选题与栏目定位

在为《计算机辅助设计与图形学学报》(Journal of Computer-Aided Design & Computer Graphics,
JCAD&CG) 动笔前，先用本技能确认选题落在本刊版图。本刊是 CCF A 类**图形学/CAD 方向专门刊**，
Journal of Computer-Aided Design & Computer Graphics 的英文刊名已点明两大支柱：计算机辅助
设计与计算机图形学。选错落点是初审"不符宗旨"退稿的首要原因，本技能帮你在投入写作前做出
判断。事实核验见 `resources/official-source-map.md`（核验日期 2026-07-09）。

## 一、本刊的版图（栏目边界）

据官网/中科院计算所页/CNKI，本刊栏目聚焦以下图形/几何方向：

| 栏目/方向 | 典型主题 |
|---|---|
| 图像视频处理 | 基于图像视频的图形技术、图像视频处理、计算摄像、AIGC(生成式图像视频) |
| 可视化与可视分析 | 信息可视化、科学可视化、可视分析(visual analytics) |
| CAD/CAE/CAM | 计算机辅助几何设计、仿真分析、智能制造、拓扑优化、设计/分析/制造一体化 |
| VR/AR/MR | 虚拟现实、增强现实、混合现实、沉浸式交互 |
| 三维图形计算 | 几何建模、网格/点云处理、渲染、动画、物理仿真 |
| 交互技术 | 人机交互、三维交互、手势/触觉、可用性 |

> 具体栏目名称与当期征稿重点、专题(special issue)是否开放，以官网当期《投稿须知》为准
> （**待核实**）。

## 二、契合度自检（先问三问）

1. **图形/几何内核**：我的核心贡献是否关于图形、几何、渲染、可视化、CAD、VR 或图像视频？
   若内核是纯算法理论、系统软件、网络或控制，本刊未必最对口。
2. **原创增量**：与最接近的已有工作相比，我的图形/几何增量是什么？（新方法、新表示、新
   评估、新应用场景？）——见 `jcadcg-related-work` 的 delta 优先。
3. **证据可达**：我能否给出本刊看重的图形证据（渲染对比、几何误差、可视化有效性、用户
   研究）？若只有榜单分数而无视觉/几何证据，需补齐。

## 三、与兄弟中文刊的分工

| 期刊 | 侧重 | 何时更对口 |
|---|---|---|
| **《计算机辅助设计与图形学学报》(JCAD&CG)** | **图形学/CAD/可视化/VR/图像视频** | 贡献内核是图形、几何、渲染、可视化、CAD、VR |
| 《计算机学报》(Chinese Journal of Computers) | 计算机全学科综合 | 跨子领域普遍意义、非专图形方向 |
| 《软件学报》 | 软件工程/系统软件/程序语言 | 贡献是软件方法、系统、工具链 |
| 《自动化学报》 | 控制、自动化、部分模式识别 | 贡献是控制/优化/机器人本体 |
| 《电子学报》 | 电子、通信、信号 | 贡献是电路、通信、信号处理 |
| 《中国图象图形学报》 | 图像图形（有交叠） | 偏图像分析/识别而非图形生成/CAD 时需权衡 |

> 若你的工作横跨图形与图像识别，需判断**内核**更偏"图形/几何生成与设计"(本刊)还是"图像
> 理解/识别"(可能更适合图像图形方向刊)。

## 四、稿件类型选择

本刊接收研究论文与综述等类型：

- **研究论文**：以一个清晰的图形/几何问题为核心，方法 + 充分实验(渲染/几何证据) + 消融。
- **综述**：需系统的分类框架、批判性比较与趋势展望；综述稿通过初审后一般由 3~4 名专家
  审阅（经验值），对系统性与全面性要求高。选综述前先确认你能提供领域级的批判性梳理而非
  文献罗列。

## 五、选题打磨清单

- [ ] 一句话能说清"针对什么图形/几何问题、提出什么、比现有好在哪(可量化)"。
- [ ] 贡献可分条，且每条能绑定到图形证据（渲染/几何误差/用户研究）。
- [ ] 与最接近的 2~3 个工作的 delta 明确，不是"换数据集重跑"。
- [ ] 选定栏目（图像视频/可视化/CAD-CAE-CAM/VR-AR/三维图形/交互）。
- [ ] 若为会议(CAD/CG、ChinaGraph 等)扩展版，能说清实质新增。

## 六、常见选题误区

| 误区 | 问题 | 修正 |
|---|---|---|
| 把纯识别/分类工作投本刊 | 内核偏图像理解而非图形/几何 | 强化图形/几何生成/设计内核或改投 |
| 只有榜单分数、无视觉证据 | 不符本刊图形证据取向 | 补渲染对比、几何误差色图、用户研究 |
| 增量是"新数据集重跑" | 原创性不足 | 提炼方法/表示/评估层面的真增量 |
| 选题过泛(如"深度学习综述") | 无本刊图形聚焦 | 收敛到图形/CAD 子方向的批判性综述 |
| 蹭热点(如泛泛 AIGC)无图形内核 | 与本刊定位错位 | 落到图像视频/渲染的具体图形问题 |

## 七、与其他技能的衔接

- 契合判断通过后 → `jcadcg-writing-style` 搭首页范式与体例。
- 增量论证 → `jcadcg-related-work`；图形证据规划 → `jcadcg-experiments`。
- 全流程时间线 → `jcadcg-workflow`；投稿前自审 → `jcadcg-submission`。

## 八、从"热点"到"可发表选题"的收敛

热点(如 AIGC、神经渲染、3D 生成)本身不是选题，需收敛到本刊可承载的**具体图形/几何问题**：

| 泛热点 | 收敛为本刊选题(示意) |
|---|---|
| AIGC | 面向可控编辑的图像/视频生成的几何一致性问题 |
| 神经渲染 | 新视角合成中的时序稳定性或几何约束问题 |
| 3D 生成 | 生成网格的拓扑正确性与可制造性问题 |
| 数字孪生 | 大规模 CAD 模型的轻量化与实时可视化问题 |
| 元宇宙/VR | 沉浸式交互的延迟-舒适度权衡问题 |

收敛判据：能否给出**可量化的图形/几何目标**与**公平可比的评测**。

## 九、选题的风险自评

- **创新性风险**：与最接近工作 delta 不足 → 高退稿风险；先补 `jcadcg-related-work`。
- **证据风险**：无法提供渲染/几何/用户研究证据 → 补实验设计 `jcadcg-experiments`。
- **契合风险**：内核偏识别/系统/控制 → 评估改投兄弟刊。
- **规范风险**：会议扩展版增量不足或未声明 → 明确增量再投。

## 十、投稿前的选题终检

- [ ] 一句话讲清"针对什么图形/几何问题、提出什么、量化好在哪"。
- [ ] 内核明确落在某一栏目(图像视频/可视化/CAD-CAE-CAM/VR-AR/三维图形/交互)。
- [ ] 与最接近 2~3 个工作的 delta 成立且可支撑。
- [ ] 计划的图形证据类型明确、可获取。
- [ ] 若为会议扩展版，实质新增可说清。
- [ ] 已确认本刊比兄弟刊更对口(或反之)。
