# 《计算机辅助设计与图形学学报》录用后定稿与校样

本技能帮助把在《计算机辅助设计与图形学学报》(Journal of Computer-Aided Design & Computer
Graphics, JCAD&CG) 已录用的稿件推进到见刊。本刊是 CCF A 类图形学/CAD 方向中文月刊。录用后的
定稿/校样阶段易在细节上出错，本技能给出核对清单。核验见 `resources/official-source-map.md`
（2026-07-09）；具体版权协议、校样轮次、出版费用等**待核实**，以编辑部通知为准。

## 一、录用后第一步：解除匿名

外审阶段为双盲，录用后应补回真实信息：

- **致谢**：感谢资助、协助者(据实)。
- **基金项目**：补全项目名称与编号。
- **作者简介**：姓名、单位、研究方向、通信作者信息。
- **代码/数据真实链接**：把匿名托管换成正式链接(仓库/归档 DOI)；确保长期可访问。
- **演示视频/附件**：换正式托管、去匿名占位。

## 二、定稿与排版

- 按编辑部要求提供定稿源文件(Word/LaTeX)与图源；确认与录用版本一致，不夹带未经审阅的新
  内容。
- 核对六件套：中英文题名、中英文摘要、关键词、中图分类号(TP)、基金/作者简介、GB/T 7714
  参考文献。
- 图题表题中英文、量与单位法定、公式编号连续。

## 三、图形结果的印刷质量（本刊尤重）

- **分辨率**：渲染图/几何图保证印刷分辨率(位图足够 DPI，线条图用矢量)。
- **色彩**：注意屏幕(RGB)与印刷(可能 CMYK)的色差；几何误差色图的色标在印刷下仍可辨；
  确认黑白印刷时图表(线型/纹理)仍可区分。
- **图内文字**：缩放后清晰、字号足够；避免过小标注。

## 四、校样(清样)校对

- 收到校样后**限期返回**；重点核对：作者姓名/单位/顺序、公式、图表编号与引用、参考文献、
  数字与单位、渲染图是否排版走样。
- 校样阶段一般**只做勘误**，不做实质性内容改动(如需重大改动须与编辑部沟通)。
- 逐字冷读一遍，尤其中英文摘要、图表标题、量与单位。

## 五、版权与出版

- 按要求签署**版权/出版许可协议**(具体条款**待核实**)。
- 确认出版相关费用与流程(**待核实**，以编辑部通知为准)。
- 确认在线优先出版/见刊时间与卷期信息。

## 六、见刊前最后一轮核对清单

- [ ] 已解除匿名：致谢/基金/作者简介/真实链接补齐。
- [ ] 定稿与录用版本一致，无夹带新内容。
- [ ] 六件套齐全、GB/T 7714 规范、中文文献中英对照。
- [ ] 渲染/几何图印刷分辨率与色彩达标、黑白可辨。
- [ ] 校样限期返回、只做勘误、逐字冷读。
- [ ] 版权协议已签、出版费用/流程已确认(按通知)。
- [ ] 代码/数据/视频正式链接长期可访问。

## 七、常见校样错误对照

| 错误 | 后果 | 规避 |
|---|---|---|
| 忘补真实致谢/基金 | 见刊信息缺失 | 录用即补解除匿名 |
| 定稿夹带新内容 | 违反流程 | 只交录用版本 |
| 渲染图印刷走样/色差 | 图形质量打折 | 提供高分辨率/矢量、校色 |
| 校样逾期 | 延误见刊 | 限期返回 |
| 匿名链接未换 | 读者打不开 | 换正式长期链接 |
| 校样阶段大改 | 编辑部拒改 | 重大改动提前沟通 |

## 八、图形结果印刷适配的细节

图形/CAD 论文的图是论文的门面，定稿阶段务必逐项检查印刷适配：

| 项目 | 检查点 |
|---|---|
| 分辨率 | 位图渲染图达到印刷 DPI 要求；线条图/示意图尽量矢量(避免锯齿) |
| 色彩空间 | 屏幕 RGB 与印刷可能的 CMYK 存在色差；关键对比色提前校色 |
| 色标可读 | 几何误差色图的 colormap 在印刷与黑白复印下仍可区分，标注数值范围与单位 |
| 图内文字 | 缩放到版面尺寸后字号足够、清晰；坐标轴与图例不糊 |
| 对比一致 | 本方法/基线/GT 并排图保持统一视角、光照、裁剪与比例 |
| 放大框 | 关键差异区域的放大框位置、边框颜色在印刷下清晰 |

## 九、演示视频与多媒体的最终提交

- 把外审阶段的匿名视频替换为**正式版本**：可加回作者/机构水印(若期刊允许)、补致谢。
- 确认编码通用(如 H.264/MP4)、体积合理、分辨率与帧率达标；随附简短说明与关键片段索引。
- 若视频托管在外部平台，确保链接**长期有效**并与正文引用一致(**具体多媒体政策待核实**)。

## 十、定稿阶段的沟通纪律

- 校样只做**勘误级**修改；如发现需实质更正(如数据笔误影响结论)，第一时间主动联系编辑部
  (`jcad@ict.ac.cn`)说明，不擅自在校样上大改。
- 多作者稿在定稿前完成署名顺序与通信作者的最终确认，录用后一般不再改动署名。
- 版权协议、出版费用与见刊时间以编辑部**书面通知**为准；本技能涉及金额/条款处均标
  **待核实**，不臆断。

## 十一、从录用到见刊的推进节奏

1. 收到录用通知 → 立即解除匿名、补真实链接与致谢、基金编号。
2. 按编辑部要求提交定稿源文件与图源(与录用版本一致)。
3. 收到校样 → 限期逐字冷读、只做勘误、返回。
4. 签署版权/许可协议、确认出版事宜。
5. 确认在线优先出版/见刊卷期，核对最终 PDF 无排版走样。

## 十二、定稿前的最后一次冷读

在提交定稿与返回校样之间，安排一次完整冷读，重点看以下易错处：

- 作者姓名、单位、顺序、通信作者与邮箱是否与最终确认一致。
- 中英文摘要、图题表题、量与单位是否在排版后仍规范。
- 参考文献序号与正文引用是否在重排后仍一一对应。
- 渲染图与几何误差色图在排版尺寸下是否清晰、色标是否可读。
- 致谢与基金编号是否补全、是否有残留的匿名占位符。
