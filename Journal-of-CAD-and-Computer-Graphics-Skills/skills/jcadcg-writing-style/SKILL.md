# 《计算机辅助设计与图形学学报》中文写作规范

本技能把《计算机辅助设计与图形学学报》(Journal of Computer-Aided Design & Computer
Graphics, JCAD&CG) 的中文论文体例落实为可执行清单。本刊是 CCF A 类图形学/CAD 方向中文月刊，
Journal of Computer-Aided Design & Computer Graphics 的读者期待规范的中文科技写作与到位的
图形证据呈现。体例不合是初审退回补正的常见原因。核验见 `resources/official-source-map.md`
（2026-07-09）。

## 一、首页范式（图形学论文）

本刊论文首页应在前几段建立：**可识别的图形/几何问题 → 现状为何不足 → 本文贡献(分条) →
图形/几何证据 → 结论与意义**。避免以"研究热点""随着…发展"空泛开场。示范见
`resources/worked-examples/01-introduction.md`。

## 二、报道性摘要的四要素

中文摘要采用**报道性摘要**，四要素齐全：

| 要素 | 写什么 | 图形学示例 |
|---|---|---|
| 问题/目的 | 针对什么图形/几何问题 | 薄壳网格简化易丢特征、薄壁穿透 |
| 方法 | 用什么技术路线 | 特征-穿透联合约束的边折叠准则 |
| 结果 | 可量化结果 | Hausdorff 误差降 x%、穿透率降至近 0 |
| 结论 | 有何一般性结论/意义 | 约束比平均误差最优更利于结构可用性 |

- 摘要中不出现"很好""重要意义"等空泛程度词，用**具体数字与量化对比**。
- **English Abstract** 与中文摘要信息一致、术语准确（如 mesh simplification、rendering、
  visualization、CAD 等图形学标准英文术语），语法校对到位。

## 三、关键词与中图分类号

- **中英文关键词**：一般 3~8 个，中英文一一对应，涵盖方法、对象、领域（如"网格简化；特征
  保持；计算机辅助设计"）。
- **中图分类号**：计算机类以 **TP** 打头。图形学常用类目参考：**TP391**（信息处理/模式识别，
  常见于图形图像）、**TP391.41**（图像处理/识别）、**TP391.7**（计算机辅助设计/图形学/
  CAD）、**TP391.9**（仿真）等，据主题选取（具体归类以最新《中图法》与编辑部意见为准，
  **待核实**细目）。

## 四、参考文献 GB/T 7714

- 采用 **GB/T 7714** 著录规则（顺序编码制为通行标准，具体呈现以官方模板为准）。
- **中文文献给出中英文对照**：中文题名后附英文题名，作者姓名、期刊名、出版信息完整。
- 文献类型标识齐全：[J] 期刊、[C] 会议、[M] 专著、[D] 学位论文、[EB/OL] 电子资源等。
- 正文引用序号与文末列表**一一对应**，无漏引、错引；优先引本领域高质量文献与本刊相关工作。

## 五、量、单位与图表规范

- **量与单位**用法定计量单位；符号、变量用斜体，单位用正体。
- **图题在图下、表题在表上，中英文对照**；图内文字随正文缩放后仍清晰。
- **渲染图/结果图**标注视角、光照、分辨率；**几何误差**用色图并给出色标范围与单位。
- 折线/柱状图在黑白打印下用线型或纹理区分而非仅靠颜色（照顾印刷）。
- **公式**用编辑器/LaTeX 排版、编号连续，正文用编号引用。

## 六、语言与结构

- 用**规范书面汉语**，术语统一（同一概念全篇同名），首次出现的英文缩写给中文全称与英文
  原词，如"生成式人工智能内容(AI-Generated Content, AIGC)"。
- 段落围绕一个论点展开；方法节给出足以复现的关键细节；实验节把主张与证据一一对应。
- **局限/威胁有效性随结果讨论**，而非结尾补一段套话。
- 路线图（"第 2 节…第 3 节…"）简短即可，不过度信号化。

## 七、写作自查清单

- [ ] 题目 ≤20 字、可检索；中英文题名一致。
- [ ] 中文摘要四要素齐全、无空泛程度词；English Abstract 术语与语法到位。
- [ ] 关键词 3~8 个、中英文对应；中图分类号(TP)选取合理。
- [ ] 首页建立可识别图形/几何问题、贡献分条、绑定图形证据。
- [ ] 参考文献 GB/T 7714、中文文献中英文对照、序号一一对应。
- [ ] 图题表题中英文、量与单位法定、渲染/误差图标注完整。
- [ ] 术语统一、缩写首次给全称、局限随结果讨论。

## 八、常见体例错误对照

| 错误 | 后果 | 修正 |
|---|---|---|
| 摘要写成指示性/空泛 | 初审印象差 | 改报道性四要素 + 具体数字 |
| 题目超 20 字或含缩写堆砌 | 补正 | 精炼到 ≤20 字、可检索 |
| 中图分类号缺失/乱填 | 补正 | 按主题选 TP 细目 |
| 中文文献无英文对照 | 不符 GB/T 7714 要求 | 逐条补英文题名与信息 |
| 渲染图无视角/色标 | 外审质疑可比性 | 标注视角/光照/分辨率/色标 |
| 局限缺失或套话 | 显得不严谨 | 随结果讨论真实边界 |

## 九、图形学术语的中英文规范

术语不统一是外审常见批注。建立术语表并全篇一致，首次出现给中文全称与英文原词：

| 中文 | 英文 | 备注 |
|---|---|---|
| 网格简化 | mesh simplification | 勿与"网格生成"混用 |
| 点云 | point cloud | 统一"点云"，勿"点集/点阵"混用 |
| 渲染 | rendering | 区分"绘制"是否等义，全篇统一 |
| 可视分析 | visual analytics | 与"信息可视化"区分 |
| 计算机辅助几何设计 | computer-aided geometric design, CAGD | 本刊常见方向 |
| 虚拟现实 | virtual reality, VR | 首次给全称 |

## 十、方法节的可复现性写作

- 方法节应给出**足以复现的关键细节**：符号定义、公式推导主线、算法伪代码、关键超参与设置。
- 用统一符号体系(变量斜体、向量/矩阵按惯例)，避免同符号多义。
- 复杂几何/渲染流程配示意图(pipeline)，图内步骤与正文小节对应。

## 十一、常见语言与排版细节

- 数字与单位间、中英文间的空格按期刊模板；标点用中文全角(正文)与英文半角(公式/代码)不混。
- 图表在正文**先引用后出现**；每图每表都在正文被讨论，不放"孤图"。
- 避免口语化与夸张("完美""极大提升")；用数据说话。
- 缩写全篇统一，出现一次即定义，之后一致使用。

## 十二、写作定稿前通读顺序

1. 只读摘要与引言，检验"问题→不足→贡献→证据→意义"是否自足。
2. 只看图表与图题表题，检验能否脱离正文大致读懂结果。
3. 通读方法节，检验可否据此复现。
4. 核对参考文献 GB/T 7714 与正文引用序号一一对应。
5. 术语与符号全局一致性扫描。
