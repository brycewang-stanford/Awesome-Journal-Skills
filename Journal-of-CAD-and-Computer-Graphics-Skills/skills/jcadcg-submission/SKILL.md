# 《计算机辅助设计与图形学学报》投稿就绪审计

在把稿件上传到《计算机辅助设计与图形学学报》(Journal of Computer-Aided Design & Computer
Graphics, JCAD&CG) 在线投稿系统之前，先跑完这份审计。本刊是由中国计算机学会(CCF)主办、中国
科学院计算技术研究所承办、创刊于 1989 年的 **CCF A 类**计算机**图形学/CAD 方向**中文月刊
(ISSN 1003-9775, CN 11-2925/TP)，以图形、几何、渲染、可视化、虚拟现实、图像视频等原创成果
为主。它按期刊稿件走同行评议，因此投稿被当作一篇要经受编辑初审、专家外审、主编终审的完整
学术论文来评判，而不是一次会议式的一轮定稿。以下事实于 2026-07-09 通过官网 `www.jcad.cn`、
中科院计算所出版页、CNKI 期刊主页与 CCF 推荐中文科技期刊目录交叉核验（见
`resources/official-source-map.md`），凡本刊未公开确认的数字均标注 **待核实**，请以投稿
系统内最新《投稿须知》为准。

## 一、投稿系统与账号

- 本刊经**官网在线投稿系统**收稿，投稿入口为 `https://www.jcad.cn/`；编辑部邮箱
  `jcad@ict.ac.cn`，电话 010-62562491 / 010-62600342。
- 在系统注册后即可投稿；注册信息（题名、作者、单位、通信作者、研究方向）决定编辑分派外审
  专家的匹配度，务必用**真实终稿信息**，不要用占位标题。
- 稿件提交后可在系统内**跟踪查询稿件处理流程与当前状态**，按编辑部要求阅读修改意见并上传
  修改稿。

## 二、格式与电子版

- 稿件应包括**题目、作者及单位、邮编、内容摘要、关键词、正文、参考文献**等标准部件。
- **文章标题一般控制在 20 字以内**（据官网口径），力求精炼、可检索。
- 电子版投稿支持 **DOC、PDF、ZIP、RAR** 格式；若用 LaTeX，建议连同源文件与图源打包。
- 具体篇幅/字数/页数上限与官方模板最新版本以系统内为准（**待核实**具体数值）。

## 三、中文科技论文的硬性组件

本刊稿件必须齐备以下中文科技期刊的标准部件（详见 `jcadcg-writing-style`）：

| 组件 | 要求 |
|---|---|
| 中文题名 + 英文题名 | 英文题名与中文一致、可独立检索；中文题名 ≤20 字 |
| 中文摘要 + English Abstract | 采用**报道性摘要**：问题与目的、方法、结果、结论四要素齐全 |
| 中文关键词 + Keywords | 一般 3~8 个，中英文一一对应，含图形/几何领域术语 |
| 中图分类号 | 计算机类以 **TP** 打头（图形学常用 TP391、TP391.41、TP391.7 等），据主题选取 |
| 作者简介与基金项目 | 注明资助项目及编号；作者简介含研究方向 |
| 参考文献 | **GB/T 7714**；中文文献需给出**中英文对照**，著录完整 |

## 四、图形学论文特有的提交细节

- **图形结果**：矢量图优先(SVG/PDF)；渲染图标注视角、光照、分辨率；几何误差用色图
  (colormap) 并给出色标范围；对比图统一比例与裁剪。
- **演示视频/多媒体附件**：图形学论文常随附演示视频；若提交，压制为常见编码(如 H.264/MP4)，
  标注时长与关键内容，去除水印与身份信息（本刊是否**硬性要求**视频规格属 **待核实**）。
- **公式**：用公式编辑器或 LaTeX 排版而非截图；对齐、编号连续；正文引用用编号。
- **文件命名**：稿件文件名不含作者姓名、单位等身份信息（双盲外审）。

## 五、学术规范与伦理红线

- **学术不端检测**：初审即查重；投稿前用查重工具把总文字复制比压到安全线以下，尤其注意
  自引与前序会议(如 CCF CAD/CG、ChinaGraph)版本的重叠（本刊官方**具体查重红线百分比**属
  **待核实**，勿套用其他刊数值为定论）。
- 若本文是作者已发表会议论文的**期刊扩展版**，必须显著声明扩展关系、指明新增贡献（新增
  方法、新增实验、新增系统模块通常需达到实质比例），并规范引用会议版本，避免一稿多投与
  自我剽窃。
- 署名、贡献、基金、数据来源须真实；用户研究须去标识并符合伦理；不得伪造、篡改数据或代写。
- 一稿多投是明确禁止项：投稿即视为该稿未在其他刊物同时评审。

## 六、退稿风险分诊表

| 审计发现 | 风险等级 | 处置 |
|---|---|---|
| 查重超标 | 初审退稿级 | 重写重叠段落、补足原创增量后再投 |
| 缺中英文摘要/关键词/中图分类号 | 初审退回补正 | 补齐全部标准部件 |
| 主题非图形/几何/可视化/CAD 方向 | 不符宗旨 | 改投综合刊(如《计算机学报》)或对口专业刊 |
| 无新意、重复前人工作、创新点模糊 | 初审退稿理由 | 回到 `jcadcg-topic-selection` 重估贡献 |
| 图形结果单薄、只有指标无视觉对比 | 证据不足 | 补渲染对比、几何误差色图、消融 |
| 参考文献不符 GB/T 7714 / 中文文献缺英文对照 | 需补正 | 逐条重排著录格式 |
| 会议扩展版未声明、增量不足 | 学术规范风险 | 显著声明并补足实质新增贡献 |
| 一稿多投 | 严重违规 | 撤回其一，确认唯一在审 |

## 七、投稿前最后一轮操作顺序

1. 冻结正文论证主线；参考文献可微调，核心贡献陈述不再改。
2. 通排一遍，确认中英文摘要、关键词、中图分类号、基金、作者简介、参考文献六件套齐全。
3. 跑查重，把总复制比压到安全线以下，逐条核对自引与会议版本重叠。
4. 校对图形结果：渲染图/几何误差色图清晰、色标与比例统一、图题表题中英文规范、量与单位
   用法定计量单位。
5. 核对演示视频/附件：编码通用、去水印、内容与正文一致。
6. 上传后重新下载 PDF 冷读一遍，确认提交的是最终版文件、无乱码、无残留批注、无身份泄露。

## 八、每轮需复核的易变项

- 具体篇幅/字数/页数上限、官方模板版本、版面费金额（**待核实**，以编辑部通知为准）。
- 是否新增代码/数据可用性声明要求、演示视频硬性规格或生成式 AI 使用披露要求（**待核实**）。
- 栏目设置与当期征稿重点、专题(special issue)是否开放。

## 九、投稿信(cover letter)与系统字段

- 投稿系统通常要求填写题名、摘要、关键词、领域分类、作者与通信作者信息、基金、推荐/回避
  审稿人等。逐项据实填写；领域分类要与稿件主题精确匹配（图像视频/可视化/CAD-CAE-CAM/VR-AR
  等），直接影响外审专家分派。
- 若填写回避审稿人，仅出于真实利益冲突，并简要说明理由。
- 投稿信简明说明：本文主要贡献、为何适合《计算机辅助设计与图形学学报》(Journal of
  Computer-Aided Design & Computer Graphics, JCAD&CG)、是否为会议扩展版及其增量、以及无
  一稿多投与利益冲突的声明。

## 十、投稿时间线上的常见踩坑

| 场景 | 后果 | 规避 |
|---|---|---|
| 领域分类填错 | 分到不对口专家、外审慢 | 精确匹配图形/几何子方向 |
| 临投稿才发现查重超标 | 被迫延期重写 | 投稿前数天先自查查重 |
| 图/视频泄露身份(水印/路径) | 违反双盲 | 提交前清理图源、视频与文件属性 |
| 只有指标没有视觉对比 | 外审质疑图形质量 | 补渲染对比与几何误差色图 |
| LaTeX 在编辑部环境编译失败 | 排版返工、延误 | 提交前用模板干净编译 |
