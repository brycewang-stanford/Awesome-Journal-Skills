## 本刊相对兄弟中文刊的特色

- **图形学/CAD 最专的阵地**：栏目聚焦图像视频处理(含计算摄像、AIGC)、可视化与可视分析
  (信息可视化、科学可视化、可视分析)、CAD/CAE/CAM(计算机辅助几何设计、仿真分析、智能制造、
  拓扑优化、设计/分析/制造一体化)、VR/AR/MR、三维图形计算与交互技术，区别于综合刊《计算机
  学报》与软件方向的《软件学报》。
- **图形/几何证据传统**：重视几何精度、渲染质量、可视化有效性、用户研究与主客观评测，而非
  纯榜单分数。
- **审稿与终审**：以同行评议为基础，编辑初审→专家外审→复审退修→栏目主编组织编委终审；综述
  稿通过初审后一般由 3~4 名专家审阅（经验值，见来源表）。
- **中文科技规范**：报道性中英文摘要、中英文关键词、中图分类号(TP)、参考文献 **GB/T 7714**
  且中文文献中英文对照、量与单位用法定计量单位。
- **与 CCF CAD/CG 全国会议同源**：本刊与 CCF 全国计算机辅助设计与图形学学术会议(CAD/CG)
  同属该学科社群，会议优秀论文常被鼓励扩展投本刊（扩展需声明增量）。

## 12 个技能

| 技能 | 作用 |
|---|---|
| [`jcadcg-topic-selection`](skills/jcadcg-topic-selection/SKILL.md) | 判断选题是否落在图形学/CAD/可视化版图、与兄弟刊分工、选栏目 |
| [`jcadcg-workflow`](skills/jcadcg-workflow/SKILL.md) | 从选题到见刊的全流程时间线与技能编排 |
| [`jcadcg-writing-style`](skills/jcadcg-writing-style/SKILL.md) | 中文体例：中英文摘要、关键词、中图分类号、GB/T 7714 |
| [`jcadcg-related-work`](skills/jcadcg-related-work/SKILL.md) | delta 优先的相关工作与文献综述，确立图形/几何创新性 |
| [`jcadcg-experiments`](skills/jcadcg-experiments/SKILL.md) | 图形与几何实验：几何精度、渲染质量、可视化有效性、用户研究 |
| [`jcadcg-reproducibility`](skills/jcadcg-reproducibility/SKILL.md) | 可复现性：环境/种子/数据集/渲染管线固定、复现包 |
| [`jcadcg-supplementary`](skills/jcadcg-supplementary/SKILL.md) | 按"决定性"划分正文与附录/补充材料/演示视频 |
| [`jcadcg-artifact-evaluation`](skills/jcadcg-artifact-evaluation/SKILL.md) | 代码与数据可用性现状与诚实声明 |
| [`jcadcg-review-process`](skills/jcadcg-review-process/SKILL.md) | 审稿流程：初审→外审→复审退修→主编终审与修回轮次 |
| [`jcadcg-author-response`](skills/jcadcg-author-response/SKILL.md) | 审稿意见答复与修回说明的组织与撰写 |
| [`jcadcg-submission`](skills/jcadcg-submission/SKILL.md) | 投稿就绪审计：系统、格式、六件套、查重、退稿风险分诊 |
| [`jcadcg-camera-ready`](skills/jcadcg-camera-ready/SKILL.md) | 录用后定稿、校样清样、版权与出版流程 |

## 快速上手

1. 先用 `jcadcg-topic-selection` 判契合、`jcadcg-writing-style` 搭骨架；
2. 用 `jcadcg-experiments` 与 `jcadcg-reproducibility` 搭图形/几何证据；
3. 在 `resources/official-source-map.md` 核实当期口径后，用 `jcadcg-submission` 做投稿自审。

## 范围说明

本包面向**中文图形学/CAD 方向期刊**，不是英文会议包，也不是经济学/统计的计量工具包。请勿把
OpenReview/PMLR 机制或 Stata/DiD/IV 计量套件搬进来；JCAD&CG 的证据是图形学与 CAD 的证据。
