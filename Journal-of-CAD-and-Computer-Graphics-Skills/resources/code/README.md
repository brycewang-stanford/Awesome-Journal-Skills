# 复现与自查脚本思路 — 《计算机辅助设计与图形学学报》(Journal of Computer-Aided Design & Computer Graphics, JCAD&CG)

本目录把《计算机辅助设计与图形学学报》(Journal of Computer-Aided Design & Computer
Graphics, JCAD&CG) 作者常用的复现与投稿前自查动作沉淀为可脚本化的思路。它刻意保持精简：
本刊的**政策**（同行评议流程、双盲外审、GB/T 7714、代码数据可用性现状）在各 `SKILL.md` 与
[`../official-source-map.md`](../official-source-map.md) 中；这里只放**跨稿件复用的工具性
检查**，尤其是图形学/CAD 论文特有的项。

## 一、复现包冒烟检查（思路）

在把复现包随稿提交前，做一次"干净环境冒烟测试"，图形学论文尤其要固定渲染/几何管线：

```bash
# 1) 从零构建
python3 -m venv .venv && . .venv/bin/activate
pip install -r requirements.txt
# 2) 一键复现主结果（含几何/渲染产物）
bash scripts/run_all.sh
# 3) 核对产物是否对上论文表图（人工比对 results/ 与论文表6/图3、渲染帧与几何误差图）
```

要点：固定随机种子、记录 GPU/驱动/渲染器版本与耗时、把大数据(网格/点云/纹理/视频)走稳定
托管而非塞进仓库（见
[`../../skills/jcadcg-reproducibility/SKILL.md`](../../skills/jcadcg-reproducibility/SKILL.md)）。

## 二、投稿前自查清单（脚本化思路）

- **查重**：投稿前用查重工具把总文字复制比压到安全线以下，重点核对自引与会议(CAD/CG、
  ChinaGraph 等)版本重叠。
- **匿名化**：清理 PDF 文档属性、致谢、基金显名、图源与视频水印；扫描仓库中的用户名、邮箱、
  机构路径。
  ```bash
  # 示意：检查提交物中可能泄露身份的痕迹
  grep -rniE '(@[a-z0-9.]+\.(edu|ac)\.cn|/home/|/Users/|课题组|实验室)' artifact/ | head
  ```
- **参考文献**：核对 GB/T 7714 著录、中文文献中英文对照、正文序号与文末列表一一对应。
- **六件套**：中英文摘要、关键词、中图分类号(TP)、基金、作者简介、参考文献是否齐全。

## 三、图形学/CAD 论文特有的检查（通用工具无法替代）

- **几何/渲染可复现**：固定网格分辨率、采样数、光照与相机参数、渲染器与随机种子；报告
  几何误差(如 Chamfer/Hausdorff)、渲染质量(PSNR/SSIM/LPIPS)时说明计算脚本与掩码。
- **图像/视频结果对齐**：论文图中的渲染帧应能由复现脚本重现；演示视频与正文数字口径一致。
- **用户研究材料**：若有主观评测，随附问卷、被试人数、统计口径与伦理合规说明(去标识)。
- **可用性声明诚实**：与实际提供的制品(代码、数据、网格、预训练模型)一致；不可公开部分
  说明原因与替代核验途径。

本目录不是计量经济学/Stata 套件，JCAD&CG 的证据也不是纯榜单分数：请勿把 DiD notebook 或
排行榜式框架搬进来。政策仍以官网当期《投稿须知》与各技能为准。
