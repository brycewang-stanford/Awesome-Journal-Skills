# English Humanities Breadth Resources

This layer supports routing across humanities venues in history, literature,
philosophy, religion, art history, classics, musicology, linguistics, and
theory. Use it when the manuscript's archive, object, method, and scholarly
conversation need to be matched to a journal before full revision work.

## What to Load

- `official-source-map.md` - official-source anchors for scope, submission, and
  style checks.
- `source-basis.md` - coverage basis and verification discipline for this
  humanities breadth bundle.
- `worked-examples/venue-routing.md` - example routing diagnoses for archival,
  interpretive, theoretical, close-reading, art-object, religion, and
  linguistics submissions.
- `exemplars/selection-patterns.md` - field-specific venue-selection patterns
  and sibling-journal traps.

## Usage Rule

Start with `en-humanities-journal-workflow` when the field boundary is unclear.
Route by object and argument first, not just topic keywords: archive, corpus,
textual object, visual object, philosophical thesis, theological tradition, or
linguistic phenomenon. Then load only the matching venue cards.
