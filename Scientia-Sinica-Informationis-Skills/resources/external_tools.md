# 外部工具与官方入口 — 《中国科学：信息科学》(SSI)

访问日期：2026-07-09

在给出投稿建议前，先重新打开《中国科学：信息科学》(Scientia Sinica Informationis, SSI)
当期页面。SSI 的栏目页数、版面费、审稿时限、参考文献与摘要要求均按当期《投稿须知》执行，
且 `scis.scichina.com` / `www.sciengine.com` 网关可能拦截直接抓取（返回 403），
需经搜索渲染读取，并与 CNKI、万方、维普交叉核实。

## 官方工作流入口

- SSI 官网首页（快速了解中国科学信息科学）: http://scis.scichina.com/
- SSI 中文投稿须知（PDF）: http://scis.scichina.com/download/ssi-info-for-authors.pdf
- SSI 在 SciEngine 平台主页（当期目录/卷期）: https://www.sciengine.com/SSI/home
- 《中国科学》杂志社官网: http://www.scichina.com/
- 作者投稿系统入口（注册作者账户、上传 PDF）: 见官网首页"作者登录/在线投稿"链接
- LaTeX 模板下载: 见官网"下载中心"（社区镜像见下方交叉核实）

## 英文姊妹刊入口（相对独立的两个刊物）

- Science China Information Sciences (SCIS) 主页: https://www.sciengine.com/SCIS/home
- SCIS 英文版 Springer 主页: https://link.springer.com/journal/11432
- 说明：SSI（中文）与 SCIS（英文）主办、学科一致但**相对独立**，应择一投稿；
  中英文互转政策**以编辑部当期说明为准（待核实）**。

## 交叉核实 surface（当网关拦截官网时）

- CNKI 中国知网《中国科学：信息科学》期刊主页（卷期、被引、栏目）。
- 万方数据 / 维普 (cqvip) 期刊详情页：https://www.cqvip.com/journal/62331/detail
- 百度百科条目（仅作二手佐证，不作首要信源）。
- CCF 推荐中文科技期刊目录（核实学科分区与推荐层级，**具体层级待核实**）。
- 社区 LaTeX 模板镜像（用于排查官方 CCT 模板在现代 TeX 发行版编译问题）：
  GitHub `xavierwoo/SCIS-latex-template`、latexstudio、TeXPage 等（**非官方，仅供排版参考**）。

## 作者侧自查

- 作者账户资料、通讯作者信息、单位全称 + 城市 + 邮编、基金资助编号。
- LaTeX 排版是否用官方模板；PDF 是否可正常上传；图表是否清晰、编号连续。
- **中、英文双语摘要**与关键词是否齐备；英文摘要是否达到国际可读水平。
- 栏目自选是否与稿件类型匹配（评述/论文/快报/学术介绍），快报是否控制在 4 页内。
- 参考文献著录格式是否统一（GB/T 7714 与否**以当期须知为准，待核实**）；中图分类号是否需要。
- 是否存在一稿多投/重复发表风险（尤其中英文姊妹刊之间）。

## 不要臆断

- 不要臆断版面费金额、审稿周期、录用率、影响因子——以官网与 CNKI 当期为准。
- 不要臆断主编姓名与编委名单——随届次变动，标注待核实。
- 不要臆断中英文互转规则、AI 使用披露政策、数据/代码强制公开与否。
- 若不同页面冲突，以官网当期《投稿须知》与投稿系统内说明为准，并记录冲突。
