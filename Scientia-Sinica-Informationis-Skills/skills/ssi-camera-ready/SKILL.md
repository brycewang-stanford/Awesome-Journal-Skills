# 《中国科学：信息科学》录用后定稿与校样（Camera-Ready）

稿件被《中国科学：信息科学》(SCIENTIA SINICA Informationis, SSI) 录用后，进入**定稿—校样—见刊**
阶段。SSI 由《中国科学》杂志社出版、月刊每月 20 日出版，编辑部会将排版后的**清样（校样）**
发回作者校对。此阶段的目标是：**只改错，不改科学内容**，并把中英文信息、著录、版面费与版权
手续一次办妥。以下流程据官网《投稿须知》与一般期刊出版惯例整理，具体手续**以录用信与
编辑部当期说明为准（待核实）**。

## 一、定稿阶段的红线

- **不得实质改动科学内容**：录用基于评审时的稿件；清样阶段只修排版、错字、著录错误，
  不新增结论、不改数据。确有必要的实质修改须**提前征得编辑部/编委同意**。
- **一切以清样为准**：编辑部排版可能调整图表位置、公式编号；以发回的清样为校对基准。
- **按时返回清样**：清样校对通常有明确时限，逾期影响见刊排期（时限**待核实**）。

## 二、清样校对清单（逐项核对）

| 项目 | 核对要点 |
|---|---|
| 标题 | 中英文标题一致、无错字、大小写规范 |
| 作者与单位 | 姓名拼写/顺序、通讯作者、单位全称 + 城市 + 邮编、上标对应正确 |
| 中英文摘要 | 双语摘要与终稿一致、英文摘要可独立阅读、无排版乱码 |
| 关键词 | 中英文关键词齐备、与摘要一致 |
| 基金 | 资助项目名称与**编号**准确、无遗漏 |
| 正文 | 章节编号、交叉引用（图/表/式/文献）指向正确 |
| 公式 | 变量正斜体、编号连续、上下标正确、无 LaTeX 转换错误 |
| 图表 | 清晰度、编号、图注、坐标轴单位、彩色/灰度呈现 |
| 参考文献 | 著录格式统一、序号与正文引用一一对应、中外文完整 |
| 中图分类号 | 是否需要、是否正确（**待核实**） |

## 三、中文排版专项

- **量与单位**：法定计量单位，变量斜体、单位正体，避免中英文标点混用。
- **术语一致**：全文术语、缩写、符号统一；中文术语与英文对照一致。
- **标点与全半角**：中文全角标点、数字与单位间距规范。
- **图注与表题**：中文图表标题规范，图注自足；避免图中文字过小。
- 官方 LaTeX 模板若在现代 TeX 发行版编译异常，注意清样中是否残留转换错误（如缺字、错位）。

## 四、版权、版面费与样刊

- **版权协议**：按编辑部要求签署版权/出版协议（形式**待核实**）。
- **版面费**：本刊对录用稿件收取版面费；按通知缴纳，保留缴费凭证（**金额待核实**）。
- **样刊**：出版后向作者**免费提供样刊**；核对邮寄地址与联系方式。
- **发表信息**：确认卷、期、页码、DOI（若有）等元数据，便于后续引用与存档。

## 五、与英文姊妹刊的区分与后续

- SSI（中文）与英文姊妹刊 **Science China Information Sciences (SCIS)** 是**相对独立的两个刊物**。
  中文见刊后，**不得**将同一成果直接译为英文另投 SCIS 或他刊而不加声明——属重复发表风险。
- 如确有中英文双语发表需求，须**事先**向编辑部说明并获同意；互转政策**待核实**。
- 记录本文的正确引用格式（中文刊名《中国科学：信息科学》+ 年卷期页），供作者自存与他引。

## 六、见刊前最后一遍

1. 用录用终稿逐字比对清样，确认无科学内容改动。
2. 走一遍第二节清单，重点核对作者/单位/基金/参考文献等易错著录。
3. 冷读一遍中英文摘要，确保英文摘要通顺、可独立阅读。
4. 检查所有交叉引用（图/表/式/文献）指向正确、无"图 ??"之类残留。
5. 汇总校对意见，按编辑部要求的格式统一返回，勿分散多次发送。
6. 完成版权协议与版面费手续，核对样刊邮寄信息。

## 七、常见遗留错误

| 错误 | 影响 | 处理 |
|---|---|---|
| 英文摘要留有机翻残句/乱码 | 损害国际可见性 | 清样阶段务必逐句核对 |
| 基金编号错误或遗漏 | 影响结题与致谢 | 与项目书核对 |
| 参考文献序号与正文错位 | 读者/引用混乱 | 逐条核对引用 |
| 交叉引用残留占位符 | 显得不严谨 | 全文搜索 ??/未定义引用 |
| 图表灰度化后不可读 | 印刷版信息丢失 | 确认灰度可读或申请彩图 |
| 单位/正斜体错误 | 规范性问题 | 按法定计量单位统一 |

## 八、清样意见的返回格式（建议）

把校对意见集中、按位置排序返回编辑部，避免口头或零散多封邮件：

```text
《中国科学：信息科学》清样校对意见 —— 稿号 <XXXX>，第 <一/二> 校
1. 第 3 页，第 2 段第 4 行："鲁棒" 误作 "鲁捧"，请改。
2. 第 5 页，式(7)：下标 i 应为斜体。
3. 第 6 页，图 4 图注：坐标轴单位缺 "ms"，请补。
4. 参考文献 [12]：年份应为 2023（原作 2013），请核。
5. 英文摘要第 2 句：拼写 "efficieny" → "efficiency"。
（如无其他修改，确认可付印。）
```

- 每条给出**页/段/行或图表式编号 + 原文 + 修改**，便于排版人员精确定位。
- 只列错误，不重述科学内容；确需实质改动的，单独标注并说明已获编委同意。
- 一次性汇总，减少校对轮次，保障月刊排期。

## 九、时间线意识

- 录用后进入排期，清样返回时限较紧，收到清样应尽快处理。
- 版权协议与版面费手续宜与清样校对并行办理，避免卡见刊。
- 若涉及在线优先出版（OnlineFirst，政策**待核实**），注意在线版与印刷版的一致性。

## 十、每次核实

- 清样校对轮次与时限、版权协议形式、版面费金额与缴纳方式（**待核实**）。
- DOI/卷期分配机制、在线优先出版（OnlineFirst）政策（**待核实**）。
- 中英文双语发表/互转政策（**待核实**）。

## 输出格式

```text
[定稿状态] 清样一校/二校 待返回 / 已返回
[核对清单] 标题/作者/单位/基金/摘要/参考文献/图表/公式——逐项通过？
[中文排版] 术语/单位/标点/交叉引用——是否零遗留？
[手续] 版权协议 / 版面费 / 样刊地址——是否完成？
[发表信息] 卷期/页码/DOI 是否确认
[遗留问题] <列出与负责人、时限>
```
