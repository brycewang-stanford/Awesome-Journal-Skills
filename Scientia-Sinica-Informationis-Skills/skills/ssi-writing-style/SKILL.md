# 《中国科学：信息科学》中文科技写作规范（Writing Style）

《中国科学：信息科学》(SCIENTIA SINICA Informationis, SSI) 刊载信息科学领域最高水平的**中文**
原创成果，写作既要符合中文科技论文规范，又要在首页把**科学价值与创新高度**讲清楚。本技能给出
从摘要到参考文献的可操作规范。具体格式细节（如是否严格 GB/T 7714、中图分类号是否强制）
**以官网当期《投稿须知》为准（待核实）**，此处给出通行做法与本刊已知要求。

## 一、首页弧线（最高杠杆）

一篇 SSI 论文的摘要 + 引言应完成：

1. **问题**：从业者/学界公认的信息科学难题，开门见山。
2. **现状为何不足**：已有工作的局限，点出空白。
3. **贡献**：以科学主张陈述创新点（新方法/新理论/新系统/新发现）。
4. **证据**：与主张相称的实验或理论依据（真实数据、公平对比、置信区间等）。
5. **意义**：对信息科学（及应用）的价值。

避免以"随着…的发展"套话开场，避免把创新点埋到第 3 页。范例见
[`../../resources/worked-examples/01-introduction.md`](../../resources/worked-examples/01-introduction.md)。

## 二、中英文双语摘要（本刊硬要求）

- **中文摘要**：概括目的、方法、结果、结论；自足，不含未在正文出现的缩写与引用。
- **英文摘要**：**可独立阅读**，是国际可见性关键，不能是中文机翻残句；术语与正文一致。
- **关键词**：中英文各若干个，覆盖问题、方法、对象；与摘要术语统一。
- 中英文摘要信息量对等，避免一详一略。

## 三、标题与结构

- 标题准确、具体、可检索；避免过度宽泛或口号式。
- 结构清晰：引言、相关工作、方法、实验/理论、讨论、结论；章节编号规范。
- 论文栏目须交代**科学思想、意义、创新点、实验和理论依据、应用前景**（须知原话要求）。

## 四、量、单位与符号

- 采用**法定计量单位**；变量**斜体**、单位**正体**、矢量/矩阵按规范（如粗体）。
- 数字与单位之间留适当间距；中文用全角标点，公式内用半角。
- 全文符号、缩写、术语**统一**；首次出现给出中文全称与必要英文缩写。

## 五、图表与公式

- 图表**清晰、编号连续、图注自足**；中文图表标题规范；坐标轴含量与单位。
- 避免正文与图表重复陈述同一数据；图表服务论证而非装饰。
- 公式编号连续，正斜体正确；关键公式在正文推导或引用附录完整推导。
- 注意灰度印刷可读性（彩图是否额外收费**待核实**）。

## 六、参考文献与著录

- 著录**格式统一、完整**；正文引用与文献表**一一对应**。
- 是否严格按 **GB/T 7714**（中国文后参考文献著录规则）**以当期须知为准（待核实）**；
  无论采用何种格式，务必全文一致。
- **中外文文献兼顾**：不要只引英文而忽略国内相关工作，也不要只引中文忽略国际前沿。
- **基金著录**：资助项目名称与**编号**完整（本刊与国家自然科学基金委关系密切，尤应准确）。
- **中图分类号**：是否需要**待核实**；若需要，按《中国图书馆分类法》信息科学类目填写。

## 七、语言与表达

- 中文表达严谨、简洁、逻辑清晰；避免口语化与过度修辞。
- 术语规范，中英文对照一致；避免中英夹杂造成歧义。
- 主张有据，不夸大（"首次""最优"须有证据支撑）。
- 威胁/局限**就地讨论**（与相应结果一起），而非结尾走过场。

## 八、体现科学价值与创新高度

SSI 门槛高，写作要主动回答审稿人心中的三问：

- **为什么重要**：这个问题对信息科学/应用意味着什么？
- **新在哪里**：与国内外已有工作的差异（delta）用一两句讲清。
- **凭什么信**：证据是否与主张相称、是否公平、是否可复现。

用"换核检验"自查：把核心技巧换成强基线，科学结论是否仍成立且有意义？若不成立，说明贡献扎实。

## 九、常见写作失误

| 失误 | 纠正 |
|---|---|
| 套话开场、创新点埋太深 | 首页即给问题与贡献 |
| 英文摘要敷衍 | 重写至可独立阅读 |
| 只引英文/只引中文 | 中外文文献兼顾 |
| 变量单位正斜体混乱 | 按法定计量单位统一 |
| 图表与正文重复堆数据 | 图表服务论证、去冗余 |
| 主张夸大无据 | 有几分证据说几分话 |
| 局限藏在结尾 | 威胁就地讨论 |

## 十、中文科技写作细节（易错点）

- **中英标点不混用**：中文正文用全角（，。；：""），公式与英文文献用半角。
- **数字与量词**：遵循出版数字用法（阿拉伯数字用于计量与统计），前后一致。
- **缩略语**：首次出现给全称并括注缩写，其后统一用缩写；摘要与正文各自首现均需展开。
- **图表引用**：正文须先引后现（"如图 3 所示"），不出现未被引用的图表。
- **时态与人称**：中文科技写作多用陈述性、客观性表达，避免第一人称过度出现。
- **算法/伪代码**：变量与正文符号一致，步骤可复现，必要时正文解释关键步骤。

## 十一、自查表

- [ ] 首页完成"问题→不足→贡献→证据→意义"。
- [ ] 中英文摘要均达标，英文可独立阅读。
- [ ] 关键词中英文齐备、与摘要一致。
- [ ] 量与单位、正斜体、符号全文统一。
- [ ] 图表清晰、编号连续、图注自足。
- [ ] 参考文献著录统一、中外文兼顾、基金编号准确。
- [ ] 创新高度可辨（换核检验通过）。

## 输出格式

```text
[首页弧线] 问题/不足/贡献/证据/意义——是否齐全
[双语摘要] 中/英文摘要是否均达标，英文可独立阅读？
[规范] 量与单位/正斜体/图表/公式——是否统一
[著录] 参考文献格式统一？中外文兼顾？基金编号准确？中图分类号(待核实)？
[创新表达] 换核检验是否通过；delta 是否一句话讲清
[待核实] GB/T 7714 强制与否、中图分类号、彩图费
```
