# 《中国科学：信息科学》选题与栏目定位（Topic Selection）

动笔前判断"该不该投《中国科学：信息科学》(SCIENTIA SINICA Informationis, SSI)、投哪个栏目"，
是回报最高的决定。SSI 是国家级大信息学科综合中文旗舰，定位高、以**重大原创与评述**为主，
强调**科学价值与创新高度**。投错刊或选错栏目，往往在初审/外审就被退。本技能给出可操作的
判断框架。

## 一、SSI 是否合适：三问

1. **重要性**：问题对信息科学（计算机/控制/通信/微电子等）是否有实质意义，而非小修小补？
2. **创新高度**：与国内外已有工作的差异（delta）是否显著？是否够得上"国家级旗舰"的门槛？
3. **科学价值**：是否有扎实的实验或理论依据支撑，经得起严格外审？

三问都为"是"→ SSI 合适；若创新高度或重要性不足，考虑兄弟刊。

## 二、换核检验（判断创新是否扎实）

- 把论文的**核心技巧换成一个强基线**，核心科学结论是否仍成立且有意义？
  - **仍成立** → 贡献扎实，适合本刊。
  - **不成立/无意义** → 可能是"技巧包装"，创新高度存疑，需重新审视或改投。
- 另一问：能否用一两句让非本子方向的信息学科同行理解创新点？不能，说明定位或表达有问题。

## 三、中文 SSI vs 英文姊妹刊 SCIS

| 维度 | 《中国科学：信息科学》SSI（中文） | Science China Information Sciences SCIS（英文） |
|---|---|---|
| 语言 | 中文（中英文双语摘要） | 英文 |
| 主办/学科 | 与 SCIS 一致 | 与 SSI 一致 |
| 关系 | **相对独立的两个刊物**，应择一投稿 | 同左 |
| 读者 | 国内为主、面向前沿 | 国际读者 |

- **择一投稿**：不可中英文一稿两投；互转/双语发表政策**以编辑部当期说明为准（待核实）**。
- 面向国内读者、希望中文首发的重大成果 → SSI；追求国际读者与英文可见性 → SCIS。

## 四、SSI vs 兄弟刊分流

| 成果特征 | 更合适的刊 |
|---|---|
| 大信息学科综合、重大原创或高水平综述 | **SSI** |
| 计算机领域深度专题 | 计算机学报 / 软件学报 |
| 控制/自动化专题 | 自动化学报 |
| 电子/器件专题 | 电子学报 |
| 通信/网络专题 | 通信学报 |

- 兄弟刊在各自子学科更聚焦；SSI 覆盖面宽、门槛高、以重大原创与评述见长。
- 增量性、子学科专门化的工作，投兄弟刊往往更匹配、更易被理解。

## 五、栏目定位

| 栏目 | 适合的成果 | 关键约束 |
|---|---|---|
| 评述 | 对某方向有长期积累、能提出独到判断与未来方向 | 约 20 页，一般编委邀请/提议，宜先沟通 |
| 论文 | 完整原创研究，科学思想+创新点+实验/理论依据+应用前景 | 主体栏目 |
| 快报 | 重要且亟需优先发表的核心成果 | ≤4 页，优先发表 |
| 学术介绍 | 客观介绍院校/实验室/重大课题 | 介绍性 |

- 评述非"文献综述堆叠"，须有观点；投前建议与相关编委沟通选题。
- 快报是"重要快讯"，不是"半成品论文"；4 页内须自洽。

## 六、大信息学科的适配考量

SSI 覆盖计算机、控制、通信、微电子等多个子学科，选题时还要考虑"综合刊的读者"：

- **跨子学科可读性**：本刊读者与审稿人来自大信息学科的不同方向，若选题过于依赖某子方向的
  背景才能理解其价值，需在写作上补足动机与意义，否则容易被判"意义不清"。
- **学科交叉是加分项**：本刊定位强调促进各学科、各行业的交叉融合，兼具理论深度与应用桥梁
  价值的选题，往往更契合本刊定位。
- **重大原创优先**：相较子学科专刊，SSI 更看重能推动学科整体发展的重大原创成果与高水平评述；
  纯粹的工程实现或增量改进，即便扎实，也可能被认为更适合兄弟刊。
- **应用前景**：论文栏目要求交代应用前景，选题时就应想清楚"这项成果对信息科学与产业意味着
  什么"，而非仅停留在指标提升。

## 七、决策流程

1. 用三问评估重要性/创新高度/科学价值。
2. 做换核检验，确认创新扎实。
3. 定中文 SSI 还是英文 SCIS（择一）。
4. 若创新高度不足或高度子学科专门化，评估兄弟刊。
5. 选栏目；若评述，先与编委沟通。
6. 形成"一句话贡献 + 目标刊 + 目标栏目 + 依据"，进入写作。

## 八、常见误判

| 误判 | 纠正 |
|---|---|
| 增量工作硬投 SSI | 评估兄弟刊，或补强创新高度 |
| 中英文一稿两投 SSI 与 SCIS | 择一；互转政策待核实 |
| 把综述堆叠当评述 | 评述须有观点与未来方向 |
| 半成品投快报 | 快报须核心自洽、4 页内 |
| 技巧包装当原创 | 换核检验不过则重新定位 |

## 九、自查表

- [ ] 三问（重要性/创新高度/科学价值）均为"是"。
- [ ] 换核检验通过，创新扎实。
- [ ] 已定中文 SSI 或英文 SCIS（择一，不两投）。
- [ ] 与兄弟刊的分流已评估。
- [ ] 栏目已选；评述已考虑与编委沟通。
- [ ] 形成一句话贡献 + 目标刊 + 栏目 + 依据。

## 十、每次核实

- 中英文姊妹刊择一投稿/互转/双语发表的当期政策（**待核实**）。
- 评述栏目的邀请/提议机制细则（**待核实**）。
- 本刊在 CCF 推荐中文科技期刊目录中的层级（**待核实**）。

## 输出格式

```text
[是否 SSI] 三问结论 + 换核检验结果
[中/英] SSI（中文）还是 SCIS（英文）——择一理由
[分流] 是否更适合兄弟刊（哪一本）
[栏目] 评述/论文/快报/学术介绍 + 依据
[产出] 一句话贡献 + 目标刊 + 栏目
[待核实] 择一/互转政策、评述机制、CCF 层级
```
