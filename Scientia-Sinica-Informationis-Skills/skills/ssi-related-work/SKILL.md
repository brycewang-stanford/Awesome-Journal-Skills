# 《中国科学：信息科学》相关工作与文献综述（Related Work）

在《中国科学：信息科学》(SCIENTIA SINICA Informationis, SSI)，相关工作不是"礼节性罗列"，
而是论证**创新高度**的关键：审稿人据此判断你与国内外已有工作的差异（delta）是否显著。
本刊定位高、以重大原创与**评述**为主，因此无论是论文的相关工作节，还是评述栏目的系统综述，
都要求客观、充分、有观点。以下给出可操作写法。

## 一、差异优先（delta-first）

- 每谈一类相关工作，落脚到**它与本文的差异**：别人做了什么、局限在哪、本文如何不同/更进一步。
- 用一两句能讲清"本文首次/首要的不同"，让非本子方向的信息学科读者也能抓住创新点。
- 避免"综述式罗列 + 一句'但他们没做我们的'"的偷懒写法；差异要具体到问题、方法或结论层面。

## 二、国内外文献均衡覆盖

SSI 是中文刊但面向国际前沿，文献覆盖要**双向均衡**：

- **不要只引英文忽略国内工作**：相关中文权威刊（计算机学报、自动化学报、电子学报、
  通信学报、软件学报等）与本刊已有相关论文应被恰当引用。
- **不要只引中文忽略国际前沿**：国际顶刊/顶会的代表性工作必须覆盖，体现对前沿的把握。
- 覆盖近三年最新进展，避免文献陈旧给人"跟进不足"的印象。

## 三、按技术脉络组织

- 以**问题/方法维度**分组，而非按时间流水账或按作者堆叠。
- 每组给出该路线的核心思想、代表工作、共同局限，再引出本文定位。
- 组间有逻辑递进，最终收敛到"本文所处空白"。

## 四、评述栏目 vs 论文相关工作节

| 维度 | 论文相关工作节 | 评述栏目 |
|---|---|---|
| 目的 | 定位本文贡献 | 为学科绘制有观点的地图 |
| 篇幅 | 紧凑（1 节） | 系统（约 20 页） |
| 要求 | 差异优先、服务本文主张 | 全面、分类、提出独到见解与未来方向 |
| 机制 | 随论文投稿 | 一般由编委邀请/提议 |

- 写评述时，综述不能只是转述，要有**作者的判断与预测**（本刊评述的核心价值）。

## 五、自引与利益关系

- 自引应**相关且必要**，不为提升引用数硬凑；过度自引会被审稿人察觉。
- 客观评价竞争性工作，不贬低、不吹捧；对存在合作/利益关系的工作保持中立表述。
- 若本文是作者前期工作的延伸，说明**新增贡献**，避免与已发表内容重复发表。

## 六、参考文献著录

- 著录格式统一（GB/T 7714 与否**以当期须知为准，待核实**），中外文文献完整。
- 正文引用序号与文献表一一对应；避免"引而不列"或"列而不引"。
- 优先引用**正式发表**的可追溯文献；预印本/网络资源注明获取方式与日期。

## 七、常见失误

| 失误 | 后果 | 纠正 |
|---|---|---|
| 只罗列不讲差异 | 创新高度无法体现 | 每类工作落到 delta |
| 只引英文/只引中文 | 覆盖失衡、显得片面 | 国内外文献兼顾 |
| 文献陈旧 | 被判跟进不足 | 补近三年进展 |
| 过度自引 | 影响客观性印象 | 自引限于相关必要 |
| 评述只转述无观点 | 达不到评述栏目要求 | 提出判断与未来方向 |
| 引而不列 / 列而不引 | 著录不规范 | 逐条核对对应 |

## 八、写作流程

1. 先列出与本文最相关的国内外代表工作（问题/方法维度分组）。
2. 每组写清核心思想、代表工作、共同局限。
3. 收敛到本文所处空白，明确 delta。
4. 补齐近三年最新进展与国内相关工作。
5. 核对自引必要性与著录格式一致性。
6. 与引言的贡献陈述对齐，确保"新在哪里"前后一致。

## 九、自查表

- [ ] 每类相关工作都落到与本文的差异（delta）。
- [ ] 国内外文献均衡覆盖，含近三年进展。
- [ ] 组织按问题/方法维度，逻辑递进。
- [ ] 自引相关且必要，无过度自引。
- [ ] 评述栏目体现作者判断与未来方向。
- [ ] 著录格式统一，引用与文献表一一对应。

## 十、差异段落范例（弱 vs 强）

同一处相关工作，"讲清 delta"与否天差地别：

- **弱**："文献[8-12]研究了边缘推理加速。与他们不同，我们的方法更好。"
  —— 未分组、未点局限、差异空泛，审稿人无法判断创新高度。
- **强**："现有边缘推理加速工作大致分两类：一类以吞吐量为目标（文献[8,9]），依赖动态调度，
  但时延长尾，难以满足实时控制；另一类追求确定性时延（文献[10]），却牺牲能效或通用性。
  本文的不同在于**首次**在满足可证明时延上界的同时保持有竞争力的能效，关键是将计算与访存
  解耦的静态调度（第 3 节）。"
  —— 分组 + 点局限 + 明确 delta + 指向本文方法，创新高度一目了然。

原则：**分类 → 各类共同局限 → 本文所处空白 → 一句话 delta**。

## 十一、与国内工作衔接的提醒

- 检索时同时覆盖 CNKI（中文）与国际数据库，避免遗漏国内课题组的相关成果。
- 若本文属某国家级项目/基金资助方向，相关国内进展尤应引用到位，体现学科脉络。
- 若同一方向已在英文姊妹刊 Science China Information Sciences (SCIS) 或本刊有相关论文，
  应恰当引用并说明本文的新增贡献，避免自我重复。

## 十二、每次核实

- 参考文献是否严格 GB/T 7714（**待核实**）。
- 预印本/网络资源的著录要求（**待核实**）。
- 评述栏目的具体篇幅与邀请机制细则（**待核实**）。

## 输出格式

```text
[定位] 每类工作是否落到 delta；本文空白是否清晰
[覆盖] 国内外均衡？近三年进展是否补齐？
[组织] 按问题/方法维度分组、逻辑递进？
[自引] 是否相关必要、无过度自引
[栏目] 论文相关工作节 / 评述——要求是否匹配
[著录] 格式统一？引用与文献表一一对应？(GB/T 7714 待核实)
```
