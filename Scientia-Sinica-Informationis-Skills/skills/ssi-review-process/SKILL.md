# 《中国科学：信息科学》审稿流程（Review Process）

理解《中国科学：信息科学》(SCIENTIA SINICA Informationis, SSI) 的审稿流程，能帮助你校准
预期、在正确的环节使力。SSI 由《中国科学》杂志社出版，坚持**公平、公正、公开、客观**的
审稿原则；部分信源提及"三审三校"制度（**报道级，以官网当期说明为准**）。以下把流程拆解为
可操作的阶段，具体轮次与时限**以官网与退修/录用信为准（待核实）**，不臆造周期数字。

## 一、流程总览（典型中文期刊链路）

```text
投稿 → 编辑部初审（形式/范围/查重/学术规范）
     → 编委分派 → 外审专家评议（通常多位）
     → 作者退修（大修/小修）→ 修改稿再审
     → 复审 / 编委意见
     → 主编或编辑委员会终审
     → 录用 / 退稿
     → 定稿校样（清样，三校）→ 见刊
```

- "三审"通常指初审、外审/复审、终审；"三校"指见刊前的多轮校对。**该表述为报道级，待核实。**

## 二、各阶段与作者空间

| 阶段 | 谁在做 | 关注什么 | 作者可行动空间 |
|---|---|---|---|
| 初审 | 编辑部 | 是否符合范围、格式、查重、学术规范 | 投前把关（栏目、双语摘要、著录、查重）可避免被退 |
| 送外审 | 编委 | 匹配合适专家 | 无法直接干预；准确的栏目与关键词有助匹配 |
| 外审 | 外审专家 | 科学价值、创新高度、严谨性、可复现性 | 事前把证据做扎实；事后靠修回答复 |
| 退修 | 作者 | 逐条回应意见、修改稿 | **最大杠杆**：见 ssi-author-response |
| 复审 | 原专家/编委 | 修改是否到位 | 保证正文与答复一致 |
| 终审 | 主编/编委会 | 综合判断是否录用 | 编委意见权重高，需认真对待 |
| 校样 | 编辑部+作者 | 排版、著录、错字 | 见 ssi-camera-ready |

## 三、评审关注的核心（本刊门槛）

SSI 定位高、以重大原创与评述为主，外审通常围绕：

- **科学价值**：问题是否重要、对信息科学是否有实质意义。
- **创新高度**：相对国内外已有工作的差异（delta）是否显著，而非增量刷点。
- **严谨性**：实验/理论依据是否充分、对比是否公平、结论是否被证据支撑。
- **可复现性**：方法与实验是否可被他人重复。
- **表达**：中文写作规范、中英文双语摘要、逻辑清晰。

不达"科学价值 + 创新高度"门槛的稿件，常被建议改投更合适的兄弟刊（如计算机学报、
自动化学报、电子学报、通信学报、软件学报）。

## 四、决定类型与含义

| 决定 | 含义 | 下一步 |
|---|---|---|
| 录用 | 达到发表标准 | 进入定稿校样 |
| 小修 | 基本认可，细节需完善 | 快速修回，通常一轮 |
| 大修 | 有价值但存实质问题 | 认真修回 + 补证据，可能多轮 |
| 退稿 | 未达标或不适合本刊 | 据意见改投兄弟刊/英文刊 |

- 大修不等于录用；把它当作严肃的 revise-and-resubmit，用证据回应质疑。

## 五、90 天条款与周期

- **重要条款**：作者若在 **90 天**内未收到编辑部关于稿件的具体意见，在通知编辑部后可改投他刊
  （据官网《投稿须知》）。这是作者的一项明确权利。
- 具体审稿周期均值、各轮时限**待核实**，不臆造数字；可礼貌查询稿件状态但避免频繁催促。

## 六、与英文姊妹刊评审的异同

- SSI（中文）与英文姊妹刊 **Science China Information Sciences (SCIS)** 主办、学科一致，
  评审都重科学价值与创新，但**语言、编委分工、周期、具体流程可能不同**，且为**相对独立的
  两个刊物**。不要用一刊的审稿经验直接套用另一刊，**以各自当期说明为准（待核实）**。

## 七、作者常见误判

| 误判 | 纠正 |
|---|---|
| 把大修当作已录用 | 大修是严肃再审，需补证据 |
| 忽略编委意见只回外审 | 编委意见权重高，单独认真回应 |
| 投错栏目/范围被初审退 | 投前对照栏目与本刊范围 |
| 频繁催稿 | 依 90 天条款理性查询 |
| 创新高度不足硬投 | 评估是否改投兄弟刊 |

## 八、如何理性查询稿件状态

在系统与 90 天条款框架内查询，避免频繁催促：

```text
尊敬的编辑部：
  您好！本人于 <日期> 投稿至《中国科学：信息科学》（稿号 <XXXX>），
  至今约 <N> 天。谨礼貌咨询稿件当前处于哪一评审阶段，是否需要作者补充材料。
  感谢您的辛勤工作。
  作者 <姓名>，<联系方式>
```

- 一般在合理周期后（或临近 90 天）再咨询；语气礼貌、只问状态、不施压。
- 若确已超过 90 天仍无具体意见，可依《投稿须知》条款，在通知编辑部后考虑改投。
- 记录每次沟通时间与答复，便于后续判断。

## 九、投稿后到录用前的自我校准

- 大修到手：先冷静，把它当作"有机会但需补强"，按 ssi-author-response 系统回应。
- 退稿到手：从意见中提取可迁移的改进点，评估改投英文姊妹刊 SCIS 或兄弟刊。
- 长时间无音讯：依 90 天条款理性处理，勿反复催稿影响关系。

## 十、每次核实

- "三审三校"的确切制度表述与轮次（**报道级/待核实**）。
- 审稿周期均值、外审专家人数、终审机制（**待核实**）。
- 各类决定的具体命名与修回时限（**待核实**）。

## 输出格式

```text
[当前阶段] 初审/外审/退修/复审/终审/校样
[决定类型] 录用/小修/大修/退稿
[核心关注] 科学价值/创新高度/严谨性/可复现性——短板在哪
[作者行动] 本阶段最高杠杆动作
[条款] 是否触及 90 天未获意见可改投
[待核实] 三审三校表述、周期、终审机制
```
