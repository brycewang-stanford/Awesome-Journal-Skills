# 《中国科学：信息科学》审稿意见答复与修回（Author Response）

收到《中国科学：信息科学》(SCIENTIA SINICA Informationis, SSI) 的退修通知后，
你需要提交两份材料：**修改稿**与**审稿意见答复说明（point-by-point）**。SSI 坚持
公平、公正、客观的审稿原则，编委与外审专家的意见是决定录用的关键；一份条理清晰、
有理有据、态度诚恳的答复，往往与修改本身同等重要。以下流程以本刊中文修回习惯为准，
具体轮次与时限**以退修信与官网当期说明为准（待核实）**。

## 一、动笔前：拆解意见清单

1. 通读**所有**审稿人与编委意见，逐条编号（如 R1-1、R1-2、R2-1、编委-1）。
2. 为每条打标签：**实质性科学质疑 / 实验或证据补充 / 写作与表述 / 格式规范 / 疑似误解**。
3. 标出**相互矛盾**的意见（如一位要求精简、一位要求补充）与**编委的倾向性意见**（权重更高）。
4. 排出修改工作量与时间表，确保在退修时限内完成（时限**待核实**，逾期需提前向编辑部说明）。

## 二、答复信结构

```text
尊敬的编委、各位审稿专家：
  感谢对本文的审阅与宝贵意见。我们已逐条修改，修改处在修改稿中以<颜色/标记>标出。
  以下逐条答复（【意见】原文摘录，【答复】说明修改，【位置】页/行/图表号）。

== 审稿人 1 ==
【意见 R1-1】（原文摘录）
【答复】我们同意/我们做了如下澄清……已在正文第 X 页第 Y 段补充……
【位置】第 X 页，第 Y 段 / 图 Z / 式(n)

……（逐条）……

== 编委意见 ==
【意见】…【答复】…【位置】…

  再次感谢各位专家使本文质量得到提升。
```

## 三、逐条回应的三种结局

对每条意见，明确落到以下之一，不含糊其辞：

- **已修改（采纳）**：说明改了什么、改在哪里，给出新增证据/图表编号；这是首选。
- **已澄清（可能误解）**：礼貌指出原文其实已有说明，并**为消除误解主动优化表述**——
  不要仅说"审稿人误解了"，而要改进正文使其不易被误读。
- **保留并说明（有据不改）**：当要求超出研究范围或有充分理由不采纳时，礼貌、有据地解释，
  必要时补充讨论或作为未来工作，并感谢建议。切忌直接拒绝或忽略。

## 四、回应质疑的原则

- **用数据与理由，不用情绪**：面对尖锐质疑，补实验、补分析、补引用，而非辩解。
- **能补则补**：SSI 看重科学价值与创新高度；对"创新性不足""对比不充分"的质疑，
  优先以新增公平对比、消融、理论分析回应。
- **诚实面对局限**：确有不足就承认并给出边界或改进方向，反而增加可信度。
- **相互矛盾的意见**：择善而从并说明取舍依据；必要时请编委裁量，措辞中立。
- **不扩大战线**：只回应被问到的问题，避免顺带引入新的争议点。

## 五、修改稿的标注

- 修改处以**清晰可辨的标记**（颜色/下划线/修订模式）标出，便于专家复核。
- 答复中的【位置】与修改稿实际位置**一一对应**，页/行/图表号准确。
- 若因修改导致图表/公式重新编号，在答复开头统一说明编号变化，避免专家迷路。
- 同步更新中英文摘要、关键词、参考文献，保证前后一致。

## 六、常见失误

| 失误 | 后果 | 纠正 |
|---|---|---|
| 只在答复里说改了，正文没真正改 | 复审发现，信任崩塌 | 确保正文与答复一致 |
| 对质疑辩解而不补证据 | 被判创新/严谨性不足 | 补实验/分析/引用 |
| 忽略编委意见 | 直接影响终审 | 编委意见优先、单列回应 |
| 语气生硬或居高临下 | 损害评审关系 | 保持中文学术礼节，先致谢 |
| 逐条位置对不上 | 专家难复核，拖慢轮次 | 核对每条【位置】 |
| 修改引入新错误/新争议 | 触发新一轮大修 | 修改后整体复读一遍 |

## 七、多轮修回

- SSI 可能有多轮修回（大修后再小修）。**每一轮都写完整答复**，并延续上一轮的编号体系。
- 第二轮起，重点回应"上一轮遗留"与"新提出"的意见，避免重复已解决条目。
- 若某审稿人持续不满而你已尽力，礼貌陈述证据并请编委综合裁量。

## 八、时限与礼节

- 在退修时限内提交；确需延期，**提前**向编辑部说明并申请（时限**待核实**）。
- 通篇保持诚恳、专业、简洁的中文学术表达；开头致谢、结尾再谢。
- 注意：本刊与英文姊妹刊 Science China Information Sciences 修回习惯相通，
  但以本刊中文流程与退修信要求为准，勿照搬英文刊的模板措辞。

## 九、答复范例对照（好 vs 差）

同一条意见"创新性相对已有方法不足、缺少与最新工作的对比"：

- **差的答复**："我们的方法是新颖的，审稿人的说法不准确。已有方法无法解决我们的问题。"
  —— 无证据、有对抗情绪、未补对比，几乎必然触发新一轮大修或退稿。
- **好的答复**："感谢指出。我们补充了与近三年 3 个代表性方法（文献[12,15,18]）在相同数据集与
  评测协议下的对比（新增表 3 与图 4），结果表明本文方法在<指标>上显著更优（报告均值与置信区间）；
  并在第 1 页明确了本文与上述工作的差异（delta）：本文首次在<约束>下给出<可证明性质>。
  相关讨论见修改稿第 X 页第 Y 段。"

原则：把"我们更好"的断言，转化为"在同一协议下的公平对比 + 明确的差异陈述 + 位置指引"。

面对"英文摘要质量不足"这类规范性意见，也应正面回应：说明已重写英文摘要使其可独立阅读，
并列出修改前后的关键差异与位置，而非仅回复"已修改"。SSI 重视中英文双语摘要，规范性意见
同样影响录用印象，不可轻视。

## 输出格式

```text
[修回类型] 大修 / 小修 / 第 N 轮
[意见清单] 共 M 条（R1:x, R2:y, 编委:z），矛盾项：<列出>
[逐条状态] 已修改 a 条 / 已澄清 b 条 / 保留并说明 c 条
[新增证据] <新实验/分析/图表编号>
[一致性] 正文—答复位置一一对应？摘要/参考文献同步？
[时限] 截止 <日期>，是否需申请延期
```
