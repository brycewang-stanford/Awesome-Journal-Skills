# Clinical Medicine Breadth Resources

This layer supports routing across specialty clinical journals. Use it when a
manuscript needs triage by clinical domain, study design, reporting guideline,
patient population, and likely reviewer standard before selecting a journal.

## What to Load

- `official-source-map.md` - official-source anchors for current author
  guidelines and journal pages.
- `source-basis.md` - coverage basis, evidence hierarchy, and verification
  discipline for the bundle.
- `worked-examples/venue-routing.md` - example routing diagnoses for trials,
  observational studies, imaging, oncology, psychiatry, surgery, and specialty
  clinical manuscripts.
- `exemplars/selection-patterns.md` - routing archetypes and sibling-journal
  traps for clinical venue selection.

## Usage Rule

Start with `en-clinmed-journal-workflow` when the specialty or article type is
uncertain. Then load only the most plausible venue cards and verify current
reporting requirements such as CONSORT, STROBE, PRISMA, ARRIVE, data-sharing,
ethics, and trial-registration expectations before final advice.
