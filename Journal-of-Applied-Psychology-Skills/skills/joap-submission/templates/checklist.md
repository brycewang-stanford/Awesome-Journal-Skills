# JAP Submission Preflight Checklist

Copy-paste this and tick each box before uploading to the **Journal of Applied Psychology (JAP)** via
**Editorial Manager** (`editorialmanager.com/apl`). Verify volatile items (length caps, abstract limit,
TOP level, the APA submission/masking checklist contents, current editor, any OA APC) on the official
APA page — they change (检索于 2026-06；以官网为准). See `../SKILL.md` (`joap-submission`) for the full
walkthrough and `../../joap-open-science-and-transparency/SKILL.md` for the data/materials/code deposit.

## Article type & length
- [ ] Type chosen: Feature Article (length commensurate with contribution) / Research Report (tighter cap, 待核实) / theory-development / integrative review / qualitative / meta-analysis
- [ ] Length appropriate to the chosen type; reduction to a Research Report considered if asked
- [ ] Structured abstract within the current limit (commonly ~250 words; 待核实)
- [ ] Abstract states the theoretical contribution, design, samples (with levels and N), and key result

## Theory & contribution
- [ ] Theoretical contribution (new mechanism / boundary / integration) stated by ~page 3
- [ ] Each hypothesis follows deductively from the stated mechanism; level of analysis labeled
- [ ] Positioned against the closest prior work, not the field at large
- [ ] Venue boundary clear (JAP, not Personnel Psychology / AMJ / OBHDP / JOB)

## Masking (anonymized review)
- [ ] No author names or affiliations anywhere in the manuscript
- [ ] Self-citations neutralized (no "as we showed in…")
- [ ] No links to personal or university websites
- [ ] Repository / preregistration links anonymized (e.g., OSF anonymized view)
- [ ] Identifying file metadata stripped (document properties, tracked-change authors, comments)
- [ ] Separate, non-anonymous title page prepared as a distinct file

## House style & exhibits
- [ ] APA 7th edition throughout (headings, bias-free language, author-date citations, references)
- [ ] Statistics with effect sizes and confidence intervals; exact p-values; fit indices for SEM
- [ ] Table 1: means, SDs, correlations, reliabilities on the diagonal, level-of-analysis notes
- [ ] Path tables/figures carry CIs and fit indices; indirect effects with bootstrap/Monte Carlo CIs
- [ ] Tables/figures self-contained (title + labels + note + N/levels/units), APA 7th format

## Measurement & analysis rigor
- [ ] Measurement model (CFA fit + reliability) reported before the structural model
- [ ] Construct validity / invariance evidence where the construct is new or contested
- [ ] Common-method-variance remedies declared (procedural + planned statistical)
- [ ] Nesting modeled (ICC[1]/ICC[2], r_wg, appropriate centering and random effects)
- [ ] Sample size justified for the carrying effect at the right level
- [ ] Full disclosure of N-determination, all exclusions, all conditions, all measures
- [ ] Confirmatory vs. exploratory analyses clearly separated

## Open science (TOP)
- [ ] Data deposited + DOI + codebook (or a justified exemption with an access path)
- [ ] Materials (scales, manipulations, instructions) deposited + DOI
- [ ] Analysis code deposited; reproduces every reported value in a fresh session
- [ ] Data-availability / transparency statement drafted per current TOP wording
- [ ] Prior-use disclosure if the dataset overlaps another manuscript
- [ ] Preregistration linked (anonymized); confirmatory/exploratory split consistent

## Final
- [ ] APA-published submission/masking checklist completed at upload
- [ ] All co-authors approve the submission
- [ ] Cover letter / metadata fields completed in Editorial Manager
- [ ] Volatile facts re-verified on the official APA JAP page (检索于 2026-06；以官网为准)
