# Games and Economic Behavior Skills 中文说明

这是面向 *Games and Economic Behavior* (GEB) 的 12 个投稿技能包。GEB 是 Elsevier
旗下、与 Game Theory Society 相关的博弈论专业期刊。

本包不是普通经济学投稿模板，而是面向博弈论及其应用：理论、机制设计、行为/实验博弈论、
严谨的应用模型和数值示例。主编、编辑名单、APC、审稿统计和 Guide for Authors 文字等易变
信息，投稿前必须重新核对官方页面。

## 技能列表

| Skill | 用途 |
| --- | --- |
| `geb-workflow` | 判断下一步该调用哪个 GEB 技能。 |
| `geb-topic-selection` | 判断论文是否真正推进博弈论。 |
| `geb-literature-positioning` | 将结果放到博弈论前沿中定位。 |
| `geb-contribution-framing` | 清楚表述博弈论贡献。 |
| `geb-identification-strategy` | 检查假设、均衡、证明和一般性。 |
| `geb-data-analysis` | 处理实验、模拟或数值示例。 |
| `geb-tables-figures` | 准备收益矩阵、博弈树、结果表和图。 |
| `geb-writing-style` | 打磨 GEB 写作、摘要、符号和 Elsevier 风格。 |
| `geb-replication-and-data-policy` | 规划可选数据/代码共享和 Elsevier 数据声明。 |
| `geb-review-process` | 理解 Editor-in-Charge 与 advisory editor 审稿路径。 |
| `geb-submission` | 进行 Editorial Manager 投稿前检查。 |
| `geb-rebuttal` | 处理 R&R 回复。 |
