# Games and Economic Behavior Skills

Twelve Codex/Claude skills for manuscripts targeted at *Games and Economic
Behavior* (GEB), Elsevier's specialist game-theory journal affiliated with the
Game Theory Society.

This is a game-theory pack, not a generic economics pack. It is tuned for papers
that advance game theory or its applications through theory, mechanism design,
behavioral/experimental work, or rigorous applied modelling. Volatile facts such
as the chief editor, editor roster, APC, review statistics, and guide-for-authors
wording should be checked on the live official pages before submission.

## Skills

| Skill | Use it for |
| --- | --- |
| `geb-workflow` | Route a manuscript to the next GEB skill. |
| `geb-topic-selection` | Test whether the paper advances game theory enough for GEB. |
| `geb-literature-positioning` | Position the result against the game-theory frontier. |
| `geb-contribution-framing` | State the game-theoretic contribution clearly. |
| `geb-identification-strategy` | Audit assumptions, equilibria, proofs, and generality. |
| `geb-data-analysis` | Analyze experiments, simulations, or numerical examples. |
| `geb-tables-figures` | Prepare payoff matrices, game trees, result tables, and figures. |
| `geb-writing-style` | Polish GEB prose, abstract, notation, and Elsevier style. |
| `geb-replication-and-data-policy` | Plan optional data/code sharing and Elsevier data statements. |
| `geb-review-process` | Understand the Editor-in-Charge / advisory-editor review path. |
| `geb-submission` | Run Editorial Manager preflight. |
| `geb-rebuttal` | Respond to referee and editor comments after an R&R. |
