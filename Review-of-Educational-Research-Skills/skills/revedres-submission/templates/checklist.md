# RER Pre-Submission Self-Check (8 sections)

A last-pass checklist for a *Review of Educational Research* (RER) **review / systematic review / meta-analysis**
before submitting via Manuscript Central (`mc.manuscriptcentral.com/rer`). Verify volatile specifics
(submission fee, current editors/contact, exact length limit, policy details) on the official SAGE/AERA
RER pages — 检索于 2026-06；以官网为准.

## 1. Venue fit (is this an RER review at all?)
- [ ] The artefact is a **comprehensive, critical, integrative review** of education research
- [ ] No **original empirical research** of the author's own — unless incorporated into a broader integrative review
- [ ] The field passed the four RER fit tests (importance / maturity / need-for-synthesis / tractable scope)
- [ ] It is not a desk-reject candidate for AERJ / *AERA Open* (original research), *Educational Researcher* (commentary), *Review of Research in Education* (commissioned volume), or *Psychological Bulletin* (psychology)

## 2. The review's contribution
- [ ] An explicit **organizing framework** (taxonomy / model / moderator architecture / tensions / constructs) — not an annotated bibliography
- [ ] The one-sentence **argument about the field** is stated up front, and it **advances theory or policy**
- [ ] Section architecture mirrors the framework's cells
- [ ] Contribution framed as an **advance** (reconciles/reframes), not merely "no prior review"

## 3. Systematic rigor (systematic reviews & meta-analyses)
- [ ] Protocol fixed **before** screening; registered on **PROSPERO/OSF** and cited; deviations flagged
- [ ] **PRISMA flow diagram** present; identification → screening → eligibility → included counts **reconcile**
- [ ] Search reported reproducibly: databases, full strings, dates; grey literature/dissertations handled
- [ ] Dual independent screening; **inter-rater reliability** (κ/% agreement) reported
- [ ] **Risk-of-bias / quality appraisal** applied with an a-priori tool

## 4. Synthesis & robustness
- [ ] Conflicting findings **weighed by estimand + design quality**, not vote-counted
- [ ] Rival perspectives **steelmanned**; emphasis is identity-blind (author's own work bracketed)
- [ ] (Meta) heterogeneity (I²/τ²), moderators, publication-bias diagnostics, and sensitivity analyses reported
- [ ] (Meta) dependent effects modeled (multilevel / robust variance), not ignored
- [ ] (Narrative) strength-of-evidence judged; a drop-the-weakest sensitivity check run

## 5. Exhibits & accessibility
- [ ] PRISMA flow, study-characteristics table (with risk-of-bias), forest/funnel (meta) embedded in the Word file
- [ ] Tables compare only **commensurable** outcomes; finding columns carry a credibility note
- [ ] A conceptual figure renders the framework for an integrative synthesis
- [ ] A broad, cross-disciplinary education reader can follow the manuscript start to finish

## 6. House style & front matter (检索于 2026-06；以官网为准)
- [ ] **APA 7th edition** throughout (citations, references, headings, tables/figures, bias-free language)
- [ ] **Abstract ≤ 150 words** with keywords, in both required locations (separately + page 2 after the blinded title page)
- [ ] **Microsoft Word** file (not PDF)
- [ ] File set: separate **unblinded title page** + APA-7 Author Note; **blinded** main manuscript; **Author Bios**

## 7. Masking & declarations
- [ ] Body **anonymized** for masked review; only **limited-circulation** self-citations (forthcoming/in press/unpublished) anonymized — ordinary published self-cites kept visible
- [ ] Coding dataset, codebook, search strings, and analysis code shared (OSF/repository, DOI) where possible
- [ ] Confirmed not under review elsewhere; AI is **not** listed as an author
- [ ] PRISMA + APA **Reporting Standards** (MARS/JARS) checklists completed and ready to attach

## 8. Volatile re-confirms (检索于 2026-06；以官网为准)
- [ ] Submission **fee** (if any) re-checked on the SAGE/AERA RER page — not asserted from memory
- [ ] Current **editors**, contact email, and Manuscript Central portal specifics re-confirmed
- [ ] Exact **length limit**, abstract rule, and formatting requirements re-confirmed on the RER pages
