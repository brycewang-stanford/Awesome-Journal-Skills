# 财经研究选题（cfe-topic-selection）

## 触发时机

- 用户说"我想冲《财经研究》，但选题不确定"
- 边际贡献写不出来 / 写了一段但和审稿人能 buy 的角度不在一个频道
- 不确定该投《财经研究》、《经济研究》还是《管理世界》
- 有数据有方法，但说不清"为什么这个问题在中国值得做"

## 三个硬约束

《财经研究》对选题的硬约束：

1. **综合财经 + 理论贡献**：选题须落在财政、产业、金融与公司财务、国际贸易、劳动、区域与发展、数字经济等方向之一，并能挂上某一支经济学 / 财经理论
2. **回应中国现实**：选题必须能在第一段说明"为什么这个现实问题在当下中国值得做"——这是《财经研究》比纯理论期刊更看重的一点
3. **边际贡献可枚举**：能拆成 3–5 条独立的、互不重叠的贡献

## 理论贡献 + 现实性自检

写完一句话摘要后，逐条自检：

- [ ] 本文挂在哪一支理论文献上？（产业组织 / 公司金融 / 公共财政 / 货币金融 / 国际贸易 / 劳动经济 / 制度经济 / 数字经济）
- [ ] 该理论文献的代表性 Top-5 论文是哪 2–3 篇？
- [ ] 本文相对该理论的边际差异是什么？（修正 / 扩展 / 引入新机制 / 提供首个证据）
- [ ] 中国情境提供了海外文献没有的**识别机会**或**制度特征**吗？（如某项政策改革、行政分权、特有市场结构）
- [ ] 一句话能说清"这个问题不解决会带来什么现实后果"吗？

少于 4 个 yes → 选题深度或现实性不够，回到理论文献与政策背景再读两轮。

## 边际贡献四句式

```
本文的边际贡献主要体现在以下四个方面：
第一，[理论维度]——本文将……纳入[理论框架]，区别于[已有研究]……。
第二，[识别策略维度]——本文利用[政策 / 制度冲击]构造……，缓解了……内生性问题。
第三，[机制维度]——本文揭示了……机制，回答了已有文献"知其然不知其所以然"的问题。
第四，[现实意义维度]——本文为理解 / 应对[中国某项重要现实议题]提供了……证据。
```

四条都齐才算合格。少于四条 → 选题深度不够。注意第四条要落到**中国现实**，不要写成空泛的"丰富了文献"。

## 期刊匹配判断

|              | 财经研究 | 经济研究 | 管理世界 |
|----------|--------|--------|------------|
| 学科        | 综合财经 | 经济学   | 管理 + 应用经济 |
| 选题取向    | 实证 + 中国现实 | 理论贡献优先 | 实践契合 |
| 理论贡献    | **必备**（服务现实问题） | **必备** | 可选 |
| 案例 / 质性 | 较少     | 较少     | 接受 |
| 结构估计 / 纯理论 | 较少 | 较多 | 较少 |
| 政策含义    | 意义层    | 意义层   | 操作层 |

如果稿子偏纯理论 / 结构估计，《经济研究》或《经济学（季刊）》可能更合适；如果偏管理 / 案例 / 可操作政策，不要投《财经研究》。

## 输出格式

```
【方向归属】财政 / 产业 / 公司财务 / 金融 / 贸易 / 劳动 / 区域 / 数字经济
【理论贡献评分】X / 4
【中国现实性】强 / 中 / 弱
【边际贡献四句】（草稿）
1. ...
2. ...
3. ...
4. ...
【期刊匹配建议】财经研究 / 经济研究 / 管理世界 / 其他
【下一步】cfe-literature-review
```

## 《财经研究》操作性加固

把本 skill 当作一次可验收的投稿前审查，而不是泛泛润色。先锁定核心问题、识别链条、机制证据和可执行的政策含义，再判断当前稿件是否真的满足《财经研究》的读者预期：中文财经学术审稿人会同时追问选题政策价值、识别可信度和本刊栏目适配。

- **Fit pass**：Score the manuscript on venue fit, novelty, evidence readiness, and audience ownership; reject a prestige-only target when a sibling venue owns the contribution more directly.
- **证据账本**：输出“主张-证据-风险”三列；每条建议必须指向正文段落、表图、附录或待补材料。
- **姊妹刊护栏**：若稿件更像《经济研究》用于更强理论/全国性贡献，《管理世界》用于管理实践与政策治理，《金融研究》用于金融专门议题，必须说明为什么仍投《财经研究》，否则给出改投路径。
- **停止条件**：若识别、机制、数据可复现或官方投稿要求没有被核实，不给“可以投稿”的结论。
