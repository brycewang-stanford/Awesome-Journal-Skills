## 用法

- **农业农村政策评估是核心。** 本刊偏好"真实三农问题 + 农户 / 村庄层面准实验识别 + 机制 +
  异质性"的实证范式：先讲现象与问题，再讲识别，机制与异质性互证，最后落到可操作的政策
  启示（`../../skills/cre-identification`、`../../skills/cre-mechanism`、
  `../../skills/cre-heterogeneity`、`../../skills/cre-policy-implication`）。
- **正视农户自选择内生性。** 入社 / 务工 / 流转 / 触网 / 采用新技术都是农户自选行为；上表
  的代表作大多以政策分批推行（DID）或中介 / 工具策略来回应选择性，本包 `../code/`
  （多时点 DID、事件研究、IV、聚类与安慰剂）正为此而备。
- **写作落到本刊房风格。** 摘要五句法、机制落到农户行为、避免空泛"意义"句式，见
  `../../skills/cre-abstract`、`../../skills/cre-style`。
- **核对刊名与卷期。** 引用前回到 `zgncjj.ajcass.com` 正刊 PDF 复核，避免与《中国农村
  观察》《农业经济问题》《经济研究》混淆。
