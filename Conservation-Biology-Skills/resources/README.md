# Conservation Biology Resources

Resources for the *Conservation Biology* (Society for Conservation Biology / Wiley) skill pack. The
`skills/` give advice; this directory lets an agent **model a Conservation Biology section** and
**benchmark against verified exemplars**. Pair these with the relevant `skills/conbio-*/SKILL.md`.

## What's here

| Resource | Use it to... |
| --- | --- |
| [`worked-examples/01-introduction.md`](worked-examples/01-introduction.md) | See a before→after rewrite of a paper **abstract + introduction** in *Conservation Biology* house style — conservation problem front-loaded, finding stated, evidence-to-action so-what, the ≤300-word abstract. **Illustrative fictional paper; no real policy.** |
| [`exemplars/library.md`](exemplars/library.md) | Benchmark against **real, web-verified *Conservation Biology* papers** (DOIs in the `10.1111/cobi…` / `10.1111/j.1523-1739…` family) organized by method × topic. Framing positioning only — no fabricated numbers; includes a sibling-journal guardrail. |
| [`official-source-map.md`](official-source-map.md) | **Conservation Biology-specific policy & facts:** article types and word/abstract caps, IMRAD order, ScholarOne submission (`cobi`), double-blind review, Wiley data-sharing tier, OA/ORCID metadata, scope, and upload-week live-check items. The authoritative pack source. |
| [`external_tools.md`](external_tools.md) | External tools and services referenced by the pack (reference managers, conservation data/evidence databases). |

## Scope note — no econometrics code kit vendored

This is a **conservation-science** venue, not an econometrics one. Unlike some economics packs in this
repo, **no causal-inference / econometrics code kit is vendored here** — a `resources/code/` directory of
Stata/Python pipelines would be off-discipline for *Conservation Biology*, whose methods span population
modeling, occupancy/PVA/SDM, spatial prioritization, and conservation social science. Use the
domain-appropriate tools listed in [`external_tools.md`](external_tools.md) instead.

## Suggested workflow

1. Scope and frame with [`../skills/conbio-topic-selection`](../skills/conbio-topic-selection/SKILL.md)
   and [`../skills/conbio-writing-style`](../skills/conbio-writing-style/SKILL.md); model the abstract +
   introduction on [`worked-examples/01-introduction.md`](worked-examples/01-introduction.md).
2. Sharpen the so-what with
   [`../skills/conbio-conservation-relevance-and-implications`](../skills/conbio-conservation-relevance-and-implications/SKILL.md).
3. Benchmark your framing against verified papers in [`exemplars/library.md`](exemplars/library.md).
4. Confirm article type, caps, submission, and data policy in
   [`official-source-map.md`](official-source-map.md).
