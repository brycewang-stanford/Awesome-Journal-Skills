# BJPS Submission Preflight Checklist

Copy-paste this and tick each box before uploading to the British Journal of Political Science (BJPS /
BJPolS) submission portal. Verify volatile items (word caps, ORCID requirement, portal URL, any OA APC)
on the official Cambridge/BJPS pages — they change. See `../SKILL.md` (`bjps-submission`) for the full
walkthrough and `../../bjps-transparency-and-data/SKILL.md` for the replication package.

## Format & length
- [ ] Format chosen: Research Article (~10,000 words) / Letter (~4,000 words) / Comment (待核实 caps)
- [ ] Word count within the chosen format's cap
- [ ] Abstract ≤ 150 words; states question + approach + finding
- [ ] A Letter is a complete short paper (not a truncated Article); a Comment engages a specific BJPS article

## Anonymity (double-blind)
- [ ] No author names or affiliations anywhere in the manuscript
- [ ] No acknowledgments or **funding notes** in the anonymized file
- [ ] No links to personal or university websites
- [ ] No obvious self-references ("as we showed in…"); self-citations neutralized
- [ ] Identifying file metadata stripped (document properties, tracked-change authors, comments)
- [ ] Separate, non-anonymous **title page** prepared as a distinct file
- [ ] Anonymized pre-analysis plan / OSF link if preregistration is referenced

## House style & format
- [ ] **Harvard author-date** in-text citations, consistent throughout
- [ ] Reference list in Cambridge format; references excluded from the word count
- [ ] Manuscript in **MS Word or LaTeX**; double-spaced; ≥ 12-point type; correct margins
- [ ] Tables sized within the Cambridge box (~190 × 120 mm); figures within (~200 × 133 mm) at ≥ 300 dpi (待核实)
- [ ] Tables/figures self-contained (title + labels + note + N/units)
- [ ] Corresponding-author **ORCID** ready (待核实 requirement)

## Transparency & compliance
- [ ] Ethics / consent / human-subjects compliance addressed
- [ ] Replication data + code staged for the **BJPolS Dataverse** (Harvard Dataverse) at acceptance
- [ ] Restricted-access data: exemption note + access instructions + synthetic data where feasible
- [ ] Master script reproduces every table/figure; seeds set; package versions pinned
- [ ] Original work; preprint status checked against Cambridge preprint policy

## Final
- [ ] All co-authors approve the submission
- [ ] Cover letter / metadata fields completed in the portal
- [ ] Volatile facts re-verified on the official BJPS/Cambridge page (检索于 2026-06；以官网为准)
