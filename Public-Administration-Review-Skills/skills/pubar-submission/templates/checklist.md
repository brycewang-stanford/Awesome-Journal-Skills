## 1. Article type & fit
- [ ] Article type chosen: Scholarly Take / Conceptualizing PA / Early Career Intel / Practically Speaking
- [ ] Paper is PAR-shaped (broad PA significance **and** practice-bridging), not JPART (pure theory) or JPAM (policy analysis)
- [ ] Field-wide PA significance stated in the introduction

## 2. Length & abstract
- [ ] Word count **≤ 8,000** including abstract, endnotes, and references
- [ ] Tables, figures, charts, and appendices correctly **excluded** from the count
- [ ] Abstract **≤ 150 words**, on a page preceding the manuscript; states question + approach + finding

## 3. Evidence for Practice (the PAR signature)
- [ ] **3–5** sentence-length takeaway points for practitioners included
- [ ] Each point follows from the analysis (no over-claim, no boilerplate)
- [ ] (Encouraged at submission; **required at revision**)

## 4. Double-blind anonymity
- [ ] No author names on the title page or in the manuscript body
- [ ] Own prior work cited in the **third person**; self-citations neutralized
- [ ] Acknowledgments, institution names, identifying URLs removed
- [ ] Identifying **file metadata** stripped (document properties, tracked-changes authors, comments)
- [ ] Separate, non-anonymous **title page** prepared as a distinct file

## 5. Format & references
- [ ] **APA author–date** formatting; one consistent citation style throughout
- [ ] Tables/figures self-contained, accessible (colorblind-safe, grayscale-legible)
- [ ] Effect sizes + intervals shown (not stars only)

## 6. Transparency (TOP guidelines)
- [ ] Data-availability statement drafted (where data live, or why restricted)
- [ ] Materials staged in a persistent repository — **Dataverse** (quant) / **QDR** (qual)
- [ ] Master script reproduces every table/figure; seeds set; versions pinned
- [ ] Reporting-standard supplement where applicable (CONSORT / STROBE / PRISMA / COREQ)
- [ ] Pre-registration / pre-analysis plan linked (anonymized) where applicable

## 7. Compliance & files
- [ ] Ethics / IRB / human-subjects compliance documented
- [ ] Original work; preprint/prior-dissemination status checked against Wiley policy
- [ ] **Live submission portal confirmed** (Editorial Manager `editorialmanager.com/par` vs. ScholarOne — 待核实)
- [ ] No submission fee assumed unless confirmed; any post-acceptance OA APC noted 待核实
