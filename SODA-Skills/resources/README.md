# SODA Resources

Working materials behind the SODA skill pack. The `skills/` directory holds the
advice; this directory holds what the advice stands on — the verified source
trail, live-surface checklists, attribution-checked exemplars, a front-matter
teaching example, and a pointer to packaging tooling for verification archives.

| --- | --- |

| [`code/README.md`](code/README.md) | Smoke-check a verification archive's packaging with the shared kit, plus the four SODA-specific checks no script performs. |

## Scope note

This is a theory-of-algorithms conference pack. There is no reproducibility
checklist to fill, no supplement ZIP to assemble, and no rebuttal-length
negotiation to game — SODA's mechanics (uncapped full versions, lightweight
double-blind, a three-day rebuttal, SIAM proceedings) are their own system.
Sibling packs for STOC, FOCS, or ML venues describe different machinery; do not
transplant their numbers here.
