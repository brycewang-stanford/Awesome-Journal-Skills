# Journal of Banking & Finance Skills 中文说明

这是面向 *Journal of Banking & Finance* (JBF) 的 12 个投稿技能包。JBF 是
Elsevier 旗下金融期刊，重点覆盖银行、金融中介、资本市场、投资、公司金融与金融监管。

本包适合处理实证银行与金融论文，也兼容有扎实理论基础的金融研究。它覆盖从选题适配、
文献定位、识别策略、数据分析、表格图形，到 Editorial Manager 投稿和 R&R 回复的完整流程。
核心政策事实已于 2026-06-20 以 ScienceDirect / Elsevier 官方页面刷新：Editorial Manager
投稿、350 美元 unsolicited-manuscript 投稿费、双匿名审稿、250 词摘要、必需的数据声明 /
Option C data deposit-or-explain 政策，以及现任 Managing Editor Christa Bouwman。

## 技能列表

| Skill | 用途 |
| --- | --- |
| `jbf-workflow` | 判断下一步该调用哪个 JBF 技能。 |
| `jbf-topic-selection` | 判断选题是否符合 JBF 的银行与金融范围。 |
| `jbf-literature-positioning` | 将论文放到银行、金融中介与金融文献中定位。 |
| `jbf-identification-strategy` | 检查识别策略、内生性处理和因果主张。 |
| `jbf-data-analysis` | 设计样本、估计、稳健性与可复现分析流程。 |
| `jbf-tables-figures` | 准备金融期刊风格的表格、事件研究图和附录展示。 |
| `jbf-contribution-framing` | 把贡献讲清楚，让编辑和审稿人看到 JBF 价值。 |
| `jbf-writing-style` | 打磨精炼的 Elsevier/JBF 写作、编号参考文献准备和结果表达。 |
| `jbf-replication-and-data-policy` | 准备数据可得性声明、代码共享和专有数据说明。 |
| `jbf-review-process` | 理解 desk screen、双匿名审稿和决策路径。 |
| `jbf-submission` | 进行 Editorial Manager 投稿前检查。 |
| `jbf-rebuttal` | 组织 R&R 回复信和修稿策略。 |
