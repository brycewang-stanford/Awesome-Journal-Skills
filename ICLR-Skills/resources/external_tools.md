# External Tools - ICLR

Access date: 2026-06-01

Use these tools only after confirming the current ICLR cycle pages. ICLR rules, template files,
OpenReview forms, and publication logistics can change each year.

## Official workflow

- ICLR 2026 CFP: https://iclr.cc/Conferences/2026/CallForPapers
- ICLR 2026 Author Guide: https://iclr.cc/Conferences/2026/AuthorGuide
- ICLR 2026 Reviewer Guide: https://iclr.cc/Conferences/2026/ReviewerGuide
- ICLR 2026 OpenReview group: https://openreview.net/group?id=ICLR.cc/2026/Conference
- ICLR Code of Ethics: https://iclr.cc/public/CodeOfEthics
- ICLR Code of Conduct: https://iclr.cc/public/CodeOfConduct
- ICLR 2026 LLM policy blog: https://blog.iclr.cc/2025/08/26/policies-on-large-language-model-usage-at-iclr-2026/

## Author-side checks

- OpenReview profile and submission access for every author.
- Current ICLR LaTeX template ZIP from the Author Guide.
- Anonymous repository or supplementary ZIP for code, data, prompts, scripts, and logs.
- PDF-diff review during discussion if a revised manuscript is uploaded.
- Ethics and reproducibility statements placed outside the main-text page count when current rules
  allow that.
- Camera-ready and poster instructions from the accepted-paper emails and iclr.cc pages.

## Do not infer

- Do not infer next-cycle deadlines from ICLR 2026.
- Do not assume the same page limit, reciprocal-reviewing requirement, LLM disclosure field, public
  discussion mechanics, or poster/video logistics without reopening official sources.
- Do not submit non-anonymous demo links or repositories that expose visitor analytics, owner names,
  private organization slugs, grant metadata, or author emails.
