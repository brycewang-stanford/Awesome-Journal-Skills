## 用法

- **学测度叙事与方法定位，不学具体数字。** 看这些论文如何把"测得准 / 分解得清"写成贡献核心，
  如何把指数编制 / 生产率分解嵌进引言（对照 `worked-examples/01-introduction.md` 与
  `../../skills/jqte-fit-positioning/SKILL.md`、`../../skills/jqte-measurement/SKILL.md`）。
- **测度以章祥荪·贵斌威(2008)为准绳。** TFP / 效率测度首选讲清方法假设与分解，而非黑箱跑软件；
  起测度部分前先读 `../../skills/jqte-measurement/SKILL.md`。
- **计量按本刊规范做满。** 时序 / 面板须先验平稳性 + 协整防伪回归，混频用 MIDAS / 桥接，动态面板
  GMM 须报工具有效性，见 `../../skills/jqte-econometric-methods/SKILL.md`；方法规范底线另见
  [`../../../shared-resources/empirical-methods/reporting-standards.md`](../../../shared-resources/empirical-methods/reporting-standards.md)。
- **引用前核对刊名。** 凡记不清是 JQTE 还是《经济研究》/《统计研究》/《中国工业经济》的篇目，
  先回 CNKI 或官网核对，再写进参考文献。

## 排除项（核实为其他venue或暂未核实，勿误记为 JQTE）

- 多数**纯计量方法名篇**与宏观 / 一般均衡题材常发在**《经济研究》**——引用前务必核对刊名，
  不要因为"方法很顶"就默认是《数量经济技术经济研究》。
- 偏**统计指标编制 / 抽样调查 / 官方统计口径**的论文常发在**《统计研究》**。
- 偏**产业组织 / 企业与创新 + 现代因果识别**（多期 DID / 试点准实验）的论文发在**《中国工业经济》**
  （如智慧城市、低碳城市试点类）——本包 `jqte-fit-positioning` 的改投对照表亦如此分流。
- 题名相近但本轮未能逐项核实卷期 / 页码的"数字经济 / 全要素生产率 / 碳排放分解"类论文——一律按
  `待核实 / TO VERIFY` 处理，不入正表。
