# Developmental Psychology Pre-Submission Checklist

A self-check before submitting to **Developmental Psychology** (APA) via **Editorial Manager** at
**editorialmanager.com/dvl**. Re-verify every specific against the current APA author guidelines —
length tiers, abstract ceiling, masked-review options, JARS/TOP wording, and the masthead (editor
Koraly Pérez-Edgar, 检索于 2026-06；以官网为准).

## 1. Fit & article type

- [ ] The contribution is about **developmental change** (age effect, within-person change, trajectory,
      or mechanism), not a single-age correlation
- [ ] Correct **length tier** selected (brief report ~4,500 / larger ~10,500 / multi-study or
      comprehensive longitudinal ~15,000 words — confirm current numbers)
- [ ] Scope is broad lifespan development (vs. a sibling venue: Child Development / Developmental Science)

## 2. Length & format (APA 7th edition)

- [ ] Total length within the chosen tier (counts cover page, abstract, text, references, tables, figures)
- [ ] **APA 7th-edition** style throughout (headings, citations, references, table/figure format)
- [ ] Developmentally precise, bias-free language (specified ages, not "kids")
- [ ] Tables and figures formatted per APA; trajectory/growth exhibits show age/time on the x-axis

## 3. Abstract & Public Significance Statement

- [ ] APA-style **abstract** within the current word ceiling; foregrounds the developmental change
- [ ] **Public Significance Statement** present: **2–3 brief sentences (~30–70 words)**, lay-accessible,
      no overclaim beyond the design, on the abstract/keywords page

## 4. Developmental credibility (JARS)

- [ ] **Age vs. cohort** confound addressed (or age-difference language used honestly)
- [ ] **Measurement invariance** (configural → metric → scalar) reported **before** interpreting change
- [ ] **Attrition** analysis (completers vs. dropouts) + principled missing-data model (FIML / MI)
- [ ] **Sample-size justification** for the change parameter (power for slope/interaction, precision, or
      a decision rule), with the assumed effect size and its source
- [ ] **Effect sizes + confidence intervals** for all major results
- [ ] Full JARS disclosure: sampling, all exclusions and reasons, all conditions and measures
- [ ] **Confirmatory vs. exploratory** analyses clearly separated; no HARKing of trajectory shapes

## 5. Ethics (minors & vulnerable populations)

- [ ] Parental/guardian **consent** and child **assent** documented where applicable
- [ ] IRB / ethics approval stated; special protections for vulnerable populations noted
- [ ] Measures age-appropriate and valid across the age range compared

## 6. Transparency (TOP) & reproducibility

- [ ] **Data-availability statement** present (what is shared, what is restricted, and why)
- [ ] Data + analysis scripts deposited with **DOIs** (or a justified restricted-access archive such as
      ICPSR / Databrary); a **data dictionary/codebook** included
- [ ] Identifiable data from minors handled via permission-based access (e.g., **Databrary**), not posted openly
- [ ] Analysis scripts reproduce the reported numbers in a **fresh session**
- [ ] **Preregistration** linked; confirmatory/exploratory split consistent with it

## 7. Masked review (if requested)

- [ ] Masked manuscript contains **no** author names, affiliations, or identifying acknowledgments
- [ ] Self-citations neutral ("prior research (Author, 2020)"), not "our earlier work"
- [ ] Repository / preregistration links **anonymized** (OSF anonymized view; Databrary view link)
- [ ] File metadata/properties scrubbed of author identity
- [ ] A masked copy is provided via a stable link or supplemental material as the portal specifies

## 8. Files for Editorial Manager

- [ ] Masked main document (if masked review) + a separate title page with authors/affiliations/funding
- [ ] Online supplemental material (robustness, full models, attrition tables, extra studies)
- [ ] Cover letter (fit with the journal, the developmental contribution, not under review elsewhere)
- [ ] High-resolution figure files if requested separately
- [ ] Correct article type / length tier selected so the paper routes appropriately

## 9. Consistency & final preview

- [ ] Hypothesis labels, construct/parameter names, and table/figure numbers consistent throughout
- [ ] Figure values match the deposited analysis script and the in-text estimates
- [ ] Built PDF previewed in Editorial Manager before final submission
