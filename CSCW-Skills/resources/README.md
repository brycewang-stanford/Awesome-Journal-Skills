# CSCW Resources

Working materials for the CSCW skill pack. The `skills/` directory advises; this
directory lets an agent benchmark a draft against CSCW's own canon, model a
group-level introduction, verify the venue's fast-changing submission mechanics, and
handle the computational slice of a paper without mishandling the human slice.

## What's here

| Resource | Use it to... |
| --- | --- |
| [`official-source-map.md`](official-source-map.md) | Trace every venue fact in this pack to a URL and access date (2026-07-08), including the fixed-cycle → rolling transition, decision statistics, and the explicit 待核实 list. |
| [`external_tools.md`](external_tools.md) | Open the live official surfaces (cscw.acm.org, PCS, Manuscript Central, PACMHCI track, dblp) and run five author-side passes before upload. |
| [`exemplars/library.md`](exemplars/library.md) | Benchmark against five verified CSCW papers spanning the Lasting Impact lineage and the PACMHCI era, with era-correct citation forms. |
| [`worked-examples/01-introduction.md`](worked-examples/01-introduction.md) | See a fictional abstract/introduction rebuilt from individual-user framing to group-level CSCW framing, with ethics on page one. |
| [`code/README.md`](code/README.md) | Apply the shared reproducibility kit to the computational slice only, with pointers to what it cannot check about human-subjects material. |

## Scope note

This pack serves a venue whose evidence is often people — interviews, fieldwork,
community traces — published through a journal (PACMHCI), not a proceedings volume.
Do not import quantitative-only reproducibility reflexes: sharing raw qualitative
data is frequently the *wrong* move here, and the resources above are written to
keep the shareable and the protected separate.

## Suggested route

1. Decide the venue and the unit of analysis with
   [`../skills/cscw-topic-selection`](../skills/cscw-topic-selection/SKILL.md), then
   study [`exemplars/library.md`](exemplars/library.md) to see which contribution
   kind you are writing.
2. Design evidence with
   [`../skills/cscw-experiments`](../skills/cscw-experiments/SKILL.md) and draft the
   opening against
   [`worked-examples/01-introduction.md`](worked-examples/01-introduction.md).
3. Before any upload, confirm the current regime (fixed-cycle residue vs. rolling)
   in [`official-source-map.md`](official-source-map.md) and run the passes in
   [`external_tools.md`](external_tools.md).
