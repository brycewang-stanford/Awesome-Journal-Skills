# 中国情境化（nbr-china-context）

## 触发时机

- 中国情境只出现在"样本来自中国某省 N 家企业"
- 文中常说"中国情境下"，但说不清情境**改变了什么**
- 直接套用西方理论，未问其前提在中国是否成立

## 核心标准：情境进入理论，而非作背景

情境化要回答**情境如何改变理论**，三条路径任选其一：
1. **改变构念含义**：同一构念在中国情境下内涵/外延不同（如"关系""面子""家长式领导"）
2. **改变机制**：X→Y 的中介路径或强度因情境（制度、文化、市场发育）而变
3. **改变边界条件**：某理论的适用范围在中国情境下被收窄/扩展/反转

## 把情境做成调节而非标签

- 用**可测的情境变量**（制度环境、市场化程度、文化价值观、所有制、数字化水平）作为调节或对比维度
- 让情境进入**假设与模型**，而不是只在引言/讨论喊口号
- 与既有（含西方主流）理论**平等对话**：继承什么、修正什么、何处因情境而不同

## 两个极端都要避免

- **全盘套用**：把中国当西方理论的"又一个样本"，无情境贡献
- **过度特殊化**：只讲"国情特殊"、拒绝任何一般化，导致无法对话、不可累积

## 情境化深度分级（先自评再动笔）

| 层级 | 形态 | 本刊接受度 |
|------|------|-----------|
| L0 | 中国只是采样地，正文论证与情境无关 | 低——常被审稿人点名"换个国家也成立" |
| L1 | 情境变量只作控制变量或稳健性分组 | 偏低——情境仍在理论之外 |
| L2 | 情境以可测调节变量进入假设与模型 | 合格起点 |
| L3 | 情境重塑构念内涵或机制本身（情境化理论建构） | 最具竞争力，呼应本刊"中国管理实践理论化"取向 |

判定口诀：把稿中"中国"全部替换成"某新兴市场"，若论证毫发无损，说明仍停在 L0–L1，需返工。

## 微型走查：差序氛围与员工建言

设想一篇稿件：用西方建言（voice）理论解释中国制造企业员工为何沉默，发现心理安全感的作用弱于既有文献预期。
- L1 写法：加一个"所有制"控制变量——情境未进入理论，预期被拒
- L2 改法：引入**差序氛围**作可测调节变量，提出假设：差序氛围强时，心理安全感→建言的路径被削弱；测量用本土差序氛围量表（衔接 nbr-measurement）
- L3 改法：论证"建言"在差序情境下分化为"对内圈进言 / 对外圈沉默"两种形态，重构构念维度并提出新命题——这是本刊最欢迎的情境化贡献

## 评审追问与应对

| 评审追问 | 修复动作 |
|----------|----------|
| "换成其它新兴市场是否同样成立？" | 指明机制依赖的**制度/文化要素**（如差序格局、关系治理），并说明其在何类社会可比 |
| "情境变量怎么测？" | 给出已发表的本土量表或客观代理（市场化指数、所有制），并实际进入模型 |
| "是不是在讲特殊论？" | 补一条一般化命题：情境作为调节变量的理论判断可在其它情境中检验 |

## 自检清单

- [ ] 情境改变了构念含义 / 机制 / 边界之一，且说清"改变了什么"
- [ ] 情境以**可测变量**进入假设或模型，不只是背景叙述
- [ ] 与既有理论实质对话（继承/修正/因情境不同各有所指）
- [ ] 提炼出可与国际文献对话的**一般化命题**，非纯特殊性
- [ ] 测量与情境匹配（量表本土化见 nbr-measurement）

## 反模式

- "本研究基于中国情境数据"——情境只在样本句
- 反复说"具有中国特色"却不指明改变了哪个理论要素
- 把任何显著结果都归因为"中国情境使然"，无机理
- 拒绝一般化，使贡献无法进入学术对话

## 已刊文章的情境化呈现（校准锚）

《南开管理评论》已刊实证文章通常在引言即点出情境张力（西方理论预期 vs 中国管理现象），假设部分至少一条涉及情境性调节，讨论部分单列"情境贡献 / 本土管理启示"小节。这一观察用于校准，不是硬性条款；栏目与篇幅细节以期刊最新投稿指南为准。

## 输出格式

```
【情境作用】改变构念含义 / 机制 / 边界
【情境变量】<制度/市场化/文化/所有制/数字化…>（是否可测、是否入模型）
【对话】继承<…> 修正<…> 因情境不同<…>
【一般化命题】<可与国际文献对话的判断>
【是否仅采样地】否（已进入理论）/ 是（需返工）
【下一步】nbr-discussion-contribution
```
