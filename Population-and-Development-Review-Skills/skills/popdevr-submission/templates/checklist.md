# PDR Pre-Submission Self-Check (7 sections)

A last-pass checklist for a *Population and Development Review* (PDR) manuscript before submitting via
ScholarOne Manuscripts. PDR uses **Free Format** for the first round, so keep formatting consistent
rather than perfect — but anonymization, the data statement, and the science must be in order. Verify
volatile specifics (abstract limit, length norms, portal link, fee/Online Open options) on the Wiley
author page. 待核实.

## 1. Fit & article type
- [ ] The paper connects a **population process** (fertility, mortality, migration, ageing, structure) to
      **social, economic, or environmental change** and/or **policy** — broad interest, not a narrow estimate
- [ ] Article type chosen: Research Article / Notes & Commentary / Data & Perspectives / Archives / Book Review
- [ ] Length within the type's norm (Research Article ~8,000–10,000 words; Note short)

## 2. Anonymity (double-anonymized review)
- [ ] All identifying information about **all co-authors** removed from the manuscript file
- [ ] No self-identifying references ("as we showed in…"); self-citations neutralized
- [ ] Identifying **file metadata stripped** (document properties, comments)
- [ ] Separate (non-anonymous) title page prepared as ScholarOne requires

## 3. Front matter & format (Free Format first round)
- [ ] Abstract and keywords present (within the current abstract limit — 待核实)
- [ ] References in one **consistent** style (APA-style house style applied at revision)
- [ ] Figures/tables self-contained, accessible, and tied to the population-and-development point

## 4. Analysis & exhibits
- [ ] Rates/exposure constructed correctly; uncertainty reported (not just point estimates/stars)
- [ ] Decomposition components defined and summing to the total; APC constraint stated if used
- [ ] Cross-country comparability addressed before any contrast is read as a development effect
- [ ] Each exhibit reproducible from the master script; numbers match the deposit

## 5. Transparency & integrity
- [ ] **Data availability statement** drafted (public / restricted / on request) with persistent IDs
- [ ] Materials staged for a **FAIR repository** (DOI/accession), not a personal URL
- [ ] Licensed data (DHS/census) handled by extract code, **not** redistributed raw microdata
- [ ] **ORCID** for the submitting author; funding / conflict-of-interest / ethics declarations complete

## 6. Reproducibility readiness
- [ ] One master script regenerates every table, figure, life table, decomposition, and projection
- [ ] Seeds set and reported for bootstrap/simulation; software/package versions pinned
- [ ] README documents data provenance/vintage and how to reproduce each exhibit

## 7. Fees & portal
- [ ] Confirmed: **no submission fee and no APC** under the standard model (note any Online Open choice)
- [ ] Submitting via **ScholarOne Manuscripts** (current link confirmed on the Wiley author page)
- [ ] On R&R: apply APA-style formatting and keep new exhibits in sync with the deposit
