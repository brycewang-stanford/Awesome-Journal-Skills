# English Natural-Science Breadth Resources

This layer supports routing across broad natural-science, biomedical, clinical,
physical-science, chemistry, mathematics, and engineering journal fit cards.

## What to Load

- `journal-roster.md` - roster of covered journals and stable official anchors.
- `official-source-map.md` - source anchors for live author-guideline checks.
- `source-basis.md` - source discipline and coverage basis.
- `worked-examples/venue-routing.md` - diagnostic routing cases.
- `exemplars/selection-patterns.md` - venue-family patterns and common traps.

## Usage Rule

Use `en-natsci-journal-workflow` for uncertain targets. Then invoke the
single-journal breadth card for quick fit, and only then move into a depth pack
when the repository has one for that venue.
