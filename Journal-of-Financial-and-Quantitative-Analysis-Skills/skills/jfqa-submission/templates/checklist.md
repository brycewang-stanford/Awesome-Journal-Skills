# JFQA Pre-Submission Checklist

Verify volatile specifics on the official pages before relying on them
(see `resources/official-source-map.md`).

## 1. Scope & contribution
- [ ] Question is in JFQA scope (corporate finance / investments / markets / institutions / methods).
- [ ] Quantitative empirical or theoretical contribution is sharp (journal prints < 9% of 1,000+).

## 2. Design & analysis
- [ ] Identification/design defended; inference (clustering / weak-IV / multiple testing) justified.
- [ ] Economic magnitudes reported, not only significance.
- [ ] Robustness and placebos included.

## 3. Abstract & writing
- [ ] Abstract is one paragraph, **≤ 100 words**, states the result with a number.
- [ ] Prose is result-forward; no over-claiming.

## 4. Formatting
- [ ] 8.5 × 11 paper, 1-inch margins, 12-pt Times New Roman, double-spaced (body + appendices).
- [ ] Single **text-searchable PDF**; exhibits embedded, numbered, with self-contained notes.
- [ ] Length not gratuitous.

## 5. Anonymization (double-anonymous)
- [ ] No names/affiliations/acknowledgments in the PDF.
- [ ] Self-citations neutral; PDF metadata scrubbed.

## 6. Disclosures & resubmission
- [ ] Prior JFQA rejection (if any) disclosed in the cover letter — avoids the one-year ban.
- [ ] Conflict-of-interest and data-source disclosures ready.
- [ ] Not under review elsewhere.

## 7. Code/data archive
- [ ] Archive plan ready for acceptance (code + raw/pseudo data + read-me roadmap → JFQA Dataverse).
- [ ] Any code-sharing **exception requested on this initial submission**.

## 8. Portal & fee
- [ ] Editorial Manager account (separate from Cambridge Core) set up.
- [ ] $350 fee ready (credit card); aware only $275 is refundable if not sent to a reviewer.
