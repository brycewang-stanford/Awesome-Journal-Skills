# JFQA Manuscript Template

A skeleton for a *Journal of Financial and Quantitative Analysis* (JFQA) empirical-finance
manuscript. Adapt freely; for theory papers, replace the data/identification sections with
model setup, propositions, proofs, and numerical examples. Format for submission:
**8.5 × 11, 1-inch margins, 12-pt Times New Roman, double-spaced**, single **text-searchable
PDF**, **anonymized** for double-anonymous review.

---

## Title Page (anonymized)

- Title (no author names/affiliations in the submission PDF).
- **Abstract**: one paragraph, **≤ 100 words**. State the question, data/design, the headline
  result **with a number** (bps, monthly alpha, elasticity, effect per one-SD), and the
  implication. No citations, no equations.
- Keywords; JEL codes.

## 1. Introduction
- The finance question and why it matters, in the first paragraph.
- The answer and its **economic magnitude**.
- The contribution vs. the closest rivals (one line each).
- The data and identification in brief; the key exhibit previewed.
- Roadmap.

## 2. Data and Sample
- Sources (CRSP / Compustat / TAQ / IBES / TRACE / etc.), filters, sample period.
- Variable definitions; **winsorizing/trimming** disclosed.
- Summary statistics (Table 1).

## 3. Research Design / Identification
- The design (sorts/Fama-MacBeth, panel FE, staggered DID, IV, RDD, event study).
- Identifying variation; threats addressed; inference (clustering / weak-IV / multiple testing).

## 4. Results
- Main result table with magnitudes, not only significance.
- Mechanism / heterogeneity.

## 5. Robustness
- Alternative samples, definitions, fixed effects, clustering; placebos/falsification.

## 6. Conclusion
- What we learn; do not over-claim beyond the design.

## References
- Internally consistent style (no named style mandated at submission — 待核实; accepted
  papers follow the JFQA Style Guide).

## Appendix
- Additional tables/figures; variable construction details. Double-spaced like the body.

---

### Pre-submission reminders
- Abstract word count ≤ 100, one paragraph.
- PDF is text-searchable; exhibits embedded and legible.
- Author identity removed from text and PDF metadata.
- If previously rejected by JFQA, disclose it in the cover letter.
- Code-sharing exception (if any) requested at this initial submission.
