# External Tools - NeurIPS

Use these tools only after verifying the current year's official instructions.

- NeurIPS conference site: https://neurips.cc/
- NeurIPS 2026 Call for Papers: https://neurips.cc/Conferences/2026/CallForPapers
- NeurIPS 2026 Main Track Handbook: https://neurips.cc/Conferences/2026/MainTrackHandbook
- NeurIPS Paper Checklist guidelines: https://neurips.cc/public/guides/PaperChecklist
- OpenReview submission and review system: https://openreview.net/
- NeurIPS proceedings: https://proceedings.neurips.cc/
- NeurIPS 2026 Reproducibility / MLRC track: https://neurips.cc/Conferences/2026/CallForReproducibility
- Papers with Code research-code release guidance linked from the NeurIPS handbook: https://github.com/paperswithcode/releasing-research-code
- Software Heritage archive for durable software identifiers: https://archive.softwareheritage.org/

Volatile items to re-check every cycle: CFP, submission portal, abstract and full-paper deadlines,
paper template, page and file-size limits, author-profile rules, track boundaries, double-blind
policy, dual-submission policy, LLM/agent policy, code/data guidance, author-response mechanics,
camera-ready upload rules, and presentation requirements.
