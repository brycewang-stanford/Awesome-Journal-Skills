# NeurIPS Skills

First-party depth pack for the Conference on Neural Information Processing Systems (NeurIPS).

This pack turns the NeurIPS main-track CFP and handbook into operational submission support:
topic fit, submission readiness, author response, camera-ready, artifacts, reproducibility,
supplementary material, review-process diagnosis, writing style, related work, experiments, and
workflow routing.

## Skills

| Skill | Use it for |
|---|---|
| `neurips-topic-selection` | Decide whether a manuscript belongs at NeurIPS and which contribution type fits. |
| `neurips-workflow` | Plan the full lifecycle from idea to submission, response, decision, and camera-ready. |
| `neurips-submission` | Audit current-cycle formatting, OpenReview, anonymity, track, and policy requirements. |
| `neurips-review-process` | Understand reviewer, AC, ethics, reciprocal-reviewing, and discussion dynamics. |
| `neurips-author-response` | Draft concise OpenReview rebuttals and discussion-period replies. |
| `neurips-camera-ready` | Prepare accepted-paper de-anonymization, final PDF, checklist, and release tasks. |
| `neurips-artifact-evaluation` | Package code, data, demos, and durable artifacts for review and public release. |
| `neurips-reproducibility` | Strengthen reproducibility evidence and decide whether MLRC/TMLR is a better route. |
| `neurips-supplementary` | Decide what belongs in the main PDF, text appendices, ZIP, or external artifact. |
| `neurips-writing-style` | Rewrite for NeurIPS-style claims, contribution framing, limitations, and checklist alignment. |
| `neurips-related-work` | Position against current-cycle ML work, preprints, contemporaneous work, and nearest siblings. |
| `neurips-experiments` | Stress-test experiments, ablations, baselines, compute, data, and negative-result framing. |

## Source Discipline

The pack was checked against official NeurIPS 2026 sources on 2026-06-01. NeurIPS policies are
cycle-volatile. Before submission-ready advice, reopen the current CFP, Main Track Handbook,
paper template, OpenReview portal, and checklist guidance.
