# NeurIPS Skills

这是 NeurIPS 主会投稿的 first-party 深度包。

本包把 NeurIPS main track 的 CFP、Main Track Handbook、paper checklist 和 OpenReview 流程拆成
12 个可调用 skill，覆盖选题定位、投稿审计、作者回复、camera-ready、artifact、可复现性、
补充材料、审稿流程、写作风格、相关工作、实验设计和全流程路由。

## Skills

| Skill | 用途 |
|---|---|
| `neurips-topic-selection` | 判断论文是否适合 NeurIPS，以及应选择哪类 contribution type。 |
| `neurips-workflow` | 规划从选题到投稿、回复、录用和 camera-ready 的全流程。 |
| `neurips-submission` | 核验当年格式、OpenReview、匿名、track 和政策要求。 |
| `neurips-review-process` | 理解 reviewer、AC、ethics review、reciprocal reviewing 和讨论机制。 |
| `neurips-author-response` | 写 OpenReview rebuttal 和 discussion-period 回复。 |
| `neurips-camera-ready` | 准备录用后的去匿名、最终 PDF、checklist 和发布任务。 |
| `neurips-artifact-evaluation` | 整理代码、数据、demo 和长期可访问 artifact。 |
| `neurips-reproducibility` | 强化可复现证据，并判断是否更适合 MLRC/TMLR 路线。 |
| `neurips-supplementary` | 决定哪些内容放正文、文字附录、ZIP 或外部 artifact。 |
| `neurips-writing-style` | 按 NeurIPS 的 claim、contribution、limitations 和 checklist 风格改写。 |
| `neurips-related-work` | 处理当前周期论文、preprint、contemporaneous work 和相邻会议定位。 |
| `neurips-experiments` | 压测实验、消融、baseline、算力、数据和 negative-result 叙事。 |

## 来源纪律

本包已在 2026-06-01 核验 NeurIPS 2026 官方来源。NeurIPS 政策每年都会变化；任何投稿级建议
都必须重新打开当年的 CFP、Main Track Handbook、模板、OpenReview 入口和 checklist 指南。
