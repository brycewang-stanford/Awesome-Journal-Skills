# 《中国管理科学》投稿前自检清单

> 易变项（费用、周期、匿名要求、模板细节）以官网 www.zgglkx.com「投稿指南」「编辑部公告」与采编系统内提示为准。本清单事实核验日期 2026-07-16，来源登记于 `../../../resources/official-source-map.md`。

## 1. 对口与定位

- [ ] 稿件属于三条主线之一（优化建模 / 数据驱动预测 / 金融工程实证），并能指认对位栏目
- [ ] 已排除姊妹刊错配（定理厚→《管理科学学报》；系统工程→《系统工程理论与实践》；案例/政策→《管理世界》；金融实证→《金融研究》）
- [ ] 方法增量可用一句话命名，引言前两页已出现

## 2. 结构与体例

- [ ] 章节顺序：标题→作者→单位（省市、邮编）→摘要→关键词→中图分类号→正文→结语→参考文献→英文块
- [ ] 摘要 100–200 字、自明五要素、无自评句
- [ ] 关键词 ≥3，覆盖问题/方法/情境；中图分类号已选
- [ ] 英文标题、作者拼音、单位、摘要、关键词与中文逐要素对应

## 3. 篇幅红线

- [ ] 论文全文 ≤12 页（官方口径）
- [ ] 冗余图表与大参数表已移附录或备查材料
- [ ] 未通过删参考文献、压缩英文块等违规手段凑页数

## 4. 模型与方法交付

- [ ] 模型四要素（变量/约束/目标/假设）显式，假设编号且有辩护
- [ ] 算法有伪代码、性质分析（最优性/收敛/复杂度）与参数表
- [ ] 对比含消融版与已发表基线，非只跟弱基准比

## 5. 检验与结果

- [ ] （预测/金融线）数据来源、频率、区间可复现；样本外方案预先声明；DM/MCS 或子区间显著性已报告
- [ ] （优化线）数值实验覆盖有效性/敏感性/边界/情境还原；参数标定有来源
- [ ] 随机结果报均值±标准差与重复次数、种子
- [ ] 管理启示逐条回指命题/实验编号，符合"条件—决策—效果"三段式

## 6. 图表与公式

- [ ] 全部表格为三线表，最优值标注规则在表注声明
- [ ] 公式经公式编译器排印、统一顺序编号；大小写、正斜体、上下角标分明
- [ ] 图件黑白可辨，图表题一句话点明结论

## 7. 参考文献

- [ ] 顺序编码制：按出现顺序编号，正文序号与文末逐条对应
- [ ] 全部为公开出版、直接阅读过的文献；格式对照本刊近期论文
- [ ] 网络首发文献注明获取路径与日期

## 8. 系统提交

- [ ] 仅经官网 www.zgglkx.com 采编系统投稿（拒绝一切代投渠道）
- [ ] 作者、单位、署名顺序经全体作者确认；通讯作者联系方式无误
- [ ] 基金项目名称与批准号标注规范
- [ ] 审稿费缴纳按系统提示完成（金额与期限以系统为准，逾期视为放弃）
- [ ] 无一稿多投；相关工作重叠（会议版、预印本）如实说明
- [ ] 费用、周期、匿名要求已按「编辑部公告」当日口径复核
