# 研究工具与数据资源（《中国管理科学》）

面向《中国管理科学》三条主线（优化建模 / 数据驱动预测 / 金融工程实证）的外部数据源、
软件与写作工具。使用前核对许可与最新版本；期刊政策类信息以
`official-source-map.md`（核验 2026-07-16）与官网为准。

## 1. 优化建模线

### 建模与求解

| 工具 | 用途 |
|------|------|
| Gurobi / CPLEX（学术许可） | LP/MIP/QP/SOCP 求解；报告版本号与求解参数 |
| COPT（杉数） | 国产求解器，大规模 LP/MIP；对国内评审语境友好 |
| Pyomo / JuMP (Julia) | 代数建模语言，模型与求解器解耦 |
| Python `networkx` / OR-Tools | 网络流、路径、调度类问题与启发式基线 |
| AnyLogic / SimPy | 离散事件与多主体仿真（应急、排队、平台情境） |

### 测试实例

- 标准测试集：OR-Library、TSPLIB、MIPLIB——自设计算法先在公共实例上站住，再上情境算例。
- 自造实例须公布生成协议（分布、参数、种子），与 `cjms-numerical-experiments` 的标定纪律一致。

## 2. 数据驱动预测线

| 资源 | 用途 |
|------|------|
| 国家统计局、行业协会公开数据 | 宏观与行业需求序列 |
| EIA / 上海石油天然气交易中心 | 能源价格序列（原油、天然气） |
| 全国/试点碳市场公开行情 | 碳价预测研究 |
| Python `statsmodels` / `arch` | ARIMA、GARCH 族经典基准 |
| `sklearn` / `pytorch` / `xgboost` | 机器学习学习器；调参协议须随稿报告 |
| `PyEMD` / `vmdpy` | EMD/VMD 分解——"分解-预测-集成"是本刊预测栏目高频形态，改进环节要可命名 |

预测检验：Diebold-Mariano 检验、Model Confidence Set（R `MCS` 包 / Python `arch`）。

## 3. 金融工程实证线

| 资源 | 用途 |
|------|------|
| Wind / CSMAR / RESSET | 国内市场行情与财务数据（机构订阅） |
| 交易所与外汇交易中心官网 | 可公开引用的规则与行情事实 |
| Python `linearmodels` / R `rugarch`、`copula` | 面板、波动率、Copula 相依结构 |
| `cvxpy` / `PyPortfolioOpt` | 组合优化与风险度量（VaR/CVaR）实现 |

因果类支线（政策冲击评估）直接使用本包 [`code/`](code/) 的 Stata + Python 骨架
（清洗→描述→DiD/IV/RDD→机制→稳健性→出表）。
