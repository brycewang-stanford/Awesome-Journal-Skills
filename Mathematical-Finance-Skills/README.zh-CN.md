# Mathematical Finance Skills 中文说明

这是面向 *Mathematical Finance* 的 12 个投稿技能包。该刊是 Wiley-Blackwell
旗下、与 Bachelier Finance Society 相关的金融数学期刊，重视严谨的金融数学理论贡献。

本包不是经验金融模板，而是 theorem-first 工作流：适用于随机分析、随机微积分、概率论、
衍生品定价、风险度量、投资组合理论和市场微观结构等方向。数值实验只有在支持理论结果时
才适合放入论文，不能替代证明。

## 技能列表

| Skill | 用途 |
| --- | --- |
| `mathfin-workflow` | 判断金融数学论文下一步该使用哪个技能。 |
| `mathfin-topic-selection` | 判断问题是否符合严谨金融数学期刊定位。 |
| `mathfin-literature-positioning` | 围绕已有定理和假设精确定位贡献。 |
| `mathfin-identification-strategy` | 检查假设、定理陈述、证明架构和一般性。 |
| `mathfin-contribution-framing` | 说明定理为什么推进金融建模。 |
| `mathfin-data-analysis` | 让数值实验服务于理论，而不是变成经验研究。 |
| `mathfin-tables-figures` | 准备支持定理的图表、数值展示和附录。 |
| `mathfin-writing-style` | 打磨定义、定理、证明和金融直觉表达。 |
| `mathfin-replication-and-data-policy` | 准备 Wiley 数据可得性声明和可复现代码说明。 |
| `mathfin-review-process` | 理解 single-blind 的编辑、AE 和审稿流程。 |
| `mathfin-submission` | 进行 Wiley Research Exchange LaTeX 投稿前检查。 |
| `mathfin-rebuttal` | 回复关于新颖性、严谨性和表达的审稿意见。 |
