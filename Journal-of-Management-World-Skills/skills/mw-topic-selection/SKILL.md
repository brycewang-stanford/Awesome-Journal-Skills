# 管理世界选题（mw-topic-selection）

## 触发时机

- 用户说"我想冲《管理世界》，但选题不确定"
- 边际贡献写不出来 / 写了一段但和审稿人能 buy 的角度不在一个频道
- 不确定该投《管理世界》、《经济研究》还是《管理学报》

## 三个硬约束

《管理世界》对选题的硬约束：

1. **中国情境嵌入**：选题必须能在第一段就回答"为什么这个问题在中国语境下值得做"
2. **边际贡献可枚举**：能拆成 3–5 条独立的、互不重叠的贡献
3. **政策落点存在**：哪怕是基础理论文章，也要能在结论部分指向具体的政策含义

## 中国情境契合度自检

写完一句话摘要后，逐条自检：

- [ ] 是否使用了中国制度 / 政策作为外生冲击或情境约束？
- [ ] 数据是否来自中国（CSMAR / Wind / CNRDS / 上市公司年报 / 省级面板）？
- [ ] 研究问题在海外文献中是否已有成熟答案？如果有，中国情境带来的边际差异是什么？
- [ ] 政策含义是否能写出至少一条可由具体政府部门执行的建议？

少于 3 个 yes 时，回到选题层面而不是写作层面调整。

## 边际贡献四句式

按以下模板逐条写满，每条 1–2 句话，**不要重复**：

```
本文的边际贡献主要体现在以下四个方面：
第一，[理论 / 文献维度]——本文……，区别于已有研究……。
第二，[识别策略维度]——本文利用[政策 / 制度冲击]构造……，缓解了……内生性问题。
第三，[机制维度]——本文揭示了……机制，回答了已有文献"知其然不知其所以然"的问题。
第四，[政策含义维度]——本文为[具体政策 / 部门]提供了……证据。
```

四条都齐才算合格。少于四条 → 选题深度不够。

## 期刊匹配判断

|              | 管理世界 | 经济研究 | 经济学（季刊） |
|----------|--------|--------|------------|
| 偏好维度    | 管理学 + 应用经济 | 经济学 + 制度宏观 | 偏方法论的经济学 |
| 政策含义    | **必须** | 必须     | 可选         |
| 案例 / 质性 | 接受     | 较少     | 极少         |
| 字数        | 1.5–2.5 万 | 1.5–2 万 | 1.5–2 万    |

如果稿子是纯方法论改进 + 跨国数据，**不要投《管理世界》**。

## 本刊选题审稿期待表

《管理世界》由国务院发展研究中心主办，是管理学与经济学并重的 CSSCI 权威顶级综合期刊，选题阶段编委最看重"是否服务国家重大战略与决策、是否扎根中国情境、边际贡献能否清晰枚举"。一个跨国数据、纯方法论的好选题在本刊也可能因定位错配被退。下表锚定本刊选题审稿期待与常见退稿模式：

| 维度 | 本刊期待 | 退稿高风险信号 |
|---|---|---|
| 战略相关性 | 对接国家战略/重大决策议题 | 选题与中国政策无关 |
| 中国情境 | 制度/政策作为外生冲击或约束 | 海外问题套中国数据 |
| 边际贡献 | 可枚举 3—5 条互不重叠 | 只说"国内尚无研究" |
| 政策落点 | 能导出具体部门可执行建议 | 纯理论无政策含义 |

### 常见退稿模式（选题维度）

- 选题是海外成熟问题的中国复制，说不清中国情境带来的边际差异
- 边际贡献只有"填补国内空白"一句，无法拆成独立四条
- 纯方法论改进 + 跨国数据，与本刊"经济学与管理学并重、本土关怀"定位错配
- 选题与任何国家战略议题都不沾边，削弱"服务国家决策"的契合度

## 微型走查示例：数字化转型与企业全要素生产率选题诊断

虚构选题，按三硬约束与四句式走查：

- **中国情境契合度自检**：使用"企业上云"试点政策作外生冲击（是）；数据来自 CSMAR + 年报文本（是）；海外已有数字化—生产率相关研究但缺中国试点准实验（中国情境带来识别优势，是）；可导出对数字化补贴政策的具体建议（是）→ 4/4。
- **战略相关性**：对接"数字中国""高质量发展"国家战略 → 达标。
- **边际贡献四句（草稿，示意）**：① 文献维度——区别于以横截面相关为主的本土研究；② 识别维度——利用试点政策构造 staggered DID 缓解内生性；③ 机制维度——打开"数字化→资源配置效率→TFP"黑箱；④ 政策维度——为数字化补贴的差异化投放提供证据。
- **期刊匹配判断**：管理学 + 应用经济、政策含义强 → 匹配《管理世界》；若改为跨国面板 + 纯方法创新，则建议改投。

## 审稿人追问模式 + 本刊语境修法

- 追问"这个问题海外早有定论，中国做有何新意？" → 修法：在选题句中明确写出中国制度/政策带来的识别优势或边际差异，而非强调"国内空白"。
- 追问"边际贡献四条是否真的互不重叠？" → 修法：逐条对照理论/识别/机制/政策四维度，删去重复表述，确保每条独立可辩护。
- 追问"政策含义在哪？" → 本刊语境下，即便理论文章也需在选题阶段预埋政策落点，否则命中率极低。

## 校准锚点

本刊已刊论文的选题普遍具备"国家战略相关性 + 中国制度情境 + 可枚举边际贡献 + 明确政策落点"四要素，且边际贡献四句式与文末政策建议、综述第三层首尾呼应。期刊匹配边界（与《经济研究》《经济学（季刊）》的分工）与字数要求以编辑部最新稿约为准。
