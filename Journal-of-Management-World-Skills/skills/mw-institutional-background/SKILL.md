# 制度背景章节（mw-institutional-background）

## 触发时机

- 实证主体已写完，但读者无法理解政策 / 制度细节
- 审稿人指出"制度背景太薄"或"政策细节不清"
- 用海外读者视角，发现自己根本没说清政策是什么

## 为什么《管理世界》要独立的制度背景章节？

AER / AEJ 通常把制度背景放在引言或脚注；《管理世界》编委预期一个**独立小节**（约 800–1500 字），原因：

- 读者覆盖政策研究者、产业研究者，不能假设他们熟悉细节
- 政策时间线本身就是识别策略的基础
- 这是审稿人判断"研究是否扎实"的第一信号

## 标准结构（四段式）

```
3.1 制度背景

第一段：政策 / 制度的总体情况
  - 起源、目标、覆盖范围
  - 在中国整体改革脉络中的位置

第二段：政策的关键时间节点
  - 试点年份 / 全面推开年份 / 调整年份
  - 时间线表格或时间轴图

第三段：政策的关键条款 / 处理机制
  - 谁被"处理"、谁是控制组
  - 处理强度的可观测变量

第四段：与本文识别策略的衔接
  - "正是基于上述政策的[外生性 / 时间错位 / 强度差异]，本文构造……"
```

## 关键细节清单

- [ ] 是否给出政策原文（《XX 通知》《XX 办法》编号 + 颁布机关）？
- [ ] 是否解释"试点 → 推广"的时间差，及其外生性来源？
- [ ] 是否区分"政策出台日 / 政策执行日 / 数据观测年"？
- [ ] 是否制作了政策时间线图或表？

少于 3 个 yes，制度背景需要重写。

## 政策时间线表模板

| 时间   | 政策文件                  | 关键内容              |
|------|----------------------|-------------------|
| 2013 | 《国务院关于 XX 的通知》| 试点启动，6 省 12 市  |
| 2015 | XX 修订意见              | 扩大至全国            |
| 2018 | XX 实施细则              | 增加处罚机制          |

时间线表通常作为正文第一张表（在主回归之前）。

## 反模式

- 把制度背景缩成引言中的一段话
- 只给政策名称，不给文件号 / 颁布日期
- 用海外文献中的描述代替政策原文
- "试点选择存在外生性"但不给出实证依据 / 不引文献

## 本刊制度背景审稿期待表

《管理世界》由国务院发展研究中心主办，编委多为政策研究与产业研究背景，对制度背景章节的期待远高于纯学术经济学期刊：它既是识别策略的地基，也是"研究是否服务国家重大战略"的判断入口。下表锚定本刊审稿期待与常见退稿模式（具体字数与格式以编辑部最新稿约为准）：

| 维度 | 本刊期待 | 退稿高风险信号 |
|---|---|---|
| 政策定位 | 把政策放进国家战略脉络（如"双碳""统一大市场""数字中国"） | 只描述政策条文，不提战略意义 |
| 文件溯源 | 给出文件全称、文号、颁布机关、生效日 | 只写政策俗称，无原文出处 |
| 时间线 | 试点—推广—调整三阶段清晰，配时间线表/图 | 一句话带过"该政策于近年实施" |
| 外生性 | 用遴选机制文件论证试点选择外生 | "我们认为外生"式断言 |

### 常见退稿模式（制度背景维度）

- 把制度背景压缩进引言一段，导致海外审稿视角无法理解处理机制
- 政策时间节点与数据观测年混淆，读者无法判断处理前后窗口
- 用二手英文文献转述中国政策，而非引用《XX 通知》原文
- 缺少"政策—识别"衔接段，制度背景与实证脱节

## 微型走查示例：营商环境改革评估

虚构一篇稿件，按四段式结构走查（数字为示意）：

- **第一段（总体情况）**：以 2019 年《优化营商环境条例》为制度起点，置于"放管服改革""建设全国统一大市场"国家战略脉络中，覆盖范围为地级市。
- **第二段（时间节点）**：2018 年 6 城试点（示意）→ 2020 年全国推开 → 2022 年增加考核问责。制作时间线表作为正文第一张表。
- **第三段（处理机制）**：以城市营商环境指数变化度量处理强度，处理组为改革推进较快城市，控制组为推进滞后城市；强调"试点遴选依据经济基础而非企业绩效"。
- **第四段（衔接识别）**：写出"正是基于改革推开的时间错位与强度差异，本文构造连续处理 DID……"，与 mw-identification 的分支 A 对接。
- **本刊语境校验**：营商环境对应"市场化、法治化、国际化营商环境"国家目标 → 战略相关性达标。

## 审稿人追问模式 + 本刊语境修法

- 追问"试点城市是否本就营商环境更好，构成选择偏误？" → 修法：引用试点遴选文件证明遴选与企业层面结果无关，并在制度背景末尾给出 pre-trend 平衡表索引。
- 追问"政策执行存在地方差异，'处理'定义是否模糊？" → 修法：补充政策落地的可观测代理变量（如行政审批时限、开办企业天数），把"处理强度"量化。
- 追问"该政策与同期其他改革叠加，如何分离？" → 本刊语境下，编委接受"主政策 + 列举并存政策 + 稳健性剔除"的处理，而非回避叠加问题。

## 校准锚点

本刊已刊政策评估类论文，制度背景普遍为独立小节（约 800—1500 字），含政策文号、三阶段时间线表、处理机制界定与"政策—识别"衔接段，且时间线表位列正文第一张表。是否要求附政策原文链接、表格置于何处等细节以编辑部最新稿约为准。

## 输出格式

```
【章节字数】X 字
【关键时间节点】X 个
【是否有时间线表】是 / 否
【外生性论证】强 / 中 / 弱
【国家战略脉络】已嵌入 / 缺
【下一步】mw-identification
```
