# 管理世界文献综述（mw-literature-review）

## 触发时机

- 综述章节全是英文文献，没有中文经典
- 综述写成了"罗列式"而非"对话式"
- 审稿人指出某中文经典文献未引

## 结构规范

《管理世界》偏好**三层结构**的综述：

1. **第一层：国际主流脉络**（英文文献为主）——建立问题的国际定位
2. **第二层：中国本土研究**（中文文献为主）——梳理中国情境下的已有发现
3. **第三层：本文位置**——清晰指出"本文与现有研究的差异"

避免一层结构（按时间线罗列）或两层结构（"国外研究 / 国内研究"两段式但无对话）。

## 中英文献配比

经验法则：

- 总引文 40–80 条
- 中文文献占比 **30%–50%**（不能 <20%，也不宜 >60%）
- 五年内文献占比 **≥ 40%**
- 顶刊文献占比 **≥ 30%**（AER / QJE / JPE / Econometrica / RES / 管理世界 / 经济研究 / JF）

## 必引避坑清单

- [ ] 是否有相关中文经典综述？（搜《经济研究》《管理世界》同主题综述）
- [ ] 是否有近 3 年顶刊的同主题论文？
- [ ] 引用是否覆盖三类：理论文献、方法文献、本土实证文献？
- [ ] 是否避免了"自引扎堆"？（自引 ≤ 3 条）

## 综述句式模板

```
[作者 (年份)] 利用[数据 / 方法]研究了[问题]，发现……。
然而，该研究[局限]，未考虑[本文要补的空白]。
与之相比，本文……。
```

每段综述都应至少出现一次"与之相比 / 区别于 / 本文延伸了 / 本文补充了"。

## 反模式

- 把综述写成"教科书式知识介绍"
- 文末才指出本文位置，前面只有罗列
- 中文文献只引一两条凑数
- 综述里出现 [27,28,29,30,31] 这种 5 个并列引用 —— 拆开按贡献分述

## 本刊综述审稿期待表

《管理世界》由国务院发展研究中心主办，管理学与经济学并重，综述既要对接国际主流脉络，也要扎根中国本土研究传统。编委对"只引英文不引中文经典""把中国研究当国际研究的应用脚注"尤为敏感。下表锚定本刊审稿期待（顶刊清单与配比阈值以编辑部最新稿约为准）：

| 维度 | 本刊期待 | 退稿高风险信号 |
|---|---|---|
| 结构 | 三层（国际脉络—本土研究—本文位置） | 按时间罗列，无对话 |
| 本土文献 | 中文经典与近 3 年本土实证齐备 | 中文文献仅一两条凑数 |
| 对话性 | 每段落出现"区别于/本文延伸了" | 综述与本文贡献脱节 |
| 战略对接 | 点出问题与国家战略/政策议题的关联 | 纯学术综述，无政策落点 |

### 常见退稿模式（综述维度）

- 综述写成"教科书式知识介绍"，看不出本文的边际位置
- 把《经济研究》《中国工业经济》等本土顶刊的同主题积累漏引，被审稿人一眼识破
- 5 个并列引用 [27-31] 不拆分，无法体现对各文献贡献的辨识
- 国际文献堆到三十条、中文文献不足五条，与本刊"经济学与管理学并重、本土关怀"定位错配

## 微型走查示例：数字化转型与企业全要素生产率

虚构稿件，按三层结构走查（条目数为示意）：

- **第一层（国际脉络）**：引用信息技术与生产率、企业数字化与组织变革的国际文献约 18 条（示意），建立"数字技术如何影响企业绩效"的国际定位。
- **第二层（本土研究）**：梳理中国情境下数字化转型测度（年报文本词频法）、与 TFP 关系的本土实证约 22 条（示意），其中《管理世界》《经济研究》《中国工业经济》同主题论文须覆盖。
- **第三层（本文位置）**：明确"区别于已有以横截面相关为主的本土研究，本文利用'企业上云'试点政策构造准实验识别，并打开机制黑箱"。
- **配比自检**：中文占比约 48%（示意，落在 30%—50% 区间），五年内文献约 55%（示意，≥40%）。
- **本刊语境校验**：把数字化转型对接"数字中国""高质量发展"战略议题，使综述自然导向政策落点。

## 审稿人追问模式 + 本刊语境修法

- 追问"为何忽略某篇《管理世界》近作？" → 修法：投稿前用主题词检索本刊与《经济研究》近 3 年目录，把同主题论文纳入第二层并做对话式区分。
- 追问"综述只罗列结论，没说清方法演进？" → 修法：按"理论文献—方法文献—本土实证"三类分述，每类点明方法或识别的代际差异。
- 追问"本文贡献到第三层才出现，前文像背景介绍？" → 本刊语境下，编委期待每段综述末尾即埋下"本文与之差异"的引线，而非集中堆在结尾。

## 校准锚点

本刊已刊实证论文的综述多为三层对话式结构，中文文献占比通常显著高于一般国际期刊，且综述结尾与边际贡献四句式（见 mw-topic-selection）严密呼应。具体配比与顶刊认定范围以编辑部最新稿约为准。
