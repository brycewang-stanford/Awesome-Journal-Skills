# 投稿前 Preflight（mw-submission）

## 触发时机

- "明天就要投了"
- 准备点提交按钮前的最后检查
- 不确定投稿系统需要哪些附件

## 投稿前 Checklist

### 格式

- [ ] 正文使用宋体小四 / 1.5 倍行距（具体看当年要求）
- [ ] 章节编号使用一、二、三（一级）→ （一）、（二）（二级）→ 1、2、3（三级）
- [ ] 图表编号连续且与正文引用对应
- [ ] 公式独立成行，居中，右侧带编号 (1) (2) ...
- [ ] 参考文献按文末统一格式：作者，年份，文章名，期刊，卷期，页码

### 字数

- [ ] 正文字数 1.5–2.5 万（含图表）
- [ ] 中文摘要 300–500 字
- [ ] 英文摘要 200–300 词
- [ ] 关键词中文 3–5 个 + 英文对应

### 双盲

- [ ] 正文中**任何能识别作者身份**的内容已去除
  - 自引使用"已有研究（XX，2020）"，不要写"作者前期研究"
  - 致谢页**单独**提交，不在正文中
  - 项目基金信息**单独**提交
  - 作者联系方式只在投稿系统中填，不写在正文
- [ ] 文件名不包含作者姓名 / 单位

### 参考文献

- [ ] 中文文献和英文文献按各自格式排列
- [ ] 中文期刊用全名（《管理世界》不写 MW）
- [ ] 每条文献完整：作者、年份、题目、期刊 / 出版社、卷、期、页
- [ ] 文中出现的每条引用都在参考文献列表中
- [ ] 参考文献列表中的每条都在文中至少出现一次

### 附件

- [ ] 致谢页（单独）
- [ ] 基金资助说明（单独）
- [ ] 作者简介（单独）
- [ ] 利益冲突声明（如有要求）
- [ ] 数据可用性声明（如有要求）

### 查重

- [ ] 知网查重 ≤ 15%（保险起见 ≤ 10%）
- [ ] 自引超过 5% → 单独说明
- [ ] 工作论文 / 会议论文版本是否已声明？

## 投稿系统操作要点

- 《管理世界》在线投稿系统：勤刷新，单次上传 ≤ 20MB
- **收费**：本刊特别声明**不收审稿费、不收版面费**（2026-06-22 复核 CNKI 出版平台及多家投稿信息页一致；务必经官方系统投稿，谨防代投/收费陷阱，**以编辑部最新声明为准**）
- 推荐审稿人 3 名（建议同领域、有相关发表、不与作者有合作）
- 不推荐审稿人 0–2 名
- 选择"研究类型"务必准确（实证 / 理论 / 综述 / 案例）

## 反模式

- 通宵改完没做双盲检查 —— 致命
- 参考文献仍是 EndNote 默认 APA 格式，未转为中文规范
- 图表编号断号 / 图 3 在正文中称"图 4"
- 投稿时把 working paper 版的致谢忘了删

## 本刊投稿前审稿期待表

《管理世界》由国务院发展研究中心主办，是规范实证大刊，初审（编辑部技术性退稿）对格式、双盲、查重、参考文献规范的把关极严，许多稿件在送外审前即因形式问题被退回。下表锚定本刊投稿前的硬性期待与常见退稿模式（具体阈值与当年要求以编辑部最新稿约为准）：

| 维度 | 本刊期待 | 初审退稿高风险信号 |
|---|---|---|
| 双盲 | 正文无任何作者可识别信息 | 自引写"作者前期研究"、致谢未单独提交 |
| 参考文献 | 中文用全名、格式统一、引用一一对应 | EndNote 默认 APA 未转中文规范 |
| 查重 | 知网查重达标、自引超标已说明 | 工作论文版本未声明 |
| 研究类型 | 在系统中准确勾选 | 实证误标为综述 |

### 常见退稿模式（投稿前/初审）

- 通宵改完未做双盲检查，正文残留单位/项目名——初审即退
- 图表编号断号或正文称呼错位（图 3 写成图 4）
- 参考文献"文中有、文末无"或"文末有、文中无"
- working paper 版本的致谢/基金信息忘删，泄露作者身份

## 微型走查示例：营商环境改革稿件投稿前

虚构一份待投稿件，按 checklist 走查（数字为示意）：

- **字数**：正文约 2.1 万字（落在 1.5—2.5 万区间，示意），中文摘要 420 字，英文摘要 260 词。
- **双盲**：全文检索单位名/基金号——发现正文脚注残留"国家社科基金（编号 XXX）"，移入单独基金说明页；自引由"我们此前的研究"改为"已有研究（XX，2021）"。
- **参考文献**：中文 31 条 / 英文 38 条（示意），核对每条文中至少出现一次；《管理世界》《经济研究》写全名未缩写。
- **查重**：知网查重 9%（示意，≤10% 保险线），自引占比 4% 未超标，无需单独说明。
- **附件**：致谢页、基金说明、作者简介、数据可用性声明分别单独上传；推荐 3 名同领域无合作关系审稿人。
- **判定**：双盲修正后形式合规 → 可提交，等待外审。

## 审稿人追问模式 + 本刊语境修法

- 初审反馈"疑似可识别作者身份" → 修法：用检索逐一清除单位、项目、自指表述，致谢与基金信息一律走单独附件。
- 初审反馈"参考文献格式不规范" → 修法：按本刊文末统一格式（作者，年份，题目，期刊，卷期，页码）逐条重排，中英文分列。
- 初审反馈"研究类型勾选与正文不符" → 本刊语境下，准确勾选实证/理论/综述/案例影响送审分流，务必与正文一致。

## 校准锚点

本刊在线投稿系统通常要求正文与可识别信息分离、参考文献遵循中文规范、并提交致谢/基金/作者简介等单独附件；查重与字数阈值随年度调整。具体附件清单、单次上传大小限制与推荐审稿人数量以编辑部最新稿约为准。
