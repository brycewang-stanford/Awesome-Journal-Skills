# 政策启示（mw-policy-implication）

## 触发时机

- 文末政策建议只有一句"建议政府重视"
- 审稿人指出"政策含义太空"
- 准备投稿但结论段没写够

## 为什么是硬约束？

《管理世界》编委的政策导向极强。即使是理论文章，编委也会期待"对治理 / 政策 / 实践有什么启示"。**没有政策启示的实证文章命中率极低**。

## 分层结构

政策建议分两个维度分层：

### 维度 1：时间层

- **短期**（1–2 年）：政策微调、执行细则
- **中期**（3–5 年）：制度配套、跨部门协同
- **长期**（5 年以上）：基础制度建设

### 维度 2：层级层

- **微观**（企业 / 个体）：经营决策、合规建议
- **中观**（行业 / 地方）：行业自律、地方试点
- **宏观**（中央 / 部委）：法律法规、宏观调控

至少写 3 条政策建议，**分别覆盖不同的时间 × 层级组合**。

## 模板

```
第一，[短期 × 宏观]——基于本文发现，建议[具体部门]在[时间窗]内，[具体可执行动作]。
具体而言，可通过[路径 1] / [路径 2] 实现，预计效果为[本文实证结果]。

第二，[中期 × 中观]——……

第三，[长期 × 微观]——……
```

每条建议都必须：
- 指向**具体部门**（"政府"不够 → 财政部 / 发改委 / 证监会 / 地方人民政府）
- 给出**可执行动作**（不是"加强 / 重视"）
- 与本文**实证证据挂钩**（不能凭空建议）

## 必查清单

- [ ] 至少 3 条建议
- [ ] 覆盖至少 2 个时间层 + 2 个层级层
- [ ] 每条建议指向具体部门
- [ ] 每条建议给出可执行动作
- [ ] 每条建议挂钩本文证据

## 反模式

- "建议政府加强监管 / 重视问题 / 完善制度"——无效建议
- 建议与本文实证无关（写了 ESG 投入但建议讨论员工持股）
- 建议过于宏大（"建议改革资本市场"）
- 仅一段话堆叠所有建议，无层次

## 本刊政策启示审稿期待表

《管理世界》由国务院发展研究中心主办，其政策导向是与一般学术期刊最显著的身份差异：编委本身即政策研究者，政策启示是"理论贡献 + 中国实践 + 政策含义"三位一体中不可省略的一极，且必须服务国家重大战略与决策需要。下表锚定本刊审稿期待与常见退稿模式：

| 维度 | 本刊期待 | 退稿高风险信号 |
|---|---|---|
| 指向 | 点名具体部门（财政部/发改委/证监会/地方政府） | "建议政府重视" |
| 动作 | 可执行动作（设定阈值、调整细则） | "加强/完善/重视"空话 |
| 证据 | 每条建议挂钩本文实证系数 | 建议与实证脱节 |
| 战略 | 对接国家战略议题（双碳、统一大市场、数字中国） | 就事论事无战略高度 |

### 常见退稿模式（政策启示维度）

- 全段只有"建议政府加强监管"，无部门、无动作、无证据
- 建议过于宏大（"建议改革资本市场"），脱离本文可支撑的范围
- 异质性结论丰富，政策建议却"一刀切"，浪费了 mw-mechanism-heterogeneity 的分层证据
- 政策含义与国家战略毫无对接，削弱本刊"服务国家决策"的定位契合度

## 微型走查示例：营商环境改革评估

承接前述营商环境稿件，把异质性证据翻译成分层政策建议（数字为示意）：

- **第一条（短期 × 宏观）**：基于改革使企业开办时限缩短带来 TFP 提升 0.04（示意），建议国务院办公厅在 1—2 年内将"开办企业天数"纳入营商环境年度考核硬指标。
- **第二条（中期 × 中观）**：基于低市场化地区效应更弱（0.020，示意），建议省级政府在 3—5 年内对营商环境薄弱地区开展跨部门协同试点，配套政务数据共享。
- **第三条（长期 × 微观）**：基于民营企业对改革更敏感（示意），建议地方政府针对中小民企建立常态化政企沟通机制，降低制度性交易成本。
- **战略对接**：三条建议统一指向"市场化、法治化、国际化营商环境"与"全国统一大市场"国家战略，使政策含义具备本刊期待的高度。
- **判定**：3 条建议覆盖短/中/长 × 宏/中/微，逐条挂钩实证、点名部门、给出动作 → 达标。

## 审稿人追问模式 + 本刊语境修法

- 追问"建议是否超出本文证据可支撑的范围？" → 修法：每条建议后用括号标注对应的本文系数或异质性结论，删去无证据支撑的宏大主张。
- 追问"建议太笼统，落不了地？" → 修法：把"加强/完善"改写为可核查的动作（设定指标、明确时限、指定主体）。
- 追问"政策含义与国家战略的关系是什么？" → 本刊语境下，编委期待显式写出建议如何服务某项国家战略或重大决策，而非仅停留在企业层面微观建议。

## 校准锚点

本刊已刊实证论文文末普遍给出 3 条以上分层政策建议，逐条点名责任部门、给出可执行动作、并与实证证据和国家战略议题挂钩；纯理论文章亦会在结论部分指向治理或政策含义。具体篇幅与部门称谓以编辑部最新稿约为准。

## 输出格式

```
【建议数】X 条
【时间层覆盖】短 / 中 / 长
【层级层覆盖】微 / 中 / 宏
【可执行性】高 / 中 / 低
【与本文证据挂钩度】高 / 中 / 低
【国家战略对接】已对接 / 缺
【下一步】mw-submission
```
