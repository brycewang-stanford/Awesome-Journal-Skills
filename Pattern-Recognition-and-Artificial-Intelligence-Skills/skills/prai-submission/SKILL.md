# 《模式识别与人工智能》投稿就绪审计

在把稿件上传到《模式识别与人工智能》(Pattern Recognition and Artificial Intelligence, PR&AI) 在线
投稿系统之前，先跑完这份审计。PR&AI 是由中国自动化学会、国家智能计算机研究开发中心与中国科学院
合肥智能机械研究所主办、科学出版社出版、创刊于 1989 年的 CCF 推荐 B 类模式识别/人工智能专业中文
月刊，以高水平原创**长文**为主。它按期刊稿件走同行评审，因此投稿被当作一篇要经受编辑初审、专家
外审、主编终审的完整学术论文来评判，而不是一次会议式的一轮定稿。以下事实于 2026-07-09 通过官方
投稿系统、CNKI 期刊主页与 CCF 推荐中文科技期刊目录交叉核验（见
`resources/official-source-map.md`），凡本刊未公开确认的数字均标注 **待核实**，请以投稿系统内最新
《投稿指南》为准。

- 注册时填写的题名、作者、单位、通信作者与研究方向决定编辑分派外审专家的匹配度，务必用**真实终稿

## 二、格式与长文篇幅

- 使用本刊**官方模板**（如提供 Word 与 LaTeX 两种，以系统内为准）。中文正文按刊物排版，提交时按
  系统要求上传。
- PR&AI 为长文体例：**全文字数要求 18000~44000 字**（含图表）。请把这理解为"给足空间把模式识别/
  机器学习的方法、模型与充分实验完整展开"，而非"凑字数"；过短或超长都需调整。
- 稿件类型可对应栏目：论文（研究与应用、论文与报告）、综述（综述与评论）等。选定后按对应体例撰写。

## 三、中文科技论文的硬性组件

《模式识别与人工智能》的稿件必须齐备以下中文科技期刊的标准部件（详见 `prai-writing-style`）：

| 组件 | 要求 |
|---|---|
| 中文题名 + 英文题名 | 英文题名与中文一致、可独立检索 |
| 中文摘要 + English Abstract | **报道性摘要**四要素齐全；中文约 **200 字**，英文约 **200 词** |
| 中文关键词 + Keywords | 一般 **3~5 个**，中英文一一对应 |
| 中图分类号 | 计算机/AI 类以 **TP** 打头（如 TP18、TP181、TP391），据主题选取 |
| 作者简介与基金项目 | 注明资助项目及编号；作者简介含研究方向 |
| 参考文献 | **GB/T 7714**；中文文献需给出**中英文对照**，著录完整 |

## 四、学术规范与伦理红线

- **重复率检测**：编辑部对稿件做查重，**未通过检测的稿件退回作者**。自查时先用查重工具压到安全线
  以下，尤其注意自引与前序会议版本的重叠。具体红线百分比以编辑部口径为准（**待核实**）。
- 若本文是作者已发表会议论文的**期刊扩展版**，必须显著声明扩展关系、指明新增贡献（新增理论、新增
  实验、新增系统模块通常需达到实质比例），并规范引用会议版本，避免一稿多投与自我剽窃。
- 署名、贡献、基金、数据来源须真实；不得伪造、篡改数据或代写。
- 一稿多投是明确禁止项：投稿即视为该稿未在其他刊物同时评审。

## 五、退稿风险分诊表

| 审计发现 | 风险等级 | 处置 |
|---|---|---|
| 重复率检测不通过 | 初审退稿级 | 重写重叠段落、补足原创增量后再投 |
| 字数不在 18000~44000 区间 | 初审退回补正 | 扩充或精简到区间内 |
| 未用官方模板 / 缺中英文摘要或中图分类号 | 初审退回补正 | 套用官方模板补齐全部部件 |
| 无新意、重复前人工作、观点模糊 | 初审退稿理由 | 回到 `prai-topic-selection` 重估贡献 |
| 参考文献不符 GB/T 7714 / 中文文献缺英文对照 | 需补正 | 逐条重排著录格式 |
| 主题偏控制/软件工程/电子，非 AI 细分 | 不符宗旨 | 改投《自动化学报》等更对口刊 |
| 会议扩展版未声明、增量不足 | 学术规范风险 | 显著声明并补足实质新增贡献 |
| 一稿多投 | 严重违规 | 撤回其一，确认唯一在审 |

## 六、投稿前最后一轮操作顺序

1. 冻结正文论证主线；参考文献可微调，核心贡献陈述不再改。
2. 用官方模板做一次通排，确认中英文摘要、关键词、中图分类号、基金、作者简介、参考文献六件套齐全。
3. 跑查重，把总复制比压到安全线以下，逐条核对自引与会议版本重叠。
4. 校对图表：图题表题中英文规范、量和单位用法定计量单位、图内文字清晰可辨，模式识别结果图/混淆
   矩阵/曲线在黑白打印下可区分。
5. **对上传文件做病毒扫描**，确认无异常。
6. 在"作者中心"逐项填写系统字段并上传全文，随后重新下载 PDF 冷读一遍，确认是最终版、无乱码、无
   残留批注。

## 七、每轮需复核的易变项

- 具体模板版本、版面费/审理费金额、审稿是否双盲（**待核实**，以编辑部通知为准）。
- 官方"6 个月内"审稿口径与二手"3~4 个月"说法的当期实际（**待核实**）。
- 是否新增代码/数据可用性声明要求或生成式 AI 使用披露要求（**待核实**）。
- 栏目设置与当期征稿重点、专题(special issue)是否开放。

## 八、图表、公式与源文件提交细节

- **图表**：矢量图优先（可无损缩放），位图保证印刷分辨率；图内文字随正文缩放后仍清晰。图题在图下、
  表题在表上，中英文对照。模式识别/机器学习常见的 ROC/PR 曲线、混淆矩阵、可视化在黑白打印下用
  线型或纹理区分而非仅靠颜色。
- **公式**：用公式编辑器或 LaTeX 排版而非截图；对齐、编号连续；正文引用公式用编号。
- **源文件**：若用 LaTeX，提交前用官方模板本地完整编译一次，确认无缺字体、无未定义引用、参考文献
  正常生成；打包 .tex、.bib、图源与自定义宏，方便编辑部排版。
- **文件命名与匿名**：若本刊要求作者信息与正文分离(**待核实**是否双盲)，稿件文件不含身份信息。

## 九、投稿信(cover letter)与系统字段

- 投稿系统通常要求填写题名、摘要、关键词、领域分类、作者与通信作者信息、基金、推荐/回避审稿人等。
  逐项据实填写；领域分类要与稿件主题精确匹配，直接影响外审专家分派。
- 投稿信简明说明：本文主要贡献、为何适合《模式识别与人工智能》(Pattern Recognition and Artificial
  Intelligence, PR&AI)、是否为会议扩展版及其增量、以及无一稿多投与利益冲突的声明。

## 十、投稿时间线上的常见踩坑

| 场景 | 后果 | 规避 |
|---|---|---|
| 上传含病毒/损坏文件 | 编辑部无法处理、退回 | 上传前病毒扫描并本地打开确认 |
| 临投稿才发现重复率超标 | 被迫延期重写 | 投稿前数天先自查查重 |
| 字数超长/过短 | 初审退回补正 | 预留时间调整到区间内 |
| LaTeX 在编辑部环境编译失败 | 排版返工、延误 | 提交前用官方模板干净编译 |
| 领域分类填错 | 分到不对口专家、外审慢 | 精确匹配主题选分类 |
