# 《模式识别与人工智能》录用后定稿与校样

本技能覆盖《模式识别与人工智能》(Pattern Recognition and Artificial Intelligence, PR&AI) 稿件
**录用之后**的工作：定稿、校样(清样)核对、缴费与出版。PR&AI 是科学出版社出版的月刊，录用后进入
定稿与排期。事实核验日期 2026-07-09，见 `resources/official-source-map.md`；版面费/审理费金额、
具体出版周期 **待核实**，以编辑部通知为准。

## 一、录用后的第一步：确认要求

- 仔细阅读录用通知与主编终审/编辑的最终意见，列出所有待办（定稿修改、材料、缴费、授权）。
- 明确各项**截止时间**；录用后流程有排期，延误会推迟见刊。
- 通过投稿系统与编辑部邮箱 `bjb@iim.cas.cn`、电话 `0551-65591176` 对接。

## 二、定稿：落实终审意见

- 逐条落实主编终审与编辑的最终修改意见。
- **恢复作者信息**：若审稿阶段匿名（是否双盲 **待核实**），此时恢复作者、单位、致谢、基金显名。
- 确认作者署名顺序、通信作者、单位与基金项目编号准确无误。

## 三、校样(清样)核对清单

校样阶段**只改错，不改内容**。逐项终检：

| 项 | 检查点 |
|---|---|
| 题名/作者/单位 | 中英文准确、署名顺序、通信作者标注 |
| 中英文摘要/关键词 | 与正文一致、四要素齐、关键词对应 |
| 中图分类号 | TP 子类正确 |
| 图表 | 编号连续、图题表题中英文、黑白可辨、清晰 |
| 公式 | 编号连续、符号一致、无排版错位 |
| 参考文献 | GB/T 7714、中文文献英文对照、序号对应 |
| 基金/作者简介 | 项目编号、研究方向准确 |
| 排版 | 无乱码、无缺字、无错行 |

## 四、缴费与授权

- **版面费/审理费**金额与缴纳方式 **待核实**，以编辑部通知为准；按通知及时缴纳以免延误排期。
- 完成**著作权/出版授权**手续（以编辑部提供的协议为准）。
- 保留缴费与授权凭证。

## 五、录用后纪律

- **不得实质改动内容**：录用基于已审版本，校样阶段擅自增删数据、结论会破坏评审基础。
- 若发现录用后必须更正的错误（如数据笔误），**主动**联系编辑部说明，由编辑部决定处理方式，不自行
  改动。
- 不追加未经审稿的新实验/新结论。

## 六、出版排期与月刊节奏

- PR&AI 为**月刊**：录用后按编辑部排期安排卷期，见刊时间以编辑部通知为准（**待核实**具体时长）。
- 关注是否有网络首发/优先出版（以官网当期为准）。
- 见刊后核对正式版 DOI、卷期页码，用于后续引用。

## 七、常见踩坑

| 场景 | 后果 | 规避 |
|---|---|---|
| 校样拖延 | 推迟见刊排期 | 收到校样尽快核对 |
| 校样阶段改内容 | 破坏评审基础 | 只改错不改内容 |
| 作者信息未恢复/有误 | 署名错误 | 定稿时逐项核对 |
| 缴费/授权延误 | 流程卡住 | 按通知及时办理 |
| 参考文献未终检 | 出版后错误 | 校样时逐条核 GB/T 7714 |

## 八、录用后材料清单

- 定稿源文件（Word/LaTeX，按系统要求）与 PDF。
- 恢复署名的正式版图表源文件。
- 缴费凭证、著作权授权协议。
- 正式署名版复现材料（若审外用匿名版，见 `prai-artifact-evaluation`）。

## 九、自检清单

1. 终审/编辑最终意见是否逐条落实？
2. 作者信息/致谢/基金是否恢复且准确？
3. 校样是否只改错未改内容？
4. 图表/公式/参考文献(GB/T 7714)是否终检？
5. 版面费(待核实)与授权是否按时办理？
6. 见刊后是否核对 DOI/卷期页码？

## 十一、校样核对的实操方法

《模式识别与人工智能》(Pattern Recognition and Artificial Intelligence, PR&AI) 校样(清样)是出版前
最后一道关，实操上建议：

- **对照终稿逐字核**：把校样与录用终稿并排，逐字核对，重点看排版转换是否引入错字、错行、公式
  错位。
- **冷读**：放一两天再读，或换人读，更易发现自己惯性忽略的错误。
- **图表专项**：确认图片未因排版压缩而模糊、颜色转印刷后仍可辨、图表编号与正文引用一致。
- **参考文献专项**：逐条核 GB/T 7714，中文文献英文对照是否完整，序号与正文一一对应。
- **元数据专项**：题名、作者、单位、基金编号、通信作者邮箱这些"出版后难改"的信息重点核。

## 十二、常被忽略的定稿细节

- **英文题名与摘要**：中英文题名一致、English Abstract 无语法错误(录用后仍值得再校一遍)。
- **中图分类号与关键词**：确认 TP 子类与关键词在定稿中未被漏排。
- **致谢与基金显名恢复**：审稿匿名(是否双盲 **待核实**)时隐去的信息，定稿时确保已正确恢复。
- **DOI 与卷期**：见刊后记录正式 DOI、卷、期、页码，用于个人主页与后续引用。

## 十三、录用后到见刊的心态与纪律

录用不是终点而是出版流程的起点。此阶段的核心纪律是"**只改错，不改内容**"：录用基于已审版本，任何
实质性的数据、结论增删都会破坏评审基础；确需更正的错误主动报编辑部，由编辑部决定处理。配合编辑部
的排期与缴费(金额 **待核实**)、授权手续，及时响应校样，是让一篇模式识别/人工智能长文平稳见刊的
关键。见刊后，正式版即成为可被引用的学术记录，与 `prai-artifact-evaluation` 的正式署名制品共同
构成完整、可核验的成果。
