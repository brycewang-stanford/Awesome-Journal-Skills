# Econometric Theory Skills

面向 **Econometric Theory (ET)** 的 12 个 agent skills。ET 是 Cambridge University
Press 的理论计量经济学期刊，强调数学与统计基础、渐近理论、证明、时间序列/非平稳、高维与非标准环境。

官方依据核验日期：2026-06-01。来源见 `resources/official-source-map.md`，包括 Cambridge
author instructions、submission system、article types、50 页上限、single-anonymous review、
APA author-date references 与 2026 年编辑部变化。

## Skills

- `ectheory-workflow`：路由 ET 全流程。
- `ectheory-topic-selection`：判断是否是 general foundational result，或应投 Miscellanea。
- `ectheory-literature-positioning`：定位 theorem 与计量理论前沿。
- `ectheory-identification-strategy`：审查 assumptions、regularity conditions、asymptotics 与 proof plan。
- `ectheory-data-analysis`：处理 Monte Carlo / numerical illustration。
- `ectheory-tables-figures`：压缩模拟表、定理辅助图与 appendix exhibits。
- `ectheory-contribution-framing`：明确理论贡献。
- `ectheory-writing-style`：处理 theorem-proof prose、APA references 与 supplementary overflow。
- `ectheory-replication-and-data-policy`：区分 voluntary supplementary material 与不可误用的强制 archive。
- `ectheory-review-process`：理解 single-anonymous review、ScholarOne 与 editorial program。
- `ectheory-submission`：投稿前核查 ScholarOne、PDF、50 页、abstract 与 figures。
- `ectheory-rebuttal`：处理 R&R 中的 assumptions、proofs 与 theorem exposition。
