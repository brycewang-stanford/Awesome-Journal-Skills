## How to use (independent of the table)

- *Econometric Theory* rewards a **theorem**: a new estimator/test stated with explicit assumptions and
  a rigorously derived limiting distribution; Monte Carlo evidence is supporting, not the contribution.
- Lead the introduction with the **methodological gap and the main theoretical result**, not an
  empirical application (see this pack's writing/topic skills).
- This is a methods-*theory* venue, so the repo's applied causal-inference `code/` kit is intentionally
  **not** vendored here; the relevant artifact is a clean proof + a focused simulation.
