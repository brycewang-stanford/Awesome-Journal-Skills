# YLJ Submission Preflight Checklist

Printable companion to [`../SKILL.md`](../SKILL.md). Walk this before uploading to **The Yale Law
Journal's own online author portal** (not Scholastica/ExpressO). Verify volatile items on the official
submissions page (检索于 2026-06；以官网为准).

## 1. Track & length

- [ ] Track chosen: Article / Essay / Feature / Note / Comment / Forum
- [ ] Submitted as **Essay or Article** so the Committee can assess it accurately
- [ ] Encouraged cap met (incl. footnotes):
  - [ ] Article < 25,000 words (~50 Journal pages)
  - [ ] Essay < 15,000 words (~30 pages)
  - [ ] Forum < 10,000 words (~20 pages)
  - [ ] (Student) Note first-time < ~20,000 words / Comment first-time < ~7,000 words
- [ ] Confirmed over-length weighs **against** acceptance — trimmed accordingly

## 2. Claim & contribution

- [ ] One-sentence, contestable, generalist thesis (see `ylj-thesis-and-contribution`)
- [ ] "What's new" paragraph present; closest prior work named (see `ylj-preemption-check`)
- [ ] Argument follows doctrine → theory → normative (see `ylj-argument-structure`)
- [ ] Strongest counterargument addressed; scope/limits stated

## 3. Anonymization (review is name-blind)

- [ ] No author name, affiliation, or acknowledgments in the manuscript
- [ ] No self-identifying references ("as I argued in…")
- [ ] Prestige / prior-publication cues removed
- [ ] File metadata (document properties, comments) stripped of identity

## 4. Footnote apparatus (source-pull readiness)

- [ ] Every law/fact assertion footnoted
- [ ] Every cite pinpointed to the page/section relied on
- [ ] Bluebook form correct by source type; signals deliberate
- [ ] Short forms (`id.`, `supra`) consistent; first-full-cite precedes short forms
- [ ] Quotations marked + pinpointed; alterations Bluebook-flagged; no unmarked paraphrase
- [ ] Load-bearing cites re-checked against the source (see `ylj-footnotes-and-cite-check`)

## 5. Prose

- [ ] Argument followable from text alone (no load-bearing reasoning stranded in footnotes)
- [ ] Terms of art defined; generalist-legible
- [ ] No footnote dwarfing its paragraph

## 6. Portal & materials

- [ ] Submitting through the **YLJ author portal**, not Scholastica/ExpressO
- [ ] Any abstract / cover info the portal requests prepared
- [ ] Multi-submission slate (other reviews) handled separately via Scholastica/ExpressO
- [ ] Expedite (if any) filed in the YLJ portal with a real offer + deadline — **no queue priority expected**
- [ ] No submission fee budgeted — YLJ charges **no submission fee / no APC** (verified 2026-06-22; 以官网为准)

## 7. Final

- [ ] Manuscript is **near-final** — not a draft to finish after acceptance
- [ ] Backup saved; version recorded
- [ ] Next: await decision → `ylj-revision-and-editing` on acceptance
