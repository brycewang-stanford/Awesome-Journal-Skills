# JPAM Pre-Submission Checklist (worksheet)

Companion to [`../SKILL.md`](../SKILL.md). Work top to bottom before uploading through **Wiley Authors /
Research Exchange**. Verify every volatile item against the current Wiley/APPAM author page and the
live Research Exchange prompts (检索于 2026-06-20；以官网为准).

## 0. Article type
- [ ] Type chosen: Feature Research Article / Methods for Policy Analysis / Policy Retrospective
      (peer-reviewed) — or Point/Counterpoint / Policy Insights (invited, **not** peer-reviewed) / Book Review
- [ ] If targeting an invited type, an invitation/agreement is in hand
- [ ] Length within current guidance for the type (Policy Insights ≤ 3,000 words; feature-article limit checked live)

## 1. Fit & contribution
- [ ] A specific policy or program is evaluated (not a descriptive "determinants of" study)
- [ ] The decision the evidence informs is stated, with the policy counterfactual
- [ ] Cross-disciplinary relevance (econ / political science / public management) is explicit
- [ ] Home-journal check done vs. AEJ: Economic Policy / J. Public Economics / PAR / JPART

## 2. Identification & analysis
- [ ] Estimand and target population stated
- [ ] Identifying assumption named and defended (RCT / DiD / RD / IV / synthetic control)
- [ ] Modern heterogeneity-robust estimator for staggered DiD; pre-trends shown
- [ ] Clustering at the assignment level; multiple-testing handled
- [ ] Cost-benefit (perspective, discount rate, sensitivity / MVPF / BCR) where central
- [ ] Distributional incidence shown — who gains, who pays
- [ ] Policy implication calibrated to the estimand; limits stated honestly

## 3. Abstract & exhibits
- [ ] Abstract states question + policy/program + design + finding + implication (within current cap)
- [ ] Main effect reported with confidence intervals in policy-relevant units
- [ ] A design-validity exhibit (event-study / RD plot / balance / SC fit) is included
- [ ] Captions self-contained; figures colorblind-safe, grayscale-legible, vector format

## 4. Anonymity (double-blind)
- [ ] No author names, affiliations, or acknowledgments in the manuscript file
- [ ] Self-references neutralized; funding/IRB identifiers removed from the main file
- [ ] File metadata stripped (document properties, comments, tracked changes)
- [ ] Separate, non-anonymous **title page** prepared as a distinct file

## 5. Fee / membership & metadata
- [ ] APPAM membership current **or** submission fee paid (USD 100 professional / USD 40 student)
      **or** an approved waiver (info@appam.org) is in hand
- [ ] Corresponding-author ORCID ready (confirm requirement on the live page)
- [ ] References in the current Wiley/APPAM house style

## 6. Transparency & compliance
- [ ] Data-availability statement drafted
- [ ] Replication package staged: master script reproduces every table/figure; README; pinned versions; seeds
- [ ] Restricted data: restriction explained + access path + synthetic data where feasible
- [ ] Ethics / IRB / human-subjects compliance stated as needed
- [ ] Preprint status checked against Wiley policy

## 7. Final
- [ ] Every number in the text matches the deposited replication output
- [ ] Files and metadata required by Research Exchange assembled
- [ ] After submission: await decision → on R&R, go to [`../../jpam-rebuttal/SKILL.md`](../../jpam-rebuttal/SKILL.md)
