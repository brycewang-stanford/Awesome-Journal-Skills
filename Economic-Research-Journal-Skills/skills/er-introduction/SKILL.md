# 引言写作（er-introduction）

引言是全文最重要的部分。审稿人 80% 的"枪毙"决定在读完引言时已经形成。引言不是综述的缩写，而是一篇"为什么要读这篇文章"的说服性短文。

## 触发时机

- 第一句从"随着……的发展""近年来，我国……"这类宏大背景写起
- 第一段就堆文献，看不出现实问题
- 边际贡献写成"丰富了相关研究""提供了新的视角"
- 引言读完不知道本文结论是什么（没有剧透）
- 贡献条目相互重叠，3 条其实是 1 条
- 用了"本文首次"却无法支撑

## 标准漏斗结构（五段式）

《经济研究》引言是一个"现实 → 缺口 → 做法 → 贡献 → 结构"的漏斗，每段一个明确任务：

| 段 | 任务 | 一句话要害 |
|----|------|-----------|
| 第 1 段 | **现实重要性 / 中国情境** | 第一句就抛出现实问题或政策冲突，禁止从遥远背景起笔 |
| 第 2 段 | **学术缺口 / 文献张力** | 与谁对话、已有研究"差在哪"，是张力不是罗列 |
| 第 3 段 | **本文做法** | 数据 + 识别策略 + 核心发现一句话剧透 |
| 第 4 段 | **边际贡献 3–5 条** | 每条独立、不重叠，相对于具体文献 |
| 第 5 段 | **全文结构** | 一句话交代后文安排 |

判断标准：删掉作者名后，引言仍能读出"什么现实问题 → 已有研究为何不够 → 本文怎么做、发现了什么 → 比谁强在哪"。

## 第一句话（hook）的写法

第一句决定审稿人的第一印象。要点：**用一个具体的、有张力的中国现实问题或数据事实开场**，不要从概念定义或宏大趋势起笔。

### 坏的开头（套话起笔，立即扣印象分）

> ❌ 随着我国经济进入高质量发展阶段，……
> ❌ 近年来，数字经济蓬勃发展，引起了学界的广泛关注。
> ❌ 劳动力市场是经济学研究的重要领域。
> ❌ 改革开放以来，我国经济取得了举世瞩目的成就。

### 好的开头（范式，可任选其一）

1. **数据事实型**：用一个反差强烈的数字开场。
   > 2013—2019 年间，中国工业机器人安装量增长了近 10 倍，但同期制造业就业并未出现普遍下降。这一反差提出一个问题：[机制]究竟如何重塑了劳动力市场？
2. **政策冲突型**：摆出一个政策悖论。
   > [政策 A]意在[目标 X]，却可能在[群体 Y]上产生与初衷相反的后果。
3. **理论争议型**：点出文献中两派对立的预测。
   > 关于[变量 X]对[结果 Y]的影响，理论上存在两种相反的预测：一派认为……，另一派认为……。哪种力量占主导，是一个有待中国数据回答的实证问题。
4. **现实悖论型**（puzzle）：呈现一个"看似不该发生却发生了"的现象。
   > 在[约束条件]下，[现象]本不应出现，但[中国情境]中却普遍存在。

## 讲一个好故事：研究问题必须有张力

没有张力的问题 = 没有故事。引言的命门是让审稿人觉得"这个问题不解决会难受"。张力的三种来源：

- **Puzzle（现实悖论）**：现象与直觉/理论相悖。
- **Policy conflict（政策冲突）**：一项政策的预期效果与实际效果可能相反，或不同群体受益相反。
- **Theory controversy（理论争议）**：两支文献给出相反预测，需要中国证据来仲裁。

错误：把研究问题写成"研究 X 对 Y 的影响"——这是选题，不是故事。
正确：把研究问题写成"在[张力]下，X 对 Y 的净效应方向不明，本文用[识别]来回答"。

## 边际贡献四句式模板

列 **3–5 条**边际贡献。每条挂靠一个维度，彼此不重叠，且都要**相对于具体文献**说话——不能是"丰富了 / 拓展了 / 补充了相关研究"这类空话。

```
第一，理论维度：区别于[具体文献/作者，年份]从[旧视角]的解释，本文从[新视角/新机制]
       揭示了[关系]，为[理论命题]提供了[中国情境]证据。
第二，识别维度：相对于已有研究多采用[弱识别，如 OLS/简单 DID]，本文利用[外生冲击/
       工具变量/断点]构造[识别策略]，缓解了[具体内生性来源]。
第三，机制维度：本文不止于估计[总效应]，而是打开黑箱，识别出[机制 A]与[机制 B]，
       回答了[已有文献悬而未决的为什么]。
第四，政策意义维度：本文发现对[具体政策设计]具有直接含义——[谁应该做什么]，
       而非泛泛而谈"政策应予重视"。
```

对照：

> ❌ 本文丰富了关于数字经济的研究。
> ✅ 区别于张勋等（2019）从普惠金融视角的分析，本文进一步分离出[机制]，回答了数字经济红利"为何在低收入群体上更大"这一问题。

每写完一条，自问：把这一条删掉，文章的贡献是否真的少了一块？答否则删。

## 与摘要、文献综述的分工

- **引言 ≠ 摘要放大版**：摘要是结论的浓缩，引言是"为什么值得做"的论证。引言要有**问题意识**和完整的说服链条。
- **引言 ≠ 文献综述的缩写**：综述系统梳理谱系（见 er-literature-review），引言只挑出与本文直接对话的**少数关键文献**，用来制造张力、定位缺口。引言里夹注 5–15 条足矣，不求全。
- **引言必须剧透结论**：在第 3 段一句话给出核心发现（方向 + 关键量级），不要"卖关子"。审稿人讨厌读到最后才知道结论。

## 引用口径（《经济研究》硬规范）

参考文献采用**著者—出版年制**，正文夹注写「(作者，年份[，页码])」，如（林毅夫，2007）。

- ✅ 已有研究表明数字普惠金融可促进包容性增长（张勋等，2019）。
- ❌ 已有研究表明……[12]。
- ❌ 大量文献关注这一问题[27,28,29]。
- ❌ 不要出现 [J]/[M]/[D] 文献类型标识，不要顺序编码。

## 对标范文（确属《经济研究》，可安全引用学习）

- 王永钦、董雯《机器人的兴起如何影响中国劳动力市场？》（2020 年第 10 期，Bartik IV）——数据事实型开头 + 强识别贡献的范本。
- 张勋等《数字经济、普惠金融与包容性增长》（2019 年第 8 期）——现实重要性 + 机制贡献的范本。
- 刘生龙等《义务教育法与中国城镇教育回报：基于断点回归设计》（2016 年第 2 期）——政策冲击 + 断点识别贡献的范本。

读这三篇引言时，重点拆解：第一句怎么写、张力从哪来、贡献怎么挂靠到具体文献。

## 必查清单

- [ ] 第一句是否抛出了具体的现实问题/数据事实/政策冲突（而非套话起笔）？
- [ ] 第一段是否聚焦现实重要性、没有提前堆文献？
- [ ] 第二段是否呈现了文献张力（对话）、而非罗列谁研究过什么？
- [ ] 第三段是否一句话剧透了核心发现（方向 + 量级）？
- [ ] 边际贡献是否 3–5 条、每条挂靠不同维度、彼此不重叠？
- [ ] 每条贡献是否相对于**具体文献**、而非"丰富了相关研究"？
- [ ] 若用了"本文首次"，识别/数据/机制上是否真有支撑？
- [ ] 第五段是否用一句话交代全文结构？
- [ ] 全文夹注是否一律著者—出版年制、无顺序编码？

## 反模式

- 从遥远的宏大背景起笔（"随着……的发展""改革开放以来……"）
- 第一段就堆文献，看不见现实问题
- 贡献写成"丰富了 / 拓展了相关研究"等空话
- 引言通篇不剧透结论，逼审稿人读到正文才知道发现
- 贡献条目相互重叠，3 条实为 1 条换三种说法
- 用"本文首次"却无识别/数据/机制支撑（审稿人会反向举证）
- 把研究问题写成"研究 X 对 Y 的影响"——只有选题、没有故事
- 把引言写成文献综述的缩写，没有问题意识

## 输出格式

```
【五段漏斗合规】是 / 否（缺哪一段：现实 / 缺口 / 做法 / 贡献 / 结构）
【第一句类型】数据事实 / 政策冲突 / 理论争议 / 现实悖论 / 套话(需重写)
【故事张力】puzzle / policy conflict / theory controversy / 无(需补)
【剧透结论】有（第3段）/ 无(需补)
【边际贡献条数】X / 3–5
【贡献维度覆盖】理论 □ 识别 □ 机制 □ 政策 □
【空话贡献命中】[...]（如"丰富了相关研究"）
【引用格式】著者—出版年制 ✅ / 出现顺序编码 ❌(需改)
【下一步】er-literature-review
```
