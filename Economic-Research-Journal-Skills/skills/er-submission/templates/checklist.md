# 《经济研究》投稿检查清单

> 口径说明：标注 **[官]** 者为已联网核实的官方/权威要求；标注 **[经验值]** 者为往年稿约或社区整理，
> 逐年可能微调，**以投稿当期官网（erj.ajcass.com「投稿须知 / 作者中心 / 下载中心」）为准**。
> 详见 `resources/official-source-map.md`。

## 投稿前格式检查

- [ ] 稿件为 DOC/DOCX 格式 [经验值，同系刊通行]
- [ ] 稿件已完全匿名化（正文中无作者、单位、课题等可识别信息）[官]
- [ ] 作者姓名、排序、单位、资助等信息置于**独立页**，不出现在正文 [官]
- [ ] 文件属性中无作者信息

## 作者信息检查 [官]

- [ ] 署名作者不超过 5 位
- [ ] 确认且仅确认一位通讯作者（未明确者以第一作者为通讯作者）
- [ ] 每位作者署名单位不超过 2 个
- [ ] 每位作者署名资助课题不超过 2 个；总资助课题数 ≤ 作者数 + 2
- [ ] 信息变更：录用前仅可变更一次，须提交签字书面说明

## 摘要与关键词检查

- [ ] 中文内容提要约 300 字 [官]
- [ ] 英文梗概（Summary）覆盖五要素：①背景与意义 ②主题与思路 ③方法/理论/数据及来源 ④主要结论及政策建议 ⑤重要创新与贡献 [官]
- [ ] 已另附英文梗概的中文对照（仅供编校，不刊出）[官]；官方未规定其字数（坊间"≥1000 字"无官方依据）[经验值]
- [ ] 中文关键词 3–5 个 [经验值]
- [ ] 英文 Key Words 与中文对应
- [ ] JEL 分类号已标注 [经验值，强烈建议]

## 正文结构检查

- [ ] 引言、文献述评、结论各部分表述不简单重复
- [ ] 结论严格演绎自前文理论框架
- [ ] 引言列出 3–5 条边际贡献
- [ ] 引言/文献述评篇幅充分、结论与政策讨论充分 [经验值；旧稿约曾有"引言≥5000字 / 结论≥1500字"，以官网为准]

## 内容质量检查

- [ ] 清晰说明与已有文献（**特别是本刊近 3 年已发表文章**）的学术贡献区别
- [ ] 机制分析可靠：只估计 D→M、M→Y 由理论论证，**未使用中介三步法/Sobel**（江艇，2022）
- [ ] 数学表达式的经济含义有文字阐述
- [ ] 政策含义落在"意义层"（制度/原理/宏观），非操作化备忘录
- [ ] 无空洞的"有意义""有价值"等表述
- [ ] 体现思想启迪性、理论创造性、政策参考性（该刊取向；反对"唯定量倾向"）[官]

## 图表与公式检查 [经验值，以官网为准]

- [ ] 三线表、无竖线；系数下方括号内为标准误并注明聚类层次
- [ ] 图形 ≥300 dpi，图题与来源说明完整
- [ ] 图表编号连续；公式独立成行、居中、带编号
- [ ] （旧稿约经验值：表 ≤6、图 ≤4、图表合计 ≤8、数学表达式 ≤20——逐年可能调整，以官网为准）

## 参考文献检查 [官：体例已核实]

- [ ] 采用**著者—出版年制**（非顺序编码制）：正文夹注（作者，年份[，页码]），如（林毅夫，2007，第 126 页）
- [ ] 同一作者同年多篇加 a/b/c；同处多篇用分号分隔，如（李四，2020；王五，2021）
- [ ] 文后参考文献**中文在前、外文在后**，各按姓氏拼音 / 字母 A–Z 排序
- [ ] **不出现** `[1][2]` 顺序编码、`[J]`/`[M]` 文献类型标识、`[27,28,29]` 并列引用
- [ ] 引用真实、完整、确切
- [ ] 条数：实证类约 30 条、理论类约 50 条 [经验值]
- [ ] 自引不扎堆（建议 ≤3 条）

## 学术规范检查

- [ ] 中文原创、本人独立完成，无一稿多发 [官]
- [ ] 已做学术不端检测，查重比例符合要求（社区经验：< 10%，以官网为准）

## 扩展资料检查（录用后按编辑部要求提交）

- [ ] 附录文件（Word 格式）
- [ ] 与统计软件匹配的数据文件或 Excel
- [ ] 软件代码文件（建议带一键复现 master，见 er-reproducibility）
- [ ] （如有）权利人限制说明
- [ ] 数据可得性声明（商业库不可公开者需说明来源与获取方式）

## 实证研究专项检查

- [ ] 数据来源点名到数据库（国泰安、Wind、税收调查、CFPS 等），非"公开渠道"
- [ ] 样本选择标准清晰、可复现；变量定义给公式而非描述
- [ ] 识别策略现代化：交叠 DID 报告异质性稳健估计量（CS/BJS/SA）+ Bacon 分解 + 事件研究图
- [ ] IV 报告 KP rk F + 有效 F + AR 稳健区间；RDD 报告稳健偏差校正 CI + rddensity
- [ ] 内生性、稳健性、机制、异质性齐备

## 投稿系统检查 [官]

- [ ] 在投审稿系统注册：<https://erj.ajcass.com/Admin/user/login>
- [ ] 系统中作者信息完整准确，稿件为最新版本
- [ ] 自 2018-10-01 起仅线上投稿，不接收纸质稿
