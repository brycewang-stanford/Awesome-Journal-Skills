# 投稿前 Preflight（er-submission）

## 触发时机

- "明天就要投了"
- 准备点提交按钮前的最后检查
- 不确定投稿系统需要哪些附件

> **口径说明**：下文标 **[官]** 者为已联网核实的官方/权威要求；标 **[经验值]** 者为往年稿约或社区整理，
> 逐年可能微调，**以投稿当期官网（erj.ajcass.com「投稿须知 / 作者中心 / 下载中心」）为准**。
> 逐条出处见 [`resources/official-source-map.md`](../../resources/official-source-map.md)。
> 细化清单与模板见 [`templates/checklist.md`](templates/checklist.md)、[`templates/manuscript_template.md`](templates/manuscript_template.md)。

## 投稿前 Checklist

### 匿名（双盲）[官]

- [ ] 正文中**任何能识别作者身份**的内容已去除
  - 自引使用"已有研究（XX，2020）"，不要写"作者前期研究"
  - 作者姓名、排序、单位、资助、致谢置于**独立页**，不在正文
  - 作者联系方式只在投稿系统中填，不写在正文
- [ ] 文件名、文件属性中不含作者姓名 / 单位

### 署名 [官]

- [ ] 署名作者 ≤ 5 位
- [ ] 确认且仅确认 1 位通讯作者（未明确者以第一作者为通讯作者）
- [ ] 每位作者署名单位 ≤ 2 个；署名资助课题 ≤ 2 个；总资助课题数 ≤ 作者数 + 2
- [ ] 信息变更：录用前仅可变更一次，须提交签字书面说明

### 摘要与关键词

- [ ] 中文内容提要约 **300 字** [官]
- [ ] 英文梗概（Summary）覆盖**五要素**（背景与意义 / 主题与思路 / 方法·理论·数据及来源 / 主要结论及政策建议 / 创新与贡献）[官]
- [ ] 已另附英文梗概的**中文对照**（仅供编校、不刊出）[官]；官方未规定其字数 [经验值：因梗概是浓缩全文的长摘要，对照通常明显长于约 300 字提要]
- [ ] 中文关键词 3–5 个 + 对应英文 Key Words [经验值]
- [ ] JEL 分类号已标注 [经验值，强烈建议——已刊文章普遍带 JEL，但官方投稿须知未见明确强制]

### 正文与篇幅

- [ ] 章节编号：一、二、三（一级）→（一）（二）（二级）→ 1、2、3（三级）
- [ ] 图表编号连续且与正文引用对应；公式独立成行、居中、右侧带编号
- [ ] 引言/文献述评篇幅充分、结论与政策讨论充分 [经验值；坊间"引言≥5000 / 结论≥1500 / 正文 1.5–2.5 万字"等数字**官方原文未见**，多源自他刊（《财贸经济》《经济研究参考》）转引，以官网为准]

### 参考文献 [官：体例已核实]

- [ ] **著者-出版年制**（非顺序编码制）：正文夹注（作者，年份[，页码]）；**不出现** `[1][2]`、`[J]/[M]`、`[27,28,29]`
- [ ] 文末**中文文献在前、外文在后**，各按姓氏拼音 / 字母 A–Z 排序
- [ ] 每条文献完整：作者、年份、题目、期刊（卷、期、页）/ 出版社；外文须同时标卷与期
- [ ] 文中每条引用都在文末列表，文末每条都在文中至少出现一次
- [ ] 理论文献齐全（《经济研究》易在此挑刺）；自引不扎堆（建议 ≤3 条）
- [ ] 条数适度 [经验值：坊间"≤50 条"官方未见，以官网为准]

### 学术规范 [官]

- [ ] 中文原创、本人独立完成；**无一稿多投**（官方处罚：故意抄袭/造假/一稿多投 → 10 年内不予发表）
- [ ] 已做学术不端检测，查重比例符合要求 [经验值：社区经验 < 10%，官方未发布数字阈值，以官网为准]
- [ ] 工作论文 / 会议论文版本是否已声明？

### 附件 / 扩展资料（多在录用后按编辑部要求提交）

- [ ] 致谢页、基金资助说明、作者简介（均**单独**提交）
- [ ] 附录（Word）；与统计软件匹配的数据文件 / Excel；软件代码文件（建议带一键复现 master，见 [`er-reproducibility`](../er-reproducibility/SKILL.md)）
- [ ] 数据可得性声明（商业库不可公开者说明来源与获取方式）；（如有）利益冲突声明、权利人限制说明

## 投稿系统操作要点 [经验值]

- 系统提交前再次确认作者信息完整、稿件为最新版本。
- 选择"研究类型"务必准确（实证 / 理论 / 综述 / 评论）。
- 推荐 / 回避审稿人、单次上传大小等具体规则**以投稿当期系统页面为准**（坊间数字官方未见，勿照搬）。

## 反模式

- 通宵改完没做双盲检查；作者信息残留在正文或文件属性
- 参考文献仍是 EndNote 默认 APA / 顺序编码格式，未转为**著者-出版年制**
- 图表编号断号 / 图 3 在正文中称"图 4"
- 把往年坊间数字（字数、查重%、文献上限）当**官方硬约束**写死——应以官网当期为准
- 投稿时把 working paper 版的致谢忘了删

## 输出格式

```
【匿名合规】通过 / 待修改：[...]
【署名规则】作者X位 / 通讯1位 / 单位≤2 / 课题合规：是·否
【摘要】中文≈300字 是·否 / 英文梗概五要素 齐·缺[...] / 中文对照 有·无
【参考文献体例】著者-出版年制 ✅ / 残留顺序编码 ❌
【学术规范】查重X% / 一稿多投声明 有·无
【附件齐全】是 / 否
【下一步】等待外审 / 收到 R&R → er-rebuttal
```
