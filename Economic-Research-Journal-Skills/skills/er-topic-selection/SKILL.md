# 经济研究选题（er-topic-selection）

## 触发时机

- 用户说"我想冲《经济研究》，但选题不确定"
- 边际贡献写不出来 / 写了一段但和审稿人能 buy 的角度不在一个频道
- 不确定该投《经济研究》、《管理世界》还是《经济学（季刊）》

## 三个硬约束

《经济研究》对选题的硬约束：

1. **理论贡献优先**：仅有"现象 + 实证 + 政策"不够，必须能挂上某一支经济学理论
2. **中国问题**：选题必须能在第一段说明"为什么这个问题在中国值得做"
3. **边际贡献可枚举**：能拆成 3–5 条独立的、互不重叠的贡献

## 理论贡献自检

写完一句话摘要后，逐条自检：

- [ ] 本文挂在哪一支理论文献上？（产业组织 / 公司金融 / 货币财政 / 制度经济 / 增长理论 / 国际贸易 / 行为经济）
- [ ] 该理论文献的代表性 Top-5 论文是哪 2–3 篇？
- [ ] 本文相对于该理论的边际差异是什么？（修正 / 扩展 / 引入新机制 / 提供首个证据）
- [ ] 中国情境是否提供了海外文献没有的识别机会？

少于 3 个 yes → 选题深度不够，回到理论文献再读两轮。

## 边际贡献四句式

```
本文的边际贡献主要体现在以下四个方面：
第一，[理论维度]——本文将……纳入[理论框架]，区别于[已有研究]……。
第二，[识别策略维度]——本文利用[政策 / 制度冲击]构造……，缓解了……内生性问题。
第三，[机制维度]——本文揭示了……机制，回答了已有文献"知其然不知其所以然"的问题。
第四，[政策意义维度]——本文为理解[中国 / 国际重要议题]提供了……证据。
```

四条都齐才算合格。少于四条 → 选题深度不够。

## 中国问题意识三问

《经济研究》反对"为计量而计量"，选题必须先过"中国问题"这一关。逐问自答：

1. **制度独特性**：这个问题的中国版本与海外版本有何制度差异？（如央地财政分成、国企双重目标、户籍约束）若答案是"没差异，只是换了中国数据"，选题会被视为复制性工作。
2. **识别机会**：中国情境是否提供了海外数据得不到的准实验变异？（试点推开、行政边界、改革时点错位）——这是把"中国数据"升级为"中国证据"的关键。
3. **理论回报**：研究结果能否反过来修正或扩展一般理论，而非只说"该理论在中国也成立"？最有力的表述是"中国情境揭示了该理论隐含假设 X 不满足时会发生什么"。

三问全部有实质答案，才谈得上"理论 + 中国问题"的双重合格。

### 一句话选题陈述（改写前 / 改写后）

> ❌ 改写前："本文研究数字化转型对企业绩效的影响。"（无理论、无中国、无识别）
>
> ✅ 改写后："本文利用某项分批推开的政策试点，考察征管技术冲击如何通过压缩避税空间改变企业要素配置——为'税收征管的资源配置效应'这一理论命题提供来自中国的准实验证据。"

## 选题红旗清单（出现任何一条先停下）

- [ ] 核心解释变量是可以随意替换的指数（换个指数故事照讲）——理论挂靠不实
- [ ] 摘要里写不出"区别于已有研究"的那半句——边际贡献未成形
- [ ] 冲击是全国同时实施、无分组变异——识别策略先天不足
- [ ] 故事只能讲到"有正向影响"，讲不出任何机制候选——写到机制节会卡死
- [ ] 问题在英文顶刊已有成熟答案，中国版本仅换数据——三问中的第 3 问不过关

## 期刊匹配判断

|              | 经济研究 | 管理世界 | 经济学（季刊） |
|----------|--------|--------|------------|
| 学科        | 经济学   | 管理 + 应用经济 | 经济学（偏方法） |
| 理论贡献    | **必备**  | 可选     | 必备         |
| 案例 / 质性 | 较少     | 接受     | 极少         |
| 字数        | 1.5–2.5 万 | 1.5–2.5 万 | 1.5–2 万    |
| 政策含义    | 意义层    | 操作层    | 可选         |

如果稿子偏 management/case，**不要投《经济研究》**。

## 近年选题主轴（命中率更高）

据 2024 年目录，该刊近年主轴：新质生产力、共同富裕 / 收入分配、数字经济 / 数据要素、双碳 / 绿色、央地财政关系、对外开放与高质量发展——可概括为"效率与公平的动态平衡 / 中国式现代化下的高质量发展"。**让你的问题挂上其中一条主轴**。

## 范本对标（确属《经济研究》）

挑选已核实的该刊范本对照你的选题站位（详见 [`resources/exemplars/`](../../resources/exemplars/)）：王永钦·董雯《机器人……劳动力市场》（2020 年第 10 期，IV）、张勋等《数字经济、普惠金融与包容性增长》（2019 年第 8 期，机制）、刘生龙等《义务教育法与教育回报》（2016 年第 2 期，RDD）。**勿误引**发表于《管理世界》《中国工业经济》的政策评估名篇。
