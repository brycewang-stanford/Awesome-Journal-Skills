# 投稿前审稿镜（er-reviewer-lens）

本 skill **不揣测某个具体审稿人的偏好**，也不模拟某位评审。它只把《经济研究》评审中**反复出现、可预判、结构性**的质疑点归纳成一面"自检镜"，供作者在投稿前自己先过一遍，提前堵住漏洞——把"投稿后才听到的批评"提前到"投稿前自己说出口"。

## 触发时机

- 初稿成形、准备进入 er-submission 之前
- 自我感觉"差不多了"但说不清还差什么
- 被退稿后想系统排查"到底踩了哪些高频雷"
- 合作者之间互审、想用统一口径挑刺

## 高频质疑清单（按论文环节）

每类给"审稿人会怎么问 → 你应如何提前预防 / 在文中回应"。

### 1. 理论定位类（本刊最高频、最致命）

| 审稿人会问 | 你应如何提前预防 |
|---|---|
| "理论贡献到底是什么？这只是一篇政策评估" | 引言第 2–3 段明确写出"本文在 X 理论上的边际推进"，而非只写"评估了 Y 政策的效果"。见 [`er-introduction`](../er-introduction/SKILL.md)、[`er-theory-hypotheses`](../er-theory-hypotheses/SKILL.md) |
| "没有和《经济研究》近 3 年同主题文章对话" | 文献综述显式引用本刊近 3 年 2–4 篇最相关文章，并写清"本文与它们的差异"。见 [`er-literature-review`](../er-literature-review/SKILL.md) |
| "几条边际贡献相互重叠 / 言过其实" | 贡献写 2–3 条、彼此正交、每条都能在正文找到对应小节；删掉"首次""填补空白"等夸张措辞 |

### 2. 识别策略类

| 审稿人会问 | 你应如何提前预防 |
|---|---|
| "为什么这项政策对个体是外生的？" | 给出制度背景证据（自上而下、与个体特征无关），而非一句"我们认为外生" |
| "TWFE + 交叠 DID 没回应异质性处理效应偏误" | 改用 CS / SA / dCDH / BJS 等稳健估计量 + Bacon 分解。见 [`er-identification`](../er-identification/SKILL.md) |
| "IV 排他性不成立 / 第一阶段弱" | 论证排他性的渠道唯一性，报告有效 F、AR 稳健 CI |
| "平行趋势只看图、不做检验" | 事前系数联合检验 + 功效 / Honest DiD 敏感性，不只贴图 |
| "有没有预期效应 / 提前反应？" | 检验处理前若干期系数、讨论 anticipation，必要时剔除"抢跑"窗口 |

### 3. 机制类

| 审稿人会问 | 你应如何提前预防 |
|---|---|
| "机制只是口头说明，没有证据" | 至少 1 张独立机制回归表。见 [`er-mechanism`](../er-mechanism/SKILL.md) |
| "中介三步法 / Sobel 不可信" | 按江艇（2022，《因果推断经验研究中的中介效应与调节效应》，《中国工业经济》第 5 期）：实证只做 D→M 回归，M→Y 用理论 + 文献论证，不直接跑 Y 对 M |
| "机制与主效应方向矛盾却不解释" | 若机制变量与主效应反向，必须给出经济学解释，否则删去 |

### 4. 异质性类

| 审稿人会问 | 你应如何提前预防 |
|---|---|
| "异质性没有理论指引，像事后挑显著的讲故事" | 每个异质性维度先有理论预期再看结果。见 [`er-heterogeneity`](../er-heterogeneity/SKILL.md) |
| "子样本太小，估计不可信" | 报告子样本量，太小的不分；必要时合并 |
| "两组系数差异没做显著性检验" | 用交互项 / Chow 检验确认组间系数差异显著，不靠"一个显著一个不显著"下结论 |

### 5. 数据与变量类

| 审稿人会问 | 你应如何提前预防 |
|---|---|
| "核心变量度量存疑" | 给出度量的文献依据 + 至少 1 个替代度量稳健性。见 [`er-data-sample`](../er-data-sample/SKILL.md) |
| "样本筛选不透明" | 列出筛选步骤与每步剔除的观测数（样本流程） |
| "描述统计里有异常值 / 量纲错误" | 缩尾或截尾并说明，核对单位与极值 |

### 6. 稳健性类

| 审稿人会问 | 你应如何提前预防 |
|---|---|
| "缺关键稳健性检验" | 对照识别策略补齐该设计的标准稳健性。见 [`er-robustness`](../er-robustness/SKILL.md) |
| "换度量 / 换样本结论就变" | 自己先跑这些扰动；若结论脆弱，先弄清原因再投，而非回避 |

### 7. 写作与规范类

| 审稿人会问 | 你应如何提前预防 |
|---|---|
| "空话套话，政策建议假大空" | 删模板句；每句话都要有信息量。见 [`er-style`](../er-style/SKILL.md) |
| "摘要没有量化结果" | 摘要写出核心效应的方向 + 量级。见 [`er-abstract`](../er-abstract/SKILL.md) |
| "参考文献体例不符" | 本刊为**著者—出版年制**：正文用（作者，年份），文末按作者—年份排，**不用 [1][2] / [J][M]** |
| "图表过载" | 主文只留核心图表，其余进附录。见 [`er-tables-figures`](../er-tables-figures/SKILL.md) |
| "语法 / 文字问题" | 通读一遍消灭错别字与病句——本刊明确把语法 / 文字作为评审标准之一 |

### 8. 政策含义类

| 审稿人会问 | 你应如何提前预防 |
|---|---|
| "政策含义太薄" | 把含义提到"意义层"：解释结论对理解某类经济机制 / 制度的启示，而非只罗列建议 |
| "政策建议太操作化，像给政府写的备忘录" | 避免"建议有关部门……"式条目；从研究发现自然推出含义。见 [`er-policy-implication`](../er-policy-implication/SKILL.md) |

## 投稿前 10 问自检（最可能被拒的 10 个问题）

逐条自问"我能不能当场答上来"：

1. 一句话说出本文的**理论贡献**（不是政策效果）——能吗？
2. 列出本文与《经济研究》近 3 年哪 2–3 篇文章对话、差异何在——能吗？
3. 为什么处理 / 政策对个体是**外生**的——有证据吗？
4. 交叠 DID 是否用了异质性稳健估计量并做了 Bacon 分解——做了吗？
5. 平行趋势 / 弱工具 / 平滑性，是否做了**检验**（而非只看图）——做了吗？
6. 机制是否有独立回归表，且没用不可信的中介三步法——满足吗？
7. 异质性维度是否**先有理论再看结果**、组间差异是否做了显著性检验——是吗？
8. 核心变量是否有替代度量、换度量后结论是否稳——验证过吗？
9. 摘要里有没有**量化**的核心结果——有吗？
10. 参考文献是否为著者—出版年制、全文有无错别字病句——核对了吗？

> 任何一问答不上来，就是高风险项，优先修补再投。

## 必查清单

- [ ] 理论贡献能一句话讲清，且不等同于政策效果
- [ ] 与本刊近 3 年同主题文章显式对话
- [ ] 边际贡献 2–3 条、互不重叠、无夸张措辞
- [ ] 识别外生性有证据；交叠 DID 用稳健估计量
- [ ] 平行趋势 / 弱工具等做了检验而非贴图
- [ ] 机制有独立表、按江艇（2022）口径处理中介
- [ ] 异质性有理论指引、组间差异做显著性检验
- [ ] 核心变量有替代度量稳健性
- [ ] 摘要含量化结果；政策含义落在"意义层"
- [ ] 参考文献著者—出版年制；语法文字通读无误

## 反模式

- 把这份清单当成"投稿后被拒了才来看"——它是**投稿前**的镜子
- 只做表面功夫应付：补一句话、贴一张图，不真正回应质疑
- 忽视理论定位类质疑，只埋头补技术细节——技术再扎实，理论站不住仍会被拒
- 自检发现结论脆弱却选择回避、赌审稿人看不出来

```
  - [机制] 仅口头说明，无回归表           | 中  | er-mechanism

```
