# 语言风格（er-style）

## 触发时机

- 通读初稿后觉得"读起来像政策文件 / 工作汇报"
- 一段里出现 ≥ 2 个"具有重要意义" / "为...提供新视角"
- 政策建议读起来像"加强、完善、推进"三件套
- 自检清单出现"经济含义未阐述"

## 对照表（黑名单 → 白名单）

| 问题类型 | 黑名单（删/改）                       | 白名单（替换为）                                        |
|----------|-----------------------------------|---------------------------------------------------|
| 空洞价值 | "具有重要的理论价值"               | "为理解 X 提供了 Y 视角下的理论框架"               |
| 空洞意义 | "对 XX 具有重要的参考价值"         | "本文结果意味着 XX 政策的传导渠道应纳入 YY"      |
| 泛泛建议 | "应加强监管 / 完善制度 / 推进改革" | "监管部门应在 X 环节建立 Y 制度，重点关注 Z 类企业" |
| 方法炫耀 | "本文使用复杂的 DSGE 模型"         | "本文构建 DSGE 模型，引入 X 摩擦以模拟 Y 情形"     |
| 贡献模糊 | "丰富了现有研究"                   | "首次利用交叠 DID 识别策略考察了 X 对 Y 的影响"   |
| 文献堆砌 | "（张三，2020；李四，2021；...）"  | 按贡献分述：理论文献一段、实证文献一段、本文位置一段 |
| 表格无解读 | "表 3 报告了基准回归结果。"       | "表 3 显示，X 上升 1 个标准差使 Y 提高 0.04 个标准差，约相当于均值的 X%" |
| 公式无含义 | 公式后直接进入下一段              | "式（3）右端第二项刻画了 X 的调整成本，当 Y > 0 时..."|
| 套话型贡献 | "为 XX 提供新的视角"               | "区别于 XXX（202X）使用的 OLS，本文..."           |

## 行文规范

### 时态

- 描述本文方法/数据 → **现在时**：本文使用 / 本文构建
- 报告实证结果 → **过去时**：研究发现 / 我们检验了
- 解释结果的经济含义 → **现在时**：这意味着 / 这一发现表明

### 人称

- 避免"笔者""我们认为"出现在结论性段落
- 可用："本文发现 / 研究表明 / 结果显示"
- 英文版避免 "We feel that..." / "It is believed that..."

### 谦逊与边界

- 不要写"本文证明了"——经验研究只能说"本文提供了证据支持"
- 不要把统计显著当作经济显著——同时报告**经济量级**

## 中文学术句法四则

1. **长句拆分**：一句话超过两行、含 ≥3 个"的"字嵌套，拆成两句——先给判断，再给条件。计量结果句尤其如此："在控制 A、B、C 并考虑 D 的情形下 X 对 Y 的影响依然显著"应改为"基准结论稳健。控制 A、B、C 后，……"。
2. **主语不悬空**：删掉"通过对……的分析，发现……"这类无主语句，改为"本文分析……，发现……"。
3. **数字与量级**：正文报告系数时同时给参照系（相对均值 / 标准差的百分比）；"约 X%"与"X 个百分点"严格区分，全篇一致。
4. **夹注融入句子**：著者-出版年夹注（作者，年份）应挂在观点后而非句首堆砌；一处夹注多于三篇文献时，考虑是否真的需要全部列出。

## 术语一致性

概念在**首次出现处定义一次，此后全篇只用同一个名字**。常见翻车点：

- "全要素生产率 / TFP / 生产率"混用——首次写"全要素生产率（TFP）"，此后统一用 TFP
- 核心解释变量在引言叫"数字化转型"、模型节叫"数字化水平"、表格叫 Digital——三处必须收敛为一个中文名 + 一个变量名
- "显著"只用于统计意义；经济上的"大 / 可观"不写作"显著"

### 微型对照（改写前 / 改写后）

> ❌ 改写前："通过使用双重差分模型进行的实证检验可以发现，政策的实施对于企业的创新水平的提升具有显著的正向的促进作用。"
>
> ✅ 改写后："双重差分估计显示，政策使企业专利申请量提高约 8%，相当于样本均值的四分之一。"

改写删掉了五个"的"、一个悬空主语和一处"显著促进"套话，并补上了经济量级。

## 一段一概括

每个段落第一句必须是**概括句**，后续句子是支撑。审稿人快速读完每段首句应能复原文章逻辑。

错误：
> 张三（2020）研究发现...。李四（2021）发现...。王五（2022）发现...。

正确：
> 既有研究就 X 对 Y 的影响达成了两种对立的判断（张三，2020；李四，2021）。一类研究认为...，另一类研究认为...。两种判断的分歧在于...。

## 行文自检清单

- [ ] 每段首句是概括性陈述
- [ ] 黑名单短语命中数 = 0
- [ ] 政策建议指明了"谁应该做什么"
- [ ] 表格后有经济含义解读，不仅报数字
- [ ] 数学公式有文字阐述经济含义
- [ ] 时态、人称在全文统一
- [ ] 显著性与经济量级同时报告

## 反模式

- 通篇"具有重要的 X 意义" —— 一次都不应出现
- 政策建议用"加强、完善、推进、深化"四个动词撑满段落
- 方法描述长于结果阐释
- 引言里塞 5 个并列文献 `[27,28,29,30,31]` —— 拆开分述
