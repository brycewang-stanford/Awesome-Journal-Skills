## 一、按识别方法

### 交叠 / 多期 DID（中国政策冲击）
- **范子英、赵仁杰（2019, 第3期）《法治强化能够促进污染治理吗？——来自环保法庭设立的证据》**
  ——环保法庭交错设立为准自然实验，283 地级市面板；平行趋势 + 机制（行政处罚/公众参与）+ 异质性。绿色/法治 DID 模板。
- **刘金科、肖翊阳（2022, 第1期）《中国环境保护税与绿色创新：杠杆效应还是挤出效应？》**
  ——三重差分 DDD（区域×行业×时间）；双碳/绿色创新标杆，识别比普通 DID 更干净。
- **刘冲、庞元晨、刘莉亚（2022, 第1期）《结构性货币政策、金融监管与利率传导效率——来自中国债券市场的证据》**
  ——把货币政策识别为微观冲击做 DID；宏观题材用微观 DID 的范本。

> ⚠ 诚实提示：广为流传的「明星政策冲击」交错 DID（沪港通、产业政策、国家高新区、低碳城市）
> **多发表于《管理世界》《中国工业经济》，不是《经济研究》**。经研内的政策 DID 更偏制度类
> （环保法庭、环境税 DDD）。勿误标——详见 source map 第三节避坑表。

### IV / 工具变量
- **王永钦、董雯（2020, 第10期）《机器人的兴起如何影响中国劳动力市场？——来自制造业上市公司的证据》**
  ——Bartik / 份额移动 IV（美国行业机器人渗透 × 初始就业结构权重），教科书级；技能异质性。（深度拆解见 README §1）
- **陈媛媛、张竞、周亚虹（2022, 第1期）《工业机器人与劳动力的空间配置》**——IV 处理内生性，推进到空间配置维度。

### RDD 断点回归
- **刘生龙、周绍杰、胡鞍钢（2016, 第2期）《义务教育法与中国城镇教育回报率：基于断点回归设计》**
  ——Sharp RDD（出生年份对 1986《义务教育法》暴露）；中文顶刊 RDD 标杆。（深度拆解见 README §3）
  > 注：35 岁就业歧视、最低工资、名校回报等 RDD 多在《经济学（季刊）》《经济学报》，非《经济研究》。

### 行政 / 准实验断点
- **范子英、田彬彬（2013, 第9期）《税收竞争、税收执法与企业避税》**
  ——2002 所得税分享改革注册时点决定国税/地税征管强度（年份略早，识别设计经典）。（深度拆解见 README §4）

### 结构估计 / 理论模型 + 实证
- **林东杰、崔小勇、龚六堂（2022, 第1期）《金融摩擦异质性、资源错配与全要素生产率损失》**
  ——理论模型刻画金融摩擦 → 量化 TFP 损失，结构估计范本。

### 宏观（生产网络 / 货币财政）
- **黄益平、徐诗语、余昌华、杜浩锋、伍晓鹰（2024, 第9期）《生产网络视角下的中国经济周期》**
  ——生产网络 + 投入产出传导；该刊近年宏观/结构方向范本。（深度拆解见 README §5）

### 机制分析（黄金模板）
- **张勋、万广华、张佳佳、何宗樾（2019, 第8期）《数字经济、普惠金融与包容性增长》**
  ——数字金融→农村创业→包容性增长，逐层拆渠道 + 多重资本异质性；封面文 + 多奖。（深度拆解见 README §2）

### ⚠ 缺口（诚实告知）
- **合成控制法（SCM）**：未找到确属《经济研究》的范本；最著名的刘甲炎、范子英《房产税试点效果评估：基于合成控制法》在**《世界经济》2013(11)**，**非《经济研究》**。需 SCM 范本时，建议在 CNKI「来源=经济研究 + 主题=合成控制」二核后再用，勿误标。
