本目录用**一个连贯的实证案例**，逐节演示一篇《经济研究》风格论文"好"应该长什么样：从引言到回复信，每一节都对应一个 `er-*` skill 的口径。这是技能包"生成"模式的参照样例——你可以让 Agent"照着这个范例，把我的稿子改到这个水平"。

> ⚠ **重要声明：本范例为教学构造。** 案例所用政策（"金税三期"工程）是真实的，但**文中所有回归系数、标准误、样本量、机制数值均为示意性虚构**，**不来自任何已发表论文，也不构成对该政策真实效应的论断**。引用的方法论文献（如江艇 2022、Callaway-Sant'Anna 2021）是真实的；引用的"本文发现"是演示用的。请勿把本范例的数字当作事实或直接搬入投稿。

## 案例设定（Case Bible —— 全库共用，务必前后一致）

**题目**：税收征管数字化与企业全要素生产率——来自"金税三期"工程的证据

**情境**：金税三期（GTP3）税收征管信息系统于 2013–2016 年分批在各省上线，形成**交错（staggered）政策冲击**。处理 `D_it = 1` 当企业 i 所在省份在 t 年已上线 GTP3。

**数据**：2010–2018 年 A 股上市公司（CSMAR）+ 各省 GTP3 上线时点（手工整理）。样本约 22,000 个企业—年观测、约 3,200 家企业。剔除金融业、ST/*ST、关键变量缺失；连续变量 1%/99% 缩尾。

**被解释变量 Y**：企业全要素生产率 `tfp_lp`（Levinsohn-Petrin 法）。稳健性另用 OP 法、OLS 残差法。

**核心发现（示意数字，全库统一）**：
- 基准 TWFE：`D` 系数 = **0.041**（se 0.013）
- 异质性稳健主结果（Callaway-Sant'Anna / BJS 插补）：ATT ≈ **0.038**（se 0.011），显著于 1%；约相当于 TFP 样本均值的 0.9%、约 0.05 个标准差。
- 事件研究：处理前各期系数不显著（无预趋势），效应在上线后出现并在 3 年内走强。
- Bacon 分解：好比较（已处理 vs 未处理）权重占主导。

**机制（江艇 2022 口径：只跑 D→M，M→Y 靠理论论证）**：
- **M1 税收遵从度↑**：GTP3 使有效税率 `etr` 上升约 **0.6 个百分点**（D→M1 显著）。理论：征管数字化压缩避税空间→减少扭曲性避税投入与寻租→资源配置效率↑→TFP↑。
- **M2 融资约束缓解**：GTP3 使 SA 指数下降（融资约束减轻；D→M2 显著）。理论：更规范的涉税财务信息降低信息不对称→外部融资可得性↑→研发与设备投资↑→TFP↑。

**异质性（理论驱动）**：
- **产权**：民企效应（0.052）> 国企（0.018），差异显著——国企软预算约束、避税动机弱。
- **市场化程度**（王小鲁指数）：市场化低地区效应更大——征管对制度环境的"替代效应"。
- **外部融资依赖**（行业层 Rajan-Zingales 度量）：高依赖行业效应更大——与 M2 机制呼应，构成机制侧证。

**主要稳健性**：替换 TFP 度量；替换处理口径；排除同期"营改增"干扰（加营改增虚拟变量）；安慰剂（随机指派上线时点 500 次，真实系数落在分布最右尾）；PSM-DID；省级聚类（31 省 < 40 → wild cluster bootstrap，p = 0.003）；Oster δ = 1.7 > 1；CS/BJS 与 TWFE 方向量级一致。

**四条边际贡献**：
1. **理论**：把"征管技术—税收遵从—资源配置效率"纳入企业 TFP 决定框架，区别于已有仅关注税率/税负水平的研究。
2. **识别**：利用 GTP3 交错上线构造准自然实验，采用异质性稳健 DID，缓解政策内生性与 TWFE 偏误。
3. **机制**：揭示税收遵从与融资约束双渠道，打开"征管→实体效率"的黑箱。
4. **政策意义**：为理解征管现代化/数字政府的微观效率含义提供证据，服务高质量发展。
