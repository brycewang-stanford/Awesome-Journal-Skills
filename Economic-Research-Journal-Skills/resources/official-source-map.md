## 二、需复核 / Live-check——须以投稿当时官网「投稿须知 / 作者中心」为准

> 以下数字此前在本技能包模板中以「硬性规定」出现，但在本次可加载的**官方来源中未找到原文依据**。
> 它们很可能源自往年《经济研究》投稿须知或经验整理，**逐年可能微调**。本技能包正文凡引用这些数字处，
> 均按「经验值 / 待官网复核」处理，不以官方硬约束口吻断言。

- [ ] **正文字数**（常见经验值「正文含图表 1.5–2.5 万字」）——官方来源未见明确条文。
- [ ] **英文梗概字符数**（此前写「约 4400 字符含空格」）——官文只给五要素 + 中文对照，**未给字符数**；勿断言 4400。
- [ ] **参考文献条数上限**（此前写「≤50 条」）——官文未见。
- [ ] **图 / 表 / 公式数量上限**（此前写「表 ≤6、图 ≤4、图表 ≤8、公式 ≤20」）——官文未见。
- [ ] **引言文献述评 ≥5000 字 / 结论政策讨论 ≥1500 字**——官文未见明确下限。
- [ ] **关键词数量、JEL 分类号、中图分类号、文献标识码**是否强制——官文未见；同系刊《财贸经济》要求 JEL + 中图分类号，但**不能直接套用到《经济研究》**。
- [ ] **审稿费 / 版面费**——官方来源未见任何金额；社科院系刊多为不收费，但《经济研究》本身未见明确条文，勿杜撰。
- [ ] **审稿周期 / 录用率**——无权威数字（二手转引「1–3 个月」不足采信）。
- [ ] **「双向匿名」措辞**——官方仅见「匿名审稿」，是否严格「双向」措辞待核。
- [ ] **DOC/DOCX 格式与单次上传大小上限**——《经济研究》本身未见明确条文（同系刊要求 DOC/DOCX）。
- [ ] 外文参考文献**逐字范例**——建议从投稿系统「下载中心」的「英文文献引用说明」house 文件取官方样例。

> 复核方式：联网后优先访问投审稿系统 https://erj.ajcass.com 「投稿须知 / 作者中心 / 下载中心」，
> 以 CNKI 刊名页核对 ISSN/CN/创刊年/收录，并将上表逐条替换为带访问日期的确证信息。

## 四、本技能包据此应保持的口径

1. 参考文献一律按**著者-出版年制**书写与示例；**删除「顺序编码制」「[J]/[M]」「[27,28,29]」等顺序编码痕迹**。
2. 凡引用「字数 / 条数 / 图表 / 公式」上限，统一加「（经验值，以投稿当期官网为准）」口径，不作官方硬约束断言。
3. 英文梗概按**五要素 + 中文对照**写，不强调具体字符数。
4. 范文只用第三节「已核实」清单中的篇目；如需 staggered DID 范本，明确标注其真实期刊（多在《管理世界》/《中国工业经济》），不冒充《经济研究》。
