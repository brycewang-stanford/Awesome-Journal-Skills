# CR Pre-Submission Checklist (worksheet)

Companion to [`../SKILL.md`](../SKILL.md). Work top to bottom before uploading to **SAGE Track**
(ScholarOne). Verify every volatile item against the current SAGE author page
(`journals.sagepub.com/author-instructions/crx`) — items marked 待核实 change (检索于 2026-06；以官网为准).

## 0. Fit & contribution
- [ ] The study is **quantitative and theory-testing** (experiment / survey-panel / content analysis)
- [ ] A **tested communication mechanism** (mediation/moderation), not a bare main effect
- [ ] Numbered **hypotheses** (and RQs) follow from the positioned theory
- [ ] Home-journal check done vs. *Journal of Communication* / *Human Communication Research* /
      *New Media & Society* (not qualitative/interpretive for this venue)

## 1. Design & measurement
- [ ] Estimand / claim stated; identifying assumption named and defended
- [ ] Experiments: a-priori power/MDE; **stimulus sampling** (message as random factor); attention/manipulation checks
- [ ] Surveys/panels: validated multi-item scales; **reliability (alpha/omega)** reported; CMV addressed
- [ ] Content analysis: documented codebook; **intercoder reliability** on a subsample; unit of analysis stated
- [ ] Causal-ordering warrant for any mediation/process claim (panel/experiment, not cross-section)

## 2. Analysis & reporting (APA)
- [ ] Right model for the design (ANOVA/regression/SEM/multilevel)
- [ ] **Effect sizes** (d, η²ₚ, β, R²) + CIs/SDs reported — not significance stars alone
- [ ] Mediation reported with **bootstrap CIs**; moderated mediation with the **index**
- [ ] Confirmatory vs. exploratory analyses separated; multiple-testing handled where needed
- [ ] Robustness probes that could break the result, with what was learned

## 3. Abstract & exhibits
- [ ] Abstract: **unstructured, <= 150 words**, placed between title and main body; states question + design + **tested mechanism** + finding (clinical-trial registration, if any, at the end of the abstract)
- [ ] Main result reported with magnitude + uncertainty in interpretable units
- [ ] A mechanism exhibit (path diagram / simple-slopes plot) included where central
- [ ] Captions self-contained; figures colorblind-safe, grayscale-legible

## 4. Anonymity (double-anonymized)
- [ ] No author names, affiliations, or acknowledgments in the **main document**
- [ ] **Supplemental materials also anonymized** (no identifying links/paths)
- [ ] **Self-citations in the third person**
- [ ] File metadata stripped (document properties, comments, tracked changes)
- [ ] Separate, non-anonymous **title page** prepared as a distinct file

## 5. Length, format & metadata
- [ ] Manuscript **≤ 45 pages, double-spaced, inclusive of references** (检索于 2026-06；以官网为准)
- [ ] Full specs, scale items, robustness grids, codebooks moved to the **online supplement**
- [ ] **APA** formatting and citation style; consistent throughout
- [ ] Corresponding-author **ORCID** ready (confirm requirement on the live page)

## 6. Transparency & compliance
- [ ] **Data-availability statement** drafted (repository + identifier, or exemption)
- [ ] Materials staged: master script reproduces every table/figure; README; pinned versions; seeds
- [ ] Restricted data: restriction explained + access path + synthetic data where feasible
- [ ] Open-materials/data deposited if claiming open practices; preregistration noted in cover letter
- [ ] Ethics / IRB / human-subjects compliance and informed consent stated

## 7. Final
- [ ] Every number in the text matches the deposited analysis output
- [ ] Files and metadata required by SAGE Track assembled
- [ ] After submission: await decision → on R&R, go to [`../../commres-rebuttal/SKILL.md`](../../commres-rebuttal/SKILL.md)
