# 资源目录 · 《通信学报》(Journal on Communications, JOC)

本目录汇集《通信学报》(Journal on Communications, JOC) 技能包引用的支撑材料。JOC 由中国通信学会
主办，聚焦通信理论与技术、无线/移动通信、网络与交换、信息安全与密码、信号处理等方向。所有事实以
`official-source-map.md` 的双源核验为准，**待核实** 项不得写成定论。

## 目录结构

| 路径 | 内容 |
|---|---|
| `official-source-map.md` | 官方信息源对照表 + 访问方式说明 + **待核实** 清单 |
| `external_tools.md` | 写作、仿真、查重、参考文献、绘图等外部工具清单 |
| `exemplars/library.md` | 通信领域范例结构与"决定性"拆解 |
| `worked-examples/01-introduction.md` | 一份符合本刊体例的引言写作范例 |
| `code/README.md` | 复现包与仿真代码的组织建议 |

## 使用约定

- 引用本刊事实前先回读 `official-source-map.md`；
- 数值（费用、周期、影响因子）若属 **待核实**，在产出中保留标注或改写为"以投稿系统内当期通知为准"；
- 英文刊名统一写作 Journal on Communications (JOC)，与《通信学报》并现。
