# 范例库 · 《通信学报》(Journal on Communications, JOC)

本库收录《通信学报》(Journal on Communications, JOC) 通信学科长文的**结构范式**与"决定性"拆解，
帮助把方法讲透、把证据摆足。范例为结构示意，不复制任何已发表论文原文。

## 一、典型研究长文骨架

| 部件 | 要点 | 占比（约） |
|---|---|---|
| 中文标题（≤20 字） | 点明问题+方法+场景 | — |
| 中英文结构式摘要 | 目的/方法/结果/结论；中文≤200 字 | — |
| 关键词（≥4） | 中英对应，覆盖方法与场景 | — |
| 引言 | 背景→问题→现有不足→本文贡献（3~4 条） | 12% |
| 相关工作 | 以 delta 为主线，非流水账 | 10% |
| 系统模型/问题建模 | 符号表、假设、信道/网络模型、优化目标 | 15% |
| 方法/算法 | 核心贡献；算法伪代码、复杂度分析、（必要时）定理与证明 | 25% |
| 实验/仿真 | RQ、基线、参数表、SNR-BER/吞吐/时延曲线、消融、置信区间 | 25% |
| 结论与展望 | 回扣贡献、局限、后续 | 5% |
| 参考文献 | GB/T 7714，中英文对照 | 3% |

## 二、三个方向的范式差异

- **物理层/信号处理**：重"信道模型 + 理论推导 + 蒙特卡洛仿真"。决定性证据 = 在标准信道（AWGN/
  Rayleigh/多径）下 BER/容量曲线优于基线，且有理论界佐证。
- **网络与交换**：重"协议设计 + 系统级仿真/实测"。决定性证据 = 在 NS-3/OMNeT++ 或真实拓扑下
  吞吐、时延、丢包、公平性优于对照，且开销可接受。
- **信息安全与密码**：重"威胁模型 + 安全性证明/形式化验证 + 性能开销"。决定性证据 = 在明确
  威胁模型下给出可归约的安全性证明或工具验证，并量化计算/通信开销。

## 三、"决定性"三问（判定入正文还是附录）

1. **移除它，主结论是否崩塌？** 是 → 正文；否 → 附录（见 `jonc-supplementary`）。
2. **审稿人复核创新性/正确性时必看吗？** 是 → 正文（如关键推导、主实验）。
3. **它只是完备性佐证吗？** 是 → 附录（如超参搜索、额外场景、冗长证明细节）。

## 四、图表范例清单

- 系统框图（发射机-信道-接收机 / 网络拓扑 / 协议时序）。
- 主结果曲线：SNR-BER、容量-SNR、吞吐-负载、时延 CDF，均标注置信区间。
- 消融/敏感性：关键参数（码率、天线数、节点数、密钥长度）扫描。
- 复杂度/开销表：时间/空间复杂度、通信轮数、能耗。

> 用法：先用第三节"决定性三问"划定正文边界，再对照第一节骨架自查部件齐备，最后按方向范式补足
> 该领域外审最在意的证据类型。这些都以《通信学报》/Journal on Communications (JOC) 的通信学科
> 评审口味为准。
