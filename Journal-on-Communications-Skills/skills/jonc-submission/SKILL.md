# 《通信学报》投稿就绪审计

在把稿件上传到《通信学报》(Journal on Communications, JOC) 的 CBPT 在线投稿系统之前，先跑完这份
审计。JOC 由中国科学技术协会主管、**中国通信学会(China Institute of Communications)** 主办，是
通信学科的中文旗舰**月刊**（每月 25 日出版），聚焦通信理论与技术、无线/移动通信、网络与交换、
信息安全与密码、信号处理。它按期刊稿件走**初审→外审→终审**三审制，因此投稿被当作一篇要经受
编辑初审、专家外审、编委会终审的完整学术论文来评判。以下事实于 2026-07-09 通过官网、CNKI 期刊
主页与 CCF 推荐中文科技期刊目录交叉核验（见 `resources/official-source-map.md`），凡本刊未公开
确认的数字均标注 **待核实**，请以投稿系统内最新《投稿须知》为准。

## 一、投稿系统与账号

- 本刊**网上投稿**，投稿入口为 CBPT 系统 `http://cbpt.infocomm-journal.com/`（作者中心登录）；
  官网 `https://www.infocomm-journal.com/txxb`；编辑部邮箱 `txxb@bjxintong.com.cn`；
  电话 010-53878169 / 53859522 / 53878236。
- 系统内投稿须**完整填写"添加作者信息"**：姓名、职称、职务、工作单位、地址、邮编、电话、电子
  邮箱、身份证号码等，缺项可能卡在初审。
- 题名、作者、单位、通信作者与研究方向决定编辑分派外审专家的匹配度，务必用**真实终稿信息**。
- 通信作者邮箱须长期可达，修回与录用通知均经此发送。

## 二、行文次序与篇幅（官方口径）

官方规定的行文次序为：**中文**（标题、作者姓名、作者单位、单位所在省市与邮编、摘要、关键词）→
**中图分类号、文献标识码、文章编号** → **英文**（标题、作者姓名、作者单位、城市名与邮编、摘要、
关键词）→ 正文 → 参考文献。要点：

- **中文标题不超过 20 个汉字**，必要时可加副标题；避免非正规缩略语、代号、结构式与公式入题。
- **中文摘要 200 字以内**，要素为①目的②方法③结果④结论⑤其他；英文摘要与之对应。
- **关键词至少 4 个**，列于摘要之后，中英文一一对应。
- **基金项目**置通信作者下方，括号内注项目编号；多项以分号隔开。
- **全文约 12000 字符（约 12 页）**，**综述可酌情延长至约 15 页**（**待核实** 精确页数上限）。

## 三、中文科技论文的硬性组件

《通信学报》的稿件必须齐备以下部件（详见 `jonc-writing-style`）：

| 组件 | 要求 |
|---|---|
| 中文题名 + 英文题名 | 中文≤20 字；英文可独立检索、与中文一致 |
| 中文摘要 + English Abstract | 结构式（目的/方法/结果/结论）；中文≤200 字 |
| 中文关键词 + Keywords | **≥4 个**，中英一一对应 |
| 中图分类号 | 通信/电子类以 **TN** 打头（如 TN915、TN918、TN929）；据主题选 |
| 文献标识码 + 文章编号 | 按本刊模板填写 |
| 作者简介与基金项目 | 注明资助项目及编号；作者简介含研究方向 |
| 参考文献 | **GB/T 7714**；中文文献需完整著录，必要时中英文对照 |

## 四、学术规范与伦理红线

- **知网查重**：已引入 CNKI 查重系统，与已发表论文**重复率超过 20% 不予收录**。自查时压到远低于
  该线，尤其注意自引与前序会议/预印本版本的重叠。
- 若本文是已发表会议论文的**期刊扩展版**，必须显著声明扩展关系、指明新增贡献（新增理论、新增
  实验、新增系统模块通常需达到实质比例），并规范引用会议版本。
- **一稿多投**明确禁止：投稿即视为该稿未在其他刊物同时评审。违规将通报作者单位，并不再接受
  第一作者的投稿。
- 署名、贡献、基金、数据来源须真实；不得伪造、篡改数据或代写。

## 五、费用（以投稿系统内当期通知为准）

| 环节 | 口径 | 说明 |
|---|---|---|
| 投稿 | **不收取稿件处理费** | 投稿本身免费 |

## 六、退稿风险分诊表

| 审计发现 | 风险等级 | 处置 |
|---|---|---|
| 与已发表论文重复率 > 20% | 初审退稿级 | 重写重叠段落、补足原创增量后再投 |
| 选题偏离通信学科（宽电子/纯计算机） | 初审退稿级 | 参见 `jonc-topic-selection` 重新定位或改投兄弟刊 |
| 中文摘要超 200 字 / 关键词 < 4 | 需修改 | 压缩摘要、补足关键词 |
| 无中图分类号/文献标识码/文章编号 | 需修改 | 按模板补齐 |
| 参考文献不合 GB/T 7714 | 需修改 | 统一著录格式 |
| 实验缺基线/置信区间 | 外审重伤 | 补对照与蒙特卡洛统计（见 `jonc-experiments`） |
| 会议扩展未声明新增贡献 | 学术规范风险 | 显式声明扩展关系与增量 |

## 六点五、栏目与稿件类型对照

投稿时需在系统内选择稿件类型/栏目，选错会影响初审匹配：

| 类型 | 适用 | 篇幅提示 |
|---|---|---|
| 学术论文 | 完整原创研究（方法+理论/系统+充分实验） | 约 12000 字符 |
| 技术报告 | 工程性强、以系统实现与实测为主 | 参照论文，突出工程价值 |
| 综述 | 系统梳理某方向、批判性归纳与展望 | 可延至约 15 页 |
| 短文 | 单点创新、篇幅较短 | 聚焦、精炼 |

若类型与内容不符（如把综述投成学术论文），初审可能要求改类型或退修。
