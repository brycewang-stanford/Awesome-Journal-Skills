# 《通信学报》录用后定稿与出版

本技能覆盖《通信学报》(Journal on Communications, JOC) 稿件录用后的收尾工作。JOC 由中国通信学会
主办，是通信学科中文旗舰**月刊**（每月 25 日出版）。录用只是起点，定稿、校样、缴费、排期都需
认真对待。以下与 2026-07-09 核验的本刊口径一致（见 `resources/official-source-map.md`；精确排期与
版权流程 **待核实**，以编辑部通知为准）。

## 一、定稿总流程

| 步骤 | 内容 | 注意 |
|---|---|---|
| 终排 | 按官方模板排版终稿 | 行文次序、中英文摘要、参考文献 GB/T 7714 |
| 校样 | 核对清样/校样 | 逐字校对，尤其公式、符号、图表 |
| 缴费 | 版面费约 600 元/面 | 以缴费通知为准；缴后开票 |
| 版权 | 签署版权/授权协议 | 按编辑部要求 |
| 排期 | 确定见刊档期 | 受月刊排期影响 |

## 二、校样核对清单

- [ ] 中文题名（≤20 字）与英文题名一致、无误。
- [ ] 中英文结构式摘要（中文≤200 字）与关键词（≥4）正确。
- [ ] 中图分类号 TN、文献标识码、文章编号无误。
- [ ] 作者姓名、单位、通信作者与邮箱正确；基金项目与编号无误。
- [ ] 公式、符号（斜体/正体）、量与单位在排版后未错位。
- [ ] 图表编号连续、清晰、被正文引用；曲线含置信区间。
- [ ] 参考文献 GB/T 7714 著录完整、序号与正文对应。
- [ ] 交叉引用（图 X、式 Y、表 Z、文献[n]）在终排后仍正确。

## 三、缴费

- 本刊**不收稿件处理费**；录用排版后**按篇幅收取版面费（约 600 元/面）**，以邮件/系统通知为准。
- 及时缴纳以便编辑部开票并安排刊出；精确当期数额 **待核实**。
- 保留缴费凭证与发票信息。

## 四、版权与授权

- 按编辑部要求签署版权转让/授权协议（**待核实** 具体条款）。
- 确认基金资助致谢、通信作者信息无误。
- 若涉及已发表会议版扩展，确认声明与引用规范。

## 五、出版与排期意识

- 本刊为月刊，每月 25 日出版；录用到见刊需排队，具体档期 **待核实**。
- 关注是否有网络首发/优先出版（**待核实**），以缩短见刊等待。
- DOI 与见刊卷期以正式出版为准。

## 六、常见收尾失误

| 失误 | 后果 | 预防 |
|---|---|---|
| 校样漏改公式/符号 | 见刊错误难追回 | 逐字校对，重点公式 |
| 缴费延迟 | 影响排期 | 收到通知即办 |
| 作者/基金信息错 | 影响署名与结题 | 终排核对 |
| 参考文献错位 | 学术规范瑕疵 | 核对序号与著录 |
| 图表模糊/黑白不可辨 | 影响可读性 | 提供矢量图、线型区分 |

## 七、定稿阶段与外审修回的区别

录用后的定稿不是再一次评审，而是**编辑加工与出版准备**。常见误区是把校样当作"再改一版内容"：

| 项目 | 修回阶段 | 定稿阶段 |
|---|---|---|
| 目标 | 回应意见、改进内容 | 核对排版、准备出版 |
| 可改动范围 | 内容可较大调整 | 一般只做校对性订正，不宜大改内容 |
| 主体 | 作者 vs 外审/编委会 | 作者 vs 编辑/排版 |
| 若发现内容硬伤 | 正常修改 | 须及时与编辑沟通，避免见刊后勘误 |

因此校样阶段重点是"抓错"（错字、错符号、错引用），而非"再优化行文"。

## 八、缴费与发票要点

- 版面费按篇幅计（约 600 元/面，以通知为准）；提前确认页数与总额。
- 准备发票抬头、税号等信息；确认开票类型（电子/纸质）。
- 保留缴费凭证；若由课题经费报销，核对报销所需材料（**待核实** 编辑部具体开票口径）。

## 九、出版形式与存档

| 形式 | 说明 |
|---|---|
| 正式见刊 | 卷期、页码、DOI 以出版为准 |
| 网络首发/优先出版 | 是否提供 **待核实**；若有可缩短等待 |
| 数据库收录 | 见刊后陆续被 CNKI/万方/EI 等收录（时点 **待核实**） |
| 机构存档 | 遵守版权协议，按允许的版本存档 |

## 十、见刊后事项

- 保存正式版 PDF、卷期与 DOI 信息。
- 按需在机构库/个人主页规范存档（遵守版权约定）。
- 更新复现包链接为正式版指向（如适用）。
- 若发现出版后错误，按编辑部流程申请勘误（errata）。

## 八、与其它技能衔接

- 从 `jonc-review-process` 的"录用"结果进入本技能。
- 校样中若发现体例问题，回查 `jonc-writing-style`。
- 图表/数据链接更新参见 `jonc-reproducibility`、`jonc-supplementary`。

## 十点五、定稿沟通礼仪

- 校样通常有**返回期限**，务必按期，逾期可能顺延到下一期。
- 订正意见集中一次反馈给编辑，避免多次零散往来。
- 对排版差异（字体、图位）尊重编辑部体例，非必要不强求个性化。
- 缴费、版权、发票等事务性沟通留存邮件记录，便于追溯。
- 对不确定的出版政策（网络首发、DOI 时点）礼貌询问编辑，勿臆断。

## 十一、要点回顾

- 定稿是出版准备而非再评审：校样重在"抓错"，一般不做内容大改。
- 逐字核对题名、中英文摘要、关键词、中图分类号 TN、公式符号、图表与参考文献。
- 版面费按篇幅（约 600 元/面，以通知为准），及时缴纳并备好开票信息。
- 按编辑部要求签署版权/授权；确认基金致谢与作者信息无误。
- 月刊排期需排队，关注是否有网络首发；见刊后规范存档、如需勘误按流程办理。
