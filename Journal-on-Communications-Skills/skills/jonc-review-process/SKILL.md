# 《通信学报》审稿流程建模

本技能对《通信学报》(Journal on Communications, JOC) 的三审制建模，帮助作者预判各环节并用好杠杆。
JOC 由中国通信学会主办，走**初审→外审→终审**，终审由编委会负责，审稿周期约 1~3 个月
（**待核实** 精确口径）。以下依据官网审稿流程页于 2026-07-09 核验（见
`resources/official-source-map.md`）。

## 一、三审制流程

| 环节 | 主体 | 审什么 | 可能结果 |
|---|---|---|---|
| 初审 | 编辑部 | 规范、对口性、知网查重、学术不端 | 通过外审 / 退修 / 退稿 |
| 外审 | 约 2~3 名专家 | 创新性、正确性、实证充分性、写作 | 录用 / 小修 / 大修 / 退稿 |
| 终审 | 编委会 | 综合评审、最终把关 | 录用 / 修改 / 退稿 |

## 二、各环节详解

### 初审
- 编辑审查：是否符合本刊通信学科宗旨、格式是否达标、**知网查重是否 < 20%**、有无学术不端。
- 初审通过后收取**审稿费 300 元/篇**（以通知为准），进入外审。
- 作者杠杆：投前把对口性（承重墙在通信学科）和格式做扎实（见 `jonc-topic-selection`、
  `jonc-submission`），避免初审即退。

### 外审
- 约 2~3 名同行专家评审，聚焦创新性与实证。
- 常见质疑：delta 不清、基线不公平、缺统计置信区间、缺消融、写作不规范。
- 作者杠杆：投前用 `jonc-experiments` 补强证据，让每条贡献都有可验证证据。

### 终审
- **由编委会负责**，综合外审意见给出最终结果，按月度节奏推进。
- 作者杠杆：修回闭环、答复专业得体（见 `jonc-author-response`）。

## 三、结果类型与应对

| 结果 | 含义 | 应对 |
|---|---|---|
| 退稿（初审） | 不对口/不规范/查重超标 | 重新定位或改投兄弟刊；压查重 |
| 退稿（外审） | 创新性或实证不足 | 补实证后改投或重构 |
| 大修 | 有价值但问题较多 | 系统性修改，常需重新送审 |
| 小修 | 接近录用 | 逐条落实，快速返回 |
| 录用 | 通过终审 | 进入定稿（见 `jonc-camera-ready`） |

## 四、审稿周期与月刊排期

- 审稿周期约 **1~3 个月**（多处记为约 3 个月，**待核实**）。
- 终审与见刊受**月刊排期**影响（每月 25 日出版）；录用到见刊仍需排队。
- 作者杠杆：为审稿+修回+排期预留提前量，按系统状态节点跟进，勿频繁催稿。

## 五、各环节作者杠杆总表

| 环节 | 杠杆 | 对应技能 |
|---|---|---|
| 投前 | 对口性、格式、查重 | `jonc-topic-selection`、`jonc-submission` |
| 外审前 | 实证补强、写作规范 | `jonc-experiments`、`jonc-writing-style` |
| 修回 | 逐条闭环、差异标注 | `jonc-author-response` |
| 终审后 | 定稿、校样、缴费 | `jonc-camera-ready` |

## 六、状态跟进建议

- 通过 CBPT 投稿系统查看稿件状态（初审/外审/退修/终审/录用）。
- 收到退修意见后尽快启动答复，避免超期。
- 对时间敏感（如毕业/结题）者，提前规划，勿以会议 deadline 心态压缩期刊周期。

## 七、常见误区

| 误区 | 纠正 |
|---|---|
| 以为一轮定稿 | 期刊常多轮修回，大修多需重新送审 |
| 忽视对口性 | 初审即可能因不属通信学科退稿 |
| 修回不逐条回应 | 易被判未认真修改 |
| 频繁催稿 | 尊重三审制节奏，按节点跟进 |

## 七点五、稿件状态语义速查

CBPT 投稿系统内的状态标签，对应三审制的不同阶段，读懂它可减少无谓催稿：

| 系统状态（示意） | 含义 | 作者应做 |
|---|---|---|
| 待编辑处理/初审中 | 编辑在做规范与对口性审查 | 等待，勿催 |
| 已送外审/外审中 | 已分派专家评审 | 耐心等待，准备修回预案 |
| 退修/修改中 | 收到意见，需作者修回 | 尽快按 `jonc-author-response` 修改 |
| 复审/再审中 | 修回稿重新送审 | 保持前后一致 |
| 终审中 | 编委会终审 | 按月度节奏等待 |
| 录用/待发表 | 通过终审 | 进入 `jonc-camera-ready` |
| 退稿 | 不予录用 | 据意见改投或重构 |

> 具体状态名称以系统实际显示为准（**待核实**）。

## 八、退稿后的路径

被退稿不等于终点，视原因选择路径：

| 退稿原因 | 建议路径 |
|---|---|
| 不对口通信学科 | 改投兄弟刊（见 `jonc-topic-selection`），或重构承重墙 |
| 查重超标 | 重写重叠内容、压到远低于 20% 后再投 |
| 创新性不足 | 补理论/实证增量，明确 delta 后再投 |
| 实证不足 | 补多场景、统计、消融后再投 |
| 写作严重不规范 | 按 `jonc-writing-style` 系统返修 |

## 九、自查清单

- [ ] 理解初审/外审/终审各审什么、谁负责。
- [ ] 读懂 CBPT 系统状态语义，按节点跟进而非盲目催稿。
- [ ] 投前已做对口性、格式、查重（< 20%）准备。
- [ ] 外审前实证已补强、写作已规范。
- [ ] 修回时逐条闭环、差异可追溯。
- [ ] 已按月刊排期预留时间。
- [ ] 若退稿，已按原因选定改投或重构路径。

## 十、要点回顾

- 三审制：初审（编辑：规范/对口/查重）→ 外审（约 2~3 名专家）→ 终审（编委会）。
- 初审即可能因不属通信学科或查重超 20% 退稿；对口性与格式是投前必做功课。
- 外审聚焦创新性与实证；投前用 `jonc-experiments` 补强证据是最有效杠杆。
- 读懂 CBPT 系统状态语义，按节点跟进而非频繁催稿。
- 审稿约 1~3 个月（**待核实**）、月刊排期需排队；退稿按原因选改投或重构路径。

> 目标：把稿件按《通信学报》/Journal on Communications (JOC) 三审制的真实节奏稳步推进，在每个环节
> 用对杠杆，减少因对口性、实证或修回疏漏导致的退稿。
