# 《通信学报》中文科技写作规范

本技能规范《通信学报》(Journal on Communications, JOC) 中文长文的体例。JOC 是中国通信学会主办的
通信学科中文旗舰**月刊**，对中英文摘要、关键词、中图分类号、参考文献有明确格式要求，初审即查
规范。以下依据官网稿件格式要求页于 2026-07-09 核验（见 `resources/official-source-map.md`），
细则以官方模板为准，未确认处标 **待核实**。

## 一、官方行文次序

> 中文（标题、作者姓名、作者单位、单位所在省市与邮编、摘要、关键词）→ 中图分类号、文献标识码、
> 文章编号 → 英文（标题、作者姓名、作者单位、城市名与邮编、摘要、关键词）→ 正文 → 参考文献。

严格按此顺序排版，缺任何一段都可能在初审被退修。

## 二、题名

- **中文标题不超过 20 个汉字**，必要时可加副标题。
- 避免非正规缩略语、字符、代号、结构式与公式入题。
- 英文题名与中文一致、可独立检索；实词首字母大写（或按模板）。

## 三、结构式摘要（中英文）

- **中文摘要 200 字以内**，采用报道性/结构式，要素：①目的②方法③结果④结论⑤其他。
- 英文摘要与中文对应，信息完整、可独立阅读。
- 摘要中不出现图表号、参考文献号、未定义缩略语。

摘要模板（示意）：
```
【目的】针对 ____ 问题……
【方法】提出 ____，通过 ____……
【结果】在 ____ 场景下，相对 ____ 基线，____ 指标提升 ____……
【结论】____，可用于 ____。
```

## 四、关键词

- **至少 4 个**，列于摘要之后，中英文一一对应。
- 覆盖方法、对象与场景，便于检索；避免过泛词（如"通信""网络"单独成词）。

## 五、中图分类号 / 文献标识码 / 文章编号

- **中图分类号**：通信/电子类以 **TN** 打头，据主题选取，可给 1~3 个：
  - TN911（信号处理理论）、TN915（通信网络）、TN918（保密通信/密码）、TN929（移动/无线通信）等。
- **文献标识码**：研究论文一般为 A（按本刊规定）。
- **文章编号**：由编辑部按模板生成/填写。

## 六、量、单位、符号与公式

- 使用**法定计量单位**；数字与单位之间留空（如 10 dB、5 GHz）。
- 变量与物理量符号用**斜体**，矢量/矩阵用**黑斜体**，单位与函数名用正体。
- 公式居中编号（右对齐括号编号），正文引用写"式(3)"。
- 首次出现的缩略语给出中英文全称（如 正交频分复用 (OFDM)）。

## 七、图表规范

- 图题在下、表题在上；中文为主，必要时中英对照。
- 曲线图坐标轴标注量与单位；多曲线用不同线型+图例，保证黑白可辨。
- 主结果曲线标注置信区间或误差棒；表格数据对齐、给单位。
- 图表须在正文被引用（"如图 3 所示"）且编号连续。

## 八、参考文献 GB/T 7714

- 采用国家标准 **GB/T 7714** 顺序编码制著录（本刊模板据此排版）。
- 常见类型示例（示意，具体标点以模板为准）：
```
[1] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页.
[2] 作者. 书名[M]. 版本. 出版地: 出版者, 年.
[3] 作者. 题名[C]//会议论文集名. 出版地: 出版者, 年: 起止页.
[4] 作者. 题名[D]. 保存地: 保存单位, 年.
```
- 中文文献著录完整；必要时给出中英文对照。文献应尽量新、权威、可溯。
- 作者 3 名以上可用"等/et al."（按模板）。

## 九、作者简介与基金项目

- **基金项目**置通信作者下方，括号内注编号；多项以分号隔开。
- 作者简介含姓名、出生年、职称/学位、研究方向；通信作者标注邮箱。

## 十、语言与文风

- 术语统一、前后一致；避免口语与冗词。
- 主动/被动得当，逻辑连接清晰；一段一意。
- 缩略语全篇统一；数字、单位、符号用法一致。

## 十点五、中文科技写作常见语病

| 语病 | 例 | 修法 |
|---|---|---|
| 长句嵌套 | "由于……使得……从而……导致……" 一句到底 | 拆成短句，一句一意 |
| 主语缺失 | "通过仿真，验证了有效性" | 补主语"本文通过仿真验证了……" |
| 术语不统一 | "信道估计"与"信道估计器"混用指同一物 | 全篇统一 |
| 中英夹杂 | 正文频繁夹英文缩写未给全称 | 首现给中英文全称，后用缩写 |
| 量与单位不规范 | "10dB""5GHz"无空格 | 数字与单位间留空 |
| 图注过简 | "图 3 结果" | 图注自足，说明场景与结论 |

## 十点六、English Abstract 常见问题

- 时态：方法/结果一般用一般现在时或过去时，保持一致。
- 冠词与单复数：中文母语作者易错，建议请同行或工具校对。
- 术语与中文摘要对应：关键术语的英文表述前后一致、与国际文献一致。
- 长度与信息量：与中文摘要对应，可独立阅读，不出现未定义缩略语。

## 十一、格式自查清单

- [ ] 行文次序符合官方规定。
- [ ] 中文题名≤20 字；中英文结构式摘要齐备；中文摘要≤200 字。
- [ ] 关键词≥4、中英对应；中图分类号 TN、文献标识码、文章编号齐全。
- [ ] 量与单位、符号、公式排版规范；缩略语首现给全称。
- [ ] 图表编号连续、被正文引用、坐标含单位、主结果含置信区间。
- [ ] 参考文献 GB/T 7714，著录完整、较新、可溯。
- [ ] 基金项目与编号、作者简介齐备。

## 十二、要点回顾

- 严格遵循官方行文次序：中文→分类信息→英文→正文→参考文献。
- 中文题名≤20 字；中英文结构式摘要（中文≤200 字）；关键词≥4、中英对应。
- 中图分类号 TN、文献标识码、文章编号齐备；量与单位、符号、公式规范。
- 参考文献 GB/T 7714、著录完整较新；基金项目与作者简介齐全。
- 语言精炼、术语统一，中英文摘要各自可独立阅读。
