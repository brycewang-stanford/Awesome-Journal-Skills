# 《通信学报》审稿答复与修回说明

本技能帮助为《通信学报》(Journal on Communications, JOC) 的审稿意见撰写高质量答复与修回材料。JOC 由
中国通信学会主办，走三审制、常有多轮修回；答复质量直接影响能否通过编委会终审。以下与 2026-07-09
核验的本刊流程一致（见 `resources/official-source-map.md`）。

## 一、答复总原则

- **逐条闭环**：每条意见都对应"回应 + 修改位置"，不遗漏、不合并糊弄。
- **礼貌专业**：感谢审稿人，客观陈述，不情绪化。
- **有据可核**：修改处给页/行号或章节号，方便复核。
- **诚实**：能改则改；不能改的有据说明，不回避。

## 二、答复条目模板

```
【意见 R1-1】审稿人：本文缺少与最新方法 X 的对比。
【回应】感谢指正。我们已在第 4.2 节新增与 X 的对比实验（表 3、图 5），在相同信道与
        参数下，本文方法在 BER 与复杂度上均优于 X。
【修改位置】第 4.2 节第 2 段；表 3；图 5；参考文献新增[18]。
```

## 三、修回说明信结构

1. 开头致谢，概述本轮修改总体情况。
2. 列出**主要修改**（大的结构/实验/理论调整）。
3. 逐审稿人、逐条答复（用上节模板）。
4. 结尾说明差异标注方式与配套材料（复现包更新等）。

## 四、差异标注

- 在修改稿中对改动处**标色**或用修订模式；正文答复引用页/行号。
- 大改（新增小节、重做实验）在说明信显著位置提示。
- 保持一份"干净稿"和一份"标注稿"（按投稿系统要求，**待核实** 是否需两版）。

## 五、处理棘手意见

| 情形 | 策略 |
|---|---|
| 两位审稿人意见矛盾 | 客观陈述权衡，说明采纳依据，必要时请编委会裁量 |
| 超出本文范围的要求 | 礼貌说明为何超范围，可列为未来工作 |
| 不认同的意见 | 有据申辩（数据/理论/文献支撑），语气克制 |
| 要求补大量实验 | 优先补最关键者，其余说明可行性与时间 |
| 质疑对口性 | 强化通信学科承重墙论证（见 `jonc-topic-selection`） |

## 六、常见外审质疑与回应要点

| 质疑 | 回应要点 |
|---|---|
| 创新性不足 | 明确 delta，对照最强基线量化增量 |
| 基线不公平 | 补 SOTA、同等调优、同信道同场景 |
| 缺统计严谨 | 补蒙特卡洛置信区间/显著性 |
| 缺消融 | 补模块消融证明必要性 |
| 写作不规范 | 按 `jonc-writing-style` 修摘要/术语/参考文献 |
| 可复现性存疑 | 提供复现包、参数与种子（见 `jonc-reproducibility`） |

## 七、多轮修回意识

- 本刊大修常**重新送审**，第二轮审稿人可能仍是原专家；保持前后一致，勿反复。
- 每轮都更新答复表与标注稿，形成可追溯的修改链。
- 按月刊排期及时返回，避免因超期影响档期。

## 八、语言礼仪

- 用"感谢审稿人指出……我们已……"的句式。
- 不使用"审稿人误解""这个问题不重要"等冒犯表达。
- 申辩时以证据说话，措辞谦和坚定。

## 九、修回说明信开头范例

```
尊敬的编辑、各位审稿专家：
    衷心感谢各位对本文的细致评审。根据意见，我们对稿件做了系统修改，主要包括：
    (1) 在第 4.2 节新增与最新方法 X 的公平对比（表 3、图 5）；
    (2) 为所有主结果补充蒙特卡洛置信区间（图 3、图 5）；
    (3) 新增消融实验（第 4.4 节）以验证各模块必要性；
    (4) 按 GB/T 7714 统一了参考文献，并补充近三年关键文献[18-21]。
    以下逐条回应各位意见，修改处已在标注稿中标色，并给出页/行号。
```

## 十、答复长度与语气把控

| 情形 | 建议 |
|---|---|
| 简单意见（错字/格式） | 一两句确认已改 + 位置 |
| 实质意见（要求补实验） | 说明做了什么、结果如何、在哪 |
| 争议意见 | 先致谢，再摆证据，最后给折中或申辩 |
| 无法满足的要求 | 诚实说明原因，列为未来工作，不硬编造 |

语气统一用第一人称复数（"我们"），避免辩解式或对抗式表达。

## 十一、自查清单

- [ ] 每条意见都有"回应 + 修改位置"闭环。
- [ ] 修回说明信结构完整、开头致谢并概述主要修改。
- [ ] 修改处已标色/标注页行号；干净稿与标注稿备齐（按要求）。
- [ ] 矛盾/超范围/不认同意见有据处理。
- [ ] 关键外审质疑（创新性/基线/统计/消融/复现）已实质回应。
- [ ] 答复长度与语气按意见性质把控得当。
- [ ] 复现包/补充材料已同步更新。
- [ ] 已按月刊排期及时返回。

## 十一点五、答复表整体组织

除逐条答复外，建议在修回材料开头放一张"意见—处置"总览表，便于编委会快速把握全貌：

| 编号 | 意见摘要 | 处置 | 修改位置 |
|---|---|---|---|
| R1-1 | 缺与方法 X 对比 | 已补对比实验 | 4.2 节、表 3、图 5 |
| R1-2 | 缺置信区间 | 已补蒙特卡洛置信区间 | 图 3、图 5 |
| R2-1 | 术语不统一 | 已全篇统一 | 全文 |

总览表让审稿人先看"改了哪些、在哪"，再看逐条细节，显著降低复核成本。

## 十二、要点回顾

- 逐条闭环是底线：每条意见都要"回应 + 修改位置"，一条不落。
- 修回说明信开头致谢并概述主要修改，正文逐审稿人逐条回应。
- 修改处标色、给页/行号；按系统要求备干净稿与标注稿。
- 争议意见先致谢再摆证据，诚实处理无法满足的要求。
- 大修常重新送审，保持前后一致，按月刊排期及时返回。

> 目标：让《通信学报》/Journal on Communications (JOC) 的外审与编委会一眼看清每条意见如何被落实，
> 把稿件顺利推过三审制的多轮修回。配合 `jonc-review-process` 预判环节，配合 `jonc-camera-ready` 收尾。
