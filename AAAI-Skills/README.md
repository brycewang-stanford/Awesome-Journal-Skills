# AAAI Skills

Twelve agent skills for AAAI Conference on Artificial Intelligence main technical track papers. The
pack focuses on OpenReview submission readiness, two-phase review, the AAAI-26 AI-assisted review
pilot, short author rebuttals, reproducibility checklists, supplementary files, and camera-ready
publication.

Official basis checked on 2026-06-01: AAAI-26 conference page, Main Technical Track CFP,
submission instructions, review process, supplementary-material rules, author policies, rebuttal
FAQ, AI-assisted review FAQ, and publication/attendance rules. See
`resources/official-source-map.md` for the exact source map.

## Skills

- `aaai-submission` - audit OpenReview, anonymity, page limit, reproducibility checklist,
  supplementary material, author limits, multiple submission, and AI-use policy.
- `aaai-author-response` - draft the short AAAI rebuttal under no-URL, no-new-results constraints.
- `aaai-camera-ready` - prepare accepted-paper source files, proceedings metadata, extra pages,
  copyright, registration, and presentation plan.
- `aaai-artifact-evaluation` - package code/data and appendices for review-time reproducibility.
- `aaai-reproducibility` - strengthen the checklist, evidence map, seeds, compute, and data details.
- `aaai-supplementary` - organize technical appendix, multimedia appendix, and code/data ZIPs.
- `aaai-review-process` - reason about Phase 1, Phase 2, AI-assisted review, SPC/AC discussion, and
  final decisions.
- `aaai-writing-style` - revise for broad AI contribution, AAAI clarity, and policy-aware claims.
- `aaai-related-work` - position against archival work, arXiv/workshop papers, and contemporaneous
  results.
- `aaai-experiments` - audit baselines, ablations, statistics, robustness, AI-system evaluation, and
  compute.
- `aaai-workflow` - manage the calendar from topic selection through proceedings publication.
- `aaai-topic-selection` - decide whether a project is AAAI-shaped or better routed to another AI
  venue.
