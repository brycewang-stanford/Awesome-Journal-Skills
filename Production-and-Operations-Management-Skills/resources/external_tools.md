Access date: 2026-06-01

Use these tools after reopening current POM official pages:

- POMS author instructions: https://www.poms.org/journal/author_instructions
- POM departments and review board: https://www.poms.org/journal/review_board/
- SAGE journal home: https://journals.sagepub.com/home/PAO

- POMS ethics PDF: https://www.poms.org/POMS_Ethics_for_Research_and_Publishing.pdf
- ScholarOne submission system, linked from the current POMS author instructions.

Before submission, verify:

- Department-editor route and department fit.
- 32-page main-manuscript compliance.
- Double-blind main file and e-companion.
- Same-data disclosure and no-resubmission-after-rejection status.
- E-companion file designation and ordering in ScholarOne.
