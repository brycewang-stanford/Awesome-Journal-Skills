Twelve agent skills for **Journal of Economic Perspectives (JEP)** articles and proposals. The pack is tuned for accessible, non-technical economics writing: choosing a topic with broad AEA-member interest, pitching a proposal or symposium contribution, building a narrative arc, translating technical evidence, designing reader-friendly exhibits, preserving balance, working with editors, submission, and revision.

**Official basis checked 2026-06**: see [`resources/official-source-map.md`](resources/official-source-map.md).

## Quick Start

```
/plugin marketplace add ./Journal-of-Economic-Perspectives-Skills
/plugin install jep-skills
```

Manual use: start with [`skills/jep-workflow/SKILL.md`](skills/jep-workflow/SKILL.md).

| # | Skill | Role |

| 6 | [`jep-evidence-without-equations`](skills/jep-evidence-without-equations/SKILL.md) | Present evidence without specialist machinery |

| 9 | [`jep-balance-and-objectivity`](skills/jep-balance-and-objectivity/SKILL.md) | Keep the article fair and even-handed |
| 10 | [`jep-editor-strategy`](skills/jep-editor-strategy/SKILL.md) | Work with editor-led shaping |
| 11 | [`jep-submission`](skills/jep-submission/SKILL.md) | Run submission/proposal preflight |
