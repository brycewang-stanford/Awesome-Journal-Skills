# Advanced Materials — Pre-Submission Checklist

> Verify all volatile specifics (page/word limits per article type, figure DPI/format,
> TOC-graphic spec, submission portal, APC, required declarations) on the official Wiley
> Advanced Materials author page before submitting.

## 1. Scope and venue

- [ ] The work clears the advance + broad-impact gate (see `advmat-scope-fit`)
- [ ] You have honestly considered Adv. Funct. Mater. / Adv. Sci. / Small / AMI as alternatives
- [ ] The single central advance is stated in one sentence (new material / mechanism / property)

## 2. Article type and length

- [ ] Correct type chosen: Communication (~4 typeset pages) or Research Article (~10 typeset pages)
- [ ] Length measured as typeset pages in the Wiley template, not manuscript pages
- [ ] Display items (figures, schemes, tables) counted together with text
- [ ] No double-column figure unless the data require it
- [ ] Communication: opening paragraph serves as the abstract
- [ ] Research Article: short abstract (~200 words) + 3–7 keywords

## 3. Format and nomenclature

- [ ] Prepared in the Wiley Advanced Materials template / accepted format
- [ ] SI units; consistent significant figures; number–unit spacing
- [ ] Every abbreviation, chemical formula, and sample name defined and consistent across text/figures/SI
- [ ] One English variant (British or American) used consistently
- [ ] References in current Wiley Advanced Materials style

## 4. Science and characterization

- [ ] Every structural/compositional claim supported by ≥2 complementary techniques
- [ ] Morphology/size claims carry statistics, not a single hero image
- [ ] Headline metric benchmarked against cited state-of-the-art systems, fairly
- [ ] Device metrics report full measurement protocol (area, conditions, scan rate)
- [ ] Headline number carries sample count and spread (n, mean ± s.d.)
- [ ] Experimental Section is reproducible (bulky detail in the SI, cited)

## 5. Figures and TOC graphic

- [ ] A coherent, essential figure set; Figure 1 conveys the advance at a glance
- [ ] Each figure closes one structure → property → function link
- [ ] Every micrograph has an in-image scale bar and names its technique
- [ ] Consistent sample/condition color-coding across figures
- [ ] Legible at two-column print size; color-blind-safe and grayscale-distinguishable
- [ ] Figures at Wiley's required resolution/format (vector line art; high-DPI raster)
- [ ] TOC graphic + blurb prepared to spec

## 6. Supporting Information

- [ ] The main text stands alone without the SI
- [ ] No decisive evidence for the advance lives only in the SI
- [ ] Every SI item cited from the main text; ordered to match citation order
- [ ] SI figures/tables use the S-prefix (Figure S1, Table S1)
- [ ] SI provided as a separate file, not appended to the main PDF
- [ ] Full reproducible recipe/protocol in the SI if condensed in the body

## 7. Files to upload

- [ ] Main manuscript (with Experimental Section) + figures
- [ ] All figure files at production resolution/format
- [ ] Supporting Information file (separate, labeled)
- [ ] TOC graphic + blurb
- [ ] Cover letter (argues advance + broad impact)
- [ ] Bibliography / reference file as required

## 8. Metadata, authors, and integrity

- [ ] Title / abstract / author list consistent with the manuscript
- [ ] ORCID provided as required (corresponding author at minimum)
- [ ] Affiliations and corresponding-author contact correct
- [ ] Funding and CRediT contributions recorded if required
- [ ] All authors meet criteria and approved submission
- [ ] Originality / not-under-consideration-elsewhere confirmed
- [ ] Prior preprint (ChemRxiv / arXiv) disclosed
- [ ] Competing-interest statement present
- [ ] Ethics / biosafety statements if biomaterials or animal/human-derived samples involved
- [ ] Data-availability statement present

## 9. Referees

- [ ] Suggested referees can judge both novelty AND characterization rigor
- [ ] At least one suggested referee from the claimed neighboring/device field
- [ ] No conflicts of interest among suggestions
- [ ] Opposed referees (short list) with brief, factual, professional rationale

## 10. Self-consistency

- [ ] Figure / scheme / table / reference numbering continuous and matching in-text callouts
- [ ] Cover-letter referee section matches the submission form
- [ ] Cross-references intact after any last-minute trimming
