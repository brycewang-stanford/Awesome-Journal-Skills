# The Econometrics Journal Skills

面向 **The Econometrics Journal (EctJ)** 的 12 个 agent skills。这个包覆盖 leading-case
econometric contribution、applied value、RES/OUP 紧凑格式、渐近识别、Monte Carlo、必需的
empirical application、Editorial Express 投稿、快速 editorial screening 与录用阶段 replication
files。

官方依据核验日期：2026-06-01。已核验 OUP general instructions、RES journal page、RES
submissions、Editorial Express、RES review process、RES submission-fee policy 与 RES replication
policy。精确来源与 `待核实` 项见 `resources/official-source-map.md`。

## Skills

- `ectj-workflow`：在选题、理论、实证、压缩、投稿、审稿、复制包与 rebuttal 之间路由。
- `ectj-topic-selection`：判断项目是否满足 EctJ 的 leading-case 与 applied-value 门槛。
- `ectj-literature-positioning`：定位 econometrics、statistics、ML 与 applied literatures。
- `ectj-identification-strategy`：审查 identification、assumptions、asymptotics、proofs 与
  leading-case theory。
- `ectj-data-analysis`：设计 Monte Carlo 与 empirical-application evidence。
- `ectj-tables-figures`：在 printed-paper constraint 下压缩图表。
- `ectj-contribution-framing`：明确方法贡献与应用价值。
- `ectj-writing-style`：处理 150-word summary、20-page discipline、template compliance 与
  theorem readability。
- `ectj-replication-and-data-policy`：准备 README、software versions、data documentation、
  seeds、restricted-data explanations 与 OUP Supporting Information。
- `ectj-review-process`：规划 one-week screen、three-month target decision 与 resubmission
  timing。
- `ectj-submission`：投稿前核查 Editorial Express、fee、cover letter、empirical application、
  proof placement 与 replication disclosure。
- `ectj-rebuttal`：为 assumptions、proofs、simulations、application、compression 与 replication
  问题撰写回复与修改计划。

## 维护说明

- 给出投稿级建议前必须重新打开 RES/OUP 官方页面。
- 不要断言 peer-review blinding model；本包将其标为 `待核实`。
- Editor、fee、页数措辞、template files 与 replication-policy 细节均可能变化。
