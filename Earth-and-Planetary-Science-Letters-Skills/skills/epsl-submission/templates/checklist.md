# EPSL Submission Checklist

> Scope note: items marked **[official]** were checked against Elsevier/EPSL pages (see
> `resources/official-source-map.md`, checked 2026-07-16); items marked **[re-check]** are volatile or
> community-experience values — confirm on the live Guide for Authors
> (sciencedirect.com/journal/earth-and-planetary-science-letters/publish/guide-for-authors) before
> submitting.

## Format & letters discipline

- [ ] Article type is a Letter; "Frontiers Paper" selected only with an editor's invitation [official]
- [ ] Main text (introduction through conclusions) within the letters cap (~6,500 words) [re-check]
- [ ] No companion manuscript submitted or planned — EPSL discourages companion papers [official]
- [ ] Title concise, process-forward, informative; abbreviations avoided where possible [official]
- [ ] Abstract self-contained: question, approach, headline number ± uncertainty, consequence
- [ ] Highlights included if the current guide requires them [re-check]
- [ ] Keywords chosen per the current guide's count [re-check]

## Scientific content

- [ ] One decisive, process-level result — not a regional case study without portable insight
- [ ] Every quoted uncertainty defines its σ-level and analytical-vs-systematic content
- [ ] Reference materials, blanks, decay constants/tracer named with citations (isotope/geochron work)
- [ ] MSWD / goodness-of-fit reported and interpreted for pooled ages and isochrons
- [ ] Model results accompanied by resolution/recovery or sensitivity tests
- [ ] Rates and fluxes carry their chronologic anchor's uncertainty
- [ ] Excluded/rejected analyses visible in figures and present in deposited tables

## Figures & tables

- [ ] Each main-text exhibit carries a claim; reference material moved to the supplement
- [ ] Location/context panel with coordinates, scale, projection or planetary datum
- [ ] Error ellipses/bars/envelopes drawn at a stated σ-level
- [ ] Perceptually uniform colormaps (no rainbow/jet); colorblind-safe; grayscale-legible
- [ ] Vector formats for line art; resolution per the guide for rasters [re-check]
- [ ] Captions self-contained: σ-level, model, data source

## Data, code & supplement

- [ ] Data-availability statement written, with resolvable DOIs/accessions [official — Elsevier policy]
- [ ] Geochemical/isotope tables → EarthChem (or PANGAEA as appropriate)
- [ ] Seismic waveform/station products → IRIS/EarthScope, cited by network DOI
- [ ] Paleomagnetic data → MagIC; planetary mission products → NASA PDS / ESA PSA
- [ ] Reduction scripts / model configs archived (Zenodo or equivalent), versions pinned
- [ ] Supplement internally consistent (Fig. S-numbers, Table S-numbers all resolve)
- [ ] Abstract numbers regenerate from the deposited data + code

## Declarations & metadata

- [ ] Declaration of competing interests [official — Elsevier]
- [ ] Authorship contributions (CRediT) drafted and agreed by all authors [re-check exact format]
- [ ] Funding sources listed
- [ ] Generative-AI use disclosed per Elsevier policy, if applicable [official — Elsevier]
- [ ] Corresponding-author ORCID linked in Editorial Manager
- [ ] All coauthors have approved this exact version
- [ ] Preprint status noted if the manuscript is on a server

## Editorial Manager entries

- [ ] Cover letter final: advance + number + portability, one page (see `epsl-cover-letter`)
- [ ] Suggested reviewers entered — spanning the paper's sub-fields, no recent co-authors,
      no shared institutions
- [ ] Opposed reviewers (if any) named with a brief factual reason
- [ ] Editor preference (where offered) matched to the paper's subject area [re-check roster]
- [ ] References in the journal's author–date house style, DOIs included [re-check formatting details]

## Final sweep

- [ ] Word count re-run after the last edit
- [ ] All cross-references (figures, tables, S-items, equations) resolve
- [ ] Manuscript, supplement, and deposited tables agree on every number
- [ ] PDF proof built by Editorial Manager inspected page by page before approval
