# External Tools - INTERSPEECH

Access date: 2026-07-08

Live surfaces for an Interspeech author, plus the verification passes to run
against them. A gateway blocked direct fetches of the ISCA and conference domains
during this pack's verification (see the access-method note in
[`official-source-map.md`](official-source-map.md)), so open each URL yourself
before acting on any date or rule.

## Official surfaces

- Interspeech 2026 (Sydney): https://interspeech2026.org/
  - Call for Papers: https://interspeech2026.org/en-AU/pages/calls/call-for-papers
  - Submission policy: https://interspeech2026.org/en-AU/pages/author-resources/submission-policy
  - Registration: https://interspeech2026.org/en-AU/pages/registration/registration
- Interspeech 2027 (São Paulo): https://interspeech2027.org/
- ISCA (the society): https://isca-speech.org/ — upcoming editions, awards, ISCApad
- ISCA Archive (open-access proceedings, all editions): https://www.isca-archive.org/
- Paper kit: distributed per cycle via the conference site and Overleaf — always
  take the current cycle's kit, never last year's.
- Submission portal: CMT for recent cycles (verify the 2026/2027 portal link from
  the official "submit a paper" page, never from a search result).

## Author-side verification passes

Run these against the live pages at each milestone:

1. **Format pass** — current page counts (4+1 regular; Long Paper rules if used),
   template version, PDF-checker requirements, media-attachment limits.
2. **Anonymity pass** — this cycle's double-anonymous wording, the anonymity-period
   start/end, preprint policy, and what counts as an identifying link (demo pages
   and repos included).
3. **Calendar pass** — portal open, paper deadline, paper-update close, rebuttal
   window, notification, camera-ready, registration tiers; note the timezone
   convention used.
4. **Post-acceptance pass** — camera-ready instructions, the author-registration
   coupling, presentation-mode instructions, and Archive metadata requirements.
5. **Community-layer pass** — the challenges and special sessions of the current
   edition (each has its own organizer, dates, and paper rules), plus the relevant
   ISCA workshop calendar (Odyssey, SSW, and task-specific workshops).

## Conflict rule

If pages disagree — conference site vs ISCA site vs aggregator — the newest text on
the current edition's own domain wins, followed by direct word from the Technical
Program Chairs. Record any conflict you find in your project notes and in
[`official-source-map.md`](official-source-map.md) if you maintain this pack.

## Do not infer

- Do not project the Sydney calendar onto São Paulo 2027; ISCA editions set their
  own dates and rules.
- Do not assume the Long Paper track recurs, keeps its page budget, or keeps its
  acceptance target — it debuted in 2026.
- Do not assume rebuttal mechanics, attachment caps, or the portal from any prior
  cycle. Interspeech's process has been changing every year in exactly these spots.
