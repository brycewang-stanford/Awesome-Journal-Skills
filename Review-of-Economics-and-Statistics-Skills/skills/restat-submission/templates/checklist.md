# REStat Pre-Submission Self-Check (8 sections)

A last-pass checklist for a *The Review of Economics and Statistics* (REStat)
manuscript before submitting via **Editorial Express**. Live-check volatile specifics
(fee, abstract/appendix limits, editors, category rules) on the official MIT Press
REStat pages before upload.

## 1. Venue fit & category
- [ ] The paper is applied economics / applied econometrics with credible identification or careful measurement
- [ ] Category chosen: full-length **Article** / **Note** / **Short Paper**
- [ ] If **Short Paper**: ≤6000 words and ≤5 exhibits
- [ ] Not better suited to AER (general-interest) or *J. Econometrics* (methods-development)

## 2. Format (hard rules)
- [ ] Single **PDF**; **12-point** font; **double-spaced throughout** (abstract, acknowledgments, footnotes, references)
- [ ] Pages numbered; figures/tables placed legibly with self-contained notes
- [ ] **Online appendix ≤20 pages**; the main manuscript is **self-contained**

## 3. Front matter
- [ ] **Abstract ≤100 words**, in JEL-suitable form
- [ ] Title page: JEL codes + keywords + author affiliations
- [ ] Cover letter states the contribution and REStat fit

## 4. House statistical presentation
- [ ] Standard errors in parentheses below coefficients; clustering level stated
- [ ] Significance stars (if used) accompany, never replace, standard errors
- [ ] Modern inference reported (clustering / weak-IV-robust / few-cluster / multiple-testing as relevant)

## 5. Identification & measurement
- [ ] The causal design (DID / RD / IV / shift-share) is credibly defended with diagnostics
- [ ] Measurement quality of key variables addressed (validation / error correction / bounds)
- [ ] Robustness suite present; headline stable across specs/sample/measurement/inference

## 6. Data & code policy (deposit before publication)
- [ ] Target archive is the **REStat Harvard Dataverse** (not openICPSR, not JAE)
- [ ] One master script regenerates every table and figure; README permits replication
- [ ] Software/package versions pinned; seeds set and reported

## 7. Proprietary / restricted data (act at submission)
- [ ] Proprietary or restricted data **indicated at submission**
- [ ] Access instructions documented for researchers who cannot get the raw data
- [ ] **Submission fee NOT paid** until the editorial office confirms policy compliance

## 8. Declarations & fee
- [ ] Original, unpublished work; not under review elsewhere
- [ ] Funding / conflict-of-interest disclosures prepared; AI not listed as an author
- [ ] Submission fee (**US$125**, nonrefundable) ready by credit card (subject to §7 hold)
