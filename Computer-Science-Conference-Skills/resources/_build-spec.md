# Build Spec - Computer-Science-Conference-Skills

This pack is a breadth bundle for 155 top computer-science conferences plus one router skill. It is ordered with AI/ML venues first, then data mining/web AI, vision, NLP/speech/IR, robotics, HCI/visualization, systems/networking/architecture/HPC, security/privacy, SE/PL/formal methods, and DB/theory.

Each conference profile is a fit, venue-selection, and re-framing tool. It must not claim to replace the current CFP or author kit. Conference facts are volatile by year; exact deadlines, page limits, template details, rebuttal length, artifact requirements, and AI-use policies belong in the official-cycle checklist unless the current live official page has been checked.

## Required sections

1. Conference positioning
2. When to trigger
3. Scope & topic fit
4. Method & evidence bar
5. Structure & house style
6. Official-cycle checklist
7. Pre-submission self-check
8. Common desk-reject triggers
9. Re-routing decision
10. Output format

## Quality rules

- Frontmatter uses only `name` and `description`.
- Folder name equals frontmatter `name`.
- Every profile must include the official anchor from `conference-roster.md`.
- Never invent current-year deadline, location, acceptance rate, page limit, fee, or chair names.
- Re-check live CFPs before submission-ready advice.
