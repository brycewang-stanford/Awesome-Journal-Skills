# Source basis - CS/AI conference breadth bundle

The pack uses stable official source anchors plus a conservative re-check rule. For high-priority AI venues, official current-cycle pages were consulted or selected as first-pass anchors on 2026-06-01; for the full 155-conference roster, each skill points to the conference or society home page so future agents can verify the current cycle.

Do not turn this file into a list of hard-coded annual facts. Conference policy changes are frequent: OpenReview/ARR/CMT/PCS workflows, page limits, AI-use rules, artifact badges, ethics forms, anonymity exceptions, registration obligations, and camera-ready release forms may all change by year.
