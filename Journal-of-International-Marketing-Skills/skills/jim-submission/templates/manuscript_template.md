# JIM Manuscript Skeleton (regular empirical article)

Two-file structure for double-anonymized review. Format: 12-pt Times New Roman, double-spaced, 1-inch margins, no page numbers; ≤ 50 formatted pages all-inclusive. Re-check current specs at journals.sagepub.com/author-instructions/jig.

---

## FILE 1 — Title Page (identity lives only here)

- Title (names the mechanism and the cross-national scope, not just the topic)
- Authors, affiliations, ORCIDs, corresponding author
- Acknowledgments, funding, ethics/IRB statement, data-availability note

## FILE 2 — Main Document (fully anonymized)

### Title + Abstract (≤ 200 words, third person) + Keywords (3–8)

Abstract order: tension → countries/design (with numbers) → findings including the cross-national pattern → the managerial rule.

### 1. Introduction (≤ 2 pages)

- The cross-border tension a global marketer or the literature actually faces
- Why domestic answers fail across borders (assumed universality / conflicting country evidence)
- Question, countries, design, and the answer — disclosed here
- Contribution preview: theory delta + managerial rule, one sentence each

### 2. Theoretical Background and Hypotheses

- The stream joined (export marketing / global branding / cross-cultural CB / entry modes)
- Baseline mechanism (H1…), then cross-national moderation with dimension-tied because-clauses
- Conceptual model figure: levels labeled, country moderator at its own level

### 3. Method

- Country selection rationale on the focal dimension + confound audit
- Samples per country (frame, N, fieldwork window, language version)
- Translation/back-translation protocol; measures with sources
- Invariance testing plan and decision rules (pre-committed)

### 4. Results

- Invariance table first (configural/metric/scalar, Δ-criteria, verdicts)
- Per-country measurement quality; then hypothesis tests with formal country-difference tests
- The money exhibit: the cross-national pattern, graphed

### 5. Discussion

- Theoretical implications: what the border-crossing changed in the theory
- Managerial implications: actor + action + magnitude + market profile
- Limitations honestly scoped (which profiles the design cannot license); research agenda

### References (AMA marketing-journal author–date style)

### Tables and Figures (numbered in order of first mention)

## FILE 3 — Web Appendix (anonymized)

Item wordings in all languages; per-country correlation matrices; invariance sensitivity runs; alternative estimators; stimuli.
