# RFS Pre-Submission Checklist

Verify journal-specific limits (word count, fee, file formats, data policy) on the official
RFS page at submission time — the items below capture durable norms, not fixed numbers.

## 1. Format & anonymization (double-blind)

- [ ] Manuscript fully anonymized: no author names, affiliations, or identifying acknowledgments
- [ ] Self-citations neutral ("Smith (2021) shows" not "in our prior work")
- [ ] Grant numbers / thanks / presentation lists removed from the blinded file
- [ ] Title page with author info uploaded as a separate file per portal instructions
- [ ] File names contain no author identity

## 2. Author information

- [ ] All coauthors listed with correct affiliations and ORCID
- [ ] Corresponding author designated
- [ ] All coauthors have approved the submission
- [ ] Conflict-of-interest disclosures prepared

## 3. Abstract, keywords, JEL

- [ ] Abstract leads with the question and the headline finding (with magnitude)
- [ ] Keywords present
- [ ] JEL classification codes present
- [ ] Abstract within the journal's length limit (verify)

## 4. Manuscript structure

- [ ] Introduction states the contribution and the delta vs. the closest paper
- [ ] Identification / research design explained with the source of variation
- [ ] Main results reported with economic magnitudes
- [ ] Mechanism / interpretation present
- [ ] Conclusion does not merely repeat the abstract

## 5. Content quality

- [ ] Novelty AND rigor both clearly established
- [ ] Endogeneity threats addressed, not waved away
- [ ] Robustness battery complete (most convincing check in the main text)
- [ ] Multiple-testing / out-of-sample discipline for any predictability claim
- [ ] Standard errors clustered/adjusted to match the data

## 6. Tables & figures

- [ ] Each main table answers one question; overflow in the Internet Appendix
- [ ] Standard-error type stated in every table note
- [ ] Summary-statistics table present
- [ ] DID/dynamic effects shown as event-study figures with confidence bands
- [ ] Figures legible in greyscale; vector format; axes labeled with units

## 7. References & Internet Appendix

- [ ] Every in-text citation appears in the reference list and vice versa
- [ ] Recent (last ~3 years) RFS/JF/JFE work cited; canonical theory cited
- [ ] Internet Appendix prepared as a separate file, cross-referenced by IA number
- [ ] Data and code availability statement consistent with current RFS policy (verify)

## 8. Portal (ScholarOne / Manuscript Central)

- [ ] Account and ORCID verified
- [ ] Cover letter states the RFS-specific novelty and names close competitors
- [ ] Suggested / opposed referees prepared (no conflicts among suggested)
- [ ] Submission fee handling confirmed (amount/policy verified on site)
- [ ] All files staged: manuscript, title page, Internet Appendix, cover letter, code/data
- [ ] No dual submission; related submissions disclosed
