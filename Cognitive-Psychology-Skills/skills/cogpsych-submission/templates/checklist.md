# Cognitive Psychology Pre-Submission Self-Check (8 sections)

A last-pass checklist for a *Cognitive Psychology* (Elsevier) manuscript before submitting via
**Elsevier Editorial Manager**. Cognitive Psychology is a leading cognitive-science journal that favors
**longer, integrative, model-driven** articles — typically a multi-experiment program combined with a
formal/computational model that is fit and compared. Verify volatile specifics (length guidance,
abstract limit, review model, current editor, declarations) on the official guide for authors —

## 1. Venue & scope fit
- [ ] The contribution is a **theoretical advance** in cognition, carried by a program (multiple experiments and/or a fitted model), not a single isolated effect
- [ ] If a model is central, it is **fit and compared** to a rival account, not fit alone
- [ ] Scope is squarely cognitive science (attention, perception, memory, learning, language, categorization, reasoning, problem-solving, judgment/decision-making)
- [ ] Not better suited to a short-report venue (Psychological Science) or a narrower-scope sibling (Journal of Memory and Language)

## 2. Theory & model
- [ ] The cognitive theory is stated in words, then **formalized** (parameters with psychological meaning)
- [ ] Rival account(s) specified in matched formal language
- [ ] A **discriminating prediction** (a signature one model predicts and the other forbids) is identified
- [ ] Confirmatory (pre-committed) vs. exploratory model work is separated

## 3. Experiment design
- [ ] Each experiment **adds inference** (rules out a confound, extends scope, tests a further prediction)
- [ ] Stimuli/items controlled and counterbalanced; low-level confounds addressed; full pool documented
- [ ] Sample size **and trials per cell** justified for the critical contrast (simulation/recovery where the contrast is a model parameter)

## 4. Analysis & model fitting
- [ ] Models **compared** under matched flexibility (AIC/BIC/Bayes factor or cross-validation), with free-parameter counts
- [ ] **Parameter recovery** and **model recovery** reported
- [ ] (Generalized) linear mixed models or hierarchical Bayesian estimation where the design demands; convergence diagnostics (R-hat/ESS) for Bayesian fits
- [ ] Effect sizes with confidence/credible intervals for key behavioral results (no stars-only reporting)

## 5. Exhibits
- [ ] Headline figure overlays **observed data + model prediction(s)** (and the rival), not bars of means
- [ ] A **model-comparison table** (criteria + free-parameter counts) is included
- [ ] Exhibits are self-contained (Ns, trials, units, intervals defined) and accessible (colorblind-safe / grayscale-legible)

## 6. Open science, code & declarations (Elsevier policy)
- [ ] **Data** deposited with a **DOI** + codebook (or a justified, documented restriction)
- [ ] **Model-fitting code** + analysis scripts deposited; seeds set, versions pinned; fits **regenerate in a fresh session** (run log included)
- [ ] **Materials** (stimuli, item pools, task/counterbalancing) deposited with a DOI
- [ ] **Preregistration** linked where applicable; confirmatory vs. exploratory consistent

## 7. Front matter & format
- [ ] Abstract present (confirm current length/structure on the official page); keywords included
- [ ] APA-style statistical reporting and references; Elsevier formatting per the guide for authors
- [ ] A continuous, integrative narrative (experiments cohere; the model is integrated, not appended)

## 8. Declarations & final
- [ ] **Declaration of competing interest** completed; funding and author-contribution statements included
- [ ] Required ethics/consent statements per Elsevier policy
- [ ] Confirmed not under review elsewhere; AI not listed as an author
- [ ] Co-author sign-off; corresponding author designated; suggested reviewers as the system requests
