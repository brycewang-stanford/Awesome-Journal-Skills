# The Journal of Finance — Pre-Submission Checklist

> Durable-norm checklist. Verify current specifics (fee, membership, formatting limits, required statements) on the official JF / AFA author pages before submitting.

## 1. Fit & framing

- [ ] The question is general-interest — relevant beyond a single subfield of finance
- [ ] The headline result is stated in economically interpretable units (bps, % of market cap, Sharpe, alpha)
- [ ] The marginal contribution vs. the closest 3–5 papers is explicit
- [ ] JF (not JFE / RFS / a field journal) is the right home for this paper's scope and length

## 2. Identification / test design

- [ ] Corporate/empirical: a named source of identifying variation, defended (not assumed) exogeneity
- [ ] Staggered DID uses a modern estimator, not bare TWFE
- [ ] IV: first-stage strength reported; exclusion restriction argued; reduced form shown
- [ ] Asset pricing: portfolio sorts + regressions; correct SE correction (Shanken / Newey–West / two-way cluster)
- [ ] Benchmark factor model stated (CAPM / FF3 / FF5 / +MOM / q-factor)

## 3. Robustness & multiple testing

- [ ] Each main-text robustness check targets a named threat to the claim
- [ ] Multiple-testing addressed if the result came from a search (adjusted hurdle, disclosed search)
- [ ] Out-of-sample / subperiod evidence where the claim is predictive
- [ ] Economic magnitude is stable across checks; instability disclosed

## 4. Exhibits

- [ ] Main tables/figures are few and self-contained (readable from caption + note alone)
- [ ] Every table note states the SE type and clustering/correction
- [ ] No vertical rules; consistent decimals; one reported R²
- [ ] Figures vector / ≥300 dpi, legible in greyscale, with units and source
- [ ] Table/figure numbers match in-text references

## 5. Internet Appendix

- [ ] Proofs, secondary tables, and exhaustive robustness moved online
- [ ] Every Internet Appendix item is cited from the main text
- [ ] IA numbering (IA.I, IA.1, …) consistent with the main text
- [ ] No new headline result appears only in the appendix

## 6. Writing & abstract

- [ ] Question + finding + magnitude in the first ~3 pages
- [ ] Accessible to a finance reader outside the subfield; jargon glossed
- [ ] Abstract matches the introduction's claims
- [ ] JEL codes and keywords included

## 7. References & disclosures

- [ ] References in the journal's style; in-text ↔ list consistency checked
- [ ] Foundational and recent JF/JFE/RFS references on the topic are present
- [ ] Conflict-of-interest / disclosure statement prepared
- [ ] Data-availability / replication statement prepared; package ready for acceptance
- [ ] If blinding is required: body, self-citations, acknowledgments, and file metadata scrubbed

## 8. Portal

- [ ] Manuscript Central / ScholarOne account active
- [ ] Current submission fee / AFA-membership requirement verified and handled
- [ ] File formats and size limits meet current portal requirements
- [ ] Suggested / opposed reviewers prepared if requested
- [ ] Final version uploaded; metadata and on-screen info checked
