# MICRO Resources

Working materials that complement the `skills/` directory: the skills advise, these
files let an agent benchmark a draft against verified MICRO papers, model the house
introduction style, and prepare a simulation artifact that survives evaluation.

## Scope note

This is a computer-architecture conference pack. Its evidence culture is
cycle-level simulation, power/area modeling, RTL, and FPGA measurement — not
econometrics and not ML-benchmark leaderboards. Use the shared ML-conference kit
only through the adapter in `code/README.md`, which lists what it cannot check.

## A suggested route through the pack

1. **Fit first:** [`../skills/micro-topic-selection`](../skills/micro-topic-selection/SKILL.md)
   against the exemplar shapes in [`exemplars/library.md`](exemplars/library.md).
2. **Build evidence:** [`../skills/micro-experiments`](../skills/micro-experiments/SKILL.md)
   and [`../skills/micro-reproducibility`](../skills/micro-reproducibility/SKILL.md),
   with the manifest discipline installed before the first sweep.
3. **Write:** [`../skills/micro-writing-style`](../skills/micro-writing-style/SKILL.md)
   beside [`worked-examples/01-introduction.md`](worked-examples/01-introduction.md);
   triage overflow with [`../skills/micro-supplementary`](../skills/micro-supplementary/SKILL.md)
   (MICRO allows no appendix).
4. **Verify and upload:** [`external_tools.md`](external_tools.md) five passes, then
   [`../skills/micro-submission`](../skills/micro-submission/SKILL.md).
5. **Afterward:** [`../skills/micro-review-process`](../skills/micro-review-process/SKILL.md),
   [`../skills/micro-author-response`](../skills/micro-author-response/SKILL.md) in the
   June window, then [`../skills/micro-camera-ready`](../skills/micro-camera-ready/SKILL.md)
   and [`../skills/micro-artifact-evaluation`](../skills/micro-artifact-evaluation/SKILL.md)
   — or the fallback lattice in [`../skills/micro-workflow`](../skills/micro-workflow/SKILL.md).
