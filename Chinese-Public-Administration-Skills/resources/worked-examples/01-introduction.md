# 引言改写示例 — 《中国行政管理》房风格（worked example）

> **示意性、纯属虚构。** 下文的论文、机构、政策细节与全部数字均为演示而杜撰，**不指向任何
> 真实研究**。本示例仅用于展示《中国行政管理》（Chinese Public Administration）的引言房风格，
> 风格判据**只来自本技能包自身的 skills**（先读 `../../skills/cpa-identification/SKILL.md`、
> `../../skills/cpa-mechanism/SKILL.md`、`../../skills/cpa-heterogeneity/SKILL.md`、
> `../../skills/cpa-style/SKILL.md`、`../../skills/cpa-abstract/SKILL.md`）。格式细节以本包
> [`../official-source-map.md`](../official-source-map.md) 为准。

《中国行政管理》是中国行政管理学会主办的**公共管理 / 公共政策权威刊**：治理实践问题先行、
理论框架贯穿全篇、识别策略（定量准实验或定性过程证据）到位、机制与异质性挂在公共管理理论上，
最终落到**对治理实践有据的贡献**（`cpa-style`、`cpa-identification`）。一篇合格的引言必须让
读者读完就知道：**现实治理问题是什么、挂在什么公共管理理论上、凭什么相信因果 / 过程、本文相对
已有研究的边际贡献在哪、对治理实践意味着什么**。

---

## ❌ Before（变量驱动、罗列文献、识别含糊、对策四件套）

> 数字政府是当前研究热点。本文研究地方政府数字化转型是否提升公共服务满意度。已有研究中，
> A（2020）发现数字化有利于效率，B（2021）发现数字化提升透明度，C（2022）发现数字化改善
> 回应性。我们用 2015—2022 年 X 省地级市数据，以政府网站办事项数量衡量数字化程度，回归满意度
> 对数字化程度，控制人口、财政与城镇化率。结果显示数字化程度越高，满意度越高（β=0.14，p<0.01）。
> 本文对推进数字政府建设、提升治理现代化水平具有重要意义。

逐条诊断：

- **像横截面相关而非有识别的设计。** 用"办事项数量"做连续解释变量、OLS 回归满意度，没有清晰
  的政策冲击与处理 / 对照结构，也没说明问题本质是"效应估计"还是"过程机制"——本刊最反感把流行
  因果方法硬套、或称因果却无识别（`cpa-identification` 的"方法服务于问题"与分支 A 红线）。
- **文献写成罗列。** "A 发现…、B 发现…、C 发现…"是流水账，看不出已有研究的缺口，也看不出本文
  从哪里切入（`cpa-style`"一段一概括"：每段首句应是概括句，文献按线索归类并梳理不足）。
- **机制缺位。** 只给相关系数，没回答"通过什么治理过程实现"——本刊要求机制挂在公共管理理论上
  （执行 / 注意力 / 央地 / 激励问责），不能只说"数字化大概起了作用"（`cpa-mechanism`）。
- **识别策略空白。** 数字化程度是地方政府**自选**的，与满意度互为因果（治理能力强的地区更可能
  推数字化、满意度也更高）；没有事件研究图、没有平行趋势、没有安慰剂（`cpa-identification` 分支 A）。
- **异质性缺位。** 笼统适用，没分析"在什么行政层级 / 治理情境下更强或更弱"（`cpa-heterogeneity`）。
- **贡献与对策停在意义层。** "具有重要意义""提升治理现代化水平"是空喊，对策没落到"谁在哪个环节
  做什么"，且是"推进 / 提升"四件套（`cpa-style` 黑名单、`cpa-abstract` 反模式）。

## ✅ After（治理问题先行 → 理论框架 → 识别 → 机制与异质性 → 贡献，公共管理视角）

> 同样推行政务服务"一网通办"，有的地区群众办事时间显著缩短、满意度上去了，有的地区线上系统
> 上线却换来"线上线下两头跑"的新负担。差别往往不在地方政府"上没上数字化"，而在它**是否把数字
> 系统嵌入了真实的办事流程、是否压减了科层间的协调环节**。然而既有公共管理文献多把数字化当作
> 一个连续的"程度"变量与服务绩效做相关，对**何时数字政府真能提升回应性、何时只是叠加了一套
> 考核留痕**这一边界条件着墨不足（沿 `cpa-style` 的概括—归类—定位结构展开述评，此处略）。
>
> 本文以**政策执行与注意力分配理论**为框架提出：数字政府提升公共服务满意度，主要经由**减少
> 科层协调环节、释放一线服务注意力**这一过程——只有当数字系统替代而非叠加于原有审批链时，
> 才能把技术能力转化为真实的回应性改进；若数字化只是新增一套留痕考核，反而挤占一线注意力，
> 于是"数字化程度"与"满意度"脱钩。这把问题从"要不要数字化"重述为"**在什么治理条件下数字政府
> 才提质**"，并预测一个**调节效应**而非单纯主效应（机制留待 `cpa-mechanism` 的机制变量 /
> 过程证据落地）。
>
> 识别上，本文利用（虚构的）**某省政务服务"一网通办"改革分批扩围**这一准自然实验：试点批次
> 由省级按既定名单分年下达，与地市当期服务满意度冲击无关，从而构造交错处理的多期 DID。本文画出
> **事件研究图**确认处理前各期系数不显著（平行趋势），用**随机化处理时点的安慰剂检验**排除偶然，
> 并因处理时点交错而采用**异质性稳健估计**（Callaway-Sant'Anna 与 Goodman-Bacon 分解互证），
> 而非直接套 TWFE（遵循 `cpa-identification` 分支 A 红线：交错处理须回应负权重与异质性偏误）。
> 机制部分把"科层协调环节数 / 一线注意力指标"当作机制变量、沿用同一识别策略估计"试点→机制"
> 这一段，并以财政能力与属地加码强度做调节（`cpa-mechanism` + `cpa-heterogeneity`）。读到这里，
> 读者已经知道现实问题（同样改革、满意度分化）、理论框架（执行 / 注意力分配）、识别（分批试点
> 的多期 DID）与机制。
>
> 本文进一步检验**行政层级与治理情境**的异质性：理论上，层级越低、属地加码越强的地区，数字化
> 越易异化为留痕负担，故预期提质效应在县区一级、财政能力较弱处更弱（`cpa-heterogeneity` 切分
> 维度 1 与 2，报告系数差异检验）。
>
> 本文的边际贡献有二：对公共管理理论，**给出数字政府提质的边界条件**（流程再造而非系统叠加），
> 厘清"数字化—满意度"脱钩之谜；对治理实践，提示把数字政府的着力点**从"上线系统、增设考核"
> 转向"压减审批环节、为一线减负"**——即不在"上不上数字化"的环节，而在"用不用得起来、减不减
> 得了负"的环节发力（`cpa-style`：对策落到谁、哪个环节、做什么）。

为什么成立：

- **治理问题先行**（同样改革、满意度分化）而非变量先行，且落点在数字政府 / 公共服务回应性 /
  基层负担，正中本刊画像（`cpa-style`、`cpa-identification` 的问题—方法匹配）。
- **理论框架贯穿**（政策执行 / 注意力分配），生成"调节效应"这一可检验命题，并预告用机制变量 +
  调节而非空口推测落地（`cpa-mechanism`、`cpa-style` 的理论—分析一致性）。
- **识别策略直面内生性**：用分批试点的外生冲击构造多期 DID，事件研究图 + 安慰剂 + 异质性稳健
  估计齐全，预先拆掉"数字化是自选、与满意度互为因果"这一审稿人核心攻击（`cpa-identification`）。
- **异质性有理论指引**：按行政层级与财政 / 属地情境切分并回扣机制，而非笼统"东中西"（`cpa-heterogeneity`）。
- **贡献写成对理论（边界条件）与实践（着力点从上线转向减负）的双重 delta**，对策指明主体与环节，
  而非"具有重要意义"或"推进 / 提升"四件套（`cpa-style` + `cpa-abstract`）。
