# External Tools - OSDI

Access date: 2026-07-08

Use these surfaces after reopening the current OSDI cycle pages. OSDI policy is re-set
per edition — two rules changed at OSDI '26 alone (author response removed, artifact
badges narrowed) — so never let a cached fact outrank a live page. Note that
`usenix.org` sometimes blocks automated fetches (HTTP 403); if that happens, read
search-engine renderings of the exact URLs and say so in your output.

## Author-side verification passes

Run each pass against the current cycle's pages, not this file:

1. **Calendar pass** — both December deadlines (registration and full paper), their
   exact Pacific-time cutoffs, notification, artifact, and final-paper dates; confirm
   which track (Research vs Operational Systems) the paper is registered under.
2. **Format pass** — page budget for the reviewed body, the reference-page exemption,
   whether appendices remain forbidden at submission, font/text-block specification,
   and the final-paper budget if accepted.
3. **Anonymity pass** — author and institution anonymization, third-person
   self-citation, and the renamed-system rule: any arXiv/tech-report/talk precursor
   must use a different project name than the submission.
4. **Policy pass** — per-author submission cap, dual-submission rules, whether an
   author-response phase exists this cycle, and any new AI-use or ethics clause.
5. **Artifact pass** — which badges the current cycle evaluates, the artifact deadline
   relative to notification, and accepted hosting (Zenodo, institutional, third-party
   permanent repositories).

## Do not infer

- Do not project OSDI '26 dates onto OSDI '27; the '27 CFP was not yet readable on
  2026-07-08 and is marked 待核实 in the source map.
- Do not assume USENIX deadlines are Anywhere-on-Earth; OSDI '26 used 2:59 pm PST.
- Do not assume the badge set or the no-response policy persists — both were changes
  introduced at OSDI '26 and could be reversed.
- Do not carry over "co-located with USENIX ATC" logistics; ATC ended after ATC '25.
- If official pages disagree (HTML CFP vs PDF CFP vs HotCRP), the newest dated revision
  or direct chair communication controls; record the conflict in your output.
