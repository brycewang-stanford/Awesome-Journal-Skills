# Chinese Social-Science Breadth Resources

This resource layer supports the 100+ Chinese social-science fit cards. Use it
when the user needs routing across journals rather than deep lifecycle guidance
for a single venue.

## What to Load

- `official-source-map.md` - official-source starting points for each journal;
  re-check live 投稿须知 / 征稿启事 before final advice.
- `source-basis.md` - source policy and verification discipline for this
  breadth bundle.
- `worked-examples/venue-routing.md` - example routing diagnoses for common
  Chinese economics, management, accounting, and public-policy manuscripts.
- `exemplars/selection-patterns.md` - reusable venue-selection patterns and
  sibling-journal traps. These are routing archetypes, not fabricated papers.

## Usage Rule

Start with `cn-journal-workflow` for uncertain targets, then open only the one
or two per-journal skills that match the manuscript. For full lifecycle work on
flagship depth packs such as 《经济研究》 or 《管理世界》, route into the dedicated
single-journal pack after the breadth skill confirms fit.
