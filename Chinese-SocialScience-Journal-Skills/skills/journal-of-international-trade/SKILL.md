# 《国际贸易问题》投稿（journal-of-international-trade）

## 期刊定位

《国际贸易问题》由**对外经济贸易大学**主办，是国际贸易与开放经济领域的权威核心期刊，属国家社科基金首批资助学术期刊。覆盖贸易政策、关税与贸易摩擦、企业出口与进口中间品、服务贸易、数字贸易、FDI 与全球价值链。本刊以**实证识别和机制检验**见长，要求问题有国际/开放经济落点、数据可核、识别干净。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 关键事实（核验日期 2026-05；以官网最新《投稿须知》为准）

- **主办**：对外经济贸易大学；国家社科基金首批资助学术期刊。
- **费用**：编辑部声明以质论稿，**不收取版面费、审稿费**等任何费用；并声明从未与任何第三方代投/代发机构合作。
- **投稿方式**：仅通过官网在线投稿系统投稿（学术刊物部 journal.uibe.edu.cn），此为唯一渠道。
- **篇幅与摘要**：标题一般 20 字以内；正文一般约 8000–10000 字；要求中英文摘要（中文摘要约 300 字以内）。
- **参考文献**：执行 **GB/T 7714-2015**《信息与文献 参考文献著录规则》。
- **审稿周期**：以官网最新为准（作者反馈通常半年左右，待核实）。

## 触发时机

- 用户明确说目标期刊是《国际贸易问题》或 Journal of International Trade
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文经管稿改写成《国际贸易问题》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 选题偏好

- 贸易政策、关税、贸易便利化和贸易摩擦
- 企业出口、进口中间品、服务贸易和数字贸易
- FDI、跨境电商和全球价值链

## 方法与证据

- 贸易、FDI、汇率、跨境资本、GVC、开放政策和国际比较是核心对象
- 常用海关微观数据、跨国面板、投入产出/GVC 分解、引力模型、shift-share 和事件研究
- 需要回应外部冲击外生性、暴露份额推断和国际可比口径

## 结构与写作

- 引言第一屏必须出现国际/开放经济维度
- 机制要说清贸易、投资、金融、汇率、预期或供应链传导渠道
- 文献对话不能只有中文政策文献，要接入国际贸易/国际金融经典与前沿

## 路由防误投

- 本刊快速定位属于“开放经济与国际问题”通道；只有当稿件的核心贡献满足“开放经济、国际贸易、GVC、汇率、FDI或国别比较必须是主贡献”时，才把 《国际贸易问题》 作为首选。
- 相近 profile 先比较：`journal-of-guangdong-university-of-finance-and-economics`（《广东财经大学学报》） / `journal-of-industrial-engineering-and-engineering-management`（《管理工程学报》） / `journal-of-jiangxi-university-of-finance-and-economics`（《江西财经大学学报》） / `journal-of-macro-quality-research`（《宏观质量研究》）。若这些刊物的读者对象更贴近，不要因为名称、地区或变量相似而强行投本刊。
- 常见误投：若只是国内政策评估，优先回到综合经济或产业经济刊。 正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从其中的官方来源锚点或同一主办/出版体系入口开始核验，并在回答中说明核验日期。
- 通过官网在线投稿系统投稿（唯一渠道）；核对正文字数、20 字标题、中英文摘要与关键词格式
- 确认国际数据来源、国家/行业口径与汇率处理
- 检索“《国际贸易问题》 投稿须知 / 征稿简则 / 作者指南”，以最新页面为准；不要沿用二手代投网站格式
- 核对 GB/T 7714-2015 参考文献体例、匿名正文、作者信息页、基金信息单列、数据/程序/补充材料、版权协议
- 再次确认审稿周期与不收费声明（上述数值可能随官网更新而变化）
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《国际贸易问题》的核心读者范围
- [ ] 引言前 3 段已经出现本刊关心的问题层级、理论/机制和主要证据
- [ ] 文献综述至少对话 2-3 篇本刊或同层级中文期刊近年同主题文章
- [ ] 标题 20 字内、正文约 8000–10000 字、中英文摘要齐全；参考文献按 GB/T 7714-2015 复核
- [ ] 已通过官网在线投稿系统提交（唯一渠道），未走第三方代投
- [ ] 政策/管理启示从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 国内产业题硬贴“开放”标签
- GVC 只做概念叙事，没有分解或数据支撑
- shift-share/引力模型不讨论识别假设

## 改投判断

更宏观的开放经济理论转 journal-of-world-economy；更金融开放转 studies-of-international-finance。

```text
【目标期刊】《国际贸易问题》

【方法证据】<当前方法是否够本刊标准>
【最大风险】<最可能导致退稿的一点>
【需核验官方要求】<投稿系统/匿名/摘要/图表/数据等>
【改投建议】<若不匹配，给出更合适期刊>
```
