# 《科学决策》投稿（scientific-decision-making）

## 期刊定位

《科学决策》由**中国航天系统科学与工程研究院、中国社会经济系统分析研究会、欧亚系统科学研究会**主办（CN 11-3472/G3），**月刊**。它面向决策科学与系统分析，覆盖**政策评估、决策模型、风险评估、治理工具、区域发展与产业政策**，带有系统工程与软科学交叉的色彩。相对偏战略咨询的《中国软科学》，本刊更看重**决策方法与系统建模**；相对偏纯学术的《科学学研究》，本刊更面向决策支持的现实问题。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《科学决策》或 Scientific Decision Making
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文经管稿改写成《科学决策》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 关键事实（核验日期 2026-05；以官网最新《投稿须知》为准）

- **主办**：中国航天系统科学与工程研究院、中国社会经济系统分析研究会、欧亚系统科学研究会（CN 11-3472/G3）；**月刊**。
- **费用**：编辑部**收取版面费**（作者反馈常见区间约 3200–5000 元，按页数计；以编辑部最新通知为准，勿照搬二手网站数字）。
- **注释体例**：采用**页下注**（脚注），用 ①②③ 标注、每页重新编号。
- **数字用法**：凡能用阿拉伯数字处一律用阿拉伯数字。
- **基金**：若为基金项目须注明来源、名称及编号。
- **审稿周期**：作者反馈约 **1–3 个月**（系统状态更新偏慢，常需主动联系编辑部）——以编辑部最新通知为准。
- 上述数值与流程以官网最新《投稿须知》为准。

## 选题偏好

- 政策决策、风险评估和治理工具
- 决策模型、系统分析和评价指标
- 科技创新、区域发展和产业政策

## 方法与证据

- 决策模型、系统建模、政策评估、仿真模拟、网络分析、案例与多准则评价均可。
- 机制要落到决策过程、风险与资源配置，模型须交代假设、参数来源与稳健性。
- 纯定性政策建议须有扎实证据支撑，避免泛泛对策。

## 结构与写作

- 引言把决策/政策问题与方法缺口连接起来，明确决策支持目标。
- 模型/指标构造需说明数据来源、权重与口径，并给出敏感性分析。
- 决策含义面向政策制定、风险治理与资源配置，从本文证据推出。

## 路由防误投

- 本刊快速定位属于“管理科学与工商管理”通道；只有当稿件的核心贡献满足“组织、战略、创新管理、管理科学模型、系统工程或企业管理机制必须是核心贡献”时，才把 《科学决策》 作为首选。
- 相近 profile 先比较：`science-of-science-and-management-of-st`（《科学学与科学技术管理》） / `science-research-management`（《科研管理》） / `scientific-management-research`（《科学管理研究》） / `securities-market-herald`（《证券市场导报》）。若这些刊物的读者对象更贴近，不要因为名称、地区或变量相似而强行投本刊。
- 常见误投：若只是宏观政策效果，比较经济/产业/公共管理刊。 正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从官方来源锚点开始核验，并在回答中说明核验日期。
- 检索“《科学决策》 投稿须知 / 征稿简则”，以官网采编系统最新页面为准；不要沿用二手代投网站格式。
- 核对：页下注体例（①②③每页重编）、阿拉伯数字用法、基金编号、摘要/关键词/参考文献格式、版面费、版权与收录授权。
- 再次确认审稿周期与投稿入口（上面数值可能随官网更新而变化）。
- 若本 skill 与最新官方要求冲突，以官方要求为准。

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《科学决策》的核心读者范围
- [ ] 引言前 3 段已出现本刊关心的决策/政策问题、方法和主要证据
- [ ] 文献综述至少对话 2-3 篇本刊或同层级中文期刊近年同主题文章
- [ ] 页下注体例、阿拉伯数字用法、基金编号与参考文献格式已按官方稿约复核
- [ ] 决策/政策启示从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 决策模型缺假设、参数来源与敏感性分析
- 政策背景和项目类型混淆
- 把决策问题写成泛泛政策评论

## 改投判断

方法/模型理论更强 → chinese-journal-of-management-science（《中国管理科学》）；政策理论/战略咨询更强 → china-soft-science（《中国软科学》）。
