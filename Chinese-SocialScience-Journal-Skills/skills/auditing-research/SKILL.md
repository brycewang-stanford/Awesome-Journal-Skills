# 《审计研究》投稿（auditing-research）

## 期刊定位

《审计研究》由**审计署主管、中国审计学会主办**，国内外公开发行，是审计领域的**权威学术核心刊（全国管理类核心期刊）**，**双月刊**。覆盖国家审计与公共资金绩效、注册会计师审计质量与独立性、内部审计与内控、审计治理与制度。本刊兼收审计理论、制度政策与规范实证，对**国家审计与政府治理**议题尤为重视，要求制度背景准确、审计构念清晰。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《审计研究》或 Auditing Research
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文经管稿改写成《审计研究》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 关键事实（核验日期 2026-05；以官网最新《投稿须知》为准）

- **主管/主办**：审计署主管、中国审计学会主办；审计学术**双月刊**，全国管理类核心期刊。
- **审稿/费用**：实行**匿名审稿**；**不收版面费**。审稿周期约 **1–3 个月**（经验值，以编辑部实际为准）。
- **摘要**：**300 字以内**，应含与正文等量信息，按**目的、方法、结果、结论**顺序书写。
- **关键词**：**中英文各 3–5 个**。
- **行文结构**：按中文题目、摘要与关键词、正文、参考文献，再英文题目、摘要与关键词的顺序排布。
- **参考文献**：凡引文一律列入“参考文献”，按正文引用先后用右上标 [1][2][3]… 标示（顺序编码）。
- **投稿方式**：**投稿系统已启用，不再受理纸质邮寄稿件**；以官网采编系统为准。

## 选题偏好

- 国家审计、公共资金绩效和审计整改
- 注册会计师审计质量、独立性和费用
- 内部审计、内控和风险治理

## 方法与证据

- 审计、内控、披露、会计信息与资本市场档案研究是核心；制度政策与规范实证并重。
- 度量要有审计/会计学依据：审计质量、独立性、审计费用、应计、稳健性、披露指数等，须交代构造。
- 审计准则、监管与国家审计制度背景必须准确；实证需处理识别与稳健性。

## 结构与写作

- 引言明确审计机制或审计治理问题，不把文章写成一般公司金融
- 机制落到审计独立性、审计质量、内控、信息不对称或审计监督
- 实务启示面向审计监管、国家审计、注册会计师行业与企业内控

## 路由防误投

- 本刊快速定位属于“会计审计”通道；只有当稿件的核心贡献满足“会计确认、审计监督、披露、内控或资本市场会计问题必须是核心贡献”时，才把 《审计研究》 作为首选。
- 相近 profile 先比较：`asia-pacific-economic-review`（《亚太经济》） / `auditing-and-economics-research`（《审计与经济研究》） / `bulletin-of-chinese-academy-of-sciences`（《中国科学院院刊》） / `business-management-journal`（《经济管理》）。若这些刊物的读者对象更贴近，不要因为名称、地区或变量相似而强行投本刊。
- 常见误投：若只是企业财务数据或控制变量，转向金融、产业或综合经管刊。 正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从官方来源锚点开始核验，并在回答中说明核验日期。
- 检索“《审计研究》 投稿须知 / 征稿简则 / 作者指南”，以官网采编系统最新页面为准；不要沿用二手代投网站格式。
- 核对：摘要 300 字以内（目的/方法/结果/结论）、中英文关键词 3–5 个、顺序编码参考文献、匿名要求、中英文要素齐全、版权协议、不收版面费声明。
- 再次确认在线投稿入口与最新格式（上面数值可能随官网更新而变化）。
- 若本 skill 与最新官方要求冲突，以官方要求为准。

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《审计研究》的核心读者范围
- [ ] 引言前 3 段已经出现本刊关心的问题层级、理论/机制和主要证据
- [ ] 摘要 300 字以内（目的/方法/结果/结论）、中英文关键词各 3–5 个
- [ ] 参考文献按顺序编码上标；审计度量已交代构造，制度背景准确
- [ ] 政策/管理启示从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 审计度量不交代构造
- 审计/会计准则条款或实施时点写错
- 只跑资本市场回归，缺审计机制贡献

## 改投判断

会计信息披露与会计准则转 accounting-research（《会计研究》）；审计的经济后果、门槛略友好转 auditing-and-economics-research（《审计与经济研究》）；高规范学术会计转 china-accounting-review（《中国会计评论》）。
