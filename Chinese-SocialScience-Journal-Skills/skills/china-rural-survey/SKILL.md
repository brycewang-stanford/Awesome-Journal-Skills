# 《中国农村观察》投稿（china-rural-survey）

## 期刊定位

《中国农村观察》由**中国社会科学院农村发展研究所**主办，是三农领域的**双月刊**（CN 11-3586/F，ISSN 1006-4583）。主发表关于农业、农村和农民问题的学术论文，研究范围横跨农村**经济、政治、社会、法律、文化、教育**等多领域，强调**原创性、思想性、前瞻性与方向性**，以创新性为首要考量。它与同所主办的姊妹刊《中国农村经济》(china-rural-economy) 形成分工：本刊更偏制度调查、农村社会与多学科观察，《中国农村经济》更偏经济学实证与政策机制。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《中国农村观察》或 China Rural Survey
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文经管稿改写成《中国农村观察》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 关键事实（核验日期 2026-05；以官网最新《投稿须知》为准）

- **主办**：中国社会科学院农村发展研究所；**双月刊**，CN 11-3586/F，ISSN 1006-4583。
- **审稿模式**：双向匿名审稿；**零版面费**，不收取审稿费/发表费用。
- **审稿周期**：来稿量大、人员有限，编辑部公布**审稿周期约 6 个月**；以系统状态为准。
- **篇幅**：论文一般控制在 **15000 字以内**（含图、表、参考文献）。
- **摘要/关键词**：内容提要约 **100–300 字**，除中文提要外**还需附英文提要**；中文**关键词 3–5 个**。
- **数据与代码**：与《中国农村经济》一道要求**公开论文数据与程序代码**等资料（CASS 农发所统一说明），实证稿须按要求提交。
- **投稿系统**：采编系统为**唯一**投稿方式，在线投稿网址 zgncgc.ajcass.com（官网 zgncgc.ajcass.com）。

## 选题偏好

- 村庄治理、土地制度和集体经济
- 农户行为、农村社会结构和乡村振兴
- 农村公共服务、共同富裕和基层实践

## 方法与证据

- 微观农户/村庄/县域数据、政策试点 DID、土地制度和农业生产率测度是常见路径
- 识别要处理地区选择、政策叠加、农户异质性和样本代表性
- 定性制度材料、田野调查与多学科证据在本刊更受欢迎，但仍需严谨论证
- 实证稿需按编辑部要求**公开数据与程序代码**

## 结构与写作

- 三农问题要先讲制度场景：土地、户籍、集体经济、农业经营主体或乡村治理
- 机制需落到农户行为、农业生产、农村金融、要素流动或基层治理
- 政策含义要可操作，但避免工作报告腔

## 路由防误投

- 本刊快速定位属于“三农与农业经济”通道；只有当稿件的核心贡献满足“农业、农村、农民、土地、乡村治理或农业技术经济必须是核心场景”时，才把 《中国农村观察》 作为首选。
- 相近 profile 先比较：`china-public-administration-review`（《公共管理评论》） / `china-rural-economy`（《中国农村经济》） / `china-soft-science`（《中国软科学》） / `chinese-journal-of-management`（《管理学报》）。若这些刊物的读者对象更贴近，不要因为名称、地区或变量相似而强行投本刊。
- 常见误投：若只是一般区域发展或产业政策，比较区域/产业经济刊。 正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从其中的官方来源锚点或同一主办/出版体系入口开始核验，并在回答中说明核验日期。
- 核对栏目是否偏农村社会观察、制度调查与多学科议题（纯经济计量更贴《中国农村经济》）

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《中国农村观察》的核心读者范围
- [ ] 引言前 3 段已经出现本刊关心的问题层级、理论/机制和主要证据

- [ ] 正文已双向匿名；中英文摘要齐备（中文 100–300 字）、关键词 3–5 个；篇幅控制在 15000 字内
- [ ] 数据与程序代码已按编辑部要求准备公开；方法、图表和参考文献格式已按最新官方稿约复核
- [ ] 政策/管理启示从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 把农村样本当普通企业/居民样本，缺三农制度解释
- 政策时点、试点名单或土地制度细节写错
- 只切东中西，忽略农户、村庄、县域和产业链异质性
- 缺乏原创性与思想性，或实证稿无法提供可复核的数据与程序代码

## 改投判断

更纯经济计量、政策机制实证转 china-rural-economy（《中国农村经济》）；农业技术效率转 journal-of-agrotechnical-economics（《农业技术经济》）；综合农业政策转 issues-in-agricultural-economy（《农业经济问题》）。
