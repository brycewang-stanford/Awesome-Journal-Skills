# 《管理评论》投稿（management-review）

## 期刊定位

《管理评论》由**中国科学院大学**主办，是综合性**管理学**学术**月刊**。覆盖战略管理、创新管理、组织行为、人力资源、营销、公司治理、运营与数字化转型等，方法上兼收问卷、档案数据、实验、多案例与混合方法。与偏数理优化的《管理科学学报》相比，本刊更看重**理论构念 + 机制推演 + 测量规范 + 中国情境解释**；档案实证也要回到管理理论，而非停留在经济政策评估。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《管理评论》或 Management Review
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文管理稿改写成《管理评论》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 关键事实（核验日期 2026-05；以官网最新《投稿须知》为准）

- **主办**：中国科学院大学；综合性管理学术月刊。
- **审稿**：**全程匿名审稿**——投稿正文须做匿名处理；编辑部已启用**网上审稿系统**，不再受理纸质投稿。
- **摘要**：中文摘要约 **200–300 字**（含研究目的、方法与结果）；英文摘要约 **300–400 词**。
- **关键词**：**3–8 个**。
- **参考文献**：采用**顺序编码标注制**（按引用先后用方括号上标数字编号，文末按序著录）。
- **图表/公式**：通篇分别编号；图题、表题须**中英文对照**。题目应简洁准确，不宜使用缩略词。

## 选题偏好

- 战略管理、创新管理与公司治理
- 组织行为、人力资源与营销
- 数字化转型、平台治理与创业

## 方法与证据

- 战略、组织、HRM、营销、创新、公司治理均可；方法包括问卷、档案数据、实验、多案例与混合方法。
- 核心是**理论构念、机制推演、测量规范与中国情境解释**——问卷/实验须交代伦理、样本、量表信效度与共同方法偏差检验。
- 档案实证也要回扣管理理论，不只做经济或政策评估。

## 结构与写作

- 引言先立**管理理论缺口**，再说明中国情境为何能推进理论。
- 假设推演逐条给机制，不写直觉相关。
- 讨论部分回扣理论贡献、边界条件与实践启示。

## 路由防误投

- 本刊快速定位属于“管理科学与工商管理”通道；只有当稿件的核心贡献满足“组织、战略、创新管理、管理科学模型、系统工程或企业管理机制必须是核心贡献”时，才把 《管理评论》 作为首选。
- 相近 profile 先比较：`journal-of-zhongnan-university-of-economics-and-law`（《中南财经政法大学学报》） / `macroeconomics`（《宏观经济研究》） / `management-science-cn`（《管理科学》） / `management-world`（《管理世界》）。若这些刊物的读者对象更贴近，不要因为名称、地区或变量相似而强行投本刊。
- 常见误投：若只是宏观政策效果，比较经济/产业/公共管理刊。 正式建议前仍需进入官方核验清单。

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《管理评论》的核心读者范围
- [ ] 引言前 3 段已出现管理理论缺口、机制和主要证据
- [ ] 正文已匿名；中文摘要 200–300 字、英文摘要 300–400 词、关键词 3–8 个
- [ ] 参考文献按顺序编码标注制；图表题中英文对照、公式编号齐全
- [ ] 量表信效度 / 共同方法偏差已检验；中国情境已进入理论而非仅作样本来源

## 高频拒稿雷区

- 显著性替代理论贡献
- 量表信效度或共同方法偏差检验缺失
- 中国情境只是样本来源，没有进入理论

## 改投判断

量化模型 / 定理 / 算法是核心贡献 → journal-of-management-sciences-china（《管理科学学报》）；问卷/案例的管理理论建构更厚 → nankai-business-review（《南开管理评论》）；偏政策评估的经管实证 → management-world（《管理世界》）。
