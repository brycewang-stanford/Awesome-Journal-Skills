# 《税务研究》投稿（tax-research）

## 期刊定位

《税务研究》由**中国税务杂志社主办**，是税务领域的学术**月刊**。覆盖税制改革与税收优惠、税收征管与数字税务、企业避税与纳税遵从、国际税收、地方税体系与税收治理。本刊兼收**税收理论、税制政策与实务、规范实证**，相对纯经济学刊更看重税收制度背景的准确性与政策落点，适合面向税务部门与财税决策读者的研究。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《税务研究》或 Taxation Research
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文财税稿改写成《税务研究》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 关键事实（核验日期 2026-05；以官网最新《投稿须知》为准）

- **主办**：中国税务杂志社；税务学术**月刊**。
- **投稿方式**：通过中国税务杂志社**投约稿系统**（tg.ctax.org.cn）在线投稿。
- **内容提要（摘要）**：一般 **300–500 字**（中英文），应体现研究对象、方法、发现与建议等要点。
- **关键词**：一般 **3–8 个**（中英文）。
- **注释/参考文献**：文内注释一般采用**页脚注（每页重新编号）**形式，**参考文献附文后**；数据、引文、图表须注明资料来源。
- **审稿周期**：一般 **3 个月**内未接到用稿通知可自行处理；**版面费**等费用条款**待核实**，以编辑部说明为准。

## 选题偏好

- 税制改革、税收优惠和税负分配
- 税收征管、数字税务和企业避税
- 国际税收、地方税体系和税收治理

## 方法与证据

- 税制设计、税收征管、纳税遵从、企业避税、国际税收与地方税体系是核心。
- 政策改革可用 DID、断点、事件研究或结构分解；税收数据要核对口径、税种与征管层级。
- 机制要落到激励约束、纳税遵从、税负转嫁或企业行为，而非停留在收入数字描述。

## 结构与写作

- 制度背景写准税法、税收政策文件、税种与征管层级
- 把税收机制与经济结果（投资、创新、就业、税负分配）连接起来，不停留在征收描述
- 政策建议面向税务部门、税制设计与征管治理

## 路由防误投

- 本刊快速定位属于“财政税收”通道；只有当稿件的核心贡献满足“财政、税收、预算、地方债、公共支出或财贸政策必须是主问题”时，才把 《税务研究》 作为首选。
- 相近 profile 先比较：`studies-of-international-finance`（《国际金融研究》） / `systems-engineering-theory-and-practice`（《系统工程理论与实践》） / `world-economic-papers`（《世界经济文汇》） / `world-economic-studies`（《世界经济研究》）。若这些刊物的读者对象更贴近，不要因为名称、地区或变量相似而强行投本刊。
- 常见误投：若只是政策背景，转向产业、公共管理或综合经济刊。 正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从官方来源锚点开始核验，并在回答中说明核验日期。
- 检索“《税务研究》 投稿须知 / 撰稿须知 / 征稿简则”，以官网（shuiwuyanjiu.cn）与投约稿系统最新页面为准；不要沿用二手代投网站格式。
- 核对：在线投约稿系统入口、内容提要 300–500 字与关键词 3–8 个（中英文）、页脚注 + 文末参考文献、税收政策文件引用与图表资料来源、是否一稿多投声明。
- 再次确认审稿周期、是否收费与最新格式（上面标“待核实”项可能随官网更新而明确）。
- 若本 skill 与最新官方要求冲突，以官方要求为准。

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《税务研究》的核心读者范围
- [ ] 引言前 3 段已经出现本刊关心的问题层级、税收机制和主要证据
- [ ] 内容提要 300–500 字、关键词 3–8 个（中英文）；注释用页脚注、参考文献附文后
- [ ] 税法/税收政策时点与适用范围核对无误，数据口径与税种清晰
- [ ] 政策建议从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 税种、税率或税收政策时点、适用范围写错
- 中央/地方税收口径混用
- 只做相关性，没有税收制度机制

## 改投判断

财政理论、预算与财政体制更强转 public-finance-research（《财政研究》）；税收的会计/盈余后果更强转 accounting-research（《会计研究》）。

## 输出格式

```text
【目标期刊】《税务研究》

【方法证据】<当前方法是否够本刊标准>
【最大风险】<最可能导致退稿的一点>
【需核验官方要求】<投稿系统/匿名/摘要/图表/数据等>
【改投建议】<若不匹配，给出更合适期刊>
```
