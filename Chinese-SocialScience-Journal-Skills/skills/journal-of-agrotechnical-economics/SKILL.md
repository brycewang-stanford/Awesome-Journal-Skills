# 《农业技术经济》投稿（journal-of-agrotechnical-economics）

## 期刊定位

《农业技术经济》由**中国农业技术经济研究会**与**中国农业科学院农业经济与发展研究所**主办、**农业农村部**主管，**1982 年**创刊（CN 11-1883/S，ISSN 1000-6370），是农业经济类核心**月刊**，专注**农业技术经济**——技术进步、效率评价、生产率测度、技术采纳与推广、机械化与数字农业、农业绿色转型、农业投资与项目评价。它与同一主办体系下偏综合农经政策的《农业经济问题》(issues-in-agricultural-economy) 共用编辑部与投稿系统，但定位更"技术经济/效率方法"。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《农业技术经济》或 Journal of Agrotechnical Economics
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文经管稿改写成《农业技术经济》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 关键事实（核验日期 2026-05；以官网最新《投稿须知》为准）

- **主办/主管**：中国农业技术经济研究会、中国农业科学院农业经济与发展研究所主办，农业农村部主管；1982 创刊，**月刊**，CN 11-1883/S，ISSN 1000-6370。
- **收录**：1998 年入选 CSSCI 经济学科来源期刊，1999 年被 CSCD 收录；为农经类核心期刊。
- **审稿模式**：实行**同行专家双向匿名审稿**；须经多道审稿环节，编辑部公布**审稿周期约 3 个月**。
- **费用**：编辑部公布**收取审稿费**（具体金额及版面费政策**以官网最新声明为准**，坊间转述不一，待核实）。
- **投稿系统**：与《农业经济问题》共用平台，须通过本社官网 **www.iaecn.cn** 在线投稿，否则不进入审稿。
- **稿件构成**：题目（一般 20 字以内）、作者及单位、摘要、关键词、正文、参考文献等齐备；摘要用第三人称、客观自含，不写"本文""作者认为"。

## 选题偏好

- 农业技术采纳、机械化和数字农业
- 农业生产率、效率测度和绿色转型
- 农业投资、项目评价和技术推广

## 方法与证据

- 微观农户/村庄/县域数据、政策试点 DID、土地制度和农业生产率测度是常见路径
- 效率/生产率分析（SFA、DEA、Malmquist、全要素生产率分解）与技术采纳模型是本刊常见方法
- 识别要处理地区选择、政策叠加、农户异质性和样本代表性
- 定性制度材料可作为机制证据，但不能替代严谨经验设计

## 结构与写作

- 三农问题要先讲制度场景：土地、户籍、集体经济、农业经营主体或乡村治理
- 机制需落到农户行为、农业生产、技术采纳、要素配置或效率改进
- 政策含义要可操作，但避免工作报告腔
- 摘要用第三人称、客观自含，避免"本文""作者认为"等主观措辞

## 路由防误投

- 本刊快速定位属于“三农与农业经济”通道；只有当稿件的核心贡献满足“农业、农村、农民、土地、乡村治理或农业技术经济必须是核心场景”时，才把 《农业技术经济》 作为首选。
- 相近 profile 先比较：`international-economics-and-trade-research`（《国际经贸探索》） / `issues-in-agricultural-economy`（《农业经济问题》） / `journal-of-business-economics`（《商业经济与管理》） / `journal-of-central-university-of-finance-and-economics`（《中央财经大学学报》）。若这些刊物的读者对象更贴近，不要因为名称、地区或变量相似而强行投本刊。
- 常见误投：若只是一般区域发展或产业政策，比较区域/产业经济刊。 正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从其中的官方来源锚点或同一主办/出版体系入口开始核验，并在回答中说明核验日期。
- 核对栏目是否偏农业技术、效率评价与生产率（综合农业政策更贴《农业经济问题》）
- 核对数据口径、抽样说明、调查伦理和变量定义，效率测度的投入产出口径要交代清楚
- 检索“《农业技术经济》 投稿须知 / 征稿简则 / 作者指南”，以官网（www.iaecn.cn）最新页面为准；不要沿用二手代投网站格式
- 核对：摘要第三人称写法、关键词、参考文献体例、匿名/盲审要求、审稿费/版面费最新口径、基金信息单列
- 若本 skill 与最新官方要求冲突，以官方要求为准

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《农业技术经济》的核心读者范围
- [ ] 引言前 3 段已经出现本刊关心的问题层级、理论/机制和主要证据
- [ ] 文献综述至少对话 2-3 篇本刊或同层级中文期刊近年同主题文章
- [ ] 效率/生产率测度的投入产出口径、模型设定与稳健性已交代清楚
- [ ] 摘要为第三人称客观写法；方法、图表和参考文献格式已按 iaecn.cn 最新官方稿约复核；审稿费口径已官方确认
- [ ] 政策/管理启示从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 把农村样本当普通企业/居民样本，缺三农制度解释
- 政策时点、试点名单或土地制度细节写错
- 只切东中西，忽略农户、村庄、县域和产业链异质性
- 效率/生产率口径不清，或只套模型不回扣农业技术经济问题

## 改投判断

三农制度经济、政策机制实证转 china-rural-economy（《中国农村经济》）；农业政策综合转 issues-in-agricultural-economy（《农业经济问题》）；偏数量经济技术方法转 journal-of-quantitative-technological-economics（《数量经济技术经济研究》）。
