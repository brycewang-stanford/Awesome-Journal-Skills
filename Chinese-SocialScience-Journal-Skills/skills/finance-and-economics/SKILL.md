# 《财经科学》投稿（finance-and-economics）

## 期刊定位

《财经科学》由**西南财经大学**主办，1957 年创刊，是西部地区最早的经济理论刊物之一（**月刊**），国内外公开发行。选题涵盖金融学、理论经济、产业经济、企业经济、区域经济、财税会计、农业经济、管理学、公共管理及交叉/新兴学科，财政、金融、产业与区域经济并重，适合财经政策与规范实证；对西部/区域特色选题更包容。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 关键事实（核验日期 2026-05；以官网最新《投稿须知》为准）

- **主办**：西南财经大学（教育部直属）；月刊，1957 年创刊，西部最早的经济理论刊物之一。
- **学科边界**：金融、理论经济、产业经济、企业经济、区域经济、财税会计、农业经济、管理学、公共管理及交叉学科。
- **关键词/参考文献**：中英文关键词 **3–8 个**；参考文献应在 **5 条及以上**，作者 1–3 位全部列出、多于 3 位列前 3 位后加“，等”，采用规范的顺序著录。
- **费用**：作者反馈**无投稿费、审稿费、版面费，亦无稿费**（待以编辑部最新通知核实）。
- **审稿周期**：作者反馈约 **两个半月**左右，且**寒暑假不审稿**（以官网/编辑部最新为准）。
- **投稿/官网**：编辑部官网 cjkxzz.cn 及知网采编平台（cjkx.cbpt.cnki.net）。

## 触发时机

- 用户明确说目标期刊是《财经科学》或 Finance and Economics
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文经管稿改写成《财经科学》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 选题偏好

- 财政税收与地方政府行为
- 金融发展、绿色金融和资本市场
- 区域经济、产业升级和共同富裕

## 方法与证据

- 金融发展、财税与地方政府行为、产业升级、区域经济与共同富裕均可，财政、金融、产业、区域并重
- 政策改革可用 DID、断点、事件研究或结构分解；财政/金融数据要核对口径和层级
- 机制要落到激励约束、金融摩擦、税收遵从、公共品供给或区域发展逻辑，而非仅做相关性

## 结构与写作

- 制度背景写准法律法规、财政/金融体制和实施层级
- 把财政或金融机制与经济结果连接起来，不停留在收入支出或市场表现描述
- 政策建议面向财政金融部门、地方政府或区域发展治理，可保留西部/区域特色落点

## 路由防误投

- 本刊快速定位属于“金融财经”通道；只有当稿件的核心贡献满足“金融机构、资本市场、监管、公司金融或国际金融机制必须是主线”时，才把 《财经科学》 作为首选。
- 相近 profile 先比较：`economic-theory-and-business-management`（《经济理论与经济管理》） / `economist-cn`（《经济学家》） / `finance-and-trade-economics`（《财贸经济》） / `financial-regulation-research`（《金融监管研究》）。若这些刊物的读者对象更贴近，不要因为名称、地区或变量相似而强行投本刊。
- 常见误投：若金融只是机制之一，优先比较产业、财政、区域或综合经济刊。 正式建议前仍需进入官方核验清单。

## 高频拒稿雷区

- 财政/金融口径与层级混用，数据无法复核
- 政策时点和适用范围写错
- 只做相关性，没有制度机制；选题超出本刊学科边界

## 改投判断

西部或区域特色可保留；顶级金融机制转 journal-of-financial-research（《金融研究》）；纯财政公共经济转 public-finance-research（《财政研究》）。
