# 《经济问题》投稿（economic-problems）

## 期刊定位

《经济问题》由**山西省社会科学院**主管主办，是综合性**经济理论**学术**月刊**，**1979 年**创刊。覆盖经济学各分支的应用与政策研究——区域经济、财政金融、产业发展、企业经营、能源资源与高质量发展——重**学术研究成果**，明确不收决策咨询、报告或新闻报导类稿件。相对国内顶刊（《经济研究》《管理世界》），它对选题的区域/政策落点更包容，但仍要求问题清楚、证据扎实、体例规范。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《经济问题》或 Economic Problems
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文经管稿改写成《经济问题》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 关键事实（核验日期 2026-05；以官网最新《投稿须知》为准）

- **主办**：山西省社会科学院；综合性经济理论月刊，1979 创刊。
- **费用**：编辑部声明**不收取审稿费、版面费、加急费**。
- **审稿周期**：投稿后**约 2 个月**无回复可自行另投（编辑部不出退稿意见、不退稿）；录用后见刊一般约 **6–9 个月**。
- **查重**：要求重复率在 **10% 以下**。
- **投稿方式**：电子版投稿（官方邮箱以官网为准，曾公布 jjwts@163.com）；官网入口 jjwt.juqk.net。

## 选题偏好

- 区域经济、产业发展与产业政策
- 财政金融、企业经营与数字经济
- 能源资源、生态经济与高质量发展

## 方法与证据

- 区域经济、产业经济、财政金融、企业管理、开放经济均可；关键是问题清楚、证据扎实、体例规范。
- 经验研究仍需处理内生性与稳健性，但相比顶刊可更强调区域特色与政策解释力。
- 作为社科院主办的理论月刊，要求规范写作、完整中英文摘要与关键词、基金信息与标准参考文献格式。

## 结构与写作

- 引言把区域 / 产业 / 政策特色转化为明确的**学术问题**，而非工作总结。
- 文献综述避免罗列，必须说明本文相对既有研究的位置。
- 结论给出适度政策启示与研究边界，从本文证据推出。

## 路由防误投

- 本刊快速定位属于“综合经济”通道；只有当稿件的核心贡献满足“经济学问题意识、机制解释和可外推的理论/政策贡献必须成立”时，才把 《经济问题》 作为首选。
- 相近 profile 先比较：`economic-aspects`（《经济纵横》） / `economic-perspectives`（《经济学动态》） / `economic-research`（《经济研究》） / `economic-review-cn`（《经济评论》）。若这些刊物的读者对象更贴近，不要因为名称、地区或变量相似而强行投本刊。
- 常见误投：若只是行业背景或地方案例，比较更窄的产业、财政、金融或区域刊。 正式建议前仍需进入官方核验清单。

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《经济问题》的核心读者范围
- [ ] 引言前 3 段已出现本刊关心的问题层级、理论/机制和主要证据
- [ ] 文献综述至少对话 2-3 篇本刊或同层级中文期刊近年同主题文章
- [ ] 重复率已控制在 10% 以下；中英文摘要、关键词、基金信息、参考文献格式按官方稿约复核
- [ ] 政策/管理启示从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 像地方工作报告、决策咨询或泛泛综述，而非学术研究成果
- 数据口径和变量定义不清
- 未按学报体例处理作者信息、基金和参考文献

## 改投判断

理论贡献更强、识别更干净 → economic-research（《经济研究》）/ china-economic-quarterly（《经济学（季刊）》）；纯理论或经济思想史 → economist-cn（《经济学家》）。
