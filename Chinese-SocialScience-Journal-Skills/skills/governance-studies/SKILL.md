# 《治理研究》投稿（governance-studies）

## 期刊定位

《治理研究》由**中共浙江省委党校（浙江行政学院）**主办，**2018 年第 1 期**由《中共浙江省委党校学报》正式更名而来，是治理理论与实践取向的社科类综合性学术刊物。覆盖国家治理、基层治理、社会治理、城市治理、数字治理、协同治理与公共政策、治理绩效等。本刊兼顾理论深度与中国治理实践，适合把治理机制讲清、有理论对话和经验证据的研究，而非政策宣传或工作总结。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《治理研究》或 Governance Studies
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文治理/公共管理稿改写成《治理研究》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 关键事实（核验日期 2026-05；以官网最新《投稿须知》为准）

- **主办**：中共浙江省委党校（浙江行政学院）；社科综合性学术刊物。
- **刊名沿革**：2018 年第 1 期起由《中共浙江省委党校学报》更名为《治理研究》。
- **费用**：编辑部声明**不收取版面费、审稿费**。
- **字数**：正文一般约 **10000–20000 字**（以官网最新栏目要求为准）。
- **摘要**：中文摘要**200 字以内**，包含研究目的、方法、结果、结论；不用评论性词语，不含图表、参考文献序号或缩写。
- **关键词/英文项**：正文须含摘要、关键词，并附英文标题。
- **审稿周期**：约 **1–3 个月**；自投稿之日起 **3 个月**内未收到处理意见可改投他刊。
- **投稿/官网**：投稿系统 zlyj.zjdx.gov.cn；编辑部邮箱以官网为准（曾公布 zlyjbjb@126.com）。

## 选题偏好

- 国家治理、基层治理和社会治理
- 城市治理、数字治理和协同治理
- 公共政策、公共服务和治理绩效

## 方法与证据

- 公共治理、行政改革、政策执行、数字政府、基层治理和公共服务绩效是核心
- 可用政策评估、案例比较、调查、文本分析、混合方法和制度分析
- 必须把政府行为、组织激励和公共价值讲清，并体现治理理论对话

## 结构与写作

- 引言围绕公共问题和治理机制，而非企业绩效或政策宣传
- 机制要落到政府层级、部门协同、公众参与、政策执行或问责
- 政策建议要可执行，并说明适用层级

## 路由防误投

- 本刊快速定位属于“公共管理与社会政策”通道；只有当稿件的核心贡献满足“政府治理、公共服务、行政改革、社会保障或数字政府机制必须是主线”时，才把 《治理研究》 作为首选。
- 相近 profile 先比较：`forum-on-science-and-technology-in-china`（《中国科技论坛》） / `frontiers-of-engineering-management-science-and-technology`（《工程管理科技前沿》） / `industrial-economics-research`（《产业经济研究》） / `inquiry-into-economic-issues`（《经济问题探索》）。若这些刊物的读者对象更贴近，不要因为名称、地区或变量相似而强行投本刊。
- 常见误投：若只是经济政策效果，转向财政、产业或综合经济刊。 正式建议前仍需进入官方核验清单。

## 官方核验清单

- 正式给出投稿格式/流程建议前，先读取 `../../resources/source-basis.md` 和 `../../resources/official-source-map.md`，从其中的官方来源锚点或同一主办/出版体系入口开始核验，并在回答中说明核验日期。
- 检索"《治理研究》 投稿须知 / 征稿简则 / 作者指南"，以官网 zlyj.zjdx.gov.cn 最新页面为准；不要沿用二手代投网站格式。
- 核对：正文字数、中文摘要 200 字以内、关键词、英文标题、匿名/作者信息、基金信息单列、不收费声明、参考文献格式、数据/补充材料、版权协议。
- 再次确认审稿周期（约 1–3 个月、3 个月可改投）与投稿系统入口（上面数值可能随官网更新而变化）。
- 若本 skill 与最新官方要求冲突，以官方要求为准。

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《治理研究》的核心读者范围
- [ ] 引言前 3 段已经出现本刊关心的问题层级、理论/机制和主要证据
- [ ] 文献综述至少对话 2-3 篇本刊或同层级中文期刊近年同主题文章
- [ ] 中文摘要 200 字以内（含目的/方法/结果/结论）、关键词、英文标题齐备；字数符合栏目要求
- [ ] 政策/管理启示从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 只写政策宣传或工作总结，缺理论和证据
- 案例材料描述多、机制抽象少
- 把企业管理框架生硬搬到公共部门
- 摘要含评论性词语或超出 200 字

## 改投判断

行政管理制度与政策实务稿转 chinese-public-administration（《中国行政管理》）；方法更厚的公共管理实证转 journal-of-public-management（《公共管理学报》）；理论评论与案例比较转 china-public-administration-review（《公共管理评论》）。
