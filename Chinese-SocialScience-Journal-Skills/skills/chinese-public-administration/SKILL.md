# 《中国行政管理》投稿（chinese-public-administration）

## 期刊定位

《中国行政管理》由**国务院办公厅主管、中国行政管理学会主办**，是行政管理与政府治理领域的权威综合性学术月刊。覆盖行政改革、政府职能、政策执行、治理现代化、数字政府、基层治理、法治政府与公共服务等。本刊兼具学术性与决策影响力，设综合性与学术性等不同板块——既要研究问题清楚、机制与证据扎实，也强调对中国治理实践的解释力，而非地方工作总结或纯政策宣传。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《中国行政管理》或 Chinese Public Administration
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文公共管理稿改写成《中国行政管理》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 关键事实（核验日期 2026-05；以官网最新《投稿须知》为准）

- **主管/主办**：国务院办公厅主管，中国行政管理学会主办；行政管理学权威月刊。
- **费用**：编辑部声明**不以任何形式收取版面费、审稿费、中介费**。
- **字数**：综合板块约 **4000–6000 字**，学术板块约 **10000 字**左右（按官网最新栏目要求为准）。
- **审稿周期**：约 **1–3 个月**（初审约 1 个月）。
- **查重**：经学术不端系统检测，与已公开文献重复率原则上不得超过 **10%**；重复率超过 **30%** 者将被列入"学术失范登记库"。
- **投稿/官网**：官网 cpaj.com.cn（采编系统入口 zgxzgl.juqk.net）；编辑部邮箱以官网为准（曾公布 cpabjb@163.com）。

## 选题偏好

- 行政改革、政府职能和治理现代化
- 数字政府、基层治理和公共服务
- 政策执行、绩效管理和法治政府

## 方法与证据

- 公共治理、行政改革、政策执行、数字政府、基层治理和公共服务绩效是核心
- 可用政策评估、案例比较、调查、文本分析、混合方法和制度分析
- 必须把政府行为、组织激励和公共价值讲清；学术板块对机制与证据要求更高

## 结构与写作

- 引言围绕公共问题和治理机制，而非企业绩效或工作汇报
- 机制要落到政府层级、部门协同、公众参与、政策执行或问责
- 政策建议要可执行，并说明适用层级

## 路由防误投

- 本刊快速定位属于“公共管理与社会政策”通道；只有当稿件的核心贡献满足“政府治理、公共服务、行政改革、社会保障或数字政府机制必须是主线”时，才把 《中国行政管理》 作为首选。
- 相近 profile 先比较：`chinese-journal-of-management`（《管理学报》） / `chinese-journal-of-management-science`（《中国管理科学》） / `chinese-review-of-financial-studies`（《金融评论》） / `comparative-economic-and-social-systems`（《经济社会体制比较》）。若这些刊物的读者对象更贴近，不要因为名称、地区或变量相似而强行投本刊。
- 常见误投：若只是经济政策效果，转向财政、产业或综合经济刊。 正式建议前仍需进入官方核验清单。

- 再次确认审稿周期与投稿系统入口（上面数值可能随官网更新而变化）。
- 若本 skill 与最新官方要求冲突，以官方要求为准。

## 投稿前自检

- [ ] 一句话说明为什么这篇稿件属于《中国行政管理》的核心读者范围
- [ ] 引言前 3 段已经出现本刊关心的问题层级、理论/机制和主要证据
- [ ] 文献综述至少对话 2-3 篇本刊或同层级中文期刊近年同主题文章
- [ ] 重复率已控制在 10% 以下；字数符合所投板块要求；摘要、关键词、基金信息、参考文献格式按官方稿约复核
- [ ] 政策/管理启示从本文证据推出，而不是泛泛口号

## 高频拒稿雷区

- 只写政策宣传或地方工作总结，缺理论和证据
- 案例材料描述多、机制抽象少
- 把企业管理框架生硬搬到公共部门
- 重复率超阈值或体例不规范

## 改投判断

更强实证、方法更厚的学术公共管理转 journal-of-public-management（《公共管理学报》）；理论评论与案例比较转 china-public-administration-review（《公共管理评论》）；治理实践与基层治理转 governance-studies（《治理研究》）。
