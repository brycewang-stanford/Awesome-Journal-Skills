# 《软科学》投稿（soft-science）

## 期刊定位

《软科学》由**四川省科技促进发展研究中心**主办，**月刊**。它面向软科学与管理决策，覆盖**创新政策、区域发展、产业管理、评价体系、政策模拟与决策支持**，强调抓住当前社会的热点和焦点问题。相对偏国家战略的《中国软科学》，本刊对区域/产业落点更包容；相对偏纯学术的《科学学研究》，本刊更看重**问题现实性与方法规范并重**。

这个 skill 是**定位 / 选刊 / 改写框架**工具，不替代该刊最新官方投稿须知。正式投稿前必须重新核对官网、采编系统或编辑部发布的最新模板。

## 触发时机

- 用户明确说目标期刊是《软科学》或 Soft Science
- 稿件主题接近本刊，但不确定问题意识、方法强度和读者对象是否匹配
- 需要把一篇通用中文经管稿改写成《软科学》更容易接受的叙事
- 投稿前需要列出本刊高频拒稿风险和改投方向

## 选题偏好

- 软科学方法、创新政策和区域经济
- 产业升级、企业管理和数字经济
- 评价体系、政策模拟和决策支持

## 方法与证据

- 企业/专利/项目数据、政策评估、网络分析、文本挖掘、案例与系统评价均可。
- 机制要落到创新资源配置、激励、知识溢出和组织能力。
- 实证须处理识别与稳健性；评价/指标类研究须交代权重与口径来源。

## 结构与写作

- 引言把科技政策问题和创新理论缺口连接起来，突出区域/产业现实意义。
- 变量构造需说明专利质量、研发投入、项目绩效或知识网络口径。
- 政策含义面向科技治理、企业创新和产业升级，从本文证据推出。

## 路由防误投

- 本刊快速定位属于“管理科学与工商管理”通道；只有当稿件的核心贡献满足“组织、战略、创新管理、管理科学模型、系统工程或企业管理机制必须是核心贡献”时，才把 《软科学》 作为首选。
- 相近 profile 先比较：`social-security-studies`（《社会保障评论》） / `sociological-studies`（《社会学研究》） / `south-china-journal-of-economics`（《南方经济》） / `studies-in-labor-economics`（《劳动经济研究》）。若这些刊物的读者对象更贴近，不要因为名称、地区或变量相似而强行投本刊。
- 常见误投：若只是宏观政策效果，比较经济/产业/公共管理刊。 正式建议前仍需进入官方核验清单。
