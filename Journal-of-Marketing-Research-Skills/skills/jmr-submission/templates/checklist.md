# JMR Pre-Submission Checklist

A self-check before submitting to the **Journal of Marketing Research (JMR)** via
ScholarOne at **mc.manuscriptcentral.com/ama_jmr**. Re-verify every specific against the
current SAGE author instructions and AMA submission-guidelines page — limits, fees, the
masthead (mid-2026 Hamilton → Thomadsen handover), and the transparency policy change.

## 1. Anonymization (double-anonymized review)

- [ ] Main document contains **no** author names, affiliations, or acknowledgments
- [ ] Self-citations are neutral ("prior research (Author 2020)"), never "our earlier work"
- [ ] Acknowledgments, funding, and author bios are on the **separate title page** only
- [ ] File metadata/properties scrubbed of author identity
- [ ] File names contain no author or institution identifiers
- [ ] Data/code links (e.g., OSF) do not de-anonymize the authors during review

## 2. Length & format

- [ ] Print paper **≤ 50 pages**, inclusive of title, abstract, keywords, text, footnotes, references, tables, figures, and print appendices (Web Appendix excluded)
- [ ] 12-point Times New Roman (or 12-pt LaTeX font)
- [ ] Double-spaced (tables and references may be single-spaced); 1-inch margins
- [ ] **No** page numbers, line numbers, headers, or footers
- [ ] Abstract **≤ 200 words, written in the third person**
- [ ] Title **≤ 25 words**; up to **8** primary keywords

## 3. Citations & statistics (AMA style)

- [ ] **AMA author-year** citations ("Thorelli 1960"; all authors up to three, "et al." for four+)
- [ ] No cap on references — both behavioral and modeling literatures cited as relevant
- [ ] Every in-text citation appears in the reference list and vice versa
- [ ] Empirical tables/text report **exact p-values to three digits** (no "p < .05", no asterisks)
- [ ] **Standard errors** reported for parameter estimates
- [ ] **Effect sizes** reported and practical magnitude discussed
- [ ] AMA number style: **no leading zero** before decimals; **≤ 3 decimal places**

## 4. Web Appendix (JMR-specific)

- [ ] Web Appendix is a **single separate PDF**, excluded from the 50-page limit
- [ ] Supplementary tables/figures use the **'W' prefix** ("Table W1", "Figure W1")
- [ ] Robustness, additional studies, full stimuli, and estimation details live here
- [ ] Web Appendix exhibits are referenced from the main text where relevant

## 5. Declarations & ethics

- [ ] **Data Availability Statement** on the **title page** (AMA Research Transparency Policy)
- [ ] Replication-enabling disclosure planned (in-text, Web Appendix, or online supplements)
- [ ] Ready to share computer code, instruments/stimuli, and materials on editor request
- [ ] **AI-generated content disclosed** in the main document (e.g., Methods) and Acknowledgments
- [ ] IRB / human-subjects approval stated where applicable (on the title page, not anonymized text)
- [ ] No concurrent submission; prior conference/working-paper versions noted in the cover letter
- [ ] Manuscript is iThenticate-clean (proper quoting, paraphrasing, citation)

## 6. Files for ScholarOne

- [ ] Anonymized main document (manuscript + main-text tables/figures as the portal specifies)
- [ ] Separate title page with authors, affiliations, contact, funding, **and the Data Availability Statement**
- [ ] **Web Appendix** as a single separate PDF
- [ ] Cover letter (fit with JMR, dual contribution, not under review elsewhere)
- [ ] High-resolution figure files if requested separately
- [ ] Correct article type and domain keywords selected so the paper routes to an appropriate Coeditor/AE

## 7. Consistency & final preview

- [ ] Hypothesis labels, construct/parameter names, and table/figure numbers consistent throughout
- [ ] Main-text vs. 'W'-prefixed Web Appendix exhibit numbering is consistent
- [ ] Built PDF previewed in ScholarOne before final submission
