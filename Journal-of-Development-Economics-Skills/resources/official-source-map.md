# JDE Official Source Map

Primary and authoritative sources behind the JDE-specific facts injected into
this skill pack. **Policy refresh date: 2026-06-20.** ScienceDirect pages were
directly readable during this refresh. Re-check the live pages before
submission-ready advice because fees, editors, special issues, and policy
wording can change.

## Official journal & policy pages

- **https://www.sciencedirect.com/journal/journal-of-development-economics**
  — ScienceDirect journal home. Verifies aims/scope, article publishing options,
  APC, subscription route, submission/review timing metrics, Editors-in-Chief,
  current calls, and the submission link.
- **https://www.sciencedirect.com/journal/journal-of-development-economics/publish/guide-for-authors**
  — Guide for Authors. Verifies scope, pre-screening/selectivity, submission cap,
  Registered Reports, short-paper route, peer review, appeal policy, file format,
  abstract/keywords/highlights/JEL requirements, Option C data policy, reference
  policy, and online submission requirements.
- **https://www.sciencedirect.com/journal/journal-of-development-economics/about/editorial-board**
  — Editorial board. Verifies current Editors-in-Chief, Co-Editors, Associate
  Editors, and board counts.
- **https://www.elsevier.com/journals/journal-of-development-economics/0304-3878/guide-for-authors**
  — Elsevier Guide mirror / official search snippet. Official Elsevier search
  result reports the current **USD 50 non-refundable submission fee for original
  manuscripts**; re-check the live fee screen before filing.
- **https://data.mendeley.com/journal/03043878** — JDE on Mendeley Data.
  Verifies JDE-linked dataset hosting on Mendeley Data.
- **https://www.bitss.org/publishing/jde/** and
  **https://afosterri.org/jdepreresults/author-guidelines/** — BITSS and JDE
  pre-results / Registered Reports author guidance.

## Checked facts

| Fact | Official source |
| --- | --- |
| JDE publishes original research on all aspects of economic development, emphasizing novel and relevant quantitative or analytical work | ScienceDirect home; Guide for Authors |
| The journal pre-screens submissions and immediately rejects some for fit, contribution, or methodological rigor | Guide for Authors |
| Approx. **1,300 submissions per year**, about **1/4 sent out for review**, and roughly **6-8% published** | ScienceDirect home; Guide for Authors |
| Under normal circumstances, an author cannot submit more than **three papers in any 12-month period** | ScienceDirect home; Guide for Authors |
| Papers rejected by JDE will not be considered again unless the decision letter explicitly leaves that possibility open | ScienceDirect home; Guide for Authors |
| Submission is through **Editorial Manager**; the system converts uploaded files into a peer-review PDF | ScienceDirect home; Guide for Authors |
| Current review model is **single anonymized**; suitable papers typically go to a **minimum of two reviewers** | Guide for Authors |
| Formal appeal possible if it meets Elsevier appeal-policy requirements; one appeal per submission and appeal decision final | Guide for Authors |
| Submission fee for original manuscripts: **USD 50 non-refundable** (re-verified 2026-06-22) | Elsevier Guide official search snippet |
| Open-access APC: **USD 4,760** excluding taxes (re-verified 2026-06-22); subscription route has no publication fee charged to authors | ScienceDirect home |
| Current Editors-in-Chief: **A. Foster** (Brown University) and **K. Macours** (Paris School of Economics) — re-verified 2026-06-22 | ScienceDirect home; editorial board |
| Co-Editors include **Dean Karlan** and other current board members listed on ScienceDirect | Editorial board |
| Short-paper route: at most **6,000 words**, **5 exhibits**, and **20-page online appendix**; manuscripts must be self-contained | ScienceDirect home; Guide for Authors |
| Short-paper refereed decisions generally occur within **6-8 weeks**; standard-format rejections may be invited into short format with a no-further-review commitment if instructions are followed | ScienceDirect home; Guide for Authors |
| Registered Reports / pre-results track available; Stage I and Stage II article types are selected in the submission portal | ScienceDirect home; Guide for Authors; BITSS/JDE pre-results guidance |
| Stage I pre-results proposal cap: **60 pages**; abstract cap: **150 words** | JDE pre-results author guidelines |
| Editable Word/LaTeX source files are required; PDF is not an acceptable source file | Guide for Authors |
| Regular article abstract must be **<=250 words** | Guide for Authors |
| **1-7 English keywords** required; **up to 6 JEL codes** requested | Guide for Authors |
| Highlights required at submission: **3-5 bullets**, **<=85 characters** each including spaces | Guide for Authors |
| Generative-AI manuscript-preparation use must be declared at first submission in a section before references; AI tools may not be authors | Guide for Authors |
| Preprint sharing is allowed under Elsevier policy; optional free SSRN posting after initial desk review has no effect on editorial outcome | Guide for Authors |
| Research-data **Option C** applies: deposit research data in a relevant repository and cite/link it, or explain why data cannot be shared | Guide for Authors |
| Data availability statement is required at submission and appears with the published article | Guide for Authors |
| References can be in any internally consistent style at submission; JDE style is applied after acceptance at proof stage | Guide for Authors |
| In-text citations are author-year; data references use `[dataset]` before the reference | Guide for Authors |

## Operational notes for this pack

- Replace older fee-free wording with the current **USD 50** original-manuscript
  fee, while still asking users to re-check the live payment screen before
  filing.
- Keep the short-paper route current: **6-8 week** refereed decisions.
- Treat data policy as both venue-specific and Elsevier-current: JDE has a
  Mandatory Replication Policy referenced in the Registered Reports section and
  Elsevier Option C data deposit/cite/link-or-explain rules in the Guide.

  cover letter.
