# Journal of Educational Psychology Pre-Submission Self-Check (8 sections)

A last-pass checklist for a *Journal of Educational Psychology* (JEP) manuscript before submitting via
**Editorial Manager** (`editorialmanager.com/edu`). JEP is the **American Psychological Association's**
flagship outlet for **original, primary psychological research on learning and instruction across all
ages** (founded **1910**; published **8×/year**; **masked review**). Verify volatile specifics (exact
word/abstract limits, current editor, badge wording, portal) on the official APA author pages —
检索于 2026-06；以官网为准.

## 1. Venue & scope fit
- [ ] The contribution is **original, primary psychological research** about learning/instruction (not a program evaluation, not a single-instrument validation)
- [ ] There is a clear **educational implication** (who in education inherits the claim)
- [ ] A learning/motivation/cognitive **mechanism** is stated and tested
- [ ] If a meta-analysis, it is **exceptionally important** and follows APA meta-analysis reporting standards

## 2. Masked review (anonymization — both author and reviewer identities masked)
- [ ] First page **omits** author names and affiliations; includes the **title** and **submission date**
- [ ] No identifying **grant numbers**, **IRB/institution/site/district** names
- [ ] Self-citations phrased in the **third person** (not "our prior work")
- [ ] Repository / preregistration links **anonymized** (e.g., OSF anonymized view); file metadata stripped

## 3. Format & front matter
- [ ] Body **≤ 12,000 words** (≈ 40 double-spaced pages), **excluding** references, tables, figures, appendices
- [ ] **Abstract ≤ 250 words** on a separate page
- [ ] **APA 7th edition**, double-spaced; keywords as requested

## 4. Design rigor (nested educational settings)
- [ ] **Unit of randomization, power, and analysis match** (cluster-level if classrooms/schools were assigned)
- [ ] Sample size justified at the **right level** (number/size of clusters, ICC, smallest meaningful effect)
- [ ] **Baseline equivalence** reported; attrition, contamination, and **implementation fidelity** addressed
- [ ] Outcome measures **validated** and capture learning/transfer, not test coaching

## 5. Analysis & statistics
- [ ] **Nesting modeled** (multilevel / SEM / growth); cluster-respecting inference; ICC reported
- [ ] **Effect sizes with confidence intervals** and an **educational interpretation** (months/percentile/variance)
- [ ] **Mechanism** (mediation/moderation) tested where hypothesized
- [ ] **Confirmatory vs. exploratory** clearly separated; principled missing-data handling (FIML/MI)

## 6. Transparency, openness & reporting standards
- [ ] **Transparency and Openness subsection** states **data**, **materials**, and **analysis code** availability with **persistent identifiers** (or justified restriction + access path)
- [ ] Analysis code **reproduces** results in a fresh session; codebook included
- [ ] **Preregistration** disclosed and linked if applicable; deviations noted
- [ ] **JARS / JARS-REC** items complete for the design (participant, context, race/ethnicity/culture reporting)
- [ ] **Open Science Badge** disclosure form signed (if applying)

## 7. Exhibits & files
- [ ] Tables/figures are **APA 7th**, self-contained, and **anonymized** (no site/district labels)
- [ ] Exhibit values **match the deposited analysis script**; secondary exhibits moved to **online supplemental material**
- [ ] Manuscript file in **Word (.docx)** or **LaTeX (.tex zip + PDF)** per Editorial Manager requirements

## 8. Declarations & final
- [ ] No **submission fee** required (free peer review); confirm any post-acceptance open-access APC
- [ ] Disclosure/conflict-of-interest and funding statements prepared (kept out of the masked body)
- [ ] Confirmed not under review elsewhere; AI not listed as an author
- [ ] Co-author sign-off; corresponding author designated
