# External Tools for Perspectives on Psychological Science

Tooling for a broad integrative review, theoretical statement, or meta-science piece. PoPS reports no
new empirical study of its own, so the kit is for **systematic search, transparent synthesis, and
reproducible meta-science** — not data collection.

## Literature search & synthesis

| Tool or source | Use |
|----------------|-----|
| **APA PsycINFO / PsycNet** | the primary psychology database — keyword + construct-term sweeps across sub-areas |
| **PubMed, Web of Science, Scopus** | adjacent fields (neuroscience, methods) and forward-citation snowballing |
| **Google Scholar** | broad forward-citation tracing and gray literature |
| **PsyArXiv (preprints)** | the frontier — recent preprints a current PoPS piece must include |
| **Rayyan / Covidence** | screening and deduplication for a systematic coverage account or meta-science corpus |
| **Saturation log + evidence matrix (spreadsheet)** | record where each sub-area's search stops yielding new must-cites; one row per study (construct / design / finding + effect size / credibility / tension) |

## Meta-science & reproducibility

| Tool or source | Use |
|----------------|-----|
| **OSF (Open Science Framework)** | preregister the sampling frame + coding scheme; deposit coded data, coding manual, and scripts |
| **AsPredicted** | lightweight preregistration of a meta-science protocol |
| **GitHub + Zenodo** | version analysis scripts; mint a citable DOI for the code release |
| **metafor / meta / dmetar (R)** | effect-size pooling, heterogeneity, and bias diagnostics — only for a genuine meta-analysis with a superordinate message |
| **p-curve / z-curve / funnel-plot tools** | publication-bias and evidential-value diagnostics for a reform/meta-science argument |
| **PRISMA flow diagram template** | document search → screening → inclusion for a transparent corpus |
| **TOP Factor / TOP guidelines** | map data/code/materials sharing to a recognized transparency standard |

## Exhibits & writing

| Tool or source | Use |
|----------------|-----|
| **APA Style (current edition)** | references, statistics, table/figure formatting (confirm edition on SAGE pages) |
| **Reference manager (Zotero / EndNote)** | accurate, complete APA references; attribute ideas to originators |
| **Diagramming tool** | the conceptual/framework figure — PoPS's signature exhibit |

## Method notes

- Keep a reproducibility manifest even when no deposit is demanded at first submission — for a PoPS reform piece, exemplary open science is the credibility.
- For meta-science, document coding rules, construct definitions, and inclusion/exclusion criteria with the same care a primary study gives its dataset; double-code subjective extractions and report reliability.
- Sibling-venue comparison set when checking fit: *Psychological Science* (single studies), *Psychological Bulletin* (formal meta-analyses), *Annual Review of Psychology* (invited surveys), *Current Directions* (short, non-technical).
