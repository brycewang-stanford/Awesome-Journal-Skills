# AISTATS Skills

面向 AISTATS 论文的 12 个 agent skills，覆盖 AI、statistics 与 machine learning 交叉
论文的选会、OpenReview 投稿、统计证据、匿名补充材料、可复现性 checklist、author-reviewer
discussion、PMLR camera-ready 与现场报告规划。

官方依据核验日期：2026-06-01。已核验 AISTATS 2026 Call for Papers、OpenReview
conference group、PMLR proceedings 页面、AISTATS Code of Conduct，以及 AISTATS / PMLR
作者出版惯例。精确来源见 `resources/official-source-map.md`。

## Skills

- `aistats-submission`：核查 OpenReview、页数、匿名、补充材料、checklist、一稿多投与
  reviewer volunteer 要求。
- `aistats-author-response`：撰写匿名、简洁、text-only 的 author-reviewer discussion
  回复，不放违规链接或无支撑的新主张。
- `aistats-camera-ready`：准备 PMLR camera-ready、去匿名、元数据、注册、现场报告与
  artifact release。
- `aistats-artifact-evaluation`：把代码、数据、证明、日志整理成匿名补充证据或录用后的公开
  artifact。
- `aistats-reproducibility`：补强 checklist、假设、证明、数据集、超参数、随机种子、算力与
  不确定性证据。
- `aistats-supplementary`：组织 appendix、proof details、code/data archives、simulation
  scripts 与匿名 artifact。
- `aistats-review-process`：解释 OpenReview 审稿、discussion、meta-review、保密与 PMLR
  录用结果。
- `aistats-writing-style`：压缩 8 页正文，强化 AI-statistics framing、假设清晰度、理论与
  实验平衡。
- `aistats-related-work`：处理 AI、ML、统计学、arXiv、workshop、concurrent 与 archival
  文献定位。
- `aistats-experiments`：审查 simulation、baseline、统计检验、不确定性、ablation、seed、
  超参数、算力与数据处理。
- `aistats-workflow`：从选会到摘要、全文、补充材料、reviews、discussion、decision、
  camera-ready 与 presentation 做项目管理。
- `aistats-topic-selection`：判断项目适合 AISTATS，还是更适合 NeurIPS、ICML、ICLR、
  UAI、COLT、JMLR、统计学期刊或应用 venue。

## 维护说明

- 涉及 deadline、页数、补充材料、discussion、PMLR forms、注册与报告要求时，必须重新打开当年
  官方页面。
- 本包中的 AISTATS 2026 事实是历史锚点，不应当被当成永久规则。
