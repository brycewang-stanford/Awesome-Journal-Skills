# External Tools - AISTATS

Access date: 2026-06-01

Use these tools after reopening the current AISTATS cycle pages. AISTATS rules can change by
year.

## Official workflow

- AISTATS 2026 Call for Papers: https://virtual.aistats.org/Conferences/2026/CallForPapers
- AISTATS 2026 OpenReview group: https://openreview.net/group?id=AISTATS.org/2026/Conference
- AISTATS 2026 official entry point: https://aistats.org/aistats2026/
- AISTATS 2026 Code of Conduct: https://virtual.aistats.org/Conferences/2026/CodeOfConduct
- PMLR proceedings: https://proceedings.mlr.press/

## Author-side checks

- OpenReview account, profile, conflicts, author list, title, abstract, subject areas, and
  reviewer-volunteer obligations.
- Current AISTATS LaTeX style, main-body page limit, appendix/reference handling, and PDF
  export.
- Anonymous paper, no acknowledgements, no identifying metadata, and anonymized supplement,
  code, data, and repository links.
- Reproducibility checklist and any required OpenReview fields.
- Supplementary material deadline, archive contents, and text-only discussion rules.
- Camera-ready package, PMLR forms, registration, in-person presentation, and public
  artifact release.

## Do not infer

- Do not infer future-cycle dates from AISTATS 2026.
- Do not assume the same page limits, author-discussion mechanics, supplement deadline,
  checklist wording, reviewer-volunteer rules, camera-ready forms, or presentation
  obligations without reopening current official pages.
- If official pages disagree, use the newest CFP, OpenReview announcement, PMLR instruction,
  or direct chair communication as the controlling source and record the conflict.
