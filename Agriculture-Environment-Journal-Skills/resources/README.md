# Agriculture and Environment Breadth Resources

This layer supports routing across agriculture, ecology, food systems, climate,
hydrology, and environmental-science fit cards. Use it when the manuscript's
best venue is uncertain or when adjacent field journals need to be separated
before deeper single-venue work.

## What to Load

- `official-source-map.md` - official-source anchors for each journal; re-check
  live author guidelines before final submission advice.
- `source-basis.md` - coverage basis, ranking logic, and source discipline for
  this breadth bundle.
- `worked-examples/venue-routing.md` - example routing diagnoses for common
  agriculture, environment, water, food, and plant-science manuscripts.
- `exemplars/selection-patterns.md` - reusable venue-selection patterns and
  sibling-journal traps.

## Usage Rule

Start with `en-agrienv-journal-workflow` when the field boundary is unclear.
Then load only the one or two venue cards that match the manuscript's main
claim. Prefer dedicated depth packs, when available, for full lifecycle work
after the breadth card confirms fit.
