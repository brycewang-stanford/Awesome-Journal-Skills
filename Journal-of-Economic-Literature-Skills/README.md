Twelve agent skills for **Journal of Economic Literature (JEL)** survey and review articles. This pack is built for proposal-first synthesis work: choosing a field worth surveying, pitching the article, building a comprehensive literature map, imposing an organizing framework, balancing schools of thought, designing review tables, writing in the JEL voice, handling JEL codes, submission, and revision.

**Official basis checked 2026-06**: see [`resources/official-source-map.md`](resources/official-source-map.md).

## Quick Start

```
/plugin marketplace add ./Journal-of-Economic-Literature-Skills
/plugin install jel-skills
```

Manual use: start with [`skills/jel-workflow/SKILL.md`](skills/jel-workflow/SKILL.md).

## Skills

| # | Skill | Role |

| 1 | [`jel-workflow`](skills/jel-workflow/SKILL.md) | Route a JEL survey project |
