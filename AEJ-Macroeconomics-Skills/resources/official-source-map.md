# AEJ: Macroeconomics Official Source Map

Primary and authoritative sources behind every AEJ: Macro-specific fact injected
into this skill pack. **Access date for all entries: 2026-06.** AEJ: Macro is one
of the four *American Economic Journals* published by the **American Economic
Association (AEA)**; it is the AEA's quarterly broad-interest macroeconomics
journal (founded 2009). Re-verify volatile items (editors, fees, page charges,
length guidance, policy effective dates) before relying on them — AEA pages
change. Items that were not confirmed from a primary fetch are marked
**需复核 (to be verified)**.

## Official AEA pages (publisher / editorial authority)

- **https://www.aeaweb.org/journals/mac/submissions**
  — AEJ: Macroeconomics Submission Guidelines. Verifies: submission as PDF;
  abstract of 100 or fewer words required; submission fee scaled by AEA
  membership and country income tier (members US$200 high-income / US$100
  middle-income / US$0 low-income; nonmembers US$300 / US$200 / US$0); papers
  rejected without review receive a 50% refund; publication fee of US$15 per
  typeset page assessed at page proofs (papers submitted after Feb 1, 2024);
  length guidance averaging ~34–35 typeset pages with a recommended maximum of
  ~40 pages (11pt, 1.5 spacing) or ~45 pages (12pt, 1.5 spacing), the limit
  including figures, tables, references, and appendices; original work not under
  consideration elsewhere; first page carries title, byline, and affiliations; a
  separate disclosure statement from each coauthor is mandatory; data/code
  required for empirical, simulation, or experimental work; field experiments
  registered in the AEA RCT Registry. Accessed 2026-06.
- **https://www.aeaweb.org/journals/mac/editors**
  — Editors of AEJ: Macroeconomics. Verifies: Editor **Ayşegül Şahin**
  (Princeton University); coeditors **Francesco Bianchi** (Johns Hopkins),
  **Şebnem Kalemli-Özcan** (Brown), and **Vincenzo Quadrini** (USC). **Re-verified
  2026-06-22 via WebFetch of the official AEA editors page** — affiliations
  (Princeton / Johns Hopkins / Brown / USC) confirmed against the live masthead;
  note some third-party listings show stale affiliations (e.g., Texas/Maryland),
  so the AEA page is authoritative. (Full associate-editor roster not transcribed
  here; **需复核** if a complete list is needed.)
- **https://www.aeaweb.org/journals/mac/editorial-policy**
  — AEJ: Macroeconomics Editorial Policy. Verifies: **single-blind** peer review
  (the author's name is revealed to the referee, while referees remain anonymous
  to authors); editorial decisions weigh the expected probability of meeting
  AEJ: Macro standards, breadth of topic, and interest to the AEJ: Macro
  audience; desk rejection possible without referees; before acceptance authors
  must provide data, code, and computational details sufficient to permit
  replication; conditionally accepted papers undergo review by the AEA Data
  Editor. Accessed 2026-06.

## AEA-wide policies (shared across AER / AEJs)

- **https://www.aeaweb.org/journals/data/data-code-policy**
  — AEA Data and Code Availability Policy. Verifies: the AEA Data Editor assesses
  compliance, conducts reproducibility checks, and verifies the accuracy of the
  information provided **prior to publication**; use of the **AEA Data and Code
  Repository** is strongly encouraged (other trusted repositories may be allowed
  with Data Editor approval); code scope is broad — data transformation/cleaning,
  analysis/estimation, **simulation**, model solution, and visualization programs
  must all be included; restricted-access data (confidential / copyrighted /
  agreement-restricted) qualify for exceptions, with 5+ year retention,
  reasonable replication assistance, public code even when data are not public,
  and disclosure of sources. Accessed 2026-06.
- **https://aeadataeditor.github.io/** and **https://aeadataeditor.github.io/about/**
  — Office of the AEA Data Editor. Verifies: the AEA Data Editor (since 2018) is
  **Lars Vilhuber** (Cornell University); the office runs the pre-publication
  reproducibility verification service and curates the AEA Data and Code
  Repository on openICPSR. Accessed 2026-06.
- **https://www.openicpsr.org/openicpsr/search/aea/studies**
  — AEA Data and Code Repository on **openICPSR**. Verifies: openICPSR is where
  AEA replication packages are deposited and disseminated. Accessed 2026-06.
- **https://www.aeaweb.org/journals/policies/disclosure-policy**
  — AEA Disclosure Policy. Verifies: each coauthor prepares a separate disclosure
  statement; the disclosure threshold for significant financial support from
  interested parties is **US$10,000** over the past three years; sources to
  disclose include grants/consulting/retainers, in-kind support (e.g., data
  access), relevant officer/director/board positions, and review-rights
  arrangements; an "interested party" has a financial, ideological, or political
  stake related to the article. Accessed 2026-06.
