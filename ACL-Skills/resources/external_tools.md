# External Tools - ACL

Access date: 2026-07-08

Open these after re-checking the current ARR cycle and the target conference's
calls. Note the access caveat in `official-source-map.md`: some gateways block
these domains directly; the ARR pages can be read from the
`acl-org/aclrollingreview` GitHub repository instead.

## Official workflow surfaces

- ACL Rolling Review portal and CFP: https://aclrollingreview.org/ and https://aclrollingreview.org/cfp
- ARR cycle calendar and venue commitment windows: https://aclrollingreview.org/dates
- ARR author guidelines: https://aclrollingreview.org/authors
- Responsible NLP checklist: https://aclrollingreview.org/responsibleNLPresearch/
- OpenReview ARR cycle groups (pattern): https://openreview.net/group?id=aclweb.org/ACL/ARR/{year}/{month}
- Current ACL edition site (pattern): https://{year}.aclweb.org/ — calls, commitment link, camera-ready instructions
- ACL Anthology (proceedings of record): https://aclanthology.org/
- Official ACL paper style files: https://github.com/acl-org/acl-style-files
- \*ACL camera-ready FAQ: https://acl-org.github.io/pubdocs/camera-ready-faq.html
- ACL member portal announcements (commitment links, policy updates): https://www.aclweb.org/portal/

## Author-side checks by stage

- **Cycle choice**: which cycles the target conference accepts; commitment
  deadline vs cycle timeline; team availability during the response window.
- **Submission**: unmodified style files; 8/4 content pages; Limitations section
  present; references/appendices placement; anonymized PDF and archive;
  preprint declaration; resubmission link + change summary if applicable.
- **Checklist**: every Responsible NLP item answered against the actual PDF,
  with section pointers.
- **Response window**: anonymous, evidence-anchored replies; review-issue
  flags for deficient reviews.
- **Commitment**: complete package (reviews + meta-review), correct venue
  group, before the venue's commitment deadline.
- **Camera-ready**: extra page used on reviewer-mandated edits; authors,
  acknowledgements, funding, and AI-assistance disclosure restored; Anthology
  metadata clean; public artifact links live.

## Do not infer

- Do not project cycle dates or commitment windows from the 2026 calendar;
  ARR has changed cycle length and count more than once.
- Do not assume page limits, checklist wording, preprint incentives, or theme
  tracks carry over between editions.
- Do not treat an ARR meta-review as an acceptance decision — only the
  committed venue decides, and Findings vs main is decided there too.
- When ARR pages, the conference site, and OpenReview forms disagree, the
  newest official announcement controls; record the conflict and, if stakes
  are high, ask the program chairs via the contact listed on the current site.
