# ACL Resources

Action-oriented companions to the ACL skill pack. The `skills/` directory gives
venue advice; this directory lets an agent model an ACL-shaped paper, benchmark
against verified ACL Anthology exemplars, and assemble the evidence package that
ACL Rolling Review actually inspects.

## What's here

| Resource | Use it to... |
| --- | --- |
| [`worked-examples/01-introduction.md`](worked-examples/01-introduction.md) | Study a before-to-after abstract/introduction rewrite for a fictional multilingual NLP paper in ACL house style. Nothing real is cited; no policy is invented. |
| [`exemplars/library.md`](exemplars/library.md) | Benchmark against six web-verified ACL main-proceedings papers, organized by contribution type, with sibling-venue (NAACL/EMNLP/NeurIPS) misattribution guards. |
| [`official-source-map.md`](official-source-map.md) | Confirm ARR-era submission, checklist, anonymity, commitment, and Anthology facts, with the access-method note for gateway-blocked domains. |
| [`external_tools.md`](external_tools.md) | Jump to ARR, OpenReview cycle groups, style files, the Anthology, and per-edition conference sites, with stage-by-stage author checks. |
| [`code/README.md`](code/README.md) | Reach the shared ML-conference reproducibility kit for supplement smoke checks. Not an econometrics code kit. |

## Scope Note

This pack covers the ACL main conference under ACL Rolling Review. Findings of
ACL, industry track, demos, and the Student Research Workshop are addressed
where they intersect the main pipeline (commitment outcomes, camera-ready),
but each has its own call — always open the current edition's pages.

## Suggested Workflow

1. Decide fit and framing with
   [`../skills/acl-topic-selection`](../skills/acl-topic-selection/SKILL.md) and
   [`../skills/acl-writing-style`](../skills/acl-writing-style/SKILL.md),
   using [`worked-examples/01-introduction.md`](worked-examples/01-introduction.md)
   as the style target.
2. Pressure-test evidence with
   [`../skills/acl-experiments`](../skills/acl-experiments/SKILL.md),
   [`../skills/acl-reproducibility`](../skills/acl-reproducibility/SKILL.md), and
   the shared kit via [`code/README.md`](code/README.md).
3. Position against [`exemplars/library.md`](exemplars/library.md), then verify
   every current-cycle rule through
   [`official-source-map.md`](official-source-map.md) and
   [`external_tools.md`](external_tools.md) before anything deadline-shaped.
