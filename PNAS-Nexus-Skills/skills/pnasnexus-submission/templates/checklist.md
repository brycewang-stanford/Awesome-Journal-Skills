> Targets reflect typical PNAS Nexus guidelines — confirm exact caps, the APC, and the license options in the current PNAS Nexus author guidelines (academic.oup.com/pnasnexus). PNAS Nexus is the gold open-access sibling of PNAS; do not reuse flagship-PNAS facts.

## 1. Fit & venue
- [ ] Broad, cross-disciplinary significance established (pnasnexus-fit; a reader from another division would care)
- [ ] PNAS Nexus is the right venue (not better at flagship PNAS or a field journal)
- [ ] If transferring from PNAS: decline was fit/priority (not soundness); prior reviews attached
- [ ] Title declarative, specific, parseable by a non-specialist

## 2. Open access, APC & license
- [ ] Gold OA understood (no subscription/hybrid escape)
- [ ] APC obtained LIVE from the OUP calculator (article category + corresponding-author region): ______
- [ ] Waiver/discount checked: ☐ LMIC full waiver ☐ LMIC discount ☐ Read & Publish ☐ membership ☐ discretionary ☐ none

- [ ] (Reminder) No Direct/Contributed track — standard editor-assigned review

## 3. Article type, structure & length
- [ ] Article type: ☐ Research Report ☐ Brief Report ☐ Registered Report (S1/S2) ☐ Perspective ☐ Review

## 4. Significance Statement & abstract
- [ ] Significance Statement 50–120 words (Research Report / Stage 2 Registered Report): ______ words
- [ ] Significance Statement written for a broad audience; distinct from abstract
- [ ] Abstract ≤ 250 words: ______ words
- [ ] Abstract: single paragraph, NO headings / no citations
- [ ] ≥ 1 quantified headline result in abstract

## 5. Figures & display items
- [ ] Within the graphical-element budget (~4 for a 6-pp Research Report; secondary items → SI): ______
- [ ] Sized to journal column widths; fonts legible at print size
- [ ] Data shown (points + n); error bars defined (SD/SEM/CI)
- [ ] Scale bars on images; raw, unprocessed image files retained
- [ ] Colorblind-safe palette; meaning not by color alone; color used where it helps (OA, online)
- [ ] Each legend: title sentence + per-panel + statistics

## 6. Statistics & reproducibility
- [ ] Per claim: effect size + 95% CI + n + test + assumptions
- [ ] Biological vs technical replication distinguished
- [ ] Sample-size rationale / power stated
- [ ] Randomization & blinding reported or justified
- [ ] Multiple-comparison correction where needed
- [ ] Registered Report considered for confirmatory work (☐ N/A)
- [ ] Analysis code + scripts public and archived with DOI

## 7. Data, code, materials, ethics
- [ ] Data, code, materials, protocols deposited in a PUBLIC repository; accession numbers / DOIs in hand: ______
- [ ] Raw, unprocessed images retained
- [ ] Data and Code Availability Statement compliant (no "on request" only for primary data)
- [ ] Datasets/software cited as references with the [dataset] tag
- [ ] Materials/reagent sharing plan (Addgene / MTA)
- [ ] Ethics: IRB ☐  IACUC ☐  field/biodiversity permits ☐  N/A ☐

## 8. References
- [ ] Any readable style accepted AT SUBMISSION — but list is consistent, complete, fully resolvable
- [ ] Authors / title / venue / year / volume / pages / DOI present
- [ ] All in-text citations resolve; no orphans, gaps, or duplicates
- [ ] Data/software cited with the [dataset] tag

## 9. Required files & metadata
- [ ] Main text (title, Significance Statement, abstract, body, Materials and Methods, references)
- [ ] Figures at required resolution + figure legends; tables
- [ ] Supporting Information (extended methods, supporting figs/tables, supplementary text); Datasets if any
- [ ] Cover letter (advance + broad significance; article type; transfer note if applicable)
- [ ] Authors, affiliations, ORCIDs, corresponding author
- [ ] Author contributions
- [ ] Competing interests + funding statements
- [ ] Classification + keywords entered in submission system (pnasnexus.msubmit.net)

## 10. Final integrity sweep
- [ ] No over-claiming beyond the evidence
- [ ] Significance Statement is broader/plainer than the abstract (not a duplicate), within 50–120 words
- [ ] Every figure n / error bar / test defined in legend
- [ ] All accession numbers / DOIs live and correct
